好的，身為權威醫事檢驗師，這份針對國考《生物化學與臨床生化學》中「生化相關之分子診斷學原理」的總結與預測，將為考生提供最高效率的複習指引。本次更新更進一步揉合了臨床生化實驗室中重要的高階分析儀器（如質譜儀、乾式生化儀）以及進階品管概念（6 Sigma），以貼近最新考題走向。

## 核心趨勢與高頻考點
本單元的出題趨勢已從傳統的 PCR、南方墨點法等基礎技術，全面轉向現代精準醫療的核心技術，並向外延伸至高階臨床生化分析與品管策略。考生必須理解，這些技術並非獨立存在，而是為了解決臨床問題而環環相扣。
其核心邏輯鏈如下：
**精準醫療需求 (如：癌症標靶治療)** → **需要更個人化、更即時的檢測工具** → **因應臨床需求，但缺乏已查驗登記的IVD產品時** → **實驗室開發檢測 (LDTs) 應運而生** → **LDTs 的實現依賴三大支柱：**
1.  **高靈敏度絕對定量:** 從 Real-time PCR 的相對定量演進到 Digital PCR 的**絕對定量**（**不需製作標準曲線**），專攻微量檢體（如 ctDNA）的拷貝數偵測。
2.  **高通量分析 (High-Throughput):** 次世代定序 (NGS) 取代傳統 Sanger 定序，能一次分析數十到數百個基因，常用於癌症基因套組檢測。
3.  **非侵入性採檢 (Non-Invasive):** 液態活檢 (Liquid Biopsy) 概念興起，以血液中的 **ctDNA** 或 **CTC** 取代傳統組織切片，用於監測治療反應與抗藥性。
**高頻考點與擴充預測：**
1.  **技術比較：** **Digital PCR vs. Real-time PCR** 的定量原理差異（dPCR **不需標準曲線**即可絕對定量），以及 **CTC vs. ctDNA** 的生物特性與臨床應用比較，是重複出現的鐵壁考題。
2.  **法規與品管：** LDTs 的法規（主管機關為**衛福部食藥署 TFDA**）、與 IVD 的差異、確效流程是必考重點。此外，實驗室品管已延伸至 **6 Sigma 策略計算**，考生必須熟記公式 `Sigma = (TEa - |bias|) / CV`。
3.  **臨床應用整合：** 題目會將技術與特定癌症連結，例如非小細胞肺癌 (NSCLC) 的 **EGFR** 突變檢測，就可能同時考到 LDTs、NGS 或液態活檢的應用。同時，經典的抑癌基因（如 **Rb**, TP53）與原癌基因的基本定義，仍是出題核心。
4.  **新興與高階分析技術原理：** 對於 aCGH（看拷貝數變異）、Microarray（看基因表現量）、Bisulfite sequencing（看**甲基化**）等技術需掌握核心原理。新加入考點則強調高階生化儀器：如 **LC-MS/MS 質譜儀**是測量**質荷比 (m/z)**而非直接測量質量；**VITROS 乾式生化分析儀**則是利用 **MicroSlide 擴散層技術**，而非傳統液體攪拌。
5.  **基礎觀念深化：** 如 **Tm 值**的定義（50%雙股解鏈溫度）與**黏合溫度 (Annealing Temp.)** 對產物專一性與產量的影響。
6.  **常規生化檢驗限制：** 例如**白蛋白 (Albumin)** 本身無色，因此**絕對不能**直接用可見光比色法測量，必須依賴 BCG/BCP 色素結合法或 UV 測量。

| :--- | :--- | :--- | :--- | :--- | :--- |
| **Real-time PCR** | 螢光探針/染料，同步放大與偵測 | **相對定量** | 病毒量偵測、基因表現量 | **Ct 值** (Cycle threshold)，與初始模板量**成反比** | 需標準曲線才能絕對定量 |
| **Digital PCR (dPCR)** | **微滴/孔盤分割 (Partitioning)**，終點偵測 | **絕對定量** (基於**二進制定量**分析) | 稀有突變偵測 (ctDNA)、拷貝數變異 (CNV) | **陽性微滴數** + **Poisson 分佈**校正 | **靈敏度極高**，**不需標準曲線**，但通量較低 |
| **次世代定序 (NGS)** | **大量平行定序 (Massively Parallel)**，邊合成邊定序 | 定序計數 (Read counts) | 癌症基因套組、全外顯子/基因體定序 | **Phred Score (Q30=錯誤率1/1000)**、覆蓋深度 | 數據分析複雜。**WGS** 成本高且易產生VUS。 |
| **液態活檢 (Liquid Biopsy)** | 分析血液等體液中的腫瘤來源分子 (ctDNA, CTC, Exosome) | - | 癌症早期篩檢、治療監控、抗藥性偵測 | **非侵入性**為最大優點，**無法**直接分析**腫瘤組織** |
| ↳ **ctDNA** | 腫瘤凋亡/壞死釋放的**片段化 DNA** | dPCR / NGS | **監控治療反應**、偵測微小殘存疾病 (MRD) | 半衰期**極短 (1-2hr)**，能即時反映腫瘤變化 | **無法**進行蛋白質層級或藥敏測試 |
| ↳ **CTC** | 進入血液循環的**完整腫瘤細胞** | 細胞計數 / 分子分析 | 預後評估、建立細胞株做藥敏測試 | **極度稀少** (1顆 / 10⁹ 血球)，捕捉困難 | 可培養，但捕捉技術是瓶頸 |
| **LDTs** | **實驗室自行開發**、驗證、使用的檢測 | - | 當缺乏合適IVD時滿足**精準醫療**需求 | - | **不需** TFDA **上市前審查查驗登記**，但受特管法規範 |
| **IVD** | 體外診斷醫療器材，**商品化套組** | - | 常規檢驗項目 | - | 需主管機關**上市前審查許可** |
| **Bisulfite Treatment** | 亞硫酸氫鹽將**未甲基化 C → U**，5-mC 不變 | - | DNA **甲基化** (Epigenetics) 分析 | C/T 比例 | DNA 修飾分析的**金標準** |
| **aCGH** | 檢體與對照組 DNA 競爭性結合至晶片探針 | - | 染色體**拷貝數變異 (CNV)** | Log2 ratio > 0 (**增益**), < 0 (**缺失**) | 解析度高於傳統染色體分析，**無法**偵測點突變 |
| **Microarray** | 檢體與對照組 cDNA/cRNA 競爭性結合 | - | **基因表現量**分析 | 紅色 (實驗組↑)、綠色 (對照組↑)、**黃色 (表現量相近)** | 已逐漸被 RNA-seq 取代 |
| **熔解溫度 (Tm)** | DNA雙股螺旋有 **50%** 解離成單股時的溫度 | - | PCR/qPCR 反應條件設計 | 受 **GC含量**、DNA長度影響 | **黏合溫度 (Annealing)** 通常設定比Tm低約 5°C。 |
| **Nonsense Variant** | 點突變造成密碼子變為**終止密碼子** | - | 遺傳疾病、癌症 | 產生**截斷的蛋白質**，功能喪失 | UAA, UAG, UGA |
| **BCR-ABL1 [t(9;22)]** | 染色體**易位 (Translocation)** | - | 慢性骨髓性白血病 (CML) | 形成**融合基因** (Fusion gene) | 為CML的**經典標靶治療**目標 |
| **抑癌基因 (Tumor Suppressor Gene)** | 如同「煞車」，需**兩個等位基因皆突變**才失去功能 (Two-hit) | - | 遺傳性癌症風險評估 | 檢測有無功能喪失性突變 (Loss-of-function) | 常見例子：**TP53, RB1 (Rb), BRCA1/2** |
| **原癌基因 (Proto-oncogene)** | 如同「油門」，**單一等位基因**突變即可致癌 (Gain-of-function) | - | 標靶治療的目標 | 檢測有無活化性突變 (Activating mutation) | 常見例子：**EGFR, HER2, RAS, MYC** |
| **LC-MS/MS** | 液相層析結合串聯質譜 (ESI 離子化) | - | 蛋白質體學、代謝體學 | **m/z (質荷比)** 圖譜 | **無法直接測量待測物「質量」**，只能測量質荷比 |
| **VITROS 乾式分析儀** | **薄軟片型 (MicroSlide)** 技術 | - | 常規臨床生化檢測 | 擴散層提供反應結構 | **不使用**攪拌棒或螺旋狀管子來混合液體 |
| **6 Sigma 品管策略** | 評估檢驗項目品質是否達標 | **Sigma = (TEa - \|bias\|) / CV** | 實驗室品質保證 (QA) | Sigma 數值 (越高代表品質越好) | 計算時 bias (誤差) 必須取**絕對值**扣除 |
| **白蛋白 (Albumin) 檢測** | 蛋白質定量分析 | - | 肝腎功能評估 | - | 白蛋白**無色**，**絕對不可**使用可見光直接比色 |
