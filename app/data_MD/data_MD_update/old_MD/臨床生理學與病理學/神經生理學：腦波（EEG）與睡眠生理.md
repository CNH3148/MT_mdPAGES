## 類群定義

> 測驗腦波圖（EEG）的原理、電極放置（10-20系統）、正常與異常波形（α, β, θ, δ波）。涵蓋不同生理狀態（清醒、睡眠）下的腦波特徵、睡眠週期（NREM、REM）的各階段波形（如紡錘波、K複合波），以及癲癇、腦死等病理狀態的腦波表現。

## 核心趨勢與高頻考點
本單元的出題核心高度集中，除了傳統的「送分題」與「細節陷阱題」，近年更加入了「技術原理題」，考生務必掌握以下四大趨勢：

1.  **波形與狀態的「配對」是絕對核心**：歷屆考題反覆測驗「什麼狀態下，出現什麼波形」。其邏輯鏈為：**生理/病理狀態 → 大腦活化程度改變 → 特定頻率/波幅的腦波出現 → 在特定腦區記錄到**。例如：閉眼放鬆 (狀態) → 枕葉活性同步 (機轉) → 出現 α 波 (波形)。未來考法會更細，例如詢問**不同年齡層（新生兒的慢波 vs 成人的α/β波）**的正常波形差異，以及**缺氧、低血糖等病理狀態**造成的腦波慢波化。

2.  **睡眠分期特徵的「鑑別」是高頻考點**：考生必須像反射動作一樣，將 **N2期** 與 **紡錘波(Spindle) + K複合波(K-complex)** 牢牢綁定；將 **N3期** 與 **Delta波 (慢波)** 綁定；將 **REM期** 與 **類清醒波形 + 肌肉張力消失 + 心率呼吸不規則** 綁定。N2期因佔總睡眠時間最長，相關考題出現頻率最高。需注意年輕人的N3期佔比相對較高。

3.  **臨床應用的「關鍵字」必須掌握**：
    *   **癲癇 (Epilepsy)** → 關鍵波形是「**棘波與慢波複合 (Spike-and-wave)**」，常用誘發方式為「**過度換氣**」（尤其對**失神性發作**最有效）與「**閃光刺激**」。**睡眠剝奪/自然睡眠**則對誘發顳葉癲癇有幫助。
    *   **腦死 (Brain Death)** → 關鍵詞是「**腦電靜默 (Electrocerebral silence)**」，定義為波幅 **< 2 µV**，波形近乎直線。
    *   **睡眠呼吸中止症 (Sleep Apnea)** → **AHI (呼吸中止/淺呼吸指數)** 是評估嚴重度的黃金標準。必須能區分**阻塞型 (OSA)**（有呼吸努力）與**中樞型 (CSA)**（無呼吸努力）。

4.  **技術細節與儀器原理題型增加 (新趨勢)**：
    *   **10-20 系統** → 不僅要會計算相對距離，更要記憶電極命名規則：**奇數在左半腦，偶數在右半腦，'z' 在中線** (例如 Fp1 是左前額，O2 是右枕葉)。
    *   **儀器設定** → 必須理解**濾波器 (Filter)** 的功能：**高頻濾波 (HFF/Low-pass)** 用於去除肌肉等快波干擾；**低頻濾波 (LFF/High-pass)** 用於去除汗液等慢波飄移。
    *   **基本定義** → 腦波的極性慣例：**波形向上為負電位** (與心電圖相反)。頻率單位 Hz 是「每秒」週數。腦波是**電位 (µV)** 訊號。

| **β 波 (Beta)** | **> 13 Hz** (最快) | 小 (<30 µV) | **額葉 (Frontal)** | **清醒、警覺**、專注思考、焦慮 | 正常清醒時的主要背景波 |
| **α 波 (Alpha)** | 8 - 13 Hz | 中 (30-50 µV) | **枕葉 (Occipital)** | **閉眼、靜息**、放鬆 | **睜眼或思考時消失 (α block)** |
| **θ 波 (Theta)** | 4 - 7.9 Hz | 變異 | 顳葉 (Temporal) | **嗜睡、淺睡 (N1, N2)**、幼兒 | 成人清醒時出現可能為異常 |
| **δ 波 (Delta)** | **< 4 Hz** (最慢) | **大 (>75 µV)** | 廣泛性 | **深睡 (N3)**、嬰兒、腦病變 | **成人清醒時出現為嚴重異常** |
| | | | | | |
| **清醒 (Awake)** | > 13 Hz | 低 | 額葉為主 | 睜眼、專注 | **β 波**為主 |
| **NREM Stage 1** | 4 - 7 Hz | 變異 | 廣泛性 | 入睡期、嗜睡 | α 波消失，**θ 波**出現 |
| **NREM Stage 2** | 背景θ波 | 中 | 頂葉 (Vertex) | 淺睡期 | **睡眠紡錘波 (Spindles)** + **K複合波**，**佔總睡眠最長** |
| **NREM Stage 3** | **< 4 Hz** | **高** | 廣泛性 | **熟睡、慢波睡眠** | **δ 波 > 20%**，**生長激素(GH)分泌高峰**，年輕人佔比較高 |
| **REM 期** | 混合頻率 | **低** | 廣泛性 | **快速動眼期** | **類清醒波形 (似β波)**、**肌張力消失**、作夢、**心率呼吸不規則** |
| | | | | | |
| **癲癇 (Epilepsy)** | 變異 | **極高** | 病灶區 | 發作期 | **棘波與慢波複合 (Spike-and-wave)** |
| **腦死 (Brain Death)** | 無 | **< 2 µV** | 全腦 | 不可逆腦幹功能喪失 | **腦電靜默 (Electrocerebral Silence)**，近乎直線 |
| **阻塞型睡眠呼吸中止** | 睡眠波形 | 正常 | - | 睡眠中 | 口鼻**氣流停止** (≥10秒)，但**呼吸努力持續**。**AHI**為嚴重度指標。 |
| **中樞型睡眠呼吸中止** | 睡眠波形 | 正常 | - | 睡眠中 | 口鼻**氣流停止**，且**呼吸努力同時停止**。 |
| **新生兒腦波** | < 4 Hz | 低 | 廣泛性 | 正常新生兒 | **低振幅、不規則的慢波** |
| **腦波極性慣例** | - | - | - | 技術原理 | **波形向上 = 負電位** (與ECG相反) |
| **ECG干擾** | - | - | - | 技術原理 | **QRS波**因波幅最大，對EEG干擾最嚴重 |
