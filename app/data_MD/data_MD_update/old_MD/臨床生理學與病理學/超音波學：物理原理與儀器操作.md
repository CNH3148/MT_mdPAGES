## 核心趨勢與高頻考點
本單元的核心仍圍繞著「物理權衡關係」，但考題已從大原則延伸至更精細的物理定義與儀器操作細節。

1.  **頻率-解析度-穿透力三角關係 (核心必考)**：此關係不變，但考法更細。
    *   **高頻率** → 波長(λ)短 → 空間脈衝長度(SPL)短 → **軸向解析度佳 (數值小，影像細緻)** → 但能量**衰減(Attenuation)快** → **穿透力差**。適用於**表淺器官**（甲狀腺、乳房、眼科，約 7.5-15 MHz）。
    *   **低頻率** → 波長(λ)長 → 空間脈衝長度(SPL)長 → **軸向解析度差 (數值大，影像粗糙)** → 但能量**衰減慢** → **穿透力佳**。適用於**深部器官**（腹部、婦產科、心臟，約 2-5 MHz）。

2.  **聲阻抗 (Acoustic Impedance, Z) 與影像生成 (細節深化)**：
    *   **公式**：Z = 介質密度(ρ) × 介質音速(c)。
    *   **成像原理**：兩種介質的**聲阻抗「差異」越大，反射回音越強**，影像越亮。更精確的說，反射強度與**阻抗差成正比，與阻抗和成反比**。
    *   **臨床應用**：
        *   **耦合膠(gel)**：排除空氣，達成**阻抗匹配 (impedance matching)**，讓音波順利傳入體內。
        *   **組織對比**：骨骼(Z最高) > 軟組織(肝>腎) > 脂肪 > 膽汁 > 肺/空氣(Z最低)。了解相對值有助於判讀影像。

3.  **杜卜勒效應 (Doppler Effect) (模式比較恆定)**：評估血流。
    *   **原理**：偵測反射回來的音波**頻率位移 (frequency shift)**。
    *   **模式比較**：
        *   **連續波(CW)**：可測量**極高速血流**（如心臟瓣膜狹窄），無混疊(aliasing)，但**無法定位深度 (Range ambiguity)**。
        *   **脈衝波(PW)**：可**精準定位特定深度**，但受限於**奈奎斯特極限 (Nyquist limit = PRF/2)**，測量高速血流時會產生**混疊假影 (aliasing)**。

4.  **儀器與影像參數 (新興考點)**：
    *   **解析度細節**：
        *   **軸向解析度**：由 **SPL** 決定。考題會出現計算題：**SPL = 週波數(n) × 波長(λ)**，而 λ = c/f。
        *   **側向解析度**：由 **音束寬度** 決定。改善方式為**聚焦(Focusing)**。
    *   **聚焦 (Focusing)**：縮窄音束、改善側向解析度的方法。主要有三種：**聲學透鏡 (acoustic lens)**、**曲面晶體 (curved crystal)**、**電子相位調控 (phasing)**。
    *   **衰減單位 (Decibel, dB)**：為對數單位。**-10 dB** 代表音波強度衰減為原來的 **10% (1/10)**；-3 dB 約為 50% (1/2)。
    *   **儀器單位**：**脈衝重複頻率 (PRF)** 的單位是 **kHz**。

| **聲阻抗 (Acoustic Impedance, Z)** | **Z = 密度(ρ) × 音速(c)** | 介質密度與硬度 | **阻抗差異大 → 強反射** (影像基礎) | **空氣-皮膚差異極大** → 需用耦合膠 |
| **衰減 (Attenuation)** | 音波強度隨距離減弱 | **頻率**、傳播距離 | 單位：dB/cm/MHz (軟組織**約 0.5**) | **頻率越高，衰減越快**，穿透力越差 |

| **軸向解析度 (Axial)** | 沿音束方向分辨兩點的能力 | **頻率**、週波數(↓) | 取決於 **SPL**；**SPL越短越好** | 與穿透力成反比；可計算 **SPL = n × (c/f)** |
| **側向解析度 (Lateral)** | 垂直音束方向分辨兩點的能力 | **頻率**、**聚焦 (Focusing)** | 取決於**音束寬度**；**音束越窄越好** | 在焦點區最好，遠近區較差 |
| **聚焦 (Focusing)** | 縮窄音束寬度的技術 | - | 改善**側向解析度** | 方法：聲學透鏡、曲面晶體、**相位調控** |
| **折射 (Refraction)** | 音波穿透介面時方向改變 | 兩種介質的**音速(c)不同** | 造成假影的原因之一 | - |
| **B-mode (亮度模式)** | 灰階二維影像 | 回音**振幅/強度** | **解剖結構成像** (腹部、乳房、婦科) | 靜態或慢速動態影像 |
| **M-mode (動態模式)** | 單一掃描線上的深度(Y軸) vs. 時間(X軸) | 介面**移動**的幅度和速度 | **心臟瓣膜/心室壁運動**評估 (時間解析度極高) | 缺乏空間結構資訊 |
| **連續波杜卜勒 (CW Doppler)** | 連續發射與接收 | 血流速度 | 測量**高速血流** (e.g., 瓣膜狹窄) | **無深度定位性** (Range ambiguity) |
| **脈衝波杜卜勒 (PW Doppler)** | 間歇式發射與接收 | 取樣容積(sample volume) | **精準定位特定血管**的血流 | **有混疊效應 (Aliasing)**，受限於PRF |
| **脈衝重複頻率 (PRF)** | 每秒發射的脈衝數 | 儀器設定 | 決定杜卜勒可測量的**最大速度** (Nyquist極限) | 單位：**kHz** |
| **探頭：線陣 (Linear)** | 晶體直線排列 | - | **高頻**、**表淺**器官 (甲狀腺、血管) | 影像為**矩形**，接觸面積大 |
| **探頭：凸陣 (Curved/Convex)** | 晶體弧形排列 | - | **低頻**、**深部**器官 (腹部、婦產科) | 影像為**扇形 (弧頂)**，視野寬廣 |
| **探頭：相控陣 (Phased/Sector)** | 小面積，電子控制相位 (**脈波延遲器**) | - | **低頻**、心臟 (可穿過肋間) | 影像為**扇形 (尖頂)** |
| **探頭：向量陣列 (Vector)** | 結合線陣與相控陣技術 | - | 兼具大視野與矩形近場 | 影像為**梯形** |
