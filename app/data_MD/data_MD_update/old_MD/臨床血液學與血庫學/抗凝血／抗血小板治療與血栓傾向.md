## 類群定義

> 測驗常用抗凝血劑（Heparin, Warfarin/Coumadin, DOACs）與抗血小板藥物（Aspirin, Clopidogrel）的作用機制與實驗室監測方法。同時涵蓋易栓症（thrombophilia）的遺傳性風險因子，如第五因子Leiden突變、抗凝血酶/蛋白質C/S缺乏等。

## 類群說明

## 核心趨勢與高頻考點
本單元的出題核心高度集中在「**藥物機轉與監測**」及「**遺傳性易栓症**」兩大主軸，兩者幾乎各佔半壁江山。

1.  **藥物作用的因果鏈**：考題邏輯非常清晰，幾乎都是「**藥物 → 作用目標 → 影響的凝血路徑 → 監測的實驗室數據**」的單向連結。例如：Warfarin → 抑制維生素K依賴因子 (II, VII, IX, X) → 主要影響外在路徑 → 以 **PT/INR** 監測。考生必須對每種藥物的這條因果鏈瞭若指掌。

2.  **抗凝血劑的分類與比較**：
    *   **Heparin vs. Warfarin**：傳統考點，比較作用速度 (Heparin快/Warfarin慢)、作用途徑 (體內/體外)、監測方式 (aPTT/PT)。
    *   **Heparin vs. LMWH**：比較分子大小、對 Xa/IIa 的抑制比例、是否需常規監測。
    *   **傳統藥物 vs. DOACs**：DOACs (Direct Oral Anticoagulants) 是新興命題熱點，強調其「直接」抑制單一凝血因子 (Xa 或 IIa) 的特性，以及通常「不需常規監測」的優勢。

3.  **Factor V Leiden (FVL) 的獨霸地位**：在遺傳性易栓症中，FVL 的出題頻率最高。考點始終圍繞在其核心病理機制：「**Factor Va 構型改變，導致對活化蛋白C (APC) 的切割產生抗性 (APC resistance)**」。考生需能直接反射出「FVL = APC resistance」，這也是最常見的篩檢方法原理。

4.  **天然抗凝系統的重要性**：**抗凝血酶 (Antithrombin)、蛋白質C (Protein C)、蛋白質S (Protein S)** 是人體內建的煞車系統。考題會測驗其缺乏 (deficiency) 會導致血栓風險增加。特別是 Heparin 的作用必須依賴 Antithrombin，因此「**Antithrombin 缺乏會導致 Heparin 抗性 (heparin resistance)**」是極重要的連動考點。

**預測未來趨勢**：
*   **DOACs 相關考題將增加**：隨著臨床應用普及，測驗各類 DOACs (Rivaroxaban, Apixaban vs. Dabigatran) 的具體作用目標 (Xa vs. IIa) 及特殊情況下的監測方法 (如 Anti-Xa assay) 將成為趨勢。
*   **情境整合題**：提供一個臨床案例（如：年輕人反覆發生深層靜脈栓塞 VTE），要求考生判斷應優先篩檢的遺傳因子（如 FVL, PC/PS/AT 缺乏），或解釋其使用的抗凝藥物監測結果。
*   **區分出血與血栓因子**：考題會故意將造成「出血」的因子缺乏（如第八、九因子）與造成「血栓」的因子缺乏（如 PC/PS/AT）混淆，測驗考生能否清晰辨別。

| Aspirin | **不可逆**抑制 **COX-1** → ↓ TXA₂ 合成 | 血小板功能分析 (PFTs) | 初級止血 | 藥效持續血小板生命週期 (7-10天) |
| Clopidogrel (保栓通) | **不可逆**抑制血小板 **ADP (P2Y₁₂) 受體** | 血小板功能分析 (PFTs) | 初級止血 | 前驅藥物，需肝臟代謝活化 |
| **抗凝血藥物 (Anticoagulant)** | | | | |
| Unfractionated Heparin (UFH) | 增強 **Antithrombin** 活性 → 抑制 **Xa** 與 **IIa (Thrombin)** | **aPTT** | 內在/共同路徑 | 作用快、半衰期短、可被 protamine sulfate 中和 |
| Low Molecular Weight Heparin (LMWH) | 增強 Antithrombin 活性 → **主要抑制 Xa** (對IIa較弱) | 通常不需監測 (特定情況測 Anti-Xa activity) | 共同路徑 | 半衰期長、生體可用率高、HIT風險較低 |
| Warfarin (Coumadin) | 抑制**維生素K環氧化物還原酶** → ↓ **II, VII, IX, X, PC, PS** 合成 | **PT/INR** (目標通常 2.0-3.0) | 外在/共同路徑 | 作用慢 (3-5天)、受飲食/藥物影響大、**FVII**最先↓ |
| DOACs (-xaban) | **直接**抑制 **Factor Xa** (e.g., Rivaroxaban, Apixaban) | 通常不需監測 (特定情況測 Anti-Xa activity) | 共同路徑 | 口服、作用快、半衰期短 |
| DOACs (-gatran) | **直接**抑制 **Factor IIa (Thrombin)** (e.g., Dabigatran) | 通常不需監測 (特定情況測 dTT, ECT) | 共同路徑 | 口服、作用快、腎臟代謝 |
| **遺傳性易栓症 (Thrombophilia)** | | | | |
| Factor V Leiden | **Arg506Gln (R506Q)** 點突變 → FVa 對 **APC** 產生**抗性** | **APC resistance test** (功能性), PCR (基因) | 天然抗凝路徑 | **最常見**的遺傳性易栓症、主要增加**靜脈**栓塞風險 |
| Antithrombin (AT) 缺乏 | AT 合成↓或功能異常 → 無法有效抑制 IIa, Xa | AT 功能/抗原分析 | 天然抗凝路徑 | 導致 **Heparin 抗性 (Heparin resistance)** |
| Protein C (PC) 缺乏 | PC 合成↓或功能異常 → 無法有效裂解 **FVa, FVIIIa** | PC 功能/抗原分析 | 天然抗凝路徑 | 可能引發 **Warfarin 皮膚壞死** |
| Protein S (PS) 缺乏 | PS 合成↓或功能異常 → 無法輔助 **APC** 作用 | PS 功能/抗原分析 (總量/游離) | 天然抗凝路徑 | PS 為 PC 的輔因子 (cofactor) |
| Prothrombin G20210A | Prothrombin基因突變 → **Prothrombin (FII)** 濃度 **↑** | PCR (基因) | 凝血路徑 | 增加 prothrombin 產生，導致凝血亢進 |
