## 類群定義

> 涵蓋T細胞的完整生命週期，從胸腺的發育成熟（正向與負向選擇）、T細胞受體 (TCR) 的基因重組與訊息傳遞、活化所需的共刺激分子，到分化為各種輔助性T細胞亞群 (Th1, Th2, Th17, Tfh) 與調節性/毒殺性T細胞 (Treg/CTL) 的功能與相關細胞激素。

## 類群說明

身為一名資深的醫事檢驗師，我將依據歷屆考題，為你深度剖析《臨床血清免疫學》中「T細胞生物學」的知識點，並預測未來的命題趨勢。

## 核心趨勢與高頻考點

T細胞生物學是免疫學的基石，考題分布廣泛但核心概念極為集中。分析歷屆試題後，可歸納出四大命題趨勢：

1.  **T細胞亞群的功能性配對：** 這是最高頻的考點。出題模式圍繞著「**刺激源 → 轉錄因子 → 代表性亞群 → 效應細胞激素 → 臨床功能**」這條核心軸線。考生必須能像反射動作般，將 **IL-12 → T-bet → Th1 → IFN-γ → 對抗胞內菌** 這樣的因果鏈串連起來。新考題更深入到特定組織的誘導因子，如 **TGF-β + 維A酸 → 腸道Treg**。未來考題可能更深入地測驗這些亞群失衡所導致的疾病（如：Th1/Th17過強與自體免疫、Th2過強與過敏）。

2.  **T細胞活化的分子機制：** 考試重點在於「訊號的啟動與調控」。**第一訊號** (TCR-MHC) 與**第二訊號** (共刺激分子，如 CD28/B7 vs. CTLA-4/B7) 的配對與功能對比是必考題。TCR下游的訊息傳遞鏈 (Lck → ZAP-70 → LAT → PLCγ) 也是常見的選項內容。近年趨勢是測驗T細胞與B細胞訊息傳遞的異同（如TCR的ITAM在CD3/ζ鏈，BCR在Igα/Igβ），以及更精細的調控，如**FoxP3會抑制IL-2基因轉錄**。

3.  **胸腺發育與篩選：** T細胞的成熟過程是基礎，但考法固定。考生需熟記發育順序 (**DN → DP → SP**)，並精準區分**正向選擇**（於皮質，辨識自身MHC以存活）與**負向選擇**（於髓質，強力辨識自身抗原而凋亡）的地點與目的。新考題會細究各階段的表面標記，例如DN期是**CD3陰性**。篩選的嚴格性（僅少數細胞存活）也是一個重點。

4.  **T細胞的遷移與組織分佈：** 這是一個新興的命題重點。除了傳統的淋巴結歸巢 (**L-selectin, CCR7, LFA-1**)，考題開始聚焦於活化後的動態調控，如 **CD69** 表現後會**抑制S1PR1**，使T細胞暫時滯留於淋巴結。特定組織的歸巢分子（如腸道的 **α4β7, CCR9, CD103**）與細胞比例（如腸道固有層 **CD4:CD8 ≈ 3:1**）也成為新的考點。

總體而言，本科目已從單純的分子背誦，轉向更重視分子間的**邏輯功能**、**動態調控**與**組織特異性**。理解比死記更重要。
