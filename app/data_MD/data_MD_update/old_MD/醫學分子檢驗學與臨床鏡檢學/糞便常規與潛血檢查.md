## 核心趨勢與高頻考點
糞便檢查的命題趨勢高度集中於**「糞便潛血試驗 (FOBT)」**與**「消化吸收功能障礙」**兩大主軸。考生必須能清晰地闡述各類潛血試驗的原理、優缺點及干擾因素，並將糞便的物理、化學、鏡檢結果與特定的臨床病理狀態（如上下消化道出血、膽道阻塞、胰臟功能不全、脂肪痢、乳糖不耐症）進行精準的因果連結。

1.  **潛血試驗原理大亂鬥：** 這是本單元的王者級考點。出題者極愛比較各種方法的原理差異。考生必須建立清晰的分類邏輯：
    *   **化學法 (Guaiac/g-FOBT):** 利用血紅素 (Heme) 的**類過氧化酶 (Pseudoperoxidase)** 活性，氧化呈色劑 (如：Guaiac, Benzidine, Ortho-toluidine)。此法**非人類特異性**，易受含過氧化酶的食物（如：紅肉、花椰菜）或維生素C干擾。Benzidine 敏感度最高，但因致癌性已少用。
    *   **免疫法 (i-FOBT/FIT):** 利用**抗人類血紅素球蛋白 (Globin) 的抗體**進行抗原抗體反應。此法具**高度人類特異性**，不受食物干擾。但 Globin 在上消化道易被分解，故對**下消化道出血**較敏感。
    *   **螢光法 (HemoQuant):** 測定糞便中由血紅素降解而來的**卟啉 (Porphyrin)**。此法可定量，但無法區分飲食來源的 Heme 或內源性出血，因此仍有飲食干擾。

2.  **顏色與型態的臨床直覺：** 這是最基礎但也必考的連結題。
    *   生理機轉 A (上消化道出血) → 檢驗表現 B (血液經胃酸作用，血紅素變性為酸性血紅素) → 臨床診斷 C (糞便呈**柏油狀 (Tarry stool)**、黑色)。
    *   生理機轉 A (膽道完全阻塞) → 檢驗表現 B (膽紅素無法排入腸道，糞膽素原無法生成) → 臨床診斷 C (糞便呈**灰白色 (Clay-colored)**)。
    *   生理機轉 A (下消化道/直腸腫瘤壓迫) → 檢驗表現 B (糞便通過狹窄處被擠壓) → 臨床診斷 C (糞便呈**鉛筆狀/帶狀 (Ribbon-like)**)。

3.  **鏡檢下的消化吸收線索：** 顯微鏡下的發現是診斷吸收不良的直接證據。
    *   **脂肪痢 (Steatorrhea):** 鏡檢可見大量**中性脂肪** (橘紅色球狀，以 **Sudan III** 染色) 與**脂肪酸** (針狀結晶)。定量標準 (>60個中性脂肪滴/HPF) 需熟記。總脂肪測定需先用**醋酸加熱**水解。
    *   **胰臟功能不全：** 因缺乏胰蛋白酶 (Trypsin)，導致蛋白質消化不良，鏡檢可見**未消化的肌肉纖維 (Creatorrhea)**，其橫紋與縱紋清晰可見。化學法則以測定**胰臟彈性蛋白酶-1 (Elastase-1)** 最具特異性。

未來考題預測將更深入地比較**不同潛血試驗在特定臨床情境下的適用性**（例如：篩檢大腸癌應首選何者？評估上消化道慢性失血用何者定量較佳？）。此外，將鏡檢發現與特定酵素缺乏（如乳糖酶、胰蛋白酶）連結的整合型題目也會是趨勢。
