## 核心趨勢與高頻考點
本單元的核心命題邏輯，在於測驗考生對痰液中「**特定異常結構物**」的辨識能力，並能將其與「**對應的臨床疾病**」進行連結。考題極少涉及複雜的生理機轉，而是高度聚焦在**視覺辨識**與**臨床意義的直接配對**。

**高頻考點**：
1.  **心臟衰竭細胞 (Heart Failure Cell)**：這是最熱門的考點。命題圍繞其本質（吞噬含鐵血黃素的巨噬細胞）、特殊染色（普魯氏藍 Prussian blue）、以及關聯疾病（鬱血性心衰竭）。
2.  **氣喘/過敏相關結構**：
    *   **Charcot-Leyden 結晶**：來源（嗜酸性球崩解）、形態（無色、菱形、雙錐體）與臨床意義（氣喘、過敏）。
    *   **Curschmann 螺旋體**：本質（黏液栓）、形態（螺旋狀）與臨床意義（氣喘）。
    *   **Creola 小體**：來源（脫落的支氣管上皮細胞團）與臨床意義（氣喘）。
    考生必須能區分這三者，並共同連結至「氣喘」。
3.  **支氣管肺泡灌洗液 (BALF)**：近年興起的考點，重點在於**白血球分類計數**的判讀。例如：嗜中性球↑ → 細菌感染；淋巴球/肥大細胞↑ → 過敏性肺炎。
4.  **檢體採集與品質**：包含誘導咳痰法（使用生理食鹽水），以及痰液的來源（下呼吸道分泌物，非唾液）。

**命題邏輯鏈範例**：
*   **疾病 (A)** → **細胞/病理變化 (B)** → **鏡下特殊發現 (C)**
*   鬱血性心衰竭 (A) → 肺泡微血管破裂，RBC滲出 (B) → 巨噬細胞吞噬形成 **心臟衰竭細胞** (C)
*   過敏性氣喘 (A) → 嗜酸性球大量浸潤、支氣管上皮脫落 (B) → 形成 **Charcot-Leyden 結晶**與 **Creola 小體** (C)

**預測未來趨勢**：
未來考題可能更深入地測驗各種結構之間的「**鑑別診斷**」，例如給定一個臨床情境（如：慢性咳嗽伴有腐敗味），要求考生選擇最可能的鏡下發現（Dittrich氏栓子）。此外，BALF的判讀比例可能增加，並與其他間質性肺病做連結。
