## 類群定義

> 專指用於快速篩檢已知或未知基因變異的技術，而非直接定序。包括限制片段長度多型性（RFLP）、單股構象多型性（SSCP）、變性梯度膠體電泳（DGGE）、高解析度熔解曲線分析（HRM）及增幅阻礙突變系統（ARMS）等。

## 核心趨勢與高頻考點
本單元的出題核心圍繞在**各種篩檢技術的「核心原理」與「應用限制」之間的比較**。考生必須清晰地理解每種技術是利用何種物理或化學特性來區分 DNA 序列的差異。

**常見問法趨勢：**
1.  **原理辨析型：** 直接提問某技術的關鍵原理，例如 ARMS 的專一性來自引子 3' 端，DGGE 依賴熔解曲線，SSCP 則看單股 DNA 的三級結構。
2.  **條件限制型：** 提問某技術成立的「必要條件」或「最不適用」的情境。例如 RFLP 的突變必須影響限制酶切位；SSCP 使用非變性膠體；DGGE 需要 GC-clamp。
3.  **方法比較型：** 將多種技術放在同一選項中，要求選出最適合／最不適合檢測特定目標（如未知突變、已知 SNP）的方法。這是鑑別度最高的題型，需要通盤理解。
4.  **結果判讀型：** 給予實驗結果圖（如 HRM 的熔解曲線），要求判斷樣本基因型（純合子 vs. 異型合子）。

**知識邏輯鏈：**
`基因序列差異` → `導致 DNA 物理/化學性質改變` (如：限制酶切位有無、熔點高低、三級結構、雙股穩定性) → `選擇對應原理的篩檢技術` → `判讀特定結果` (如：電泳條帶有無/位移、熔解曲線變化、雜交訊號) → `推斷基因型`。

**高頻考點預測：**
未來考試將更著重於**情境應用與整合比較**。例如，給定一個臨床情境（如篩檢癌症組織中的低頻率突變），要求考生選擇靈敏度最高、最適合的方法。此外，**HRM** 因其快速、閉管、高通量的特性，在臨床應用日益廣泛，其原理與圖譜判讀的相關考題出現頻率預計會持續升高。ARMS 與 RFLP 作為經典技術，其原理細節（如 Taq 酶的特性、引子設計）仍會是必考基礎。

## 關鍵字反射表

| 檢驗技術 | 核心原理 | 檢測目標 | 關鍵試劑／條件 | 結果判讀 | 優點／缺點與限制 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **RFLP** | DNA序列差異造成**限制酶切位**的產生或消失 | **已知**突變 | **限制酶** | 電泳膠上**片段長度**改變 | 簡單 / **突變須在酶切位點** |
| **SSCP** | 單股DNA因序列不同而有不同的**三級構象** | **未知**突變 | **非變性**聚丙烯醯胺膠體 (PAGE) | 電泳膠上**移動速率**不同 | 簡單、不需特殊儀器 / 靈敏度較低、對大片段不適用 |
| **DGGE** | DNA雙股在**變性梯度**中部分熔解，導致泳動變慢 | **未知**突變 | **變性梯度膠體**、**GC-clamp** (引子5'端) | 電泳膠上**停滯位置**不同 | 靈敏度高 / GC-clamp設計、膠體製備複雜 |
| **HRM** | DNA序列影響雙股熔解溫度 (**Tm值**) | **未知/已知** | **飽和螢光染料** (如EvaGreen)、即時PCR儀 | **熔解曲線**的形狀與Tm值 | **閉管操作**、快速、靈敏度高 / 需特殊儀器與軟體 |
| **ARMS / PCR-SSP** | 引子(Primer) **3'端**與模板完全互補才能有效擴增 | **已知**突變 | **等位基因特異性引子**、缺乏3'校對功能的Taq酶 | PCR產物**有無** | 快速、簡單、專一性高 / 每個突變點需設計一組引子 |
| **ASO** | **等位基因特異性探針(Probe)**與目標DNA的雜交專一性 | **已知**突變 | **寡核苷酸探針**、嚴謹的雜交與沖洗條件 | 膜上**雜交訊號**有無 | 可高通量化 (Array) / 操作繁瑣、耗時 |
| **HA** (Heteroduplex Analysis) | **異源雙股(mismatch)**結構扭曲，導致泳動變慢 | **未知**突變 | **非變性**膠體 | 電泳膠上出現**額外條帶** | 操作簡單 / 靈敏度中等 |
| **CCM** (Chemical Cleavage) | 化學藥劑能辨識並**切割鹼基錯配(mismatch)**處 | **未知**突變 | OsO₄ (切 T,C 錯配), Hydroxylamine (切 C 錯配) | 電泳膠上出現**被切割的小片段** | 可定位突變位置 / 使用**劇毒化學品**、繁瑣 |

```Anki
RFLP (限制片段長度多型性) 技術檢測基因變異的根本前提是？;<ans>基因變異必須<b>創造或破壞</b>一個<b>限制酶辨識序列</b></ans><br><br>⚠️陷阱：若突變發生在辨識序列之外，即使只差一個鹼基，RFLP也無法檢測到。

ARMS (增幅阻礙突變系統) 的等位基因專一性主要取決於？;<ans>PCR引子 (Primer) 的 <b>3' 端鹼基</b>是否與模板DNA互補</ans><br><br>📌原理：依賴缺乏 <b>3'→5' 校對功能 (proofreading)</b> 的Taq聚合酶，一旦3'端不匹配，延伸效率會極低，導致無PCR產物。

SSCP (單股構象多型性) 分析為何必須使用「非變性」膠體電泳？;<ans>為了維持單股DNA摺疊後形成的<b>三級立體結構差異</b></ans><br><br>💡反思：若使用「變性」膠體（如尿素），所有單股DNA都會被拉直，失去結構差異，也就無法根據泳動速率不同來區分序列。

DGGE (變性梯度膠體電泳) 技術中，在引子5'端加上GC-clamp的目的是？;<ans>防止DNA片段在膠體中<b>完全解離</b>成單股</ans><br><br>📌作用：GC-clamp有極高的熔點(Tm)，確保DNA只會「部分熔解」，其泳動速率的改變才完全取決於序列本身的熔解行為，從而停在不同的變性劑濃度位置。

HRM (高解析度熔解曲線分析) 圖譜中，異型合子 (Heterozygote) 的曲線相較於純合子 (Homozygote) 有何特徵？;<ans>熔解溫度(Tm)通常<b>較低</b>，且曲線<b>形狀較寬</b>或呈現雙峰</ans><br><br>🔬原因：異型合子樣本在PCR後會形成鹼基錯配的<b>異源雙股 (Heteroduplex)</b>，其穩定性較差，故在較低溫度下就開始熔解。

下列何者最「不適合」用來篩檢一個基因中「未知」的突變？ (RFLP/SSCP/DGGE/HRM);<ans>RFLP</ans><br><br>💡辨別：RFLP需要預知突變會影響「特定」的限制酶切位，因此適用於「已知」突變的基因分型。SSCP、DGGE、HRM則是利用序列改變造成的物理性質變化來篩檢，不需預知突變位點。

ARMS技術的產物分析，最「不適合」使用下列何種儀器？ (毛細管電泳/水平式膠體電泳/微流體晶片);<ans>毛細管電泳</ans><br><br>💡理由：ARMS的結果是判斷PCR產物的「有無」，而非精確的「大小」或「序列」。使用簡單的水平式膠體電泳即可，毛細管電泳主要用於定序或片段分析，過於昂貴且非必要。

等位基因特異性檢測技術中，何者是在「雜交(Hybridization)」層面產生專一性，而非「擴增(Amplification)」層面？;<ans>ASO (等位基因特異性寡核苷酸雜交)</ans><br><br>⚠️區別：ARMS是利用<b>特異性引子</b>在PCR擴增步驟就進行篩選；ASO則是先將目標DNA擴增出來，再利用<b>特異性探針</b>去雜交來辨識。

Heteroduplex Analysis (HA) 的判讀依據為何？;<ans>在非變性膠體中，異源雙股 (Heteroduplex) 比同源雙股 (Homoduplex) 的<b>泳動速率更慢</b>，會出現<b>額外的條帶</b></ans><br><br>📌原理：鹼基錯配 (mismatch) 處會使DNA雙股結構產生扭結 (kink)，增加了泳動時的阻力。

哪一種技術的核心原理與HRM最為相似，都是基於DNA雙股的熔解特性？;<ans>DGGE (變性梯度膠體電泳)</ans><br><br>💡比較：DGGE是利用「空間」上的化學變性劑梯度來分離不同Tm值的DNA；HRM則是利用「時間」上的溫度梯度，監測螢光變化來分析Tm值。兩者皆利用<b>Tm = f(序列)</b>此一原理。
```
