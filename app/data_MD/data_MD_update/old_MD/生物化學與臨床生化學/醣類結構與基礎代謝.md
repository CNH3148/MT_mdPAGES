## 核心趨勢與高頻考點
綜合分析歷屆考題，本單元的命題趨勢有四大主軸：

1.  **基礎定義與結構辨識**：這是基本盤，每年必有。考題形式多為「下列何者為...」或「...的結構特性為何」。重點在於快速辨認單醣（如六碳酮糖：**果糖**）、雙醣（**乳糖**→半乳糖+葡萄糖）及多醣（肝醣 vs. 澱粉的**分支度**、纖維素的 **β-1,4 鍵結**）。近年新增了**醣蛋白**的考點，特別是醣基與胺基酸的鍵結方式（如 **N-糖苷鍵**）。

2.  **核心代謝路徑的「關鍵酵素」與「調控點」**：這是最大宗的考點。命題者極度偏愛將生化路徑與臨床情境結合。出題邏輯鏈為：
    *   **生理機轉 → 關鍵酵素 → 產物/結果**
        *   *範例*：糖解作用（Glycolysis）中，**Hexokinase** 催化第一步，**Aldolase** 裂解 F-1,6-BP，**Enolase** 是 **NaF** 的抑制目標。
    *   **疾病 → 酵素缺陷 → 代謝異常**
        *   *範例*：**GSD Type I (von Gierke)** → 缺乏 **Glucose-6-phosphatase** → 肝醣無法分解為葡萄糖釋放 → 嚴重**空腹低血糖**。
        *   *新增範例*：**Pyruvate carboxylase 缺乏** → 丙酮酸無法進入糖質新生 → 堆積並轉化為乳酸 → **乳酸/丙酮酸比值上升**。
    *   **荷爾蒙 → 訊息傳遞 → 代謝調節**
        *   *範例*：**胰島素** → 刺激 **GLUT4** → 骨骼肌/脂肪細胞攝取葡萄糖↑。胰島素同時也促進肝醣、脂肪合成，並**抑制酮體生成**。
        *   *新增範例*：不同轉運蛋白有不同功能，如 **GLUT5** 主要負責運輸**果糖**。

3.  **檢驗醫學的生化應用**：這是醫檢師國考的特色，將理論與實務緊密結合。考題圍繞在檢體處理與檢測原理。
    *   *範例*：為何血糖管要用灰頭管（含 **NaF**）？因為 NaF 抑制 **Enolase**，終止糖解作用，避免血糖偽性降低。另一個抑制劑 **Iodoacetate** 則作用於 **G3PDH**。
    *   *範例*：血糖測定法原理比較。**Glucose oxidase (GO-POD) 法**專一性針對 **β-D-glucose**；而**己糖激酶 (Hexokinase) 法**則透過 G6PD 作用，偵測 **NADPH** 的生成量。
    *   *新增範例*：乳酸/丙酮酸檢測，其反應利用 **LDH** 酵素，輔酶為 **NADH**，需在低溫下盡速處理檢體。

**預測未來趨勢**：除了反覆出現的 GSD Type I，未來可能更深入考 GSD Type II (Pompe) 或其他類型，並結合新生兒篩檢情境。另外，糖質新生（Gluconeogenesis）的前驅物與不可逆反應的酵素（如 **Pyruvate carboxylase**）將是潛在的重點，特別是其**缺陷造成的代謝物比值變化**。酮體代謝（肝臟無法利用的原因、糖尿病酮酸中毒 DKA）與不同**血糖測定方法學**的比較，也極具命題價值。

| **GSD Type II (Pompe)** | **Acid α-glucosidase** (缺乏) | 溶小體 (Lysosome) 內肝醣堆積 | 心肌酵素 (CK-MB) ↑ | **心肌肥大**, 肌肉無力, 新生兒篩檢項目 | - |

| **胰島素作用** | **GLUT4** (轉運蛋白) | 促進轉運蛋白移動至細胞膜, **抑制酮體生成** | 肌肉/脂肪組織葡萄糖攝取 ↑ | 降血糖 | - |
| **醣蛋白 (Glycoprotein)** | - | 轉譯後修飾 | **N-糖苷鍵 (Asn)**, O-糖苷鍵 (Ser/Thr) | 細胞辨識、訊息傳遞 | - |
| **血糖檢驗 (灰頭管)** | Enolase (目標酵素) | **NaF** 與 Mg²⁺ 結合 | 糖解作用停止 | 避免檢體存放時血糖**偽性降低** | 溶血 (K⁺ 偽高) |
| **血糖測定法 (GO-POD)** | Glucose oxidase | **專一性**氧化 **β-D-glucose** | H₂O₂ → 有色產物 | 常規血糖檢測 | 維生素C (Ascorbic acid) → 偽性降低 ↓ |
| **血糖測定法 (Hexokinase)** | **Hexokinase**, G6PD | 偶聯反應 (Coupled reaction) | **NADPH** (於 340 nm 偵測) | 參考方法 (Reference method), 專一性高 | 嚴重溶血 (釋放 G6PD) |
