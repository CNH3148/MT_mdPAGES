## 核心趨勢與高頻考點
本單元的出題核心圍繞在**「特定癌症 ↔ 特定分子標記/病原體/致癌物」**的配對記憶。考生必須建立一個清晰的知識網路，將疾病名稱與其對應的致癌基因 (Oncogene)、腫瘤抑制基因 (Tumor Suppressor Gene, TSG)、DNA修復基因、融合基因、染色體異常或致癌病毒直接連結。近年來，新增了對**「器官特異性基因圖譜」**（如不同種族的肺腺癌基因差異）與**「荷爾蒙致癌」**機轉的考察。

**常見問法變化：**
1.  **直接配對型：** 「A癌症與哪個B基因/病毒有關？」(最常見)
2.  **分類辨識型：** 「下列何者是/不是致癌基因/腫瘤抑制基因？」要求考生理解基因的功能分類。
3.  **功能機轉型：** 「病毒A如何導致癌症？」或「瓦爾堡效應的意義為何？」考到更細的分子機轉。
4.  **反向排除型：** 「下列癌症與基因配對，何者錯誤/除外？」需要更廣泛且準確的知識儲備。

**知識邏輯因果鏈：**
`致癌因子 (病毒/化學/輻射/荷爾蒙) → 特定基因突變/功能失常 (Oncogene活化 / TSG失活 / DNA修復缺陷) → 細胞核心功能失控 (細胞週期↑ / 細胞凋亡↓ / 基因組不穩定 / 代謝重編程) → 惡性轉化與增殖 → 特定癌症`

**未來考法預測：**
預計考題會從單純的記憶配對，逐漸走向**「標靶治療」**的連結。例如，不僅要知道GIST與c-KIT有關，更要連結到其標靶藥物Imatinib (Glivec)；知道肺腺癌與EGFR有關，就要聯想到EGFR抑制劑。此外，對於**基因功能**的理解會更加重要，例如區分RAS/BRAF（信號傳導）、MYC（轉錄因子）、CDK4/Cyclin D（細胞週期調節者）等不同致癌基因的角色。
