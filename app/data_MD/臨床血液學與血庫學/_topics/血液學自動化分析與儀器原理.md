## 核心趨勢與高頻考點

本單元的命題核心圍繞在兩大主流技術的「**原理對比**」與「**異常干擾判讀**」。出題者期望考生不僅能背誦原理，更能將其應用於解決臨床常見的偽性數據問題。

1.  **原理對比是基礎**：考生必須清晰區分**電阻抗法 (Impedance)** 僅依賴「**體積 (Volume)**」進行計數與分類，而**光學/流式細胞法 (Optical/Flow)** 則利用「**大小 (Forward Scatter, FSC)**」與「**內部複雜度/顆粒性 (Side Scatter, SSC)**」等多維度資訊，因此分類更為精準。這是所有變化的基礎考點。

2.  **圖形判讀是高頻重點**：直方圖 (Histogram) 與散佈圖 (Scatterplot) 的判讀極為重要。考題常以圖形或文字描述圖形特徵（如：雙峰、右偏、拖尾），要求考生連結至對應的臨床狀況。
    *   **邏輯鏈**：**臨床狀態** (如：輸血後、缺鐵性貧血治療初期) → **血球族群變化** (如：出現兩種大小的紅血球) → **直方圖/散佈圖異常** (如：RBC Histogram 出現**雙峰 (Dimorphic)**) → **數值參數變化** (如：**RDW ↑**)。

3.  **偽性數值判斷是決勝點**：這是最具鑑別度的考題類型。核心在於某些「非目標細胞」因其物理特性（大小、通透性等）被儀器「**誤判**」。
    *   **常見邏輯**：
        *   **巨大血小板 (Giant PLT)** → 體積過大，超出PLT discriminator → 被誤判為RBC → **偽性血小板減少 (PLT↓) + 偽性紅血球增多 (RBC↑)**。
        *   **EDTA 引起的血小板凝集** → 多個血小板結成一團 → 被誤判為一顆白血球 → **偽性血小板減少 (PLT↓) + 偽性白血球增多 (WBC↑)**。
        *   **紅血球碎片 (RBC fragments)** → 體積小，落入PLT計數區 → **偽性血小板增多 (PLT↑)**。
        *   **有核紅血球 (NRBC)** → 細胞核被儀器誤判為淋巴球 → **偽性白血球增多 (WBC↑)** (需校正)。

預測未來考題將更著重於「**整合性判讀**」，例如提供一份包含異常數據與圖形的 CBC 報告，要求考生不僅要指出可能的干擾源，還要能推斷潛在的疾病，並說明應採取的下一步確認動作（如：製作血液抹片鏡檢）。
