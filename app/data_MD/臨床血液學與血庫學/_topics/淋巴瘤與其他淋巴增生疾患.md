## 類群定義

> 涵蓋源於淋巴組織的腫瘤。包括何杰金氏淋巴瘤（Hodgkin lymphoma）的Reed-Sternberg細胞，以及非何杰金氏淋巴瘤（NHL）中常見的類型，如濾泡性淋巴瘤（t(14;18)）、被套細胞淋巴瘤（t(11;14)）、Burkitt淋巴瘤（t(8;14)）等，測驗其病理特徵、免疫表型與相關基因重組。

## 類群說明

身為資深醫事檢驗師，我將為您剖析《臨床血液學與血庫學》中「淋巴瘤與其他淋巴增生疾患」的命題脈絡，並提供高效率的備考材料。

## 核心趨勢與高頻考點
淋巴瘤單元的出題趨勢極為明確，已從單純的疾病定義背誦，轉向**「特徵整合與鑑別診斷」**。考生必須建立一個清晰的知識網絡，將特定疾病與其獨特的「分子標記、細胞形態、病毒關聯」三者緊密連結。

**常見問法與邏輯鏈：**

1.  **基因 → 疾病：** 給定一個染色體轉位 `t(11;14)` 或基因突變 `BRAF V600E`，直接詢問對應的疾病名稱。這是最直接、最高頻的考法。
    *   邏輯鏈：`t(11;14)` → `Cyclin D1` 過度表現 → 細胞週期失控 → **被套細胞淋巴瘤 (Mantle cell lymphoma)**。

2.  **臨床/鏡檢 → 診斷：** 提供臨床情境（如：皮膚紅疹、高 IgM 血症）或顯微鏡下特徵（如：Reed-Sternberg cell、滿天星 starry sky），要求做出最可能的診斷。
    *   邏輯鏈：周邊血見「細胞質有毛狀突起」細胞 → 懷疑**毛狀細胞白血病 (Hairy cell leukemia)** → 檢驗 `BRAF V600E` 突變與 TRAP 染色確認。

3.  **疾病 → 免疫表型/病毒：** 給定一個淋巴瘤，測驗其關鍵的 CD marker 組合或相關病毒。
    *   邏輯鏈：診斷**何杰金氏淋巴瘤 (Hodgkin lymphoma)** → 其腫瘤細胞 (RS cell) 來源於B細胞，但表型特殊 → 檢驗呈 **CD30(+)、CD15(+)**，但 **CD20(-)、CD45(-)**。

**預測未來考法趨勢：**

*   **整合型案例題增加：** 類似考題「78歲男性...IgM...骨髓...CD marker...」，會給予更多線索（如流式細胞儀散點圖、蛋白質電泳圖），要求考生綜合判斷，鑑別診斷 LPL/Waldenström's、漿細胞骨髓瘤或 MGUS。
*   **治療標靶入題：** 如「DLBCL...單株抗體」考題，未來可能測驗更多標靶藥物與其對應的 CD marker，例如 Brentuximab vedotin 標靶 **CD30**，適用於何杰金氏淋巴瘤與 ALCL。
*   **T細胞淋巴瘤比重提升：** B細胞淋巴瘤考點已相對飽和，未來將更著重於 T 細胞淋巴瘤的鑑別，如 Mycosis Fungoides/Sézary Syndrome (皮膚侵犯、腦迴狀細胞核) 與 Anaplastic Large Cell Lymphoma (ALCL) (CD30+, ALK+/-) 的區別。
