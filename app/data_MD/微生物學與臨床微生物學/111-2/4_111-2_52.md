## 題目

52.β-內醯胺酶（β-lactamase）依Ambler分類，可分為class A～D。其中何種β-lactamase之活性受EDTA抑制？

Class A
Class B
Class C
Class D

> [!success]- 正確解答
> **B**

---

## 筆記與詳解

### (1) 快速破題

看到題目關鍵字「Ambler 分類」與「受 **EDTA** 抑制」，最直覺的聯想就是 **Metallo-β-lactamase (金屬-β-內醯胺酶)**。在生化與檢驗常識中，EDTA 是一種強效的「金屬螯合劑」，會奪走酵素活性所需的金屬離子。在 Ambler 的 A~D 四大類中，只有 **Class B** 的活性中心是依賴金屬離子（鋅）來運作，其餘三類皆為絲胺酸 (Serine) 基礎。因此，看到 EDTA 抑制，直接秒殺選 Class B。

### (2) 詳解預備：關鍵字與知識點

* **Ambler 分類** → 依據酵素活性中心的胺基酸序列與結構特徵，將 β-lactamase (β-內醯胺酶) 劃分為 Class A、B、C、D 四大類。
* **EDTA (Ethylenediaminetetraacetic acid)** → 乙二胺四乙酸，一種極強的雙價金屬離子螯合劑（如 $Ca^{2+}$, $Mg^{2+}$, $Zn^{2+}$）。
* **Class B β-lactamase** → 全名為 Metallo-β-lactamase (MBL，金屬-β-內醯胺酶)。這類酵素（如 NDM、IMP、VIM）的活性中心需要鋅離子 ($Zn^{2+}$) 參與催化水解反應。臨床常利用 **EDTA 雙紙錠協同試驗**，觀察抑菌圈擴大的特徵[🖼️](https://www.google.com/search?q=%22EDTA+double+disk+synergy+test%22+MBL&tbm=isch&udm=2) 來確認 MBL 的存在。

### (3) 詳解

本題測驗各類 β-lactamase 的生化特性與抑制劑對應關係。

* **(A) Class A (錯誤)**：活性中心為 **Serine (絲胺酸)**。代表性酵素為廣效性 β-內醯胺酶 (ESBL) 與 KPC。它們會被 Clavulanic acid 等傳統抑制劑抑制，但**不受 EDTA 影響**。
* **(B) Class B (正確)**：活性中心為 **Zinc ($Zn^{2+}$)**。加入 EDTA 會將其中心的鋅離子螯合剝奪，導致酵素完全失去水解抗生素的能力而失活。
* **(C) Class C (錯誤)**：活性中心為 **Serine**。代表性酵素為 AmpC。它對傳統抑制劑 (Clavulanic acid) 具有抵抗性，也**不受 EDTA 影響**。
* **(D) Class D (錯誤)**：活性中心為 **Serine**。代表性酵素為 OXA (Oxacillinase)。通常對傳統抑制劑反應微弱，且**不受 EDTA 影響**。

### (4) 核心軸線與邏輯因果

抗藥性細菌帶有 Class B (MBL) 基因 → 產生以 $Zn^{2+}$ 為核心的金屬水解酶 → 水解絕大多數 β-lactam 類抗生素 (含 Carbapenem) → 實驗室加入 **EDTA (金屬螯合劑)** → 奪走酵素中心的 $Zn^{2+}$ → MBL 結構瓦解/失去活性 → 細菌恢復對抗生素的敏感性 (抑菌圈恢復) → 藉此確診為 Class B 產酶菌株。

### (5) 橫向比較：Ambler 分類法與 β-lactamase 特性統整

|**Ambler 分類**|**活性中心**|**代表性酵素 (舉例)**|**被傳統抑制劑抑制(Clavulanic acid / Tazobactam)**|**受 EDTA 抑制(金屬螯合)**|**臨床重點標靶藥物抵抗力**|
|---|---|---|---|---|---|
|**Class A**|**Serine** (絲胺酸)|**ESBL** (CTX-M, SHV)<br>  <br>**KPC** (Carbapenemase)|**(+) 是**|(-) 否|ESBL 剋 Cephalosporins<br>  <br>KPC 剋 Carbapenems|
|**Class B**|**Zinc ($Zn^{2+}$)**|**MBL** (NDM, IMP, VIM)|(-) 否|**(+) 是**|剋 Carbapenems<br>  <br>_(例外：對 Aztreonam 無效)_|
|**Class C**|**Serine** (絲胺酸)|**AmpC**|(-) 否|(-) 否|剋 Cephalosporins (特別是第一至第三代)|
|**Class D**|**Serine** (絲胺酸)|**OXA** (Oxacillinase)|+/- (通常效果差)|(-) 否|剋 Carbapenems (如鮑氏不動桿菌常見)|

### β-lactam 抗生素的四大神獸

這四大類抗生素的共通點，就是它們的化學結構核心都有一個「**β-lactam 四元環**」。這個環就像是炸藥包，只要它能保持完整並與細菌的細胞壁合成酵素（PBP）結合，就能干擾細菌蓋城牆，讓細菌破裂死亡。

---

#### 1. 青黴素類 (Penicillins)：開國元老

這是人類最早發現的抗生素家族。因為出來得早，細菌對它們也最熟悉，所以極容易被細菌分泌的 β-lactamase（也就是前面提的 Class A~D 酵素）破壞。

為了反制細菌，科學家不斷修改它的側鏈，發展出幾個重要分支：

- **天然青黴素：** 對付梅毒、化膿性鏈球菌還是很有效。
    
- **抗葡萄球菌青黴素 (如 Oxacillin)：** 稍微改變結構，讓早期的 Class A 酵素咬不動，專門用來對付一般的金黃色葡萄球菌。（這也是為何 Class D 酵素被稱為 Oxacillinase，因為 Class D 演化出能切斷它的能力）。
    
- **廣效性青黴素 (如 Amoxicillin)：** 增加了對革蘭氏陰性菌的穿透力，是診所最常開的感冒併發細菌感染用藥。
    
- **抗綠膿桿菌青黴素 (如 Piperacillin)：** 專攻難纏的綠膿桿菌，通常會搭配抑制劑（Tazobactam）一起使用，變成醫院病房的重症神藥（俗稱 Tazocin）。
    

#### 2. 頭孢菌素類 (Cephalosporins)：主力部隊


- **第一代** (G1, 英文開頭常 **"Cefa-"** 或 **"Cepha-"**)：骨科手術前預防感染、皮膚與軟組織感染（主打陽性菌）。**Cefazolin**（常見注射劑，台灣手術前預防雷打不動的主力）、**Cephalexin**（常見口服藥，俗稱「Keflex」）。

- **第二代** (G2, **Cefuroxime**、**Cefoxitin**（常歸類於此，對厭氧菌有效）)：承上啟下，陽性、陰性菌都能殺一點，常演變成肺炎、骨盆腔感染的經驗性用藥。

- **第三代** (G3, 英文結尾常 **"-one"**、**"-ime"** 或 **"-ten"**)：*【Class A (ESBL) 與 Class C 的最愛】* 陰性菌的殺手，能穿透腦血屏障，常用於嚴重的敗血症、腦膜炎、院內感染。**Ceftriaxone**（臨床極常用，一天只需打一針的明星藥）、**Ceftazidime**（對抗頑強「綠膿桿菌」的特效藥）、**Cefotaxime**。
> 遇上 **Class A (ESBL)**：ESBL 突變後的活性口袋變大，這代的所有藥物會被它通通切碎失效。
>
> 遇上 **Class C (AmpC)**：AmpC 本身的大口袋就是為了這代藥物的；一、二、三代碰上 AmpC 通通陣亡。

- **第四代 (G4, 如 Cefepime)：** 結構升級，不僅對陰性菌（如綠膿桿菌）殺傷力極強，
>還能抵抗 Class C (AmpC) 酵素的水解，
>
>但依然會被 Class A (ESBL) 破壞。
    
- **第五代 (G5, 如 Ceftaroline)：** 這是個特例！它是所有 β-lactam 抗生素中，**唯一能夠擊殺 MRSA（超級金黃色葡萄球菌）** 的武器。
    

#### 3. 碳青黴烯類 (Carbapenems)：最後防線

也就是字尾有 "-penem" 的重裝武器（如 Meropenem）。結構極其堅固，絕大多數的酵素都拿它沒轍，只有演化到頂點的 KPC (Class A 突變) 或 MBL (Class B 金屬酵素) 才能破壞它。

當細菌對上述的第三代 Cephalosporins 產生抗藥性時，臨床醫師就會祭出這顆大炸彈。它們的結構極度緊密，立體障礙強，一般水解酶根本咬不動它。

- **代表藥名特徵：** 英文藥名一律以 **"-penem"** 結尾。
    
- **代表藥物：**
    
    - **Imipenem**（通常與 Cilastatin 合劑，常簡稱 IMI）
        
    - **Meropenem**（臨床極常用，副作用較低，常簡稱 Mero）
        
    - **Ertapenem**（一天打一次，俗稱「魚排」，但對綠膿桿菌無效）
        
    - **Doripenem**
        
- **抗藥性關聯：** * 只要細菌帶有 **Class B (MBL，如 NDM)** 或 **Class A 突變型 (KPC)**，這類最後防線的藥物就會被瞬間水解，變成無藥可醫的「超級細菌」。

#### 4. 單環類 (Monobactams)：獨行刺客

只有一個孤單的 β-lactam 環（Aztreonam）。專殺陰性菌，對陽性菌完全無效。最大優點除了能躲過 Class B (MBL) 的追殺外，它也很適合用於對「青黴素嚴重過敏」的患者，因為結構差異夠大，較不會引起交叉過敏反應。

這個家族在臨床上成員極少，它最大的特色是結構只有「單環」，不像前面的青黴素或頭孢菌素是「雙環」。

- **代表藥物：** **Aztreonam** (亞茲崔南)
    
- **抗藥性關聯的關鍵：** * **Class B (MBL)** 這種金屬酶在水解抗生素時，雙手（結構）必須死死抱住抗生素的「雙環結構」才能好催化。
    
    - 因為 **Aztreonam 只有單環**，Class B 酵素根本「無處下手」，因此 **Aztreonam 成了唯一能逃過 Class B 毒手的 β-lactam 抗生素！**（不過它對 Class A 或 Class C 依然沒有抵抗力）。
