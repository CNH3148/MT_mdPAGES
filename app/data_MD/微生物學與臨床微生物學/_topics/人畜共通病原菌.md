## 類群定義

> 將主要經由動物傳播給人類的病原菌歸為一類。涵蓋布氏桿菌（Brucella）、法蘭西斯氏菌（Francisella tularensis）、鼠疫耶氏菌（Yersinia pestis）、巴斯德氏菌（Pasteurella multocida）、貝氏考克斯菌（Coxiella burnetii）等的傳播途徑、所引起的疾病（如布氏桿菌病、兔熱病、鼠疫、Q熱）及實驗室操作安全。

## 類群說明

## 核心趨勢與高頻考點
人畜共通病原菌單元的核心考點高度集中，主要圍繞著「病原菌–宿主/媒介–疾病」三者間的連結。出題者期望考生能像偵探一樣，從動物接觸史（如：牛、羊、豬、馬、兔、貓狗）或病媒（如：蚤、蜱）的線索，直接聯想到對應的病原菌與其引發的典型疾病。

**常見問法與邏輯鏈：**
1.  **宿主/媒介 → 病原菌：** 最常見的考法。給定一種動物或節肢動物媒介，要求選出最相關的病原菌。
    *   *邏輯鏈範例：* 接觸牛羊或其乳製品 → 懷疑 **布氏桿菌 (Brucella)** → 引起波浪熱。
    *   *邏輯鏈範例：* 接觸野兔或被蜱叮咬 → 懷疑 **法蘭西斯氏菌 (Francisella)** → 引起兔熱病。
    *   *邏輯鏈範例：* 接觸豬或其製品 → 懷疑 **小腸耶氏菌 (Y. enterocolitica)** → 引起腸胃炎。
    *   *邏輯鏈範例：* 接觸馬、騾、驢 → 懷疑 **鼻疽伯克氏菌 (Burkholderia mallei)** → 引起鼻疽病。
2.  **疾病名稱 → 病原菌：** 直接考察特定疾病的致病原。
    *   *邏輯鏈範例：* 臨床診斷為「貓抓病」 → 病原菌為 **韓瑟勒巴通氏菌 (Bartonella henselae)**。
3.  **生化特性 → 病原菌：** 針對重點菌種（尤其是*Yersinia*）考察其關鍵的生化鑑別特徵。
    *   *邏輯鏈範例：* 動力試驗呈現 **25°C 有動力 / 37°C 無動力** → 指向 **小腸耶氏菌 (Y. enterocolitica)** 或 **偽結核耶氏菌 (Y. pseudotuberculosis)**，並藉此與恆無動力的鼠疫桿菌區分。
4.  **臨床表徵 → 病原菌：** 較進階的考法，從症狀反推病原菌。
    *   *邏輯鏈範例：* 非典型肺炎 + 屠宰場/酪農工作史 → 高度懷疑 **貝氏考克斯菌 (Coxiella burnetii)** → 引起 Q 熱。

**未來趨勢預測：**
除了傳統的「配對」題型，未來考題可能更著重於「鑑別診斷」與「實驗室安全」。例如，給定一個非典型肺炎的案例，選項中同時出現*Coxiella burnetii*、*Francisella tularensis*、*Chlamydia psittaci*（鸚鵡熱），要求考生根據更細微的流行病學線索（如動物接觸史是牛羊、野兔還是鳥禽）做出判斷。此外，對於傳播媒介的精準記憶也日益重要，考生需能區分**蚤 (flea)、蜱 (tick)、蟎 (mite)**等不同節肢動物所傳播的疾病。這些病原菌多數具有高度傳染性，**生物安全等級 (BSL-3)** 的概念將成為必考重點，尤其在區分哪些檢體操作需要在生物安全櫃中進行。
