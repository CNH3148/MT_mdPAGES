## 類群定義
> 涵蓋細菌對抗生素產生抗藥性的各種分子機制。包括酵素水解或修飾藥物（如β-內醯胺酶、Aminoglycoside 修飾酶）、改變藥物作用標的（如MRSA/肺炎鏈球菌的PBP突變、VRE的vanA/B基因）、降低細胞膜通透性（如Porin缺失）及主動排出藥物（如Efflux pumps）。也探討不同菌種的內生性與後天性抗藥特徵。

## 類群說明
身為權威醫事檢驗師，我為您分析《微生物學與臨床微生物學》國考中「細菌抗藥性機轉」此一知識點，並提供高效率的總結與考題預測。
## 核心趨勢與高頻考點
此單元的出題核心圍繞在「**病原體 - 抗藥機制 - 關鍵基因/產物 - 對應藥物**」四者之間的配對。考生必須建立清晰的因果鏈，並精準區分各類機轉的細節，近年考題越來越強調不同酵素與機制的細微差異。
1.  **四大抗藥機制的分類與辨別**：
    *   考題常要求判斷特定抗藥性屬於哪一類（酵素修飾/水解、標的改變、通透性下降、主動排出）。
    *   **Aminoglycoside 抗藥機轉**：主要為**酵素修飾** (如 Acetyltransferase, AAC)、**降低攝取**、或**核糖體標靶改變**。國考特別考出陷阱：**「抗生素結合蛋白大量表現 (Target overproduction)」不屬於** Aminoglycoside 的抗藥機制。
    *   **Penicillin 抗藥機轉**：革蘭氏陽性菌（如肺炎鏈球菌 *S. pneumoniae* 與腸球菌 *Enterococcus*）對 Penicillin 的抗藥性，絕大多數肇因於 **PBPs 結構突變（改變標的）**，而非產生 ESBL。
2.  **明星菌株與其招牌機制**：
    *   **金黃色葡萄球菌 (MRSA/VRSA/VISA)**：`mecA`基因 → PBP2a → 改變標的 (MRSA)。`vanA`基因 → D-Ala-D-Lac → 改變標的 (VRSA)。**細胞壁增厚** → Vancomycin 穿透受阻 (VISA)。**觀念澄清**：金葡菌為革蘭氏陽性菌，**沒有外膜**，絕對不會有 Porin 缺失的抗藥機轉。
    *   **腸內菌科 (ESBL vs. AmpC vs. CRE)**：
        *   **ESBL**：能水解第三代頭孢菌素，但**無法水解 Carbapenems**，且**可被 Clavulanate 抑制**。
        *   **AmpC β-lactamase**：常見於 **MYSPACE** 菌群（其中 M 代表 ***Morganella morganii***），多由染色體攜帶，屬於 Serine-protease。其最大特色是**無法**被一般的 β-lactamase inhibitors (如 Clavulanate) 抑制。（註：國考曾將「具備水解 carbapenems 功能」列為 AmpC 的相對適當選項，因在特定條件下可能展現微弱活性，需依題意判斷）。
        *   **Carbapenemase (CRE/CPE)**：如 **KPC** (Serine class A) 或 **NDM** (New Delhi metallo-β-lactamase, Class B，需金屬鋅離子參與)，能強效水解 Carbapenems。
    *   **綠膿桿菌 (*P. aeruginosa*)**：`oprD`基因突變 → Porin 缺失 → 通透性下降 → 對 Carbapenems (尤其 Imipenem) 抗藥。
    *   **結核分枝桿菌 (*M. tuberculosis*)**：第一線感受性測試用藥包含 INH, RIF, EMB, PZA。對應抗藥基因必考：`rpoB`→Rifampin, `katG`/`inhA`→Isoniazid, `pncA`→Pyrazinamide, `embB`→Ethambutol。
3.  **內生性 (Intrinsic) vs. 後天性 (Acquired) 抗藥性**：
    *   **內生性範例**：GNB 對 Vancomycin (外膜屏障)、厭氧菌對 Aminoglycosides (需氧氣運輸)、**`Leuconostoc` spp. 對 Vancomycin** (天生細胞壁為 D-Ala-D-Lac)。
    *   **新興考點**：***Stenotrophomonas maltophilia*** 因天生具備 **L1 金屬內醯胺酶基因**，對 **Imipenem** 具有強大的內生性抗藥性；且身為 GNB，亦對 Vancomycin 具內生抗藥性。
## 未來考點預測與易混淆陷阱
*   **易混淆陷阱**：將 GNB 的抗藥機制 (如 ESBL、Porin 缺失) 套用在 GPC 上，或將 AmpC 與 ESBL 對抑制劑的反應搞混。
*   **未來考點預測**：Carbapenemase 的分類將考得更細，例如區分 Ambler class A (KPC), class B (NDM, VIM, IMP 需金屬離子), class D (OXA)。以及新型基因捕捉元件（Integron 整合子透過 `attI` 位點捕捉基因卡匣）在抗藥性傳播上的角色。
