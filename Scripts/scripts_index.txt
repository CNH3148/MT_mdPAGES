# 全域腳本索引 (Scripts Index)

這是本專案自動化腳本的「大腦」。請在每次新增或修改分類資料夾/腳本時更新本文件。

## 分類目錄一覽

### 1. Build-Website
- **用途概述**: 負責將原本基於本機伺服器的應用程式，轉換與打包成純靜態網頁格式，以利部署至 GitHub Pages 等靜態伺服器。
- **包含腳本**:
  - `build_website.py`: 
    - **核心功能**: 自動複製靜態檔案、呼叫 `cache_builder` 將 Markdown 轉為 JSON 快取、並透過正則表達式自動註解掉 JS 中的後端 API 呼叫，同時修正 HTML 的相對路徑。
    - **歷史使用紀錄**: 初始化建立，用於 GitHub Actions 自動化部署流程。
