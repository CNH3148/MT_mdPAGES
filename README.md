## 📌 程式簡介
這是一個隨插即用的「免安裝版」醫檢師刷題系統。
內含所有的題庫資料與獨立的執行環境，您可以直接將整個資料夾放在隨身碟中，帶到任何 Windows 電腦（如圖書館公用電腦）上執行，完全不需要安裝任何軟體！

## 🚀 如何啟動與關閉

**【 如何啟動 】**
1. 雙擊本資料夾中的 `Start_App.bat`。
2. 系統會彈出一個黑色的終端機視窗（這是背景伺服器，**請勿關閉它**）。
3. 程式會自動幫您打開瀏覽器並進入系統，即可開始刷題！

**【 如何關閉 】**
1. 刷題結束後，請先關閉瀏覽器的刷題分頁。
2. 回到黑色的終端機視窗，點擊右上角的「X」關閉即可停止伺服器。

## 🎨 切換介面風格
您可以隨時執行資料夾中的 `Change_Style.bat` 來切換不同的介面風格。
即使在伺服器啟動之後也可以隨時切換；但請記得，切換之後需要到瀏覽器按下 `Ctrl+F5` 強制重新整理網頁，新風格才會生效。

## ⌨️ 快捷鍵指南 (卡片練習模式)

為提升刷題效率，本系統在「卡片模式」下支援豐富的鍵盤快捷鍵。系統會根據您是否正在打字，自動切換快捷鍵邏輯：

### 🎯 情境一：一般瀏覽與答題狀態
當您正在看題目，且**沒有**在輸入框打字時：

* **作答與切換**
  * `1` / `2` / `3` / `4`：選擇對應選項 A / B / C / D
  * `←` (方向鍵左)：切換至「上一題」
  * `→` (方向鍵右)：切換至「下一題」
* **輔助功能**
  * `0`：開啟「查看原始 PDF」
  * `.` (小數點)：切換「收藏」狀態（右上角星號）
  * `Space` (空白鍵)：觸發畫面下方的「寫筆記」按鈕（展開詳解區）
  * `T`：快速將游標對焦到「輸入自訂標籤區」，方便直接打字新增標籤
* **筆記編輯**
  * `N`：觸發「編輯筆記（鉛筆圖示）」
    * 若編輯區未開啟：自動展開編輯區，並將游標**自動對焦**到文字最後方，馬上開始打字。
    * 若編輯區已開啟：會再次幫您把游標抓回編輯區，防止游標跑掉。

### ✍️ 情境二：正在打字輸入的狀態
當您的游標已經在**筆記編輯區**或**自訂標籤輸入框**內打字時，一般快捷鍵會暫停：

* `ESC` 鍵：取消對焦並暫停輸入（若標籤不想輸入了，或打完筆記想先放著不存檔，按 ESC 即可跳出）。
* `Enter` 鍵：在標籤輸入框內打完標籤後，按下即可新增。
* `Ctrl + S`：快速儲存筆記（打完筆記後不需動用滑鼠，直接按下即可儲存並關閉編輯模式）。

## 💡 學習小訣竅：結合 Gemini AI 快速產生高質感詳解

目前題庫中 110~115 年的題目皆已附上一行簡單的「破題關鍵 (key concept)」。若您需要更完整的詳解，強烈建議您使用 Google Chrome 瀏覽器開啟本程式，並於「卡片模式」下搭配 Chrome 內建的「問問 Gemini」側邊欄，利用「個人技能 (Gemini Skills)」功能來快速產生優質詳解，並直接複製貼上至您的個人筆記區！

**【 操作步驟 】**
1. 在 Chrome 網址列輸入 `chrome://skills` 進入技能設定頁面。
2. 點擊新增一項技能，將技能名稱設定為 `/test_explainer_1Q`，並將下方的提示詞 (Prompt) 完整貼上。
3. 在本程式的「卡片模式」下顯示某一題時，按下 `Alt+G` 快速開啟 Gemini 側邊欄。
4. 於對話框輸入 `/` 並選擇 `/test_explainer_1Q` 技能，接著輸入對應的正確答案（例如：「`/test_explainer_1Q ans: C`」），送出即可獲得圖文並茂的優質解析！

**👇 請複製以下區塊內的所有文字，作為技能的提示詞 (Prompt)：**

```text
請依照以下原則解說畫面中題目：

【全局排版與搜圖規則】
若在解說或表格中提及「視覺特徵」（如：微生物聚落型態、細胞病變 CPE、血液抹片細胞、尿液沉渣或臨床表徵等），請**直接在該特徵描述後方緊接著**附上 Google 圖片搜尋的 Markdown 超連結，**絕對不可以**在文末獨立生成一個段落來放連結。
1. 連結顯示文字請極致精簡，只使用圖示：`[🖼️]`。
2. 搜尋詞必須轉為「精準英文名詞」（專一特徵請加雙引號）。
3. 視情況加上輔助詞：實驗室/鏡檢特徵請加上檢驗方法（如 Gram stain, blood smear 等）；臨床表徵可加 clinical sign。
4. 網址空白替換為 `+` 號。網址格式請嚴格使用（包含 udm=2 參數）：`[🖼️](https://www.google.com/search?q=英文關鍵字1+英文關鍵字2&tbm=isch&udm=2)`

【解題步驟】
(1) (快速破題) 首先告訴我最快速、直接的「解題思路」；
比如「看到題目說 A (某關鍵字)、B...，並且要找『錯誤』的 → 就要想到 C」；或者，「題目問『P』，選項 A、B、D 都是 Q (某類別/特色)，只有 C 是 R (某種與其他選項區別的特質) → 因此可以猜到答案是 C」；又或者「在 X (某領域常識) 中，最為人所知的就是 Y (某個重點)，因此看到這『些關鍵字 (條列可能出現的)』 → 就知道大概要考...」

(2) (詳解預備) 拆解出這個題目 (包含題幹/選項) 中的「專有名詞」或「關鍵字」列表，接著以「關鍵字 → 知識點」的方式給予最直接的解釋或重點提醒。

(3) (詳解) 針對問題本身以及所有選項，解釋為何某些選項正確、某些錯誤；徹底解釋整個題目。

(4) 務求正確不冗長，將零散的知識點梳理出核⼼軸線與邏輯因果（例如：⽣理機轉 A → 檢驗數值變化 B → 臨床診斷 C）(縱向聯想)；如果有以表格統整比較的價值 (除了本題內容以外，其他高度相關但題目未提及等一起比較) (橫向比較)，請在文字說明之後，使表格盡量符合 (5) 精神。

(5) 表格排版精神：
1. 矩陣式對比：請以「疾病/病原體/生理狀態」或「檢驗項目」為縱軸；橫軸必須是具體的比較維度（如：致病機轉、生化指標變化、尿液試紙反應、鏡檢特徵、干擾物質）。請自行調適表格的欄位盡量把所有考試重點統整成「一個表」中；不過如果有特殊需求也可以另外製作表格。
2. 關鍵字強化：請將表格內「最具有鑑別度」的字眼（例如：**特定圓柱體名稱**、數值的**上升/下降**、**偽陰性**的成因）加上**粗體**。
3. 極致精簡：盡量不要使用完整的敘述句！請一律轉換為「名詞、箭頭符號 (↑/↓)、數據短語」。表格必須乾淨俐落。
