## 類群說明

身為一名權威醫事檢驗師，我將針對《醫學分子檢驗學與臨床鏡檢學》中「表觀遺傳學與蛋白質體學應用」此一單元，提供深入的考點分析與趨勢預測。

## 核心趨勢與高頻考點
本單元的命題趨勢涇渭分明，可分為「表觀遺傳學」與「蛋白質體學」兩大主軸，兩者幾乎不會混合出題。

1.  **表觀遺傳學：聚焦 DNA 甲基化分析**
    *   **核心邏輯**：檢測 DNA 甲基化的關鍵在於「**創造可被偵測的差異**」。其邏輯鏈為：
        `原始 DNA (甲基化 C vs. 未甲基化 C) → 化學處理 (重亞硫酸鹽) → 未甲基化 C 轉變為 U，甲基化 C 不變 → 序列產生差異 → 利用 PCR 或定序技術偵測此差異`
    *   **高頻考點**：
        *   **原理題**：必定測驗**重亞硫酸鹽 (Bisulfite)** 的作用機制，即「未甲基化 C → U」。這是所有衍生技術的基礎，為必考中的必考。
        *   **方法學比較**：常測驗各種甲基化分析方法的原理與特性，如 **MethyLight** (定量的 Real-time PCR)、**MSP** (定性的 PCR)，以及較傳統的**甲基化敏感限制酶 (MSRE)** 法，並比較其靈敏度。

2.  **蛋白質體學：聚焦分離與鑑定技術**
    *   **核心邏輯**：分析複雜蛋白質混合物的標準流程為「**分離 → 鑑定**」。
    *   **高頻考點**：
        *   **二維電泳 (2D-PAGE)**：極度熱門的考點。必須牢記其兩階段分離依據：第一維是**等電點聚焦 (IEF)**，依據蛋白質的**等電點 (pI)**；第二維是 **SDS-PAGE**，依據**分子量 (MW)**。
        *   **質譜法 (MS)**：**MALDI-TOF** 在臨床上的應用（特別是**細菌鑑定**）是必考重點。此外，樣品前處理的**酵素水解**（尤其是 **Trypsin** 的切割位點）也是常見的計算題型。
        *   **新技術比較**：常將新舊技術進行比較，例如 **DIGE** 如何解決傳統 2D-PAGE 的「膠間誤差」問題；**SELDI-TOF** 的「蛋白晶片」特色為何。

總體而言，考生需掌握各技術的「**核心原理**」與「**應用場景**」，並能比較其優劣。
