## 核心趨勢與高頻考點
綜合分析歷屆考題，本單元命題趨勢高度集中在「**鑑別診斷**」與「**功能與形態的連結**」。出題者期望考生能從一張血液抹片、一個檢測數據或一段臨床描述，反推出背後的細胞生理或病理機制。

1.  **反應性 vs. 腫瘤性：** 「**類白血病反應 (Leukemoid Reaction)**」是絕對的命題核心。考題圍繞其與慢性骨髓性白血病 (CML) 的鑑別，形成一個黃金考點組合：
    *   **生理機轉 (嚴重感染/發炎) → 血液學變化 (白血球↑、核左移、毒性變化) → 關鍵檢測 (LAP score ↑↑) → 鑑別診斷 (排除 CML)**。

2.  **遺傳性功能/形態異常：** 這是另一大命題熱區，重點在於「**基因缺陷 → 細胞異常 → 臨床表現/診斷測試**」的因果鏈。
    *   **CGD (慢性肉芽腫病)**：**NADPH oxidase** 基因缺陷 → 呼吸爆發失敗 (殺菌力↓) → **NBT 試驗陰性**。
    *   **Chédiak-Higashi 症候群**：**LYST** 基因缺陷 → 溶酶體運輸融合障礙 → 細胞內**巨大顆粒**、部分白化症。
    *   **Pelger-Huët 異常**：Lamin B receptor 基因變異 → 核分葉不全 → **雙葉核 (夾鼻眼鏡)**，但**功能正常**。

3.  **傳染性單核球增多症 (Infectious Mononucleosis)**：此為獨立的明星考點，常見的陷阱題設計如下：
    *   **病原體 (EBV) → 感染標的 (B 細胞) → 免疫反應 (CD8+ T 細胞活化增生) → 抹片所見 (反應性/非典型淋巴球)**。考生必須清楚知道，抹片上看到的並非被感染的 B 細胞，而是反應性的 T 細胞。

4.  **細胞發育與表面標記：** 這是基礎題，但必考。重點包含顆粒球的成熟順序（特別是**有絲分裂池**與**儲存池**的分界）、各階段顆粒的出現時機，以及 T/B/NK/單核球的關鍵 **CD 標記**。
