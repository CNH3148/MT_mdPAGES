## 核心趨勢與高頻考點
本單元的出題核心圍繞著「**分級與比較**」，無論是物理/化學滅菌方法的選擇，還是生物安全等級的劃分，都強調在特定情境下做出「最適當」的判斷。考生必須清晰地建立一套邏輯因果鏈：

1.  **病原體風險等級 → 生物安全等級 (BSL) → 防護設備與設施要求**
    *   例如：病原體為**結核分枝桿菌** (具空氣傳播高風險) → 必須在 **BSL-3** 實驗室操作 → 因此需要使用 **Class II 以上的生物安全操作櫃 (BSC)**，且實驗室需為**負壓定向氣流**設計，對應到臨床端則是**負壓隔離病房**。

2.  **物品/器械的特性 → 滅菌/消毒方法的選擇**
    *   例如：器械為**不耐高溫**的手術內視鏡 → 不可使用高壓蒸汽滅菌 → 應選擇**化學滅菌法**，如**戊二醛 (Glutaraldehyde)** 或環氧乙烷 (ETO)。

高頻考點高度集中在：
*   **高壓蒸汽滅菌法**的原理、參數（溫度、壓力、時間）及**限制**（如不適用於含熱敏感成分的培養基）。
*   **生物安全等級 (BSL 1-4)** 的代表病原體與操作規範的配對，特別是 **BSL-2 (醫院常規實驗室)** 與 **BSL-3 (結核菌、炭疽桿菌)** 的區分。
*   **生物安全操作櫃 (BSC Class I, II, III)** 的防護對象（人員、檢體、環境）與適用情境的比較，其中 **Class II 為最常考**，**Class III 提供最佳人員防護**。
*   **HEPA filter** 作為 BSC 過濾空氣的核心組件及其**過濾極限**（無法濾除「所有」微生物）。
*   **傳播途徑的區分**：**空氣傳播 (Airborne)** vs. **飛沫傳播 (Droplet)** vs. **接觸傳播 (Contact)**，以及對應的隔離措施（負壓病房、N95口罩、手部衛生）。
*   **傳染性物質的運送規範**，特別是**三層包裝系統**的原則。

未來的趨勢預測將更著重於**情境應用與反向思考題**，例如給定一個實驗流程或臨床檢體，要求考生判斷其中不符合生物安全規範的環節，或是為特定檢體選擇最適合的消毒運送方式。也可能出現反向提問，如給定一個BSL-3實驗室的配置，詢問哪種病原體「不適合」在此操作（如BSL-4病毒）。

| | 乾熱滅菌 | 高溫氧化 | 玻璃器皿、油、粉末 | 160-170°C, 2小時 | 穿透力差、時間長；QA指示劑: ***Bacillus atrophaeus*** |

| | 輻射法 | 游離輻射 (γ-ray) / UV | 醫療耗材 (塑膠培養皿、針筒) | 破壞 DNA | UV穿透力差，僅用於**表面**消毒 |
| **化學滅菌** | **環氧乙烷 (ETO)** | 烷基化使蛋白失活 (氣體) | **怕熱怕濕**物品 (精密儀器) | 穿透性強 | **具毒性**、致癌性、易燃，需長時間通氣 |
| | **戊二醛 (Glutaraldehyde)** | 烷基化使蛋白失活 (液體) | 手術器械、**內視鏡** | **對器械腐蝕性低** | 需長時間浸泡 (10小時) 達滅菌效果 |
| **生物安全等級 (BSL)** | **BSL-1** | 開放式操作台 | *Bacillus subtilis*, *E. coli* K-12 | 標準微生物操作 | 對健康成人**無致病性** |
| | **BSL-2** | 需 **BSC** (若產生氣溶膠) | *Staphylococcus aureus*, *Salmonella spp.* | 中等潛在危害 | 預防**經口、皮膚**感染 |
| | **BSL-3** | **所有操作皆在 BSC** 內 | ***M. tuberculosis***, ***B. anthracis***, ***F. tularensis*** | 呼吸道傳播，嚴重/致命疾病 | **負壓定向氣流**，管制進出 |
| | **BSL-4** | Class III BSC / 正壓防護衣 | Ebola virus, Marburg virus | **無**有效疫苗或治療方法 | 最高等級防護，獨立建築 |
| **生物安全操作櫃 (BSC)** | **Class I** | 負壓氣流，無菌空氣**不**吹向檢體 | 低、中度風險操作 | **保護：人員、環境** | **不保護檢體** |
| | **Class II** | **HEPA 過濾** + 垂直層流 | BSL-2, BSL-3 病原體 | **保護：人員、環境、檢體** | **最常用**類型 (A2, B2) |
| | **Class III (Glove box)** | **完全密閉**系統 | BSL-4 病原體 | **最高等級人員防護** | 操作不便 |
| **傳播途徑** | 飛沫傳播 | > 5 µm 微粒，短距離 | 流感病毒、腦膜炎雙球菌、百日咳 | 外科口罩 | - |
| | **空氣傳播** | **< 5 µm** 微粒 (飛沫核)，長距離懸浮 | **結核分枝桿菌**、麻疹病毒 | **N95 口罩** + **負壓隔離病房** | 易造成院內感染 |
| | **接觸傳播** | 直接/間接接觸 | ***Clostridium difficile***, MRSA | 隔離衣、手套、**手部衛生** | 易透過受汙染的環境表面傳播 |
