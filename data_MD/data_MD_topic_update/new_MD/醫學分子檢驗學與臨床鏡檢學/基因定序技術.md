## 核心趨勢與高頻考點
基因定序技術的命題核心，圍繞在**傳統桑格定序 (Sanger) 與次世代定序 (NGS) 的全方位比較**。考生必須清晰地掌握兩者在「原理、通量、讀長、成本、應用情境」上的根本差異。
出題趨勢已從單純的原理背誦，轉向**臨床應用情境的判斷**。例如：特定臨床問題（如：檢測腫瘤未知突變、評估腫瘤突變負荷 TMB、鑑定順式/反式 SNP）最適合使用何種技術？這背後的邏輯鏈是：**臨床需求 (高敏感度/廣度篩檢/長讀長) → 技術選擇 (NGS/TGS) → 解釋其關鍵優勢 (高通量/可偵測低頻突變/跨越重複序列)**。
NGS 的**實驗流程與品管指標**也是高頻考點。包含「文庫製備」中的 **index/barcode**（目的為 multiplexing 多工處理）與能大幅降低擴增錯誤率的 **UMI** (Unique Molecular Identifier)；以及數據分析中的 **Coverage (覆蓋率)** 與 **Depth (深度)** 的概念區分。此外，**定序圖譜判讀**是實務能力的體現，包含桑格定序的 Indel 重疊圖譜、以及焦磷酸定序的 Pyrogram 皆為經典必考題型。
## 未來考點預測與易混淆陷阱
隨著 NGS 普及與標準化，**臨床應用規範與表觀遺傳檢測**成為近年新興考點：
1. **品質規範指引：** 美國分子病理學會 (AMP) 對 NGS 的臨床應用指引強調定序深度、品質與分析流程，但**刻意「未限定」機台種類**，此為常考的觀念陷阱。
2. **【易混淆陷阱】平台原理細節對比：** 須釐清 Illumina 使用的**可逆性中斷子 (Reversible terminator)** 雖會阻斷合成並發布螢光，但它能透過「化學試劑切除」螢光及 3'-端阻斷基團以繼續定序；這與 Sanger 定序所使用之「不可逆」雙去氧核苷酸 (ddNTP) 截然不同。
3. **【易混淆陷阱】突變頻率 (VAF) 判讀：** 評估突變來源時，生殖細胞系突變 (Germline mutation) 通常分佈於 **50% (雜合)** 或 **100% (純合)**；而腫瘤的體細胞突變 (Somatic mutation) 常受限於腫瘤異質性而呈現**低頻率 (<15-20%)**。因此，Somatic 突變探索必須依賴高深度的 NGS，而 Germline 的單一位點突變則可考慮更具成本效益的 Sanger 定序。
4. **【潛在考點】DNA 甲基化定序：** 結合重亞硫酸鹽定序 (Bisulfite sequencing) 判讀。核心邏輯為：「**未甲基化**的 C 會被脫氨轉化為 U（定序結果顯示為 **T**）」；若定序結果維持為 C，才代表該位點具有甲基化修飾。

## Anki 聯想卡
```Anki
建構NGS文庫時，將DNA片段接上 index 或 bar code 的最重要目的為何？;<ans><b>多工處理 (Multiplexing)</b></ans><br><br>📌解釋：允許多個不同來源的樣本在同一次定序流程中混合進行，再藉由 index 序列將數據區分回原始樣本，大幅降低單一樣本的定序成本。
Sanger定序法在分析異質性高的腫瘤樣本時，最大的限制是什麼？;<ans><b>靈敏度不足</b></ans><br><br>📌解釋：Sanger定序對於突變等位基因頻率 (VAF) 低於 <b>15-20%</b> 的突變偵測能力很差，容易造成偽陰性結果。腫瘤樣本常含有低頻率的體細胞突變。
在NGS技術中，何種方式最能有效降低因PCR擴增造成的定序錯誤率？;<ans><b>使用獨特分子識別碼 (UMI)</b></ans><br><br>📌解釋：在文庫製備初期就為每個原始DNA分子標上獨特的UMI標籤。定序後，具有相同UMI的讀取 (reads) 應具有完全相同的序列，若出現差異，則可視為PCR或定序錯誤並予以校正，可精準定量並提高準確性。
次世代定序 (NGS) 與第三代定序 (TGS) 最根本的差異為何？;<ans><b>是否需要PCR增幅 (單分子定序)</b></ans><br><br>📌解釋：NGS (第二代) 需要透過PCR增幅訊號，而TGS可直接對<b>單一DNA/RNA分子</b>進行定序，因此讀長極長，且能保留原始的甲基化修飾資訊。
Sanger定序圖譜中，若發生單一核苷酸的插入(insertion)或刪除(deletion)突變，圖譜會呈現何種特徵？;<ans>在突變點後，出現<b>雙重訊號重疊且無法判讀</b>的混亂波形 [🖼️](https://www.google.com/search?q=Sanger+sequencing+chromatogram+insertion+deletion+frameshift&tbm=isch&udm=2)</ans><br><br>📌原因：發生移碼突變 (Frameshift)，導致突變點之後的所有序列讀取都錯位，產生兩組重疊的訊號。
下列何種情況最「不」適合使用次世代定序 (NGS)？;<ans><b>單一位點已知突變</b>的家族性遺傳病篩檢</ans><br><br>📌考量：這種情況下，目標明確、變異單純，使用 Sanger 定序或 Real-time PCR (ARMS-PCR) 更具成本效益且速度更快。NGS適用於未知突變的探索或多基因同時檢測。
焦磷酸定序 (Pyrosequencing) 的原理是偵測什麼物質的生成？;<ans><b>焦磷酸 (Pyrophosphate, PPi)</b></ans><br><br>📌連動機制：DNA聚合酶每加入一個正確的dNTP，就會釋放一個PPi。ATP硫酸化酶 (ATP sulfurylase) 利用PPi將APS轉化為ATP，接著螢光素酶 (Luciferase) 利用ATP催化螢光素 (Luciferin) 發光，進而判斷加入的鹼基。
評估腫瘤突變負荷 (Tumor Mutation Burden, TMB) 最適合的檢測方法為何？;<ans><b>全外顯子定序 (WES) 或大型基因套組 (Large Panel) NGS</b></ans><br><br>📌解釋：TMB定義為腫瘤基因組中每百萬個鹼基 (Mb) 的體細胞突變數量。這需要對廣泛的基因編碼區進行定序才能準確評估，故需使用NGS技術。
何種定序平台在文庫建構後，不需用到PCR進行大量增幅？;<ans><b>第三代定序平台 (如 PacBio 或 Oxford Nanopore)</b></ans><br><br>📌核心概念：第三代定序的特點是<b>單分子即時定序</b>，直接觀察單一分子的聚合反應或物理特性，因此無需先將模板進行PCR擴增。
NGS數據分析中的「深度(Depth)」與「覆蓋率(Coverage)」有何區別？;<ans><b>深度</b>：特定鹼基位置被定序讀取(reads)<b>重複測到的次數</b>。<br><b>覆蓋率</b>：目標區域中，至少被測到一次的鹼基<b>所佔的百分比</b>。</ans><br><br>📌比喻：深度像「照片的清晰度」，次數越多越可信；覆蓋率像「拼圖完成度」，百分比越高代表目標區域看得越完整。
已知兩個SNP位點相距僅5個核苷酸，要簡易分辨其為順式(cis)或反式(trans)排列，何種技術最適合？;<ans><b>桑格定序 (Sanger Sequencing) 或長讀長定序 (TGS)</b></ans><br><br>📌原理：只要設計的PCR產物能同時包含這兩個SNP位點 (讀長足夠長)，單次定序就能確定它們是否在同一條DNA分子上。短讀長的NGS會將DNA打斷，難以直接判斷連鎖關係。
AMP (美國分子病理學會) 公布的次世代定序臨床應用守則，主要規範定序深度、品質及分析流程，但「未限定」什麼條件？;<ans><b>機台種類</b></ans><br><br>📌解釋：各實驗室可根據臨床需求選擇不同的定序平台，只要通過確效並符合品質與深度的規範要求即可。
可逆性中斷子定序法 (如 Illumina) 每次加入單一鹼基後，必須進行什麼步驟才能繼續下一個鹼基的定序？;<ans><b>加入化學試劑切除螢光訊號與阻斷基團</b></ans><br><br>📌陷阱：選項常以「加入酸性溶液」或「化學試劑」描述此切除步驟。請注意它使用的是「可逆」中斷子，不同於 Sanger 所使用「不可逆」的雙去氧核苷酸(ddNTP)。
在重亞硫酸鹽定序 (Bisulfite sequencing) 中，若定序結果顯示原本的 Cytosine (C) 變成了 Thymine (T)，代表該 CpG 位點處於何種狀態？;<ans><b>未甲基化 (Unmethylated)</b></ans><br><br>📌連動：Bisulfite 會將「未甲基化」的 C 脫氨轉成 U，後續 PCR/定序讀取為 T；若 C 發生甲基化，則受到保護不被轉換，定序結果仍為 C。
基因變異檢測中，若某變異位點的等位基因突變頻率 (VAF) 接近 <b>50%</b> 或 <b>100%</b>，最可能屬於何種突變？;<ans><b>種系突變 (Germline mutation)</b></ans><br><br>📌對比：體細胞突變 (Somatic mutation) 僅存在於部分腫瘤細胞，其 VAF 常為低頻 (如 <10-20%)，故腫瘤體細胞基因檢測高度依賴 NGS 的高深度定序。
```
