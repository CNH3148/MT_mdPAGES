## 類群定義
> 測驗基於核酸探針（Probe）與目標序列（Target）專一性結合的各種技術，包括南方墨點法（Southern Blotting）、北方墨點法（Northern Blotting）、螢光原位雜交（FISH）及各種微陣列/生物晶片（Microarray/Biochip）技術，如比較基因組雜交（aCGH）等。
## 核心趨勢與高頻考點
歷屆考題高度集中在**微陣列/生物晶片 (Microarray/Biochip)**，特別是**微陣列比較基因組雜交 (aCGH)** 的原理、應用與極限。考生必須極度清晰地認知到，aCGH 是一種「計數」的概念，它擅長偵測基因片段的**數量增減 (Copy Number Variation, CNV)**，但對於總數量不變的**平衡性轉位 (Balanced translocation)** 或單點突變 (SNP) 則無能為力。這是最關鍵、也是最反覆出現的考點。
針對大量單點突變，若要檢測「大量**已知** SNP」，則最適合使用**寡核苷酸微陣列 (Oligonucleotide microarray)**。這裡常將「DNA 晶片」與「次世代基因定序 (NGS)」拿來比較：**DNA 晶片**的探針設計依賴於**已知**序列，只能抓出預設的變異；而 **NGS** 則具有發現**未知**基因變異的強大能力。另外，進行 DNA 晶片分析時，若檢體 DNA 量充足，理論上「不一定需要」進行 PCR 擴增。
其次，**螢光原位雜交 (FISH)** 著重於其「原位 (in situ)」特性，能在細胞形態下直接觀察特定染色體或基因的變異（如特定轉位、擴增或缺失）。考題特別強調 FISH 的標的物主要是細胞內的 **DNA 與 RNA**，**絕對無法**用於偵測蛋白質（Protein）。
最後，關於雜交反應的**基本物理化學條件——嚴謹度 (Stringency)** 更是必考基礎：**鹽類濃度越高**，會中和核酸骨架負電荷，導致**嚴謹度下降**、容許鹼基配對錯誤（mismatch），進而使**偵測特異性下降**。
總結趨勢：從單一目標的墨點法，演進到細胞層級的 FISH，再到全基因組掃描的 aCGH。考題的邏輯鏈是：**臨床情境 → 預期基因變異類型 (CNV? 轉位? 已知/未知SNP?) → 選擇最適合的雜交技術 → 理解該技術的偵測極限與流程**（例如反向雜交的 RT-PCR 操作順序）。

## Anki 聯想卡
```Anki
aCGH 主要偵測何種基因變異？;<ans>拷貝數變異 (Copy Number Variation, CNV)</ans><br><br>📌白話文：偵測基因片段的「多一份」或「少一份」。應用於偵測微小缺失/重複症候群 (microdeletion/duplication syndromes)。
aCGH「無法」偵測的兩大類基因變異？;<ans><b>平衡性重組</b> (Balanced rearrangements) 與 <b>單點突變/變異</b> (SNP/SNV)</ans><br><br>⚠️陷阱：因為 aCGH 的原理是比較 DNA 的「總量」，平衡性轉位 (balanced translocation) 或倒置 (inversion) 的 DNA 總量不變，故無法檢出。
偵測特定染色體「轉位」，如費城染色體 t(9;22)，最適合的原位雜交技術？;<ans>螢光原位雜交 (FISH)</ans><br><br>📌連動：FISH 使用不同顏色的探針標記 t(9;22) 的 ABL 與 BCR 基因，若發生轉位，兩個螢光訊號會融合成一個（Fusion signal），可直接在細胞中觀察到。
南方墨點法 (Southern) 與北方墨點法 (Northern) 偵測的目標分子分別為何？;<ans>Southern → <b>DNA</b><br>Northern → <b>RNA</b></ans><br><br>🧠記憶法：Northern blot 用於分析基因「表現 (expression)」，而基因表現的第一步就是轉錄成 RNA。
在雜交反應中，何種條件會造成「低嚴謹度 (Low stringency)」，容許探針與目標序列有部分不匹配 (mismatch)？;<ans><b>高鹽濃度</b>、<b>低溫</b></ans><br><br>📌原理：高濃度的陽離子 (如 Na+) 會中和 DNA 骨架上的負電，減少磷酸根之間的排斥力，使不完全配對的雙股結構也能穩定存在。
在雜交反應中，何種條件會造成「高嚴謹度 (High stringency)」，要求探針與目標序列完美配對？;<ans><b>低鹽濃度</b>、<b>高溫</b></ans><br><br>📌應用：在進行 SNP 晶片分析時，需利用高嚴謹度條件，才能區分僅差一個鹼基的探針結合差異。
DNA 晶片製程中，「光引導原位合成法 (light-directed synthesis)」的探針特性？;<ans>探針較<b>短</b> (寡核苷酸)、密度<b>高</b>、序列設計固定</ans><br><br>📌代表廠商：Affymetrix。其技術核心為光蝕刻法 (Photolithography)。
DNA 晶片製程中，「點樣法 (spotting)」的探針特性？;<ans>探針較<b>長</b> (cDNA/PCR產物)、密度<b>低</b>、設計彈性高</ans><br><br>📌優點：研究者可依需求自行合成探針再點上晶片，客製化程度高。
在 aCGH 的結果圖中，Log2 ratio > 0 代表什麼？;<ans>基因片段<b>增益 (Gain)</b></ans><br><br>📌解釋：代表「病患檢體」的螢光訊號強度，高於「正常對照組」的螢光訊號強度，表示該段 DNA 在病患身上數量較多。
在 aCGH 的結果圖中，Log2 ratio < 0 代表什麼？;<ans>基因片段<b>缺失 (Loss)</b></ans><br><br>📌解釋：代表「病患檢體」的螢光訊號強度，低於「正常對照組」的螢光訊號強度，表示該段 DNA 在病患身上數量較少。
檢測「大量已知 SNP」最適合使用何種生物晶片？;<ans>寡核苷酸微陣列 (oligonucleotide microarray)</ans><br><br>📌連動：寡核苷酸微陣列可透過光引導合成法製造高密度短探針，非常適合大規模的 SNP 分型。
DNA 晶片與次世代基因定序 (NGS) 在偵測基因變異最大的限制差異為何？;<ans>DNA 晶片只能測<b>已知</b>序列；NGS 可鑑定<b>未知</b>變異</ans><br><br>📌原理：晶片必須預先設計並固定對應的探針，所以無法偵測設計外的突變位點。
原位螢光雜交法 (FISH) 最主要可偵測細胞內的哪些分子？;<ans><b>DNA</b> 與 <b>RNA</b> (臨床最常用於染色體 DNA)</ans><br><br>⚠️陷阱：絕對無法偵測蛋白質 (Protein)，蛋白質的定位需使用免疫螢光染色 (IFA) 或 Western blot。
反向雜交法 (Reverse hybridization) 針對 RNA 檢體的標準操作順序為何？;<ans>檢體製備 → <b>反轉錄 (RT)</b> → <b>PCR 擴增</b> → <b>探針雜交</b> → 呈色反應 → 判讀</ans><br><br>📌原理：反向雜交是將「探針」固定在固態載體上，讓擴增過後的「檢體」游離去尋找互補探針結合。
在核酸雜交反應中，鹽類濃度「越高」，對偵測特異性 (Specificity) 的影響為何？;<ans>特異性<b>越低</b></ans><br><br>📌連動：高鹽濃度 = 低嚴謹度 (Low stringency) = 容許配對錯誤 (Mismatch) = 導致特異性下降。
```
