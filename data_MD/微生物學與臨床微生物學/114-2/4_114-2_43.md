## 題目

下列何種細菌會產生腸毒素,通常會造成人類壞死性腸炎 type C型的食物中毒?

A.Clostridioides difficile
B.Clostridium botulinum
C.Clostridium perfringens
D.Clostridium tetani

> [!success]- 正確解答
> **C**

---

## 筆記與詳解

### (1) 快速破題

看到題目關鍵字「**壞死性腸炎 (necrotizing enteritis) type C**」與「**食物中毒**」，在梭菌屬 (Clostridium spp.) 的考題中，直接鎖定 **產氣莢膜梭菌 (*Clostridium perfringens*)**。其他常見選項中，肉毒桿菌與破傷風桿菌主要引發神經毒性症狀（麻痺、痙攣），而困難梭菌的關鍵字則是「抗生素使用」與「偽膜性腸炎」，皆與 type C 壞死性腸炎無關。

---

### (2) 詳解預備：關鍵字與知識點拆解

* **壞死性腸炎 (Enteritis necroticans / Pigbel)** → 專指由 *C. perfringens* Type C 分泌的 **β-toxin (β毒素)** 所引起的嚴重腸道出血壞死。常發生於蛋白質攝取不足且食用含有胰蛋白酶抑制劑（如甘薯）的地區，導致毒素無法被腸道酵素分解而致病。
* **腸毒素 (Enterotoxin)** → 由 *C. perfringens* (多為 Type A) 孢子形成時釋放，破壞腸道細胞膜，引發一般常見的水瀉型食物中毒。
* **Type C 型** → *C. perfringens* 根據其產生的主要致命毒素（α, β, ε, ι）分為 A~E 五型。Type C 產生 α 及 β 毒素。
* ***Clostridioides difficile*** → 關鍵字：廣效抗生素使用後、Toxin A (腸毒素)/Toxin B (細胞毒素)、偽膜性腸炎。
* ***Clostridium botulinum*** → 關鍵字：罐頭食品、嬰兒吃蜂蜜、抑制乙醯膽鹼 (ACh) 釋放、鬆弛性麻痺。
* ***Clostridium tetani*** → 關鍵字：深部傷口感染、抑制 GABA/Glycine 釋放、痙攣性麻痺。

---

### (3) 詳解與選項辨析

* **(A) *Clostridioides difficile* (困難梭菌)：錯誤。**
* **辨析：** 主要引起抗生素相關性腹瀉 (Antibiotic-associated diarrhea)。當正常腸道菌叢被破壞後，該菌大量繁殖並產生毒素，導致結腸黏膜發炎壞死，內視鏡下可見黃白色的偽膜附著於腸壁 (pseudomembranous colitis)[🖼️](https://www.google.com/search?q=%22pseudomembranous+colitis%22+colonoscopy+clinical+sign&tbm=isch&udm=2)。不會引起 Type C 壞死性腸炎。


* **(B) *Clostridium botulinum* (肉毒桿菌)：錯誤。**
* **辨析：** 引起肉毒桿菌中毒，主要透過攝入預先形成的毒素（食源性）或嬰兒攝入孢子。毒素作用於周邊神經突觸，阻斷神經傳導物質釋放，臨床表徵為眼瞼下垂、吞嚥困難及嬰兒鬆弛性麻痺 (floppy baby syndrome)[🖼️](https://www.google.com/search?q=%22floppy+baby+syndrome%22+botulism+clinical+sign&tbm=isch&udm=2)。無腸毒素造成的壞死性腸炎。


* **(C) *Clostridium perfringens* (產氣莢膜梭菌)：正確答案。**
* **辨析：** 該菌不僅以 α-toxin 引起氣性壞疽 (Gas gangrene)，其產生的腸毒素 (Enterotoxin) 會導致一般食物中毒；而 **Type C 菌株** 產生的 **β-toxin** 則是引發人類壞死性腸炎 (Enteritis necroticans, 又稱 Pigbel) 的元兇。鏡檢特徵為無鞭毛、呈現火車廂狀的革蘭氏陽性桿菌 (boxcar-shaped Gram-positive bacilli)[🖼️](https://www.google.com/search?q=%22boxcar-shaped%22+Gram+stain+Clostridium+perfringens&tbm=isch&udm=2)，且在血瓊脂培養基上會呈現獨特的雙層溶血環 (double zone of hemolysis)[🖼️](https://www.google.com/search?q=%22double+zone+of+hemolysis%22+Clostridium+perfringens+blood+agar&tbm=isch&udm=2)。


* **(D) *Clostridium tetani* (破傷風桿菌)：錯誤。**
* **辨析：** 透過傷口感染，孢子發芽後產生破傷風痙攣毒素 (Tetanospasmin)。鏡檢特徵為孢子位於菌體末端，呈鼓槌狀或網球拍狀 (drumstick-like terminal spores)[🖼️](https://www.google.com/search?q=%22drumstick+spores%22+Gram+stain+Clostridium+tetani&tbm=isch&udm=2)。臨床表徵為肌肉強直收縮，如角弓反張 (opisthotonos)[🖼️](https://www.google.com/search?q=opisthotonos+tetanus+clinical+sign&tbm=isch&udm=2) 及牙關緊閉，無腸道感染症狀。



---

### (4) 核心邏輯因果與記憶口訣

* **疾病生理機轉軸線：**
* **壞死性腸炎 (Pigbel)：** 攝入 Type C *C. perfringens* 及高量胰蛋白酶抑制劑 (如甘薯) → β-toxin 無法被腸道酵素分解 → 毒素在腸道內蓄積並破壞腸壁細胞 → 腸黏膜缺血、壞死出血 → 臨床診斷：壞死性腸炎。
* **破傷風/肉毒神經機轉：**
* *C. tetani* 毒素 → 逆行性中樞神經傳導 → **阻斷抑制性物質 (GABA/Glycine)** → 運動神經元持續放電 → **痙攣性麻痺 (↑)**。
* *C. botulinum* 毒素 → 周邊神經突觸前 → **阻斷興奮性物質 (ACh)** → 肌肉無法收縮 → **鬆弛性麻痺 (↓)**。




* 💡 **醫檢/醫師國考記憶口訣：**
**「破痙攣，肉鬆弛；產氣雙溶壞死腸，困難偽膜抗生素」**
（破傷風=痙攣；肉毒=鬆弛；產氣莢膜=雙層溶血/壞死性腸炎；困難梭菌=偽膜性腸炎/抗生素引起）
