## 題目

下列那些為有性孢子?①arthrospore ②sporangiospore ③ascospore ④basidiospore

A.①②
B.③④
C.①③
D.②④

> [!success]- 正確解答
> **B**

---

## 筆記與詳解

### (1) 快速破題

看到題目要求區分「真菌的有性與無性孢子」，這屬於微生物學的純記憶基本題。解題時直接聯想「有性孢子」的三大天王：**Ascospore (子囊孢子)**、**Basidiospore (擔孢子)**、**Zygospore (接合孢子)**。只要記住這三者，其他帶有 `-spore` 字尾的絕大多數皆為無性孢子。題目出現的 ③ascospore 與 ④basidiospore 屬於有性，①② 則為無性，故秒選 B。

---

### (2) 詳解預備

* **Arthrospore (關節孢子)** → 關鍵字：「菌絲斷裂」。屬於**無性孢子**。
* **Sporangiospore (孢子囊孢子)** → 關鍵字：「包覆於囊內」。屬於**無性孢子**。
* **Ascospore (子囊孢子)** → 關鍵字：「子囊、減數分裂」。屬於**有性孢子**。
* **Basidiospore (擔孢子)** → 關鍵字：「擔子、蕈菇類」。屬於**有性孢子**。

---

### (3) 詳解

真菌的繁殖依賴孢子，臨床鑑定與分類上高度依賴孢子的顯微形態與生成方式：

* **① arthrospore (關節孢子)：錯誤（為無性孢子）**
由原本的營養菌絲直接從中隔 (septum) 處斷裂，形成連續排列的矩形或方形細胞[🖼️](https://www.google.com/search?q=arthrospore+microscopy&tbm=isch&udm=2)。此過程未經過減數分裂或基因重組，屬於無性繁殖。代表病原體如：粗球黴菌 (Coccidioides immitis)。
* **② sporangiospore (孢子囊孢子)：錯誤（為無性孢子）**
產生於菌絲頂端一個封閉的薄壁孢子囊 (sporangium) 內[🖼️](https://www.google.com/search?q=sporangiospore+microscopy&tbm=isch&udm=2)。細胞核經過有絲分裂產生，當孢子囊破裂時釋放，屬於無性繁殖。常見於接合菌門（如毛黴菌 Mucor、根黴菌 Rhizopus）。
* **③ ascospore (子囊孢子)：正確（為有性孢子）**
真菌的兩個細胞核融合並進行減數分裂後，在稱為「子囊」(ascus) 的囊狀構造內形成的孢子[🖼️](https://www.google.com/search?q=ascospore+microscopy&tbm=isch&udm=2)（通常一個子囊內含 8 個孢子）。這是典型的有性繁殖產物。
* **④ basidiospore (擔孢子)：正確（為有性孢子）**
基因重組後，在棍棒狀的「擔子」(basidium) 頂端向外突起著生的孢子[🖼️](https://www.google.com/search?q=basidiospore+microscopy&tbm=isch&udm=2)（通常一個擔子著生 4 個孢子）。臨床上最著名的代表為新生隱球菌 (Cryptococcus neoformans) 的有性世代。

---

### (4) 核心軸線與橫向比較

**【核心邏輯軸線】**
環境適宜/養分充足 → **無性繁殖**啟動 → 進行有絲分裂 → 產生大量無性孢子 (Conidia/Sporangiospore) → 快速擴張真菌聚落。
環境惡劣/養分耗盡 → **有性繁殖**啟動 → 菌絲融合/減數分裂 → 產生有性孢子 (Ascospore/Basidiospore/Zygospore) → 基因重組 ↑ → 提升環境適應力與抗逆境能力。

| 繁殖分類 | 孢子名稱 (中/英) | 鏡檢特徵 / 生成方式 | 臨床代表性真菌 |
| --- | --- | --- | --- |
| **有性孢子**<br>(Sexual) | **Ascospore**<br>(子囊孢子) | 囊狀構造 (ascus) 內含數個孢子[🖼️](https://www.google.com/search?q=ascospore+microscopy&tbm=isch&udm=2) | 大多數酵母菌、黴菌 (子囊菌綱) |
|  | **Basidiospore**<br>(擔孢子) | 棍棒狀擔子 (basidium) 頂端著生[🖼️](https://www.google.com/search?q=basidiospore+microscopy&tbm=isch&udm=2) | 隱球菌 (*Cryptococcus*)、蕈菇類 |
|  | **Zygospore**<br>(接合孢子) | 兩菌絲融合形成**巨大厚壁**孢子[🖼️](https://www.google.com/search?q=zygospore+microscopy&tbm=isch&udm=2) | 毛黴菌 (*Mucor*)、根黴菌 |
| **無性孢子**<br>(Asexual) | **Conidiospore**<br>(分生孢子) | 暴露於分生孢子柄頂端，呈**鏈狀或簇狀**[🖼️](https://www.google.com/search?q=conidiospore+microscopy&tbm=isch&udm=2) | 麴菌 (*Aspergillus*)、青黴菌 |
|  | **Sporangiospore**<br>(孢子囊孢子) | 包覆於密閉的**孢子囊**內[🖼️](https://www.google.com/search?q=sporangiospore+microscopy&tbm=isch&udm=2) | 毛黴菌 (*Mucor*) |
|  | **Arthrospore**<br>(關節孢子) | 菌絲中隔斷裂，呈**矩形排列**[🖼️](https://www.google.com/search?q=arthrospore+microscopy&tbm=isch&udm=2) | 粗球黴菌 (*Coccidioides*) |
|  | **Blastospore**<br>(芽生孢子) | 母細胞以**出芽 (budding)** 方式生成[🖼️](https://www.google.com/search?q=blastospore+microscopy&tbm=isch&udm=2) | 念珠菌 (*Candida*) |
|  | **Chlamydospore**<br>(厚膜孢子) | 菌絲內形成之**厚壁抗逆境**休眠細胞[🖼️](https://www.google.com/search?q=chlamydospore+microscopy&tbm=isch&udm=2) | 白色念珠菌 (*C. albicans*) |

> 💡 **必殺記憶口訣：【有接子擔】** (諧音：有結帳擔保)
> **有**性孢子 = **接**合 (Zygospore) + **子**囊 (Ascospore) + **擔** (Basidiospore)。
> *(只要選項不在這三個裡面，在醫檢/護理國考中直接歸類為無性孢子)*
