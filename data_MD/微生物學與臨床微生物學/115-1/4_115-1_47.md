## 題目

下列何者試驗與黴漿菌最無關?

A.PPLO 培養基生長
B.hemadsorption test
C.tetrazolium test
D.porphyrin test

> [!success]- 正確解答
> **D**

---

## 筆記與詳解

### (1) 快速破題

看到題目問「黴漿菌 (*Mycoplasma*)」且要尋找「最無關」的試驗，最快的方法是掃描選項，找出「其他常見細菌的專屬鑑定試驗」。選項 D 的 porphyrin test 是臨床微生物學中，用來鑑定「嗜血桿菌 (*Haemophilus*)」是否需要 X factor 的經典專屬試驗，與黴漿菌無任何交集，因此秒選 D。

### (2) 詳解預備

* **黴漿菌 (*Mycoplasma*)** → 缺乏細胞壁（對 $\beta$-lactam 類抗生素天然抗藥性），細胞膜含固醇 (sterol)，生長極度緩慢。
* **PPLO 培養基** → Pleuropneumonia-like organism（類肋膜肺炎微生物），為黴漿菌舊稱。此專屬培養基需添加馬血清（提供膽固醇）與酵母萃取物，菌落典型特徵為「荷包蛋型」中心緻密邊緣扁平的微小菌落 [🖼️](https://www.google.com/search?q=%22fried+egg%22+colony+Mycoplasma&tbm=isch&udm=2)。
* **Hemadsorption test (紅血球吸附試驗)** → 肺炎黴漿菌 (*M. pneumoniae*) 具表面黏附素 (P1 adhesin)，能特異性吸附天竺鼠或人類的紅血球至其菌落周圍，形成紅血球覆蓋的玫瑰花環狀視覺特徵 [🖼️](https://www.google.com/search?q=hemadsorption+test+Mycoplasma+pneumoniae+guinea+pig+RBC&tbm=isch&udm=2)。
* **Tetrazolium test (四唑試驗)** → 肺炎黴漿菌能將無色的 tetrazolium (TTC) 還原為紅色的 formazan 沉澱 [🖼️](https://www.google.com/search?q=tetrazolium+test+positive+red+formazan+bacteria&tbm=isch&udm=2)。
* **Porphyrin test (紫質試驗)** → 測定細菌是否能利用 $\delta$-aminolevulinic acid ($\delta$-ALA) 自行合成紫質。陽性結果會在紫外光下呈現紅色螢光 [🖼️](https://www.google.com/search?q=porphyrin+test+fluorescence+Haemophilus+UV&tbm=isch&udm=2)，代表該菌不需依賴外加的 X factor (hemin)。

### (3) 詳解

* **A. PPLO 培養基生長 (正確/相關)**：PPLO broth/agar 是培養黴漿菌的標準配方，因其細胞膜成分特殊，培養基中必須富含 cholesterol 以穩定其細胞膜。
* **B. hemadsorption test (正確/相關)**：此為鑑定 *Mycoplasma pneumoniae* 的重要致病特徵試驗。其表面的 P1 蛋白質除了在感染時結合人類呼吸道上皮，也能在實驗室體外試驗中吸附紅血球。
* **C. tetrazolium test (正確/相關)**：此試驗專門用於區分「肺炎黴漿菌 (*M. pneumoniae*)」與「其他人體共生黴漿菌」。肺炎黴漿菌具備在有氧環境下還原 TTC 的能力，產生的紅色變化可作為鑑別依據。
* **D. porphyrin test (錯誤/最無關)**：此為鑑定 **嗜血桿菌屬 (*Haemophilus*)** 的專屬生化試驗。例如流感嗜血桿菌 (*H. influenzae*) 缺乏相關酵素，無法自行合成 X factor，故 porphyrin test 為陰性；而副流感嗜血桿菌 (*H. parainfluenzae*) 能自行合成，故為陽性。與黴漿菌毫無關聯。

### (4) 核心軸線與縱向聯想

* ***Mycoplasma pneumoniae* 檢驗邏輯軸線：** 無細胞壁 → 膜需外源固醇穩定 → **PPLO 培養基**培養 → 生長出 Fried egg colonies [🖼️](https://www.google.com/search?q=%22fried+egg%22+colony+Mycoplasma&tbm=isch&udm=2) → 致病需結合宿主細胞 → 具 P1 adhesin → 體外表現為 **Hemadsorption (+)** → 具特殊氧化還原酵素 → **Tetrazolium test (+)**。
* ***Haemophilus* 檢驗邏輯軸線 (選項 D 延伸)：** 確認需氧條件 → 測定 X factor (Hemin) 需求 → 使用 **Porphyrin test** → 若細菌能將 $\delta$-ALA 轉為紫質 → 紫外燈下紅螢光 [🖼️](https://www.google.com/search?q=porphyrin+test+fluorescence+Haemophilus+UV&tbm=isch&udm=2) → 代表自給自足，**不需**外加 X factor。

> **💡 記憶口訣：** > 黴漿菌專屬套餐：「**P** (PPLO) **M** (Mycoplasma) 下午**吸** (Hemadsorption) **紅** (Tetrazolium) 蛋」。
