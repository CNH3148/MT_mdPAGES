## 題目

有關利用心臟超音波評估心臟功能,下列何者錯誤?

A.利用彩色杜卜勒(color Doppler)測量左心室的血流量
B.利用 M mode 測量左心室直徑改變
C.利用 2D 測量左心室面積改變
D.利用3D測量左心室體積改變

> [!success]- 正確解答
> **A**

---

## 筆記與詳解

### (1) 解題思路

看到題目考「超音波各模式的測量目標」，請直接聯想幾何維度的推進：**1D測線段（直徑）、2D測平面（面積）、3D測立體（體積）**。當遇到「杜卜勒 (Doppler)」時，必須立刻反應它的核心功能是偵測血球的移動。**彩色杜卜勒 (Color Doppler)** 只能呈現血流的「方向」與「相對流速」，絕對不能直接測量出實體的「血流量 (Volume)」。因此，看到「彩色杜卜勒測量血流量」即可判斷為錯誤選項。

---

### (2) 關鍵字拆解與知識點

* **彩色杜卜勒 (Color Doppler)** [🖼️](https://www.google.com/search?q=%22Color+Doppler%22+echocardiography&tbm=isch&udm=2) → **知識點：** 將血流方向與流速以顏色編碼疊加在 2D 影像上（紅色朝向探頭、藍色遠離探頭），用於定性觀察異常血流（如瓣膜逆流、分流），**不具備直接計算總體積流量的功能**。
* **M mode (Motion mode)** [🖼️](https://www.google.com/search?q=%22M-mode%22+echocardiography&tbm=isch&udm=2) → **知識點：** 1D 空間加上時間軸的紀錄方式。時間解析度極高，專門用來測量特定軸線上結構的**直徑或厚度**變化（例如左心室收縮/舒張末期內徑）。
* **2D 模式** [🖼️](https://www.google.com/search?q=%222D+echocardiography%22&tbm=isch&udm=2) → **知識點：** 掃描出心臟的二維切面，可描繪心室內膜邊界來測量**面積**。
* **3D 模式** [🖼️](https://www.google.com/search?q=%223D+echocardiography%22&tbm=isch&udm=2) → **知識點：** 立體空間重建，不需依賴幾何假設，直接測量真實**體積**，對心室形狀變異的病人特別準確。

---

### (3) 詳解辨析

* **(A) 利用彩色杜卜勒(color Doppler)測量左心室的血流量 → 錯誤（本題答案）**。彩色杜卜勒僅能顯示血流的「方向」與「均速」，是視覺化的定性/半定量工具。若要精準計算「血流量（心搏出量，Stroke Volume）」，必須使用**脈衝波杜卜勒 (PW Doppler)** 測出左心室出口道 (LVOT) 的血流速度時間積分 (VTI)，再搭配 2D 超音波測量的 LVOT 截面積，利用公式 $SV = VTI \times Area$ 才能算出血流量。
* **(B) 利用 M mode 測量左心室直徑改變 → 正確**。M-mode 穿透單一射線，記錄結構隨時間的位移，是測量心室腔室**直徑 (Diameter)** 及心室壁厚度最傳統且精確的方法。
* **(C) 利用 2D 測量左心室面積改變 → 正確**。在 2D 切面上圈出心室腔的輪廓，即可計算出收縮期與舒張期的**面積 (Area)**，臨床常藉此代入 Simpson's rule 估算心室容積。
* **(D) 利用 3D 測量左心室體積改變 → 正確**。3D 超音波提供空間資訊，跳脫傳統 2D 面推算體積的誤差，能最直接且精準地評估左心室的**體積 (Volume)**。

---

### (4) 核心邏輯與記憶口訣

* **縱向聯想：** 物理維度升級 → 超音波測量指標升級。1維看線（M mode）→ 2維看面（2D）→ 3維看體（3D）。超音波若加上都卜勒效應（Doppler），處理的就不是靜態幾何，而是動態流體（速度/方向）。
* **記憶口訣：** **「M線、2面、3體積，彩度看『向』不看『量』」**
* **M** mode = **線** (直徑)
* **2**D = **面** (面積)
* **3**D = **體積**
* **彩度** (Color Doppler) = 看血流方**向**，不測血流**量**。

### (5) 心臟超音波模式鑑別矩陣

| 超音波模式 | 測量維度/物理量 | 核心臨床應用 | 視覺特徵/測量目標 |
| --- | --- | --- | --- |
| **M mode** | 1D + 時間 | 測量心室肥厚、收縮分率 (FS) | 單一軸線的隨時間波動圖 [🖼️](https://www.google.com/search?q=%22M-mode%22+echocardiography&tbm=isch&udm=2) / **直徑 (Diameter)** |
| **2D mode** | 2D 平面 | 評估整體結構、射血分數 (LVEF) | 灰階扇形橫截面 [🖼️](https://www.google.com/search?q=%222D+echocardiography%22&tbm=isch&udm=2) / **面積 (Area)** |
| **3D mode** | 3D 立體 | 複雜瓣膜疾病、不規則心室容積評估 | 立體心臟動態模型 [🖼️](https://www.google.com/search?q=%223D+echocardiography%22&tbm=isch&udm=2) / **體積 (Volume)** |
| **Color Doppler** | 血流方向/均速 | 偵測**瓣膜逆流、心內分流 (VSD/ASD)** | **紅迎藍離 (BART)** 色彩疊加 [🖼️](https://www.google.com/search?q=%22Color+Doppler%22+echocardiography&tbm=isch&udm=2) / 定位異常血流 |
| **Spectral Doppler (PW/CW)** | 絕對血流速度 | 測量**狹窄壓力差、計算心搏出量 (SV)** | 頻譜波形圖 [🖼️](https://www.google.com/search?q=%22Spectral+Doppler%22+echocardiography&tbm=isch&udm=2) / **流速積分 (VTI)** |
