## 題目

下列何者與子宮內膜之漿液性癌(serous carcinoma of endometrium)最有關?

A.PTEN基因突變
B.TP53基因突變
C.暴露於大量雌激素
D.合併子宮內膜增生

> [!success]- 正確解答
> **B**

---

## 筆記與詳解

### (1) 快速破題

看到題目考「子宮內膜癌」，應立即聯想到其兩大分類：**Type I (內膜樣癌)** 與 **Type II (漿液性癌)**。題目問的是「漿液性癌 (serous carcinoma)」，此型態的特點為**非**雌激素依賴、發生於年長女性的萎縮內膜上，且惡性度極高。選項中的 A (PTEN突變)、C (雌激素暴露)、D (內膜增生) 全部都是 **Type I (內膜樣癌)** 的典型特徵。只有 B (*TP53*基因突變) 是 Type II (漿液性癌) 最核心的標誌性突變。因此答案秒選 B。

### (2) 關鍵字與知識點拆解

* **子宮內膜之漿液性癌 (serous carcinoma of endometrium) →** 子宮內膜癌 Type II。特色：侵襲性高、與雌激素無關、背景常為萎縮的子宮內膜。
* **PTEN 基因突變 →** 子宮內膜癌 Type I (內膜樣癌) 最常見的早期基因突變。
* **TP53 基因突變 →** 子宮內膜癌 Type II (漿液性癌) 的核心標誌物，發生率 >90%。
* **暴露於大量雌激素 →** 子宮內膜癌 Type I 的主要致病驅動因子（來源如肥胖、未曾生育、停經晚或單獨使用雌激素）。
* **合併子宮內膜增生 →** 子宮內膜非典型增生是 Type I (內膜樣癌) 的經典癌前病變。

### (3) 詳解

* **正確選項 B (TP53基因突變)：** 幾乎所有的子宮內膜漿液性癌皆由 *TP53* 抑癌基因突變所驅動。這個突變發生在非常早期，甚至在癌前病變的階段——即「漿液性子宮內膜上皮內癌 (SEIC)」[🖼️](https://www.google.com/search?q=%22serous+endometrial+intraepithelial+carcinoma%22+histopathology&tbm=isch&udm=2) 即可發現 p53 蛋白的異常過度表現 [🖼️](https://www.google.com/search?q=p53+immunohistochemistry+serous+carcinoma&tbm=isch&udm=2)。
* **錯誤選項 A (PTEN基因突變)：** *PTEN* 基因的去活化突變會導致 PI3K/AKT 傳遞路徑持續活化，這是 Type I 內膜樣癌 (endometrioid carcinoma) 最常見的基因變異，與 Type II 漿液性癌無關。
* **錯誤選項 C (暴露於大量雌激素)：** 長期受不受拮抗的雌激素 (unopposed estrogen) 刺激，是 Type I 內膜樣癌的主因。Type II 漿液性癌則好發於較高齡、體型較瘦的女性，其生長環境通常是萎縮的子宮內膜 (endometrial atrophy) [🖼️](https://www.google.com/search?q=endometrial+atrophy+histopathology&tbm=isch&udm=2)，與雌激素無關。
* **錯誤選項 D (合併子宮內膜增生)：** 子宮內膜增生 (尤其非典型增生) [🖼️](https://www.google.com/search?q=atypical+endometrial+hyperplasia+histopathology&tbm=isch&udm=2) 是 Type I 癌的癌前病變。而 Type II 癌的癌前病變並非增生，而是伴隨內膜萎縮的 SEIC。

### (4) 核心軸線與邏輯因果

* **Type I 發病軸線：** 雌激素過度暴露 → 子宮內膜過度刺激 → 子宮內膜非典型增生 [🖼️](https://www.google.com/search?q=atypical+endometrial+hyperplasia+histopathology&tbm=isch&udm=2) → 累積 ***PTEN*** 等突變 → 發展為 **Type I 內膜樣癌 (Endometrioid carcinoma)** [🖼️](https://www.google.com/search?q=endometrioid+carcinoma+histopathology&tbm=isch&udm=2) (預後較佳)。
* **Type II 發病軸線：** 老化/萎縮的子宮內膜 [🖼️](https://www.google.com/search?q=endometrial+atrophy+histopathology&tbm=isch&udm=2) → 早期發生 ***TP53*** 突變 → 形成漿液性上皮內癌 (SEIC) [🖼️](https://www.google.com/search?q=%22serous+endometrial+intraepithelial+carcinoma%22+histopathology&tbm=isch&udm=2) → 發展為 **Type II 漿液性癌 (Serous carcinoma)** [🖼️](https://www.google.com/search?q=endometrial+serous+carcinoma+histopathology&tbm=isch&udm=2) (常具乳頭狀結構與砂粒體 [🖼️](https://www.google.com/search?q=psammoma+bodies+serous+carcinoma&tbm=isch&udm=2)，預後極差)。

> **💡 記憶口訣：**
> * **Type 1 (一)**定有**雌激素** (E) → 發生於**增生**內膜 → ***PTEN*** 突變 → 預後較好。
> * **Type 2 (惡)**性度高 → **老**人/內膜萎縮 → ***TP53*** 突變 → **漿**液性 (Serous)。
>

### (5) 表格重點統整

| 比較維度 | Type I 內膜樣癌 (Endometrioid) | Type II 漿液性癌 (Serous) |
| --- | --- | --- |
| **雌激素依賴性** | **高** (肥胖、未生育、HRT) | **無** |
| **臨床背景** | 停經前後 (較年輕) | 高齡 (停經後晚期) |
| **癌前病變** | **子宮內膜非典型增生** | **漿液性上皮內癌 (SEIC)** / 內膜萎縮 |
| **核心基因變異** | ***PTEN***、*PIK3CA*、*KRAS*、MSI | ***TP53*** (>90%) |
| **鏡檢特徵** | 腺體呈背靠背排列 [🖼️](https://www.google.com/search?q=endometrioid+carcinoma+histopathology&tbm=isch&udm=2) | **乳頭狀結構**、高度細胞異型性、**砂粒體** [🖼️](https://www.google.com/search?q=endometrial+serous+carcinoma+papillary+psammoma+body&tbm=isch&udm=2) |
| **腫瘤行為/預後** | 侵襲性↓、預後**佳** | 侵襲性↑↑、易腹腔擴散、預後**極差** |
