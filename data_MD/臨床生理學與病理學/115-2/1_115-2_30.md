## 題目

肺功能檢查結果顯示吸氣儲備容積（IRV）1.5 L、潮氣容積（TV）0.5 L、呼氣儲備容積（ERV）1.1 L、肺餘容積（RV）0.6 L，功能餘氣量（FRC） 1.7 L，此時總肺量（TLC）約為多少 L？

A.3.1
B.3.7
C.2
D.4.8

> [!success]- 正確解答
> **B**

---

## 筆記與詳解

### (1) 快速破題

看到題目要求計算「總肺量（TLC）」，直接套用最根本的定義：肺部能容納的所有氣體體積，即**四個基本容積相加**。
公式：`TLC = IRV + TV + ERV + RV`
代入題目數值：1.5 + 0.5 + 1.1 + 0.6 = **3.7 L**，直接選出答案 B。題目給的功能餘氣量（FRC = 1.7 L）是作為驗算或干擾條件（FRC = ERV + RV = 1.1 + 0.6 = 1.7 L，完全吻合）。

### (2) 詳解預備

* **IRV (吸氣儲備容積)** → 平靜吸氣後，能再「用力吸入」的最大氣量。
* **TV (潮氣容積)** → 平靜狀態下，每次呼吸的氣量。
* **ERV (呼氣儲備容積)** → 平靜呼氣後，能再「用力呼出」的最大氣量。
* **RV (肺餘容積)** → 拼命呼氣到極限後，肺內「永遠吐不出來」的死區殘留氣量。
* **FRC (功能餘氣量)** → 平靜呼氣末期，肺內殘留的氣量 (`ERV + RV`)。
* **Capacity (容量)** vs **Volume (容積)** → 任何「容量」都必定是由 2 個或以上的「容積」組合而成。

### (3) 詳解

總肺量 (TLC) 的核心概念是「肺部最大容積」，其計算有兩種常見途徑，本題兩者皆可證實答案為 3.7 L：

1. **基礎加總法**：TLC = IRV + TV + ERV + RV = 1.5 + 0.5 + 1.1 + 0.6 = **3.7 L**
2. **容量代入法**：TLC = 吸氣容量 (IC) + 功能餘氣量 (FRC)。
* 先算 IC = IRV + TV = 1.5 + 0.5 = 2.0 L
* TLC = 2.0 + 1.7 (題目給的 FRC) = **3.7 L**



**選項剖析與陷阱：**

* **選項 A (3.1 L) 錯誤**：此為**肺活量（VC）**。算法：IRV (1.5) + TV (0.5) + ERV (1.1) = 3.1 L。這是人體能主動控制的最大呼出氣體量。
* **選項 B (3.7 L) 正確**：為總肺量（TLC）。
* **選項 C (2.0 L) 錯誤**：此為**吸氣容量（IC）**。算法：IRV (1.5) + TV (0.5) = 2.0 L。
* **選項 D (4.8 L) 錯誤**：此為盲目加總陷阱。若將算出的 VC (3.1) 加上題目給的 FRC (1.7) = 4.8 L。此錯誤在於 **ERV 被重複計算了兩次**（VC 包含 ERV，FRC 也包含 ERV）。

### (4) 核心軸線與邏輯因果

呼吸肌作功 → 改變胸腔體積 → 產生各階段之「肺容積 (Volume)」 → 兩者以上容積組合成「肺容量 (Capacity)」 → 應用於臨床 Spirometry 判別**阻塞性**或**限制性**通氣障礙。

* **記憶口訣**：
* **「四積合一為總量」**：IRV + TV + ERV + RV = TLC。
* **「含殘(RV)不可直測」**：只要公式內包含 RV 的項目（RV、FRC、TLC），都**無法**使用一般肺量計 (Spirometer) 直接吹氣測得，必須改用氦氣稀釋法 (Helium dilution) 或體箱測量法 (Body plethysmography)。
