## 題目

在一常規12導程心電圖中,右束支傳導阻斷(right bundle branch block, RBBB)最不可能呈現下列何者?

A.V1呈現rSR’波形
B.V5呈現qR波形
C.QRS寬度為0.12秒
D.PR區間為0.08秒

![[臨床生理學與病理學_exam6_q7_9.jpeg]]

> [!success]- 正確解答
> **D**

---

## 筆記與詳解

### (1) 快速破題

看到題目問「右束支傳導阻斷 (RBBB)」並要找「最不可能」的選項，請直接檢查心電圖的基本區間標準值。
心電圖中 **PR 區間的正常值為 0.12～0.20 秒**。選項 D 描述 PR 區間為 0.08 秒，這屬於異常短的 PR 區間（常見於 WPW 症候群等預激症候群），這與束支傳導阻斷完全無關，因為束支阻斷發生在心室，不會影響心房到房室結的傳導時間。因此可以直接判定 D 是最不可能出現的現象。

### (2) 詳解預備

* **右束支傳導阻斷 (RBBB)** → 傳導系統右側分支阻斷，導致右心室去極化延遲。左心室先收縮，右心室後收縮。
* **QRS 寬度** → 正常為 0.06～0.10 秒。若數值 **≥ 0.12 秒**，代表心室內傳導出現嚴重延遲，為「完全性」束支阻斷的診斷要件。
* **PR 區間** → 代表心房去極化至心室開始去極化的時間。正常值為 **0.12～0.20 秒**。
* **rSR' 波形** → 俗稱兔耳波[🖼️](https://www.google.com/search?q=%22rSR%27+pattern%22+ECG+V1&tbm=isch&udm=2)，是 V1 導程在 RBBB 的標準視覺特徵，代表右心室延遲去極化產生的向右前向量。

### (3) 詳解

* **選項 A (V1 呈現 rSR' 波形)**：合理。RBBB 時，心室去極化順序為：中隔正常由左向右去極化 (產生小 r 波) → 左心室正常去極化 (產生 S 波) → 右心室延遲去極化，產生強烈朝向右胸 V1 導程的向量 (產生高聳 R' 波)。因此 V1 會出現典型的 rSR' (兔耳) 波形[🖼️](https://www.google.com/search?q=%22rabbit+ears%22+ECG+V1+RBBB&tbm=isch&udm=2)。
* **選項 B (V5 呈現 qR 波形)**：合理。V5 位於左胸，在 RBBB 中，左心室的早期去極化不受影響，因此會正常產生反映中隔去極化的小 q 波與左心室去極化的 R 波（即初期呈現 qR 波形）。雖然其後經常會伴隨因右心室延遲去極化所造成的寬大遲緩 S 波[🖼️](https://www.google.com/search?q=slurred+S+wave+V5+V6+ECG&tbm=isch&udm=2) (整體為 qRs)，但前半段呈現 qR 屬於正常的病理生理表現。
* **選項 C (QRS 寬度為 0.12 秒)**：合理。完全性右束支傳導阻斷 (Complete RBBB) 的核心診斷標準就是 QRS 波變寬，大於等於 0.12 秒 (≥ 3 小格) [🖼️](https://www.google.com/search?q=wide+QRS+complex+ECG+strip&tbm=isch&udm=2)。
* **選項 D (PR 區間為 0.08 秒)**：**最不可能 (本題解答)**。RBBB 的病灶位於希氏束 (Bundle of His) 的右分支，僅影響心室內的傳導，**完全不會**影響心房到房室結 (SA node → AV node) 的傳導速度。因此 RBBB 患者的 PR 區間應維持在正常的 0.12～0.20 秒。0.08 秒屬於病理性的 PR 區間過短，代表存在副傳導路徑（如 WPW syndrome）。

### (4) 核心軸線與邏輯因果

* **核心病理邏輯**：右束支受阻 → 右心室無法獲得原本的快速傳導 → 改由左心室的肌肉細胞慢慢傳遞電流過來 → **去極化時間拉長 (QRS ≥ 0.12 秒)** 且 **最後的電氣向量朝向右側 (右側導程 V1 出現向上的 R' 波，左側導程 V5/V6 出現向下的寬 S 波)** → 但上方房室結傳導完全正常，故 **PR 區間不變**。
* **心電圖記憶口訣 (MaRroW vs WiLLiaM)**：
* **MaRroW (RBBB)**：在 V1 呈 **M** 型 (rSR' 波)[🖼️](https://www.google.com/search?q=%22rSR%27+pattern%22+ECG+V1&tbm=isch&udm=2)，在 V6 呈 **W** 型 (寬大的 S 波)。
* **WiLLiaM (LBBB)**：在 V1 呈 **W** 型 (寬大的 QS 或 rS 波)[🖼️](https://www.google.com/search?q=LBBB+V1+QS+pattern+ECG&tbm=isch&udm=2)，在 V6 呈 **M** 型 (寬大有缺口 notched 的 R 波)[🖼️](https://www.google.com/search?q=LBBB+V6+notched+R+wave+ECG&tbm=isch&udm=2)。
