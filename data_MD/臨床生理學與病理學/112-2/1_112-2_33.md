## 題目

有關心臟超音波臨床應用,下列何者錯誤?

A.可以偵測心包膜積水
B.M mode超音波常用以測量心臟收縮分率
C.利用2D來偵測瓣膜逆流
D.利用Doppler來評估瓣膜狹窄程度

> [!success]- 正確解答
> **C**

---

## 筆記與詳解

### (1) 快速破題

看到題目問「2D」和「瓣膜逆流」，並且要找「錯誤」的 → 就要想到超音波不同模式的物理限制。2D 超音波只能看「結構剖面」（肉）無法看「血流」（血）。**逆流屬於血流動力學異常**，必須要有「都卜勒 (Doppler)」功能才能偵測。因此可以直接判斷 C 選項張冠李戴，答案為 C。

### (2) 詳解預備

* **2D 超音波 (2D Echo)** → **知識點**：即 B-mode (Brightness)，顯示心臟解剖結構切面。可看心室大小、心肌厚度、心包膜積水與瓣膜開闔形態，但不含血流資訊。
* **M mode 超音波 (Motion mode)** → **知識點**：一維超音波加上時間軸。具極高時間解析度，取單一直線觀測心臟壁與瓣膜隨時間的運動軌跡。常用於測量左心室舒張末期與收縮末期內徑，代入公式計算**心臟收縮分率 (EF)**。
* **瓣膜逆流 (Valvular Regurgitation)** → **知識點**：血液反向流動。需利用**彩色都卜勒 (Color Doppler)** 標示血流方向（迎向探頭為紅、遠離為藍）來偵測與定量。
* **瓣膜狹窄 (Valvular Stenosis)** → **知識點**：瓣膜開口變小導致通過之血流加速。需利用**頻譜都卜勒 (Spectral Doppler，包含 CW/PW)** 測量最大血流速度 ($v$)，並代入簡化柏努力方程式 ($\Delta P = 4v^2$) 計算壓力梯度。

### (3) 詳解

* **(A) 可以偵測心包膜積水**：正確。心包膜積水在 2D 超音波下會呈現心臟外圍一圈無回音區（黑色空間）[🖼️](https://www.google.com/search?q=pericardial+effusion+echocardiography&tbm=isch&udm=2)，這是目前臨床診斷與評估心包膜積水最快速且標準的工具。
* **(B) M mode超音波常用以測量心臟收縮分率**：正確。M mode 影像呈現波浪狀的動態軌跡 [🖼️](https://www.google.com/search?q=M-mode+echocardiography+ejection+fraction&tbm=isch&udm=2)，臨床上常測量左心室內徑，利用 Teichholz formula 等數學公式估算左心室容積變化，進而得出收縮分率 (LVEF)。
* **(C) 利用2D來偵測瓣膜逆流**：**錯誤**。2D 只能看到瓣膜本身「有沒有關緊/脫垂」等結構特徵，**無法看見實體血流**。要明確偵測並觀看「逆流」的範圍及方向，必須切換成**彩色都卜勒 (Color Doppler)**，觀察心房/心室端出現的馬賽克或逆向彩色血流 [🖼️](https://www.google.com/search?q=color+Doppler+valvular+regurgitation&tbm=isch&udm=2)。
* **(D) 利用Doppler來評估瓣膜狹窄程度**：正確。瓣膜狹窄會產生局部高速血流。利用連續波或脈衝波都卜勒 (CW/PW) 測量狹窄處血流頻譜圖 [🖼️](https://www.google.com/search?q=spectral+Doppler+valvular+stenosis&tbm=isch&udm=2) 取得最大流速，藉由流速換算壓力差 (Pressure gradient)，是評估狹窄嚴重程度（如主動脈瓣狹窄 AS）的核心指標。

### (4) 核心軸線與邏輯因果

**超音波訊號處理邏輯**：探頭發射超音波 → 接收組織反射訊號 → 依照處理模式轉化為不同臨床應用：

* **單一波束 + 時間變化** → M mode → **精準測量內徑/心室壁運動** (EF值)
* **多波束掃描 + 灰階亮度** → 2D mode → **解剖結構異常** (積水、腫瘤、瓣膜形態)
* **都卜勒效應 + 色彩疊加** → Color Doppler → **血流方向與範圍** (逆流缺損)
* **都卜勒效應 + 頻譜分析** → Spectral Doppler → **精確血流速度** (狹窄壓力差)

| --- | --- | --- | --- |
| **M mode** (運動模式) | 一維線條 + 時間軸 (冰河波浪狀) | **測量心室內徑**、計算 **EF / FS**、觀測瓣膜高頻震動 | 時間位移曲線 [🖼️](https://www.google.com/search?q=M-mode+echocardiography&tbm=isch&udm=2) |
