## 核心趨勢與高頻考點

心臟傳導障礙是心電圖學的必考題型，鑑別診斷是得分的唯一途徑。歷屆考題顯示，命題趨勢高度集中，幾乎每年都會出現。

1.  **房室傳導阻斷 (AV Block) 是絕對核心**：
    *   **第一度 vs. 第二度 vs. 第三度** 的基本定義必須滾瓜爛熟。特別是第一度 AVB 的臨界值 **PR > 0.20 秒（心電圖紙上 1 大格）**。
    *   **第二度 AV Block 中 Mobitz I (Wenckebach) 與 Mobitz II 的鑑別** 是高頻中的高頻。考題常以「文字描述心電圖變化」（如：PR漸長、P波與QRS波數量關係）或直接給圖的方式呈現。
    *   解題的邏輯鏈為：**觀察 P 波與 QRS 波的關係 → 判定 PR 間期的變化（固定 vs. 漸長） → 做出精確診斷**。

2.  **束支傳導阻斷 (Bundle Branch Block, BBB) 的圖像辨識**：
    *   重點在於 **QRS 波寬度 (≥0.12 秒)** 與 **特定導程的形態學**。
    *   RBBB 的 V1 導程 **rSR' (兔子耳朵)** 與 LBBB 的 V6 導程 **單相寬R波 (M型)** 是圖像判讀的黃金法則。
    *   **重要觀念**：BBB 是心室內的傳導問題，因此**不會影響 PR 間期**。考題會測驗此觀念，例如「RBBB 不可能出現 PR 區間為 0.08 秒」這類敘述。

3.  **鑑別診斷 QRS 波寬度**：
    *   這已成為新的命題焦點。考生必須清楚區分哪些情況會導致 QRS 波變寬 (≥0.12s)，哪些不會。
    *   **會變寬**：**束支傳導阻斷 (RBBB/LBBB)**、**心室早期收縮 (VPC)**、**WPW 症候群**、心室節律。
    *   **不會變寬 (<0.12s)**：**分支傳導阻斷 (Fascicular Block / Hemiblock)**，如左前分支阻斷 (LAHB)。它主要造成**心電軸偏移**，而非 QRS 明顯增寬。

4.  **臨床意義與後續處置**：
    *   除了診斷，考題已開始延伸至臨床預後。考生必須知道哪些阻斷是相對良性的（如：第一度、Mobitz I），哪些是惡性的、可能需要置放心律調節器的（如：**Mobitz II、第三度**）。
    *   生理機轉與阻斷部位的連結也是一個潛在考點：**AV 結**的阻斷（如 Mobitz I）通常 QRS 波較窄、預後較好；**His 束-Purkinje 系統**的阻斷（如 Mobitz II）則 QRS 波較寬、預後差。

**預測未來考法**：除了傳統的定義與圖像題，未來將更強調 **情境整合與鑑別診斷**。例如，給定一段心電圖，要求進行 **複合式判讀**（如 RBBB 合併一度 AVB）。此外，**2:1 AV Block** 因無法單純從 PR 間期判斷為 Mobitz I 或 II，其鑑別診斷的模糊地帶，以及「下列何者 *不會* 造成 QRS 變寬？」這類反向思考的題目，也成為極具潛力的命題點。
