## 類群定義

> 專注於肺部、呼吸道與胸膜的疾病。包括阻塞性肺病（COPD、氣喘）、侷限性肺病（肺纖維化）、肺部感染（大葉性/支氣管性肺炎）、肺癌的各種類型（腺癌、鱗狀細胞癌、小細胞癌）及其危險因子，以及胸膜疾病（如間皮瘤與石綿暴露）。

## 核心趨勢與高頻考點
呼吸系統病理學的命題焦點高度集中，具備清晰的因果邏輯與鑑別診斷脈絡。掌握以下趨勢，便能抓住絕大多數分數：

1.  **肺癌分類是絕對核心**：歷屆考題反覆圍繞四大肺癌（腺癌、鱗狀、小細胞、大細胞）的**鑑別**。考法極度模式化，主要圍繞「**好發位置**」、「**危險因子**」與「**臨床特徵**」。
    *   **邏輯鏈 A (中央型)**：**吸菸** → 基因突變 → **中央/肺門**氣管上皮病變 → **鱗狀細胞癌** (產生 PTHrP 導致高血鈣) 或 **小細胞癌** (神經內分泌特性，產生 ADH/ACTH)。小細胞癌惡性度高，早期轉移，故**治療以化療/放療為主**，極少手術。
    *   **邏輯鏈 B (周邊型)**：**非吸菸/女性** → **EGFR** 等基因突變 → **肺部周邊**腺體細胞病變 → **肺腺癌**。
    *   未來趨勢將更深入**分子病理**，例如除了已考出的 **EGFR**，與特定癌症相關的 **ALK**, **ROS1** (腺癌)、**TP53** (幾乎所有) 突變，以及惡性間皮瘤的 **CDKN2A 基因缺失 (deletion)** 也可能成為考點。

2.  **感染性疾病重在病原體與病理分期**：肺炎的考點單純直接，鎖定「**最常見致病菌**」與「**病理特徵**」。
    *   **邏輯鏈 A (大葉性肺炎)**：社區民眾 → 感染 → **社區型肺炎 (CAP)** → 最常見病原體：**肺炎鏈球菌 (S. pneumoniae)**。其典型病理過程有**四個分期**，順序必考：**充血期 → 紅色肝變期 → 灰色肝變期 → 消解期**。
    *   **邏輯鏈 B (間質性肺炎)**：病毒/黴漿菌感染 → 炎症反應侷限於**肺泡壁/間質** (而非肺泡腔內) → 造成侷限性肺病。SARS 和 COVID-19 即為此類代表。

3.  **特定暴露史與疾病的強連結**：這是「送分題」，看到關鍵字就要能立即反應。
    *   **邏輯鏈 A**：**石綿 (Asbestos)** 纖維暴露 → 刺激**胸膜/腹膜間皮細胞** → 惡性病變 → **惡性間皮瘤 (Mesothelioma)**。
    *   **邏輯鏈 B**：**EB 病毒 (EBV)** 感染 → 潛伏於鼻咽部淋巴上皮 → 促使細胞癌變 → **鼻咽癌 (NPC)**。

4.  **疾病的定義與核心病理特徵**：對於 ARDS、慢性支氣管炎等疾病，考試著重於其**最根本的病理變化**或**臨床定義**。
    *   **ARDS**：多重病因 → 肺泡微血管內皮 & 肺泡上皮細胞**瀰漫性損傷 (DAD)** → 蛋白質滲出液在**肺泡腔內**形成**透明膜 (Hyaline membrane)**。考點在於透明膜的**位置**，是在肺泡內，而非微血管壁上。
    *   **慢性支氣管炎**：定義為臨床症狀（長期咳痰），而非特定肺功能數值。
    *   **支氣管擴張症**：其重要病因之一的**囊性纖維化 (Cystic Fibrosis)** 是一種**遺傳性疾病**，並非後天感染造成。
