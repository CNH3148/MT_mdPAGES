## 核心趨勢與高頻考點
本單元的核心在於理解**循環系統的失衡**。命題趨勢圍繞著三大因果鏈，考生必須能將其串聯：

1.  **壓力失衡 → 液體外滲：** 考試重點在於區分**靜水壓 (Hydrostatic pressure) ↑** 與**膠體滲透壓 (Oncotic pressure) ↓** 如何導致水腫。臨床情境題（如心衰竭、肝硬化、腎病症候群）會與這些基本機轉結合，是高頻考點。
    - **邏輯鏈：** 心衰竭 → 靜脈回流受阻 → 靜水壓 ↑ → 全身性水腫。
    - **邏輯鏈：** 肝硬化/腎病 → 白蛋白合成↓/流失↑ → 血漿膠體滲透壓 ↓ → 全身性水腫。

2.  **血流異常/凝血啟動 → 血栓形成與後果：** **Virchow's triad (魏修氏三要素)** 是此部分的聖經，必須熟記。考題會將其應用在動、靜脈血栓的比較上，並延伸至後續的**栓塞 (Embolism)** 與**梗塞 (Infarction)**。
    - **邏輯鏈：** Virchow's triad (e.g., 靜脈鬱滯) → **深層靜脈血栓 (DVT)** → 脫落形成栓子 → **肺栓塞 (Pulmonary Embolism)**。
    - **邏輯鏈：** 心房顫動 → 左心房血流紊亂 → **壁血栓 (Mural thrombus)** → 脫落形成栓子 → **系統性栓塞 (Systemic Embolism)** → 腦/腎/脾梗塞。

3.  **灌流不足 → 全身性休克：** 這是最高頻的考點。核心在於**比較不同類型休克的根本原因與代償反應**。特別是**敗血性休克**，其獨特的「低血管阻力」是絕對的鑑別重點。
    - **邏輯鏈：** 病因 (心肌梗塞/大出血/嚴重感染) → 觸發特定休克機轉 (幫浦衰竭/容積不足/血管麻痺) → **心輸出量 (CO) 與 全身血管阻力 (SVR) 的特異性變化** → 全身組織灌流不足 → 多重器官衰竭。

未來的考法將更傾向於**整合性題組**，例如給定一個心肌梗塞病人的情境，選項可能同時包含：(A) 心因性休克的機轉 (B) 左心室壁血栓的風險 (C) 肺鬱血造成的水腫表現 (D) 系統性栓塞的可能性。考生需具備跨子章節的整合能力。

## 關鍵字反射表

| 狀態 / 疾病 | 致病機轉 / 核心缺陷 | 病理/臨床特徵 | 血行動力學變化 (休克) |
| :--- | :--- | :--- | :--- |
| **水腫 (Edema)** | 靜水壓 ↑ / **膠體滲透壓 ↓** / 淋巴管阻塞 | 組織間液 ↑、**指壓凹陷 (pitting)** | - |
| **充血 (Hyperemia)** | **主動過程**、小動脈擴張 (發炎、運動) | 組織**鮮紅**、氧合血 ↑ | - |
| **鬱血 (Congestion)** | **被動過程**、靜脈回流 ↓ (心衰竭) | 組織**暗紅/發紺 (cyanotic)**、缺氧血 ↑、**荳蔻肝 (nutmeg liver)** | - |
| **動脈血栓** | **內皮損傷**、血流紊亂 (湍流) | 好發於冠狀動脈/腦動脈、**富含血小板**、灰白色、可見** Zahn's lines (詹氏線)** | - |
| **靜脈血栓 (DVT)** | **血流鬱滯 (stasis)** | 好發於下肢深層靜脈、**富含紅血球/纖維蛋白**、紅色、常造成**肺栓塞** | - |
| **肺栓塞 (PE)** | 栓子來源：**95% 為下肢 DVT** | 呼吸困難、胸痛、猝死 | 右心衰竭 |
| **系統性栓塞** | 栓子來源：**80% 為心臟壁血栓** | 好發於下肢、腦、內臟 | 造成遠端器官梗塞 |
| **白色梗塞 (White)** | **實質器官**、單一血管供應 (心、脾、腎) | 組織蒼白、**凝固性壞死** | - |
| **紅色梗塞 (Red)** | **疏鬆組織** (肺)、**雙重血源**、靜脈阻塞 | 組織暗紅、出血、**液化性壞死 (腦)** | - |
| **心因性休克** | **心臟幫浦衰竭** (e.g., MI) | 皮膚濕冷、發紺 | CO ↓、PCWP/CVP **↑↑**、SVR **↑ (代償)** |
| **低血容性休克** | **血液/體液容積 ↓** (e.g., 出血、燒傷) | 皮膚濕冷、脈搏細速 | CO ↓、PCWP/CVP **↓↓**、SVR **↑ (代償)** |
| **敗血性休克** | 全身性感染 → **血管劇烈擴張**、內皮損傷 | **早期皮膚溫暖 (warm shock)** | CO **↑ (早期)**、PCWP/CVP ↓、SVR **↓↓ (主要缺陷)** |
| **神經性休克** | 交感神經張力喪失 (e.g., 脊髓損傷) | - | CO ↓、PCWP/CVP ↓、SVR **↓↓** |
| **過敏性休克** | IgE 介導、肥大細胞釋放組織胺 | 全身性水腫、支氣管痙攣 | CO ↓、PCWP/CVP ↓、SVR **↓↓** |
