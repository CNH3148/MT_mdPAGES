## 題目

滲出液(Exudates)常見於下列何種疾病?

A.充血性心臟衰竭
B.肝硬化
C.胰臟炎
D.腎病症候群

> [!success]- 正確解答
> **C**

---

## 筆記與詳解

### 【快速破題】

看到題目探討體液積聚（胸水/腹水），請立刻將疾病分為兩大陣營：

1. **全身性系統疾病（心、肝、腎）** → 造成物理性壓力失衡（靜脈水壓高、血漿蛋白低）→ 產生**漏出液 (Transudate)**。
2. **局部性病理變化（發炎、感染、腫瘤）** → 造成微血管被破壞、通透性增加 → 產生**滲出液 (Exudate)**。

題目問「滲出液」，選項中 A (心)、B (肝)、D (腎) 皆為全身性壓力失衡的漏出液。只有 C (胰臟炎) 屬於強烈的局部發炎反應，因此答案秒選 C。

---

### 【詳解預備】關鍵字與知識點拆解

* **滲出液 (Exudates)：** **微血管通透性↑** → 血液中的大分子蛋白質與發炎細胞大量漏出 → 常見於局部**發炎、感染、惡性腫瘤**。
* **漏出液 (Transudates)：** **微血管靜水壓↑** 或 **血漿膠體滲透壓↓** → 單純水分濾出，蛋白質含量極低 → 常見於**心衰竭、肝硬化、腎病症候群**。
* **充血性心臟衰竭 (CHF)：** 靜脈回流受阻 → 微血管靜水壓 **↑**。
* **肝硬化 (Cirrhosis)：** 肝臟合成白蛋白能力受損 → 血漿膠體滲透壓 **↓**。
* **胰臟炎 (Pancreatitis)：** 消化酵素外漏引發周邊組織強烈發炎 → 微血管通透性 **↑**。
* **腎病症候群 (Nephrotic Syndrome)：** 腎絲球受損導致大量白蛋白從尿液流失 → 血漿膠體滲透壓 **↓**。

---

### 【詳解】選項辨析

臨床上常使用 **Light's criteria (萊特準則)** 來精準區分兩者。只要體液滿足以下**任一**條件，即為「滲出液」，若皆不滿足則為「漏出液」：

1. 體液蛋白質 / 血清蛋白質 > **0.5**
2. 體液 LDH / 血清 LDH > **0.6**
3. 體液 LDH > 血清 LDH 正常上限值的 **2/3**

* **(A) 充血性心臟衰竭 (錯誤)：** 右心室或左心室無法有效泵血，導致系統靜脈或肺靜脈壓力上升。高靜水壓將水分「擠出」血管外，因血管壁完好，蛋白質無法通過，形成**漏出液**。
* **(B) 肝硬化 (錯誤)：** 肝臟功能衰退，白蛋白合成大幅減少，導致血管內膠體滲透壓下降。水分無法被保存在血管內而流失至腹腔（腹水）或肋膜腔，形成**漏出液**。
* **(C) 胰臟炎 (正確)：** 胰臟酵素（如澱粉酶、脂肪酶）不正常活化並外漏，引發強烈的局部發炎反應。發炎介質（如組織胺、白介素）使微血管通透性大增，富含蛋白質與白血球的液體滲漏至周邊，形成**滲出液**。
* **(D) 腎病症候群 (錯誤)：** 腎絲球基底膜受損，大量蛋白質從尿液流失造成嚴重的低白蛋白血症，導致血漿膠體滲透壓驟降，水分外漏形成**漏出液**。

---

### 【縱向聯想】邏輯因果與記憶口訣

* **縱向病理機轉：**
* **系統壓力失衡** (心衰竭、肝硬化、腎病) → 靜水壓↑ 或 滲透壓↓ → 血管壁完整，僅水分濾出 → **漏出液**
* **局部組織破壞** (肺炎、胰臟炎、肺結核、肺癌) → 發炎介質釋放 → 微血管間隙撐大 → 大分子蛋白質/細胞漏出 → **滲出液**


* 💡 **國考記憶口訣：**
* **漏出液** 找 **「心、肝、腎」** (台語諧音：新肝腎)，這三個器官壞掉都是系統性水腫。
* **滲出液** 找 **「炎、感、癌」** (發炎、感染、長腫瘤)。

### 【橫向比較】漏出液 (Transudate) vs. 滲出液 (Exudate) 核心鑑別重點

| 比較維度 | 漏出液 (Transudate) | 滲出液 (Exudate) |
| --- | --- | --- |
| **致病機轉** | 靜水壓 **↑** 或 膠體滲透壓 **↓** | 微血管通透性 **↑** 或 淋巴回流受阻 |
| **常見疾病** | **心衰竭、肝硬化、腎病症候群** | **發炎(如胰臟炎)、感染、惡性腫瘤** |
| **肉眼外觀** | 清澈、淡黃色且透明 [🖼️](https://www.google.com/search?q=pleural+transudate+gross+appearance+clear&tbm=isch&udm=2) | 混濁、膿性或血性 [🖼️](https://www.google.com/search?q=pleural+exudate+gross+appearance+cloudy&tbm=isch&udm=2) |
| **凝固性** | **不會凝固** | **易有自發性凝塊** (含纖維蛋白原) [🖼️](https://www.google.com/search?q=pleural+fluid+spontaneous+clotting+fibrin&tbm=isch&udm=2) |
| **比重 (Specific Gravity)** | **< 1.015** | **> 1.015** |
| **總蛋白質 (TP)** | **< 3.0** g/dL | **> 3.0** g/dL |
| **體液/血清 TP 比值** | **< 0.5** | **> 0.5** |
| **體液/血清 LDH 比值** | **< 0.6** | **> 0.6** |
| **白血球數量 (WBC)** | **< 1000** /µL (多為單核球) | **> 1000** /µL (急性期多為**嗜中性白血球**) |
| **Rivalta Test (黏蛋白試驗)** | **陰性 (-)** | **陽性 (+)** |
