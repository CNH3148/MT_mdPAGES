## 題目

以次世代定序進行遺傳基因檢測,利用已知變異的參考物質,計算可偵測範圍中的每種變異的準確度,整理檢測結果如圖所示。下列計算何者正確?

A.陽性一致率:A/(A+C+E)
B.陽性預測值:A/(A+C)
C.偽陰性率:1-[A/(A+C)]
D.偽陽性率:1-[D/(B+D+F)]

![image](https://img.yamol.tw/file/687ee478c72b7.jpg)

> [!success]- 正確解答
> **C**

## 筆記與詳解

**(1) 快速破題**
看到題目探討「次世代定序 (NGS) 準確度計算」並出現 A、C、E 等矩陣代號，必須直覺聯想到標準的**檢驗與參考標準比較矩陣 (Contingency Table)**。在評估檢驗效能時，**絕對必須排除「無法判定 (No Call / Indeterminate)」的數據 (即選項推測出的 E、F)** 才能正確計算一致率或預測值。選項 A、B、D 的公式分母均放錯位置或包含了不該加的 E、F，只有選項 C 的算法能完美推導出標準的偽陰性率公式 `C / (A+C)`。

**(2) 詳解預備 (關鍵字 → 知識點)**

* **已知變異的參考物質** → 作為「黃金標準 (Gold Standard)」，定義檢體真實的「陽性 (有變異)」與「陰性 (無變異)」。
* **3×2 矩陣代號 (A~F)** → 標準 NGS 確效矩陣配置：
* **縱軸 (行)**：參考物質狀態 → 陽性 (A, C, E) / 陰性 (B, D, F)
* **橫軸 (列)**：NGS 檢測結果 → 陽性 (A, B) / 陰性 (C, D) / 無法判定 (E, F)
* **A**：真陽性 (TP)；**B**：偽陽性 (FP)
* **C**：偽陰性 (FN)；**D**：真陰性 (TN)


* **一致率 (Agreement)** → 用於取代「敏感度/特異度」(因參考物質可能非絕對完美的真理，但在國考中可視為同義詞)。

**(3) 詳解**

* **(A) 錯誤**：陽性一致率 (PPA) 旨在評估「成功檢測」的情況下抓出真實陽性的能力。計算時應**排除**檢測失敗的 E。正確公式為 `A / (A+C)`。
* **(B) 錯誤**：陽性預測值 (PPV) 指「所有檢測為陽性的結果中，真正是陽性的比例」。分母應為所有「測出陽性」的總數 (A+B)。正確公式為 `A / (A+B)`。選項寫的 `A / (A+C)` 實際上是 PPA。
* **(C) 正確**：偽陰性率 (FNR) 定義為「真實陽性卻測成陰性」的機率，即 `C / (A+C)`。根據數學推導：`1 - [A / (A+C)] = (A+C)/(A+C) - A/(A+C) = C / (A+C)`。公式完全吻合偽陰性率定義。
* **(D) 錯誤**：偽陽性率 (FPR) 定義為「真實陰性卻測成陽性」的機率。與 PPA 相同，計算必須排除無法判定的 F。正確公式為 `B / (B+D)`，而非 `1 - [D/(B+D+F)]`。

**(4) 縱向聯想與核心邏輯**

* **邏輯軸線**：儀器確效 → 排除 Invalid/No call (E, F) → 計算 PPA/NPA 評估**分析效能** → 導入群體盛行率評估 PPV/NPV (**臨床效能**) → 決定該 NGS Panel 是否可發出臨床報告。
* **記憶口訣**：
* **「一 (一致率) 直往『下』看」**：PPA/NPA 的分母是同一**行**的 `A+C` 或 `B+D` (參考物質總數，縱向加總)。
* **「預 (預測值) 備向『右』走」**：PPV/NPV 的分母是同一**列**的 `A+B` 或 `C+D` (檢測結果總數，橫向加總)。
* **絕對不加 E、F**：算準確度指標，一律踢除 No call。



**(5) NGS 檢驗效能與干擾物質橫向比較矩陣**
(對比 NGS 與傳統分子檢測黃金標準 Sanger 定序之優劣)

| 檢驗效能指標 | 矩陣公式 | 核心臨床意義 | NGS 偽陰/偽陽成因與干擾 | Sanger 定序 (黃金標準) 特徵對比 |
| --- | --- | --- | --- | --- |
| **陽性一致率 (PPA) / 敏感度** | **A / (A+C)** | 抓出真實變異的能力 | **↓下降**：定序深度 (Depth) 不足、**GC-rich** 擴增失敗 | 整體靈敏度較低 (需變異豐度 **>15-20%**) |
| **陰性一致率 (NPA) / 特異度** | **D / (B+D)** | 排除無變異的能力 | **↓下降**：**偽基因 (Pseudogene)** 誤導 Mapping、**PCR 重複** | 可藉由色譜圖單一銳利波峰特徵 (sharp peak Sanger sequencing chromatogram)[🖼️](https://www.google.com/search?q=sharp+peak+Sanger+sequencing+chromatogram&tbm=isch&udm=2) 輕易排除雜訊 |
| **陽性預測值 (PPV)** | **A / (A+B)** | 報告陽性時的可信度 | 受群體**盛行率**影響；**偽陽性↑ → PPV↓** | **專一性極高**，為確認 NGS 陽性變異的首選工具 |
| **偽陰性率 (FNR)** | **C / (A+C)** | 漏診風險 (**偽陰性**) | **偽陰性 (C)↑**：腫瘤純度↓、大片段 **CNV** 偵測不良 | 遇引子結合位突變 (Allelic dropout) 亦會產生偽陰性 |
| **偽陽性率 (FPR)** | **B / (B+D)** | 誤診風險 (**偽陽性**) | **偽陽性 (B)↑**：FFPE 檢體之 **C>T 脫氨基假影** | 幾無脫氨基假影干擾，排除偽陽性最佳工具 |
| **無法判定 (No Call)** | **E 或 F** | 檢驗無效/品管失敗 | 核酸降解呈現洋菜膠體電泳核酸抹片特徵 (DNA smear agarose gel)[🖼️](https://www.google.com/search?q=DNA+smear+agarose+gel&tbm=isch&udm=2)、**濃度過低** | 若品質不佳會呈現波峰重疊雜亂特徵 (messy overlapping peaks Sanger sequencing)[🖼️](https://www.google.com/search?q=messy+overlapping+peaks+Sanger+sequencing&tbm=isch&udm=2) |
