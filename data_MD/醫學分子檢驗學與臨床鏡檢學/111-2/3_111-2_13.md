## 題目

使用HemoQuant方法進行糞便潛血試驗的敘述,下列何者錯誤?

A.相較於傳統的聯苯胺(benzidine)呈色劑方法,其可以降低偽陽性反應
B.係檢測檢體的血紅素轉變成紫質類(porphyrin)的量
C.利用特殊化學試劑結合紫質類(porphyrin)以呈色
D.不受飲食中維生素C之干擾

> [!success]- 正確解答
> **C**

---

## 筆記與詳解

### (1) 快速破題

看到題目討論「HemoQuant」與「糞便潛血試驗 (FOBT)」，並且要找『錯誤』的敘述 → 就要想到 HemoQuant 的核心專屬字眼是「**螢光法 (Fluorescence)**」。
題目問錯誤選項，掃描到選項 C 描述「特殊化學試劑結合...以『呈色』」，這是傳統化學法（如 Guaiac 或 Benzidine）的特徵，明顯與 HemoQuant 的螢光原理衝突 → 因此可以秒殺答案是 C。

### (2) 關鍵字 → 知識點

* **HemoQuant** → 基於螢光分析的定量法。將糞便中的血紅素轉化為紫質 (Porphyrin) 後，測定其發出的紫質螢光 (通常呈紅色) [🖼️](https://www.google.com/search?q=porphyrin+fluorescence+red&tbm=isch&udm=2)。
* **聯苯胺 (Benzidine) / 癒創木脂 (Guaiac)** → 傳統化學法。利用血紅素的「類過氧化物酶 (Pseudoperoxidase)」活性催化 H₂O₂ 產生呈色反應 (陽性為藍色) [🖼️](https://www.google.com/search?q=guaiac+fecal+occult+blood+test+positive+blue&tbm=isch&udm=2)。
* **偽陽性干擾 (False Positive)** → 飲食中的紅肉（動物血）或富含過氧化物酶的植物（如蘿蔔、花椰菜）會干擾傳統化學法造成偽陽性。
* **維生素 C (Vitamin C)** → 強還原劑。會抑制傳統化學法的氧化呈色反應，導致「偽陰性」。對 HemoQuant 螢光法或免疫法無影響。

### (3) 詳解

* **A. 正確：** 傳統呈色法容易受到植物性過氧化物酶或飲食鐵劑的干擾。HemoQuant 測量的是血紅素轉化後的紫質，專一性較高，能大幅降低因植物過氧化物酶引起的偽陽性。
* **B. 正確：** HemoQuant 的核心機轉是加入草酸並加熱，將血紅素（Heme）脫去中心的鐵離子，使其化學結構轉變成具天然螢光特性的紫質類（Porphyrins）。
* **C. 錯誤（本題答案）：** HemoQuant 是利用紫質受激發後產生螢光的特性，透過「**螢光分析儀 (Fluorometry)**」來測定，並**不是**加入特殊試劑讓它「呈色 (Colorimetry)」。呈色是化學法 (Guaiac/Benzidine) 的專利。
* **D. 正確：** 維生素 C 是一種強還原劑，會干擾依賴「氧化還原」來呈色的傳統化學法（導致偽陰性）。HemoQuant 是定量轉換後的螢光物質，不牽涉此類氧化還原反應，因此不受維生素 C 干擾。

### (4) 核心邏輯與比較

**縱向邏輯軸線：**
檢體血紅素 (Heme) → (加入草酸+加熱) 去鐵反應 → 轉變為紫質 (Porphyrin) → 激發後散發螢光 → 儀器定量螢光強度 → 不受還原劑 (Vit C) 抑制、不受植物酶干擾。

**記憶口訣：**
**Hemo** **Quant** → **螢** (Fluor) 光 **定**量 (Quant) → 量測**紫** (Porphyrin) 光。

**橫向統整比較：糞便潛血試驗 (FOBT) 方法學比較**

| 比較維度 | 化學呈色法 (Guaiac/Benzidine) | 螢光定量法 (HemoQuant) | 免疫分析法 (FIT/iFOBT) 檢驗卡匣 [🖼️](https://www.google.com/search?q=fecal+immunochemical+test+cassette&tbm=isch&udm=2) |
| --- | --- | --- | --- |
| **檢測目標** | 血紅素之**類過氧化物酶活性** | 轉化後之**紫質 (Porphyrin)** | 人類**完整血紅素 (hHb)** 抗原 |
| **偵測原理** | 氧化還原**呈色** (藍色) | **螢光**強度定量 | 抗原抗體**免疫結合** |
| **維生素C干擾** | **偽陰性** (強還原劑抑制反應) | **無干擾** | **無干擾** |
| **飲食干擾 (紅肉/植物酶)** | **偽陽性** (需限制飲食) | 僅受動物紅肉干擾 (植物酶無影響) | **無干擾** (專一性極高) |
| **出血部位靈敏度** | 上、下消化道 | 上、下消化道 | **下消化道**為主 (上消化道hHb易被胃酸降解) |
| **特徵與限制** | 便宜快速 / 易受飲食影響 | 可準確定量 / 需儀器較費時 | 不需限制飲食 / **大腸癌篩檢首選** |
