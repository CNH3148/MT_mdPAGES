## 題目

關於分子檢驗實驗室的品質保證作法,下列何者最不適當?

A.試劑加至檢體的順序為陰性品管檢體、病人檢體、陽性品管檢體,以避免陰性品管檢體以及病人檢體被污染
B.使用有濾網(filter)的微量吸管尖(pipet tip),防止檢體或核酸污染微量吸管
C.每天使用10%的漂白水擦拭工作檯面,破壞殘留的核酸
D.當有核酸污染疑慮時,使用棉棒沾取無菌水,對工作區域表面進行wipe test

> [!success]- 正確解答
> **A**

## 筆記與詳解

### (1) 快速破題

在分子檢驗（如 PCR）中，防範污染（避免偽陽性）是最高指導原則。看到考題提及「順序」與「污染」，請直覺聯想 **「單向工作流 (Unidirectional flow)」**。
題目選項 A 描述「試劑加至檢體」，這在分生實驗室是**絕對禁止的逆向操作**。永遠必須是「先分裝無污染的試劑 (Master mix)」，再將「檢體/核酸模板」加入試劑中。

### (2) 關鍵字與知識點拆解

* **試劑與檢體添加順序**：
* **物質順序**：試劑 (Master Mix) 先行分裝 → 再加入核酸模板。
* **模板順序**：陰性品管 (NTC) → 病人檢體 (Patient samples) → 陽性品管 (PTC)。(從最低濃度/無核酸 → 最高濃度，避免高濃度污染低濃度)。


* **Filter pipet tip (濾網微量吸管尖)**：物理性阻擋移液過程中產生的 **氣膠 (Aerosol)**，防止核酸抽吸時進入 Pipette 內部軸心造成交叉污染。
* **10% 漂白水 (次氯酸鈉)**：強氧化劑，能化學性破壞、打斷 DNA/RNA 結構。**70% 酒精無效**（酒精只能殺菌或沉澱核酸，無法降解核酸）。
* **Wipe test (擦拭測試)**：環境品管監測。定期以無菌水擦拭檯面、儀器把手，並將擦拭液進行 PCR，檢查環境是否有殘留的擴增產物 (Amplicon) 污染。

### (3) 詳解

* **A (最不適當，為本題答案)**：錯在「試劑加至檢體」。若試劑加至已含核酸的檢體管中，Tip 極容易接觸到高濃度核酸，進而污染試劑母液 (Stock)。正確做法是：在獨立乾淨的「試劑準備區」先將試劑分裝至所有微量管中，蓋好後移至「檢體處理區」，再依序 (NTC → 病人 → PTC) 將檢體加入管內。
* **B (適當)**：核酸擴增技術靈敏度極高，極微量氣膠即可導致偽陽性。使用具濾網的 Tip 可有效隔離氣膠，是分子實驗室的標準配備。
* **C (適當)**：每天工作結束後以 10% 漂白水擦拭檯面，利用去氧核糖核酸或核糖核酸對強氧化劑敏感的特性，徹底清除環境殘留的核酸，隨後通常會再用無菌水或 70% 酒精擦拭以去除漂白水殘留物以免腐蝕金屬檯面。
* **D (適當)**：Wipe test 是發現隱性污染源的標準主動監測手法，能確保工作區域表面是 clean 的狀態，未受核酸污染。

### (4) 核心軸線與邏輯因果

**預防微量核酸放大 (核心目的)** → 空間與動線管制 (單向流) → 物理阻絕 (濾網 Tip) → 化學破壞 (10% 漂白水 / UV 照射) → 驗證與監測 (Wipe test) → 若監控失敗：引發擴增產物氣膠污染 → 導致後續批次出現電泳非預期偽陽性條帶[🖼️](https://www.google.com/search?q=agarose+gel+false+positive+band&tbm=isch&udm=2) → 造成臨床誤診。

### (5) 分子實驗室品質保證與污染防制比較表

| 防護維度 | 具體品管措施 / 檢驗方法 | 作用機轉與目的 | 鑑別特徵與考試重點 |
| --- | --- | --- | --- |
| **動線管制** | 建立 **單向工作流 (Unidirectional flow)** | 試劑準備區 → 檢體萃取區 → 擴增分析區 | **絕對禁止逆向**；試劑先行分裝 |
| **加樣順序** | 依核酸濃度：**NTC → Patient → PTC** | 避免高濃度 target 污染低濃度或陰性管 | **NTC 必須最先加**；PTC 最後加 |
| **物理防護** | 使用 **Filter tip** (濾網微量吸管尖) | 阻絕 **Aerosol (氣膠)** 進入微量吸管內部 | 防範移液過程的隱性交叉污染 |
| **化學/物理去污** | **10% 漂白水** / UV 紫外燈照射 | 強氧化力 / UV 造成 DNA Thymine dimer | **漂白水可破壞核酸，酒精無效** |
| **環境監測** | **Wipe test** (環境擦拭測試) | 偵測工作檯面/門把殘留之 Amplicon | 預防大規模假陽性；定期例行公事 |
