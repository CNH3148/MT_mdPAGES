## 核心趨勢與高頻考點
藥物基因體學的考題核心，圍繞在「特定基因型」與「特定藥物反應」之間的因果關係。命題趨勢高度集中，反覆測驗考生是否能將 **基因-藥物-反應** 三者正確配對。

常見的命題邏輯與變化不出以下三種類型：
1.  **基因 → 藥物不良反應 (ADR)：** 此為最常見的考法。給定一個基因型（如 `HLA-B*1502`），測驗其與何種藥物（`Carbamazepine`）的嚴重過敏反應（`SJS`）相關。反之，亦可給定藥物與不良反應，反問應檢測何種基因。
    -   *因果鏈：特定HLA基因型 → T細胞辨識藥物-胜肽複合物 → 觸發強烈免疫反應 → 嚴重皮膚不良反應 (SCARs)。*

2.  **基因 → 藥物代謝速率/療效：** 測驗代謝酵素基因多型性（如 `CYP2C9`, `UGT1A1`）如何影響藥物在體內的濃度，進而改變其療效或毒性。
    -   *因果鏈 (代謝下降型)：基因變異 (如 `UGT1A1*28`) → 酵素活性↓ → 藥物清除率↓ → 血中藥物濃度↑ → 毒性反應風險↑ (如 `Irinotecan` 造成腹瀉)。*
    -   *因果鏈 (前驅藥活化型)：基因變異 (如 `CYP2D6` PM) → 酵素活性↓ → 前驅藥 (Prodrug) 無法活化 → 活性代謝物濃度↓ → 療效↓ (如 `Tamoxifen`)。*

3.  **基因 → 藥物作用標的/抗藥性：** 測驗與藥物作用機制直接相關的基因，其表現量或變異如何影響治療效果。
    -   *因果鏈：基因表現量↑ (如 `ERCC1`) → DNA修復能力↑ → 化療藥物 (`Oxaliplatin`) 造成的DNA損傷被修復 → 癌細胞存活 → 抗藥性↑、療效↓。*

**預測未來考法：**
- **整合型考題：** 將多個基因整合於一題，例如 Warfarin 的考題會同時出現 `CYP2C9` (代謝) 和 `VKORC1` (標的) 的選項，要求考生判斷何種組合需要最低劑量。
- **定義深化：** 除了 PM (Poor metabolizer) 外，可能開始測驗 IM (Intermediate), EM (Extensive), UM (Ultrarapid) 的基因型定義。
- **新興藥物基因：** 考題可能逐漸納入更多臨床指引已涵蓋的藥物-基因配對，但核心邏輯不變。
