## 題目

人類細胞純化的RNA跑膠後,須有兩條明顯的bands,代表該RNA的完整性,此兩條bands分別是:

A.8S rRNA、18S rRNA
B.18S rRNA、28S rRNA
C.28S rRNA、60S rRNA
D.60S rRNA、80S rRNA

> [!success]- 正確解答
> **B**

---

## 筆記與詳解

### (1) 快速破題

看到題目提到「人類細胞 (真核生物)」、「純化 RNA 跑膠」、「兩條明顯的 bands [🖼️](https://www.google.com/search?q=intact+total+RNA+agarose+gel+18S+28S&tbm=isch&udm=2)」，就要想到真核細胞內含量最豐富的 RNA 是 **rRNA (核糖體 RNA)**。真核生物核糖體特徵為偶數 (80S = 60S + 40S)，其組成的核心大分子 rRNA 分別為 **28S** (存在於大次單元) 與 **18S** (存在於小次單元)。因此萃取總 RNA 跑電泳時，這兩條會是最明顯的指標帶。選項中出現 60S、80S 的皆為蛋白質與 RNA 的複合體(或稱次單元)，並非純 RNA。因此直接鎖定 B。

### (2) 詳解預備

* **rRNA (Ribosomal RNA) → 核糖體的主要成分**：細胞內占比最高的 RNA (約 80%)，總 RNA 萃取後跑電泳，最明顯的 bands 必定是 rRNA。
* **S (Svedberg unit) → 沉降係數**：代表分子在離心時的沉降速度，與質量、密度、形狀有關，**不可直接相加** (例如 60S + 40S = 80S，而非 100S)。
* **RNA 完整性 (RNA Integrity) → 28S 與 18S 的比例**：未降解的真核 RNA 在電泳膠片上應呈現 **28S : 18S 亮度約為 2:1** 的兩條清晰主帶 [🖼️](https://www.google.com/search?q=RNA+integrity+gel+electrophoresis+2+to+1+ratio&tbm=isch&udm=2)。若出現拖尾 (smear) 或小分子帶變亮，代表 RNA 已發生降解。

### (3) 詳解

* **(A) 8S rRNA、18S rRNA**：錯誤。真核生物核糖體中並不包含 8S rRNA。
* **(B) 18S rRNA、28S rRNA**：**正確**。人類 (真核生物) 萃取出的 total RNA，經電泳分離後，兩條最明顯的帶即為 18S rRNA 與 28S rRNA。
* **(C) 28S rRNA、60S rRNA**：錯誤。60S 是大次單元 (Large subunit)，是由 28S、5.8S、5S rRNA 與多種蛋白質共同組成的複合結構，不是單一 rRNA 種類。
* **(D) 60S rRNA、80S rRNA**：錯誤。60S 是大次單元，80S 是完整的真核核糖體 (Ribosome)，兩者皆含有蛋白質，在「純化 RNA」的跑膠結果中不會以 60S 或 80S 的形式存在。

### (4) 核心軸線與邏輯因果

真核細胞 RNA 萃取 → 大宗為 rRNA (>80%) → 變性凝膠電泳分離 → **28S 與 18S rRNA** 分子量最大且佔比最高 → 呈現兩條明顯主帶 [🖼️](https://www.google.com/search?q=RNA+agarose+gel+28S+18S+bands&tbm=isch&udm=2) → 觀察亮度比 (28S:18S ≒ 2:1) → 判定 **RNA 完整性 (無降解)**。

**💡 記憶口訣**：

* **真核**生物愛**偶數**：**8**0S 體 = **6**0S 大 + **4**0S 小 (主要 rRNA：**28**S, **18**S, 5.8S, 5S)
* **原核**生物愛**奇數**：**7**0S 體 = **5**0S 大 + **3**0S 小 (主要 rRNA：**23**S, **16**S, 5S)

| 比較維度 | 真核生物 (人類細胞) | 原核生物 (細菌) |
| --- | --- | --- |
| **完整核糖體** | **80S** | **70S** |
| **大次單元 (Large Subunit)** | 60S | 50S |
| **大次單元 rRNA 組成** | **28S**, 5.8S, 5S | **23S**, 5S |
| **小次單元 (Small Subunit)** | 40S | 30S |
| **小次單元 rRNA 組成** | **18S** | **16S** |
| **Total RNA 跑膠主要 Bands** | **28S**, **18S** | **23S**, **16S** |
| **RNA 完整性指標亮度比** | 28S : 18S ≒ **2 : 1** | 23S : 16S ≒ **2 : 1** |
| **分生鑑定標的基因** | **18S** rDNA / ITS 區域 (黴菌) | **16S** rDNA (細菌) |
