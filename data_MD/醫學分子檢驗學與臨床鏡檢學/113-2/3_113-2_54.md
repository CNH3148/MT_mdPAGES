## 題目

微陣列(microarray)生物晶片的製程可分成光引導原位合成法(light-directed synthesis)與點樣法(spotting),關於二者之敘述,下列何者最不適當?

A.點樣法的探針可以是寡核苷酸(oligonucleotide)探針或cDNA探針
B.點樣法產生的晶片,其探針密度可達數十萬點/平方公分
C.光引導原位合成法是以合成DNA探針為主
D.Taiwan Precision Medicine Initiative(TPMI)採用的晶片是使用光引導原位合成法的製程

> [!success]- 正確解答
> **B**

---

## 筆記與詳解

### (1) 快速破題

看到題目比較「光引導原位合成法」與「點樣法」，並且要找「最不適當(錯誤)」的選項 → 就要直覺想到兩者的最大區別：**探針密度 (解析度)**。
「點樣法」是仰賴機械手臂將液體微滴「點」在玻片上，受限於物理液滴體積與表面張力，密度極限僅在數千至數萬點之間；而「光引導原位合成法」是利用半導體微影光刻技術直接合成，因此密度極高。當看到選項 B 敘述點樣法密度達「數十萬點/平方公分」時，即可判斷該數值被張冠李戴了，因此答案是 B。

### (2) 詳解預備

* **微陣列 (Microarray)** → 檢測大量基因表現或變異的高通量平台，雜交後透過雷射掃描產生的螢光訊號 [🖼️](https://www.google.com/search?q=Microarray+fluorescence+scan&tbm=isch&udm=2) 判讀結果。
* **點樣法 (Spotting)** → 先在外部合成好探針，再用機器手臂以微細針頭將液體點到載玻片上。
* **光引導原位合成法 (Light-directed synthesis)** → 結合半導體光罩技術與化學合成，利用光線移除保護基，逐個鹼基 (base-by-base) 直接在晶片表面建構探針。
* **cDNA 與寡核苷酸 (Oligonucleotide)** → 探針的兩種主要形式；cDNA片段較長（適合看基因總體表現），寡核苷酸較短且具高度專一性（適合偵測單核苷酸多型性 SNP）。
* **TPMI (台灣精準醫療計畫)** → 大規模基因體檢測計畫，採用客製化的高密度 SNP 陣列 (Taiwan Biobank Array)。

### (3) 詳解

* **(A) 正確：** 點樣法的探針是「預先準備好」的墨水，因此不論是透過 PCR 擴增的長片段 cDNA，或是人工合成的短片段寡核苷酸 (Oligonucleotide)，都可以作為探針被點漬在晶片上。
* **(B) 最不適當 (本題答案)：** 點樣法的物理極限受限於針孔大小及液滴擴散範圍（斑點直徑約 100~150 μm），其密度極限通常約為 **1,000 ~ 10,000 點/平方公分**。要達到「數十萬甚至百萬點/平方公分」的高密度，必須使用光引導原位合成法。
* **(C) 正確：** 光引導原位合成法（例如經典的 Affymetrix GeneChip）是在晶片基質上，利用光反應一個一個加上核苷酸，因此產物必定是人工化學合成的短鏈單股 DNA (寡核苷酸探針為主)。
* **(D) 正確：** TPMI 計畫為了在單一晶片上偵測近百萬個針對台灣人設計的 SNP 位點，需要極高的探針密度。目前採用的客製化晶片（如 Thermo Fisher Axiom 平台）正是採用基於半導體製程的光引導原位合成法製造。

### (4) 知識梳理與邏輯因果

* **核⼼軸線：** 核心製程原理 → 物理空間限制 → 探針密度高低 → 臨床/研究應用場景。
* **機械液體點滴 (點樣法) →** 液滴無法無限制縮小 → 密度**低** → 適合實驗室自製、小規模或針對特定疾病群的基因表現檢測。
* **半導體光刻曝光 (光引導法) →** 奈米級精準度控制反應 → 密度**極高** → 適合全基因組掃描 (GWAS)、百萬級別 SNP 定型檢測。


* **記憶口訣：**「**光高點低**」 (光引導密度極高、點樣法密度低)。

### (5) 微陣列晶片製程技術比較

| 比較維度 | 點樣法 (Spotting / Printed Microarray) | 光引導原位合成法 (Light-directed In-situ Synthesis) |
| --- | --- | --- |
| **製程原理** | 機械手臂微細針頭**點漬液滴** | 半導體**光罩微影** + DNA化學合成 |
| **探針來源** | 外部**預先合成** (如 PCR 產物) | 晶片基質上**直接逐段合成** |
| **探針類型** | cDNA、短/長寡核苷酸 | **寡核苷酸** (單股短鏈 DNA) |
| **探針長度** | 較長 (數百 ~ 數千 bp) | **極短** (25 ~ 70 mer) |
| **探針密度** | **低** (< $10^4$ 點/$cm^2$) | **極高** (> $10^5$ ~ $10^6$ 點/$cm^2$) |
| **製造成本** | 初始建置低 / **自訂更動彈性高** | **光罩開模成本極高** / 適合大規模量產 |
| **主要應用** | 特定基因表現體學、實驗室自製檢測 | **全基因體關聯分析 (GWAS)**、高密度 SNP 陣列、**TPMI 計畫** |
| **代表平台** | 傳統 cDNA 微陣列玻片 | Affymetrix (現 Thermo Fisher)、Illumina BeadChip (原理相似的高密度) |
