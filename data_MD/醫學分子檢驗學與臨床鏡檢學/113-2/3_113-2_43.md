## 題目

下列何者是intron之5' splice site、branch site 與 3' splice site的保留序列(consensus sequence)?

A.5'-AG----------A-------GU-3'
B.5'-GU-------------------AG-3'
C.5'-AG-------------------GU-3'
D.5'-GU------------G-------AC-3'

> [!success]- 正確解答
> **B**

---

## 筆記與詳解

### (1) 快速破題

* **解題思路**：看到題目關鍵字「intron」、「splice site (剪接位)」，必須立刻反射想到遺傳學中最經典的 **GU-AG 規則 (GU-AG rule)**。
* 在真核生物的 pre-mRNA 中，內含子 (intron) 的兩端具有極度保守的序列：5' 端開頭必定是 **GU**，3' 端結尾必定是 **AG**。此外，靠近 3' 端的分支位點 (branch site) 必定有一個 **A**。因此整體序列特徵為 **5'-GU...A...AG-3'**，直接尋找兩端符合 GU 與 AG 的選項即為正解。

### (2) 詳解預備

* **Intron (內含子)**：pre-mRNA 中不具蛋白質編碼功能、需被剪接體移除的片段。
* **5' splice site (5' 剪接位 / 供位)**：Intron 開頭，保留序列為 **GU** (若為 DNA 層次則為 GT)。此處為 U1 snRNP 的辨識與結合位。
* **Branch site (分支位點)**：位於 intron 內部、靠近 3' 端 (約上游 20-50 個核苷酸處)，核心序列為高度保守的腺嘌呤 **A** (Adenine)。
* **3' splice site (3' 剪接位 / 受位)**：Intron 結尾，保留序列為 **AG**。

### (3) 詳解

* **標準序列**：正確的 pre-mRNA 剪接保留序列，由 5' 端至 3' 端依序為 5' 剪接位 (**GU**) → 分支位點 (**A**) → 3' 剪接位 (**AG**)。
* **選項剖析**：
* **(A) 5'-AG----------A-------GU-3'**：錯誤。5' 與 3' 的特徵序列完全顛倒。
* **(B) 5'-GU-------------------AG-3'**：**正確**。完美符合 5' 端為 **GU**、3' 端為 **AG** 的剪接規則。即使該選項排版未將中間的 A 標示出來，但其頭尾特徵已具備唯一正確的鑑別度。
* **(C) 5'-AG-------------------GU-3'**：錯誤。方向顛倒，GU 應在 5'、AG 應在 3'。
* **(D) 5'-GU------------G-------AC-3'**：錯誤。雖然 5' 端正確，但 branch site 應為 A (非 G)，且 3' 端應為 AG (非 AC)。



### (4) 邏輯與表格統整

* **邏輯軸線**：基因轉錄 pre-mRNA → 剪接體啟動 → **U1** 結合 **5'-GU**，**U2** 結合 **Branch site-A** → 分支位 **A 的 2'-OH** 攻擊 5' 端 → 形成套索結構 (Lariat structure)[🖼️](https://www.google.com/search?q=%22lariat+structure%22+splicing+RNA&tbm=isch&udm=2) → 5' 外顯子游離的 3'-OH 攻擊 **3'-AG** → 切除 Intron 並拼接 Exon。
* **記憶口訣**：**「瓜哥 (GUA-AG)」** → 5' 是 **GU**，中間 Branch site 是 **A**，3' 是 **AG**。

| 剪接組件 / 序列 | 結合位置 / 標的序列 | 核心功能與生化機制 |
| --- | --- | --- |
| **5' Splice site** | Intron 5' 端 (**GU**) | 剪接起始點；供 **U1 snRNP** 辨識結合 |
| **Branch site** | 近 3' 端 (**A**) | 供 **U2 snRNP** 結合；以 **2'-OH** 發動初次親核攻擊 → 形成**套索結構**[🖼️](https://www.google.com/search?q=%22lariat+structure%22+splicing+RNA&tbm=isch&udm=2) |
| **3' Splice site** | Intron 3' 端 (**AG**) | 剪接終點；接受 5' Exon 攻擊 → 釋放 Intron |
| **U1 snRNP** | **5' 端 (GU)** | 辨識 5' 剪接位 → 啟動剪接體 (Spliceosome) 組裝 |
| **U2 snRNP** | **Branch site (A)** | 結合分支位 → 迫使 **A** 凸出 (bulge) → 活化其化學活性 |
| **U4/U6 snRNP** | 剪接體核心 | **U6** 具實際催化活性 (取代 U1)；**U4** 扮演抑制 U6 之角色 |
| **U5 snRNP** | Exon 邊界 | 牽引並對齊 5' 與 3' Exon → 促進外顯子連接 |
