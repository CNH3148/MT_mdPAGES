## 題目

關於核酸分離技術的敘述,下列何者最不適當?

A.liquid phase extraction是樣本分子在分層中藉由其不同的吸附作用,對樣本分子進行分離
B.solid phase size exclusion separation是一種藉由樣本分子大小不同,在基質(matrix)中以不同速度移動的分離技術
C.solid phase ion exchange chromatography利用樣本分子的表面帶電性之差異來進行分離
D.solid phase surface adsorption (or affinity chromatography)利用樣本分子間的親和性不同來進行分離

> [!success]- 正確解答
> **A**

---

## 筆記與詳解

### (1) 快速破題

看到「液相萃取 (Liquid phase extraction)」，必須立刻聯想到傳統的「酚-氯仿萃取法 (Phenol-chloroform extraction)」；其分離的核心原理是基於分子在兩種互不相溶液體間的「溶解度 (Solubility)」**與**「極性 (Polarity)」差異，而非吸附作用。「吸附作用 (Adsorption)」是「固相 (Solid phase)」技術 (如矽膠管柱、磁珠) 的專屬特徵。因此，選項 A 將液相萃取與吸附作用連在一起，是明顯的張冠李戴，為本題要找的錯誤選項。

### (2) 關鍵字與知識點拆解

* **Liquid phase extraction (液相萃取)：** → 知識點：利用**溶解度/分配係數**差異。核酸親水 (留於水相)，蛋白質經有機溶劑變性後疏水 (沉澱於交界面或進入有機相)。
* **Solid phase extraction (固相萃取)：** → 知識點：利用固體基質 (如 Silica 矽膠) 在特定化學環境下，與核酸產生**表面吸附 (Adsorption)**。
* **Size exclusion (尺寸排除/分子篩)：** → 知識點：利用多孔性基質。分離依據為**分子大小 (Size)**。
* **Ion exchange (離子交換)：** → 知識點：利用**表面電荷 (Charge)** 差異。核酸帶負電，常以帶正電的陰離子交換樹脂捕捉。
* **Affinity chromatography (親和性層析)：** → 知識點：利用分子間的**高度專一性結合 (Specific affinity)**，如互補核酸序列 (Oligo-dT 抓取 Poly-A)。

### (3) 選項詳解與核心邏輯

**核酸分離核心邏輯 (縱向聯想)：** 細胞裂解釋放核酸 (帶負電/水溶性) → 選擇介質：
【液相路徑】加入有機溶劑 (Phenol/Chloroform) → 蛋白質變性疏水化 → **溶解度差異**使核酸保留於上層水相 → 加入高濃度酒精與鹽類 → 析出白色絲狀核酸沉澱[🖼️](https://www.google.com/search?q=%22DNA+precipitation%22+ethanol+%22white+precipitate%22&tbm=isch&udm=2)。
【固相路徑】加入離液鹽 (Chaotropic salts) 或調整 pH 值 → 核酸與固體介質表面發生**電荷、大小、吸附或專一性交互作用** → 雜質洗去 → 改變離子濃度洗脫純化核酸。

* **A. (最不適當/本題答案)：** 錯誤。Liquid phase extraction (液相萃取) 是藉由樣本分子在不同液相分層間的「**溶解度**」或「**極性**」差異進行分離，並非藉由「吸附作用 (Adsorption)」。吸附作用發生在固相-液相交界面。
* **B. (適當)：** 正確。Solid phase size exclusion (固相尺寸排除) 使用多孔微粒基質。**大分子**無法進入微孔，移動路徑短，**最先流出**；**小分子**進入微孔，移動路徑長，**最後流出**。
* **C. (適當)：** 正確。Solid phase ion exchange (固相離子交換) 運用核酸骨架上磷酸根帶**強負電**的特性，使其與帶正電的固相介質 (陰離子交換樹脂) 結合而分離。
* **D. (適當)：** 正確。Solid phase surface adsorption / affinity (固相表面吸附/親和層析) 利用特定分子間的親和力。例如：Silica (二氧化矽) 在高鹽下吸附 DNA (表面吸附)；或利用 Oligo(dT) 探針專一性捕捉 mRNA 的 Poly(A) 尾巴 (親和層析)。

### (4) 橫向比較統整

**記憶口訣：** 液相看**溶**解 (極性分層)
固相看**吸**附 (矽膠抓取)
大小用**篩**的 (**大**先**小**後)
電荷用**換**的 (核酸負電)

**【常見核酸分離/純化技術比較矩陣】**

| 技術分類 | 具體方法/介質 | 核心分離原理 | 關鍵特徵 / 條件 / 結果 |
| --- | --- | --- | --- |
| **Liquid Phase** (液相) | Phenol-Chloroform 萃取 | **極性 / 溶解度差異** | 上層水相：**核酸**<br>交界面：變性蛋白質<br>下層有機相：脂質/雜質 |
| **Solid Phase** (固相) | Silica spin column (矽膠管柱/磁珠) | **表面吸附** (Surface adsorption) | 需**高濃度離液鹽** (Chaotropic salts) 破壞水合層<br>核酸吸附於 Silica 表面 |
| **Solid Phase** (固相) | Size exclusion (分子篩/尺寸排除) | **分子大小** (Size) | 微孔基質 (Matrix)<br>洗脫順序：**大分子先出，小分子後出** |
| **Solid Phase** (固相) | Ion exchange (離子交換) | **表面電荷** (Charge) | 使用**陰離子交換樹脂** (帶正電)<br>利用**高鹽濃度溶液** (High salt) 競爭洗脫核酸 |
| **Solid Phase** (固相) | Affinity chromatography (親和性) | **專一性親和力** (Specific binding) | 專一性最高<br>例：**Oligo(dT) 捕捉 mRNA** |
