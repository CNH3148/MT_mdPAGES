## 題目

關於尿試紙測尿糖原理的敘述,下列何者錯誤?

A.尿試紙檢測的試劑含有葡萄糖還原酶,因此對葡萄糖有專一性
B.結果以半定量呈現
C.尿中維生素C大於50 mg/dL可造成偽陰性結果
D.只能測葡萄糖

> [!success]- 正確解答
> **A**

---

## 筆記與詳解

### (1) 快速破題

看到題目考「尿試紙測尿糖」的「原理」且要求找錯誤敘述，請直接鎖定「酵素名稱」與「常見干擾物」。尿試紙檢測葡萄糖的核心在於「葡萄糖**氧化酶** (Glucose oxidase)」，絕非還原酶。選項 A 試圖以「還原酶」混淆視聽，這在醫檢與護理國考中是極為經典的陷阱題，一眼即可判定 A 為錯誤。

### (2) 關鍵字解析

* **尿試紙 (Urine dipstick)** → 利用雙重連續酵素反應，快速、半定量評估尿液成分的檢測工具。
* **葡萄糖氧化酶 (Glucose oxidase)** → 試紙第一步反應的專一性酵素，**絕對不會**與果糖、半乳糖等其他醣類反應。
* **半定量 (Semi-quantitative)** → 檢驗結果以濃度級距（如：Negative, Trace, 1+, 2+ 等）呈現，而非給出絕對精準的數值。
* **維生素 C (Ascorbic acid)** → 強大的**還原劑**。會消耗檢測反應中用來呈色的過氧化氫 (H2O2)，是導致尿糖試紙出現**偽陰性**的頭號嫌疑犯。

### (3) 詳解

* **選項 A (錯誤，本題解答)：** 尿試紙測糖使用的是「葡萄糖**氧化酶** (Glucose oxidase)」，而不是「葡萄糖還原酶」。完整的反應機轉為：
1. 葡萄糖 + O2 → (經葡萄糖氧化酶催化) → 葡萄糖酸 + **H2O2**
2. **H2O2** + 色原 → (經過氧化物酶 Peroxidase 催化) → 氧化態色原 (試紙色塊變色 [🖼️](https://www.google.com/search?q=urine+dipstick+glucose+color+change&tbm=isch&udm=2)) + H2O
雖然其「對葡萄糖有專一性」的敘述正確，但酵素名稱完全錯誤。


* **選項 B (正確)：** 試紙依據顏色深淺對照標準色板來判讀，給出的結果為一個濃度範圍，屬於半定量分析。
* **選項 C (正確)：** 維生素 C 是強還原劑。當尿中維生素 C 濃度高（如 > 50 mg/dL），它會在上述第 2 步反應中，搶先將 H2O2 還原成水，導致色原無法被氧化呈色，最終造成**偽陰性 (False negative)**。
* **選項 D (正確)：** 由於葡萄糖氧化酶對葡萄糖分子的立體結構具有極高專一性，因此該試劑**只能**測出葡萄糖，無法檢測乳糖、半乳糖或果糖等其他還原糖。

### (4) 核心邏輯與統整

* **縱向聯想：** 血糖濃度 ↑ (超過腎閾值 160-180 mg/dL) → 葡萄糖溢出至尿液 → 試紙**氧化酶**反應產生 H2O2 → 試紙色原氧化變色 → 對照色板得出半定量結果。若同時服用高劑量維他命 C → H2O2 被消耗 → 試紙不變色 → 檢驗出**偽陰性**。
* **記憶口訣：** 「**氧**化專測**葡**，維 C 搶氧變**偽陰**」(試紙用**氧**化酶、專一測**葡**萄糖；維他命 C 會搶走氧/H2O2導致**偽陰**性)。

### (5) 比較表格

針對尿糖檢測，常將「試紙法」與測量小兒先天性代謝異常的「銅還原法 (Clinitest)」進行橫向比較：

| 比較維度 | 尿試紙法 (Dipstick) | 銅還原試驗 (Clinitest) |
| --- | --- | --- |
| **核心原理** | 葡萄糖**氧化酶**法 (雙重酵素反應) | **本內氏 (Benedict's)** 銅離子還原法 |
| **專一性** | **極高** (僅針對葡萄糖) | **無** (針對所有還原物質) |
| **可測物質** | **只有**葡萄糖 | 葡萄糖、果糖、半乳糖、乳糖、戊糖 |
| **視覺特徵** | 試紙色塊由黃變藍/褐 | 產生**磚紅色沉澱** [🖼️](https://www.google.com/search?q=Clinitest+brick+red+precipitate&tbm=isch&udm=2) |
| **干擾：維生素 C** | 造成 **偽陰性** (消耗 H2O2) | 造成 **偽陽性** (強還原劑還原銅離子) |
| **干擾：漂白水/強氧化劑** | 造成 **偽陽性** (直接氧化色原) | 造成 **偽陰性** (干擾還原反應) |
| **主要臨床應用** | 常規糖尿病篩檢 | 嬰幼兒**半乳糖血症**篩檢 |
