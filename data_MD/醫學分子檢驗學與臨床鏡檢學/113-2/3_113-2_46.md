## 題目

聚合酶連鎖反應(PCR)的擴增曲線由三個階段所組成,下列何者不在此三個階段中?

A.exponential phase
B.elongation phase
C.linear phase
D.plateau phase

> [!success]- 正確解答
> **B**

---

## 筆記與詳解

### (1) 快速破題

看到題目問「PCR的**擴增曲線 (Amplification curve)**」且要找「不在此三個階段中」的選項 → 就要想到擴增曲線描述的是 Real-time PCR 中「整體產物量隨循環數增加的**宏觀動態變化**」，包含指數、線性、高原三期；而選項 B (elongation phase) 是描述 PCR 單一循環內的「**微觀溫度步驟**」(變性、黏合、延長)。將「巨觀曲線階段」與「微觀溫度步驟」混淆是考題常見陷阱，故答案秒選 B。

### (2) 詳解預備：關鍵字 → 知識點

* **擴增曲線 (Amplification curve)** [🖼️](https://www.google.com/search?q=qPCR+amplification+curve&tbm=isch&udm=2) → qPCR 中螢光訊號隨 Cycle 數增加而繪製出的 S 型曲線。
* **Exponential phase (指數期)** → 產物呈 $2^n$ 完美倍增，試劑極度充足，為計算 Ct 值 (Threshold cycle) 的最佳區間。
* **Linear phase (線性期)** → 反應物開始消耗，聚合酶效率下降，倍增效率 $<2$，呈線性緩慢上升。
* **Plateau phase (高原期)** → 試劑 (dNTPs, Primers) 耗盡或產物自我黏合抑制，螢光訊號達飽和不再上升。
* **Elongation phase (延長/延伸期)** → PCR 溫度循環 (Thermal cycling) 步驟之一，Taq polymerase 於 72°C 進行 DNA 股合成。

### (3) 詳解

* **正確解答辨析 (B 為錯誤敘述，故選 B)：** 題目問的是「擴增曲線的階段」。擴增曲線是反應整體產量隨時間(循環數)變化的趨勢。Elongation phase (延長/延伸期) 是每次 cycle 中的一個「溫度步驟」(Denaturation → Annealing → Elongation)，屬於反應機轉，而非螢光產量動態曲線的巨觀階段。
* **選項 A (exponential phase) 正確：** 擴增曲線的第一階段。此時模板量少但試劑充足，每個 cycle 產物穩定翻倍，為最準確的定量區間。
* **選項 C (linear phase) 正確：** 擴增曲線的第二階段。隨著反應進行，dNTPs 和 primers 被大量消耗，酵素活性逐漸降低，產物增加速率減緩。
* **選項 D (plateau phase) 正確：** 擴增曲線的最終階段。試劑完全耗盡或副產物累積抑制反應，螢光訊號達到天花板而平緩。

### (4) 核心軸線與橫向比較

**核心邏輯因果：**
微觀 PCR 單次循環步驟 (變性 → 黏合 → 延長) 反覆進行 → 巨觀螢光訊號累積形成擴增曲線 → 隨反應資源消耗產生三階段變化 (**Exponential ↑↑** → **Linear ↑** → **Plateau ─**) → 取指數期計算 Ct 值用於精準定量。

**記憶口訣：**

* 曲線三階段：**指、線、高** (指數、線性、高原 → 想像一條線指到高處)。
* 循環三步驟：**變、黏、長** (變性、黏合、延長)。
* 解題心法：「曲線」看整體產量變化，「循環」看單次溫度變化。
