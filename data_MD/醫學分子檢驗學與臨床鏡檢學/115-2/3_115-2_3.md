## 題目

糖尿病患尿液採集後直接置放室溫 2 小時，最可能會產生下列何種變化？

A.酮體濃度增加
B.pH 值降低
C.顏色變淡
D.混濁度降低

> [!success]- 正確解答
> **B**

---

## 筆記與詳解

### (1) 快速破題

看到題目出現「尿液置放室溫」，就要直覺想到「尿液未保存/久放的化學與物理變化」。
這題的破題關鍵在於「糖尿病患」**這個先決條件。一般正常尿液久放，細菌會分解尿素產生氨，導致 pH 值上升；但糖尿病患的尿液富含葡萄糖，室溫久放會促使細菌與酵母菌大量增生並**發酵葡萄糖產酸，導致 pH 值反而下降。
快速檢視選項：酮體會揮發 (下降)、顏色因氧化變深、混濁度因細菌孳生而增加。因此秒選 B。

### (2) 關鍵字與知識點拆解

* **室溫置放 2 小時 (未保存尿液)** → 知識點：尿液中的細胞會崩解、化學物質會揮發或被光分解、細菌會大量增殖。超過 2 小時未檢驗的尿液必須冷藏。
* **糖尿病患 (Diabetes Mellitus)** → 知識點：臨床表徵為多尿、尿液常出現高濃度的葡萄糖 (Glucosuria) 與酮體 (Ketonuria)。
* **酮體 (Ketones)** → 知識點：包含丙酮、乙醯乙酸、β-羥丁酸。其中丙酮具極高揮發性，乙醯乙酸易被細菌代謝。
* **混濁度 (Turbidity)** → 知識點：正常新鮮尿液多為清澈，久放變混濁的兩大主因是「細菌增生」與「溫度改變導致結晶析出」。

### (3) 詳解

* **選項 A (酮體濃度增加)：錯誤。**
酮體中的丙酮 (Acetone) 在室溫下極易**揮發**散失至空氣中，且孳生的細菌也會消耗代謝乙醯乙酸 (Acetoacetate)，因此尿液久放會導致酮體濃度**下降**（造成偽陰性）。
* **選項 B (pH 值降低)：正確。**
一般人的尿液久放，細菌尿素酶 (Urease) 分解尿素產生氨 (NH₃)，會使 pH 值**上升**。但此題特指「糖尿病患」，其尿液中含有大量葡萄糖，細菌或酵母菌[🖼️](https://www.google.com/search?q=yeast+urine+sediment&tbm=isch&udm=2)會**發酵葡萄糖產生酸性代謝產物**，反而會導致尿液 pH 值**下降**。這是一個常見且極具鑑別度的國考陷阱。
* **選項 C (顏色變淡)：錯誤。**
尿液室溫久放會發生氧化還原反應，例如無色的尿膽素原 (Urobilinogen) 被氧化成有色的尿膽素 (Urobilin)，或是其他代謝物氧化，皆會導致尿液顏色**變深**。
* **選項 D (混濁度降低)：錯誤。**
尿液置於室溫會成為絕佳的培養基，導致細菌大量增生[🖼️](https://www.google.com/search?q=bacteria+urine+sediment&tbm=isch&udm=2)；此外，尿液冷卻至室溫可能促使無定形尿酸鹽[🖼️](https://www.google.com/search?q=amorphous+urate+crystals+urine+sediment&tbm=isch&udm=2)（酸性尿）或無定形磷酸鹽[🖼️](https://www.google.com/search?q=amorphous+phosphate+crystals+urine+sediment&tbm=isch&udm=2)（鹼性尿）結晶析出，這些都會導致混濁度**增加**。

### (4) 核心軸線與橫向比較

* **縱向聯想（糖尿病尿液久放機轉）：**
尿液富含葡萄糖 → 室溫放置 >2 小時 → 細菌/酵母菌生長 → 糖解發酵作用產酸 → **pH↓**
* **縱向聯想（一般尿液久放機轉）：**
尿中含尿素 → 細菌尿素酶作用 → 分解產氨 (NH₃) → **pH↑** → 鹼性環境加速紅血球[🖼️](https://www.google.com/search?q=RBC+urine+sediment&tbm=isch&udm=2)與圓柱體[🖼️](https://www.google.com/search?q=urinary+casts+sediment&tbm=isch&udm=2)崩解。

**記憶口訣：**
「未保存尿液變化：**色深濁度高，氨升(pH↑)亞硝酸生；糖酮膽素降，細胞圓柱破**。唯獨**糖尿酸(pH↓)**！」

### (5) 未保存尿液置放室溫超過 2 小時之變化總表

| 檢驗項目 / 狀態 | 數值/濃度變化 | 核心機轉與干擾物質 |
| --- | --- | --- |
| **pH (一般尿液)** | **↑ (變鹼)** | 細菌分解尿素產生**氨 (Ammonia)** |
| **pH (糖尿病尿液)** | **↓ (變酸)** | 細菌/酵母菌發酵**葡萄糖**產酸 |
| **葡萄糖 (Glucose)** | **↓** | 遭細菌及細胞(如白血球[🖼️](https://www.google.com/search?q=WBC+urine+sediment&tbm=isch&udm=2))之**糖解作用**消耗 |
| **酮體 (Ketones)** | **↓** | **丙酮揮發**；細菌代謝乙醯乙酸 |
| **膽紅素 (Bilirubin)** | **↓** | 照光發生光氧化，轉為**膽綠素** (偽陰性) |
| **尿膽素原 (Urobilinogen)** | **↓** | 氧化為**尿膽素** (Urobilin) |
| **亞硝酸鹽 (Nitrite)** | **↑** | **硝酸鹽還原菌**大量增生 (過久亦可能還原成氮氣散失導致↓) |
| **紅/白血球、圓柱體** | **↓** | 在**稀薄 (低比重)** 或 **鹼性 (pH↑)** 尿液中發生溶解破壞 |
| **混濁度 (Turbidity)** | **↑** | **細菌增生**；**無定形結晶**析出 |
| **顏色 (Color)** | **變深** | 代謝產物 (如尿膽素原) **氧化** |
| **氣味 (Odor)** | **阿摩尼亞味** | 尿素分解產氨 (若是糖尿則可能帶有**發酵酸臭味**) |
