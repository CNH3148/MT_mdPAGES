## 題目

循環核酸(Circulating nucleic acid)可以做為腫瘤核酸標誌檢驗的標的。下列有關循環核酸分離技術的敘述,何者正確?

A.Heparin為適合的抗凝血劑
B.循環核酸適合以分離高分子量核酸的技術取得
C.血漿是腫瘤循環核酸標誌較佳的樣品來源
D.適合以全血的方式做長期的檢體保存

> [!success]- 正確解答
> **C**

---

## 筆記與詳解

### (1) 快速破題

看到題目討論「循環核酸 (cfDNA/ctDNA)」的採檢與分離，解題思路須圍繞其三大特性：**微量、片段化、極易受背景基因體 DNA (gDNA) 污染**。
選項 A 提及 Heparin → 想到分子 PCR 檢驗「禁用」Heparin；選項 B 提到高分子量 → 想到 cfDNA 是凋亡產生的「短片段」；選項 C 比較血漿與血清 → 想到血清凝血會導致白血球破裂釋放 gDNA 造成「污染與稀釋」，故「血漿」較佳；選項 D 提到全血長期保存 → 想到白血球久置會破裂，必須「盡速離心」。因此一眼即可判定 C 為正確答案。

### (2) 關鍵字與知識點拆解

* **循環核酸 (cfDNA / ctDNA)** → 腫瘤細胞或正常細胞凋亡/壞死後，釋放入血流的游離 DNA。
* **Heparin (肝素)** → 強效的 **PCR 抑制劑**，分子檢驗常規禁用。
* **高分子量核酸 vs 短片段** → 完整細胞內的 DNA 為高分子量；cfDNA 因經歷細胞凋亡 (Apoptosis) 的核酸內切酶切割，呈現約 **166 bp** 及其倍數的短片段特徵。
* **血漿 (Plasma) vs 血清 (Serum)** → 血清製備過程需經歷凝血，此過程會破壞白血球，釋放大量背景 gDNA；血漿（使用 EDTA 等抗凝血劑）可保持白血球完整，背景干擾最低。
* **全血保存** → 血液離體後白血球會逐漸降解破裂，導致 gDNA 釋放。

### (3) 詳解與選項辨析

* **A 錯誤**：Heparin 會強烈抑制 Taq DNA polymerase 的活性，導致後續 PCR 或 NGS 擴增失敗。分子檢驗標準抗凝血劑應為 **EDTA** (紫頭管) 或專用游離核酸保存管 (如 Streck tube)。
* **B 錯誤**：循環核酸主要源於細胞凋亡，DNA 會被切割纏繞在核小體上，形成約 160~170 bp 的小片段。若使用「分離高分子量核酸」的技術，反而會萃取到來自白血球破裂污染的完整基因體 DNA (gDNA)，漏失真正的目標 cfDNA。因此必須使用專門針對「低分子量/短片段」最佳化的萃取套組。
* **C 正確**：**血漿 (Plasma) 是絕對首選**。若採集血清，血液在凝固的過程中會造成白血球及其他血球細胞破裂，釋放大量正常的 gDNA 進入檢體。這會大幅「稀釋」血液中原本就極微量的腫瘤循環核酸 (ctDNA)，降低檢測的靈敏度與準確性。
* **D 錯誤**：全血**不可**直接做長期保存。全血在室溫放置過久，白血球會發生凋亡與溶解，釋出大量 gDNA 污染檢體。常規 EDTA 管採血後必須在 **2~4 小時內盡速離心**分離出血漿，並將血漿冷凍於 -80°C 保存；若需長時間運送，必須使用含白血球穩定劑的特殊 cfDNA 專用管。

### (4) 核心軸線與邏輯因果 (縱向聯想)

腫瘤細胞凋亡/壞死 → 釋放短片段 DNA 進入血液 → 形成微量 ctDNA → **[採檢挑戰]** 抽血若引發凝血或白血球破裂 → 正常白血球釋放大量長片段 gDNA → 稀釋目標 ctDNA，導致**偽陰性 (靈敏度↓)** → **[對策]** 使用 **EDTA 抗凝血**取得血漿 + **盡速雙重離心**去除細胞 + 使用**短片段核酸純化技術**。

### (5) 橫向比較統整表格

| 比較維度 | 一般基因體 DNA (gDNA) 檢測 | 循環游離核酸 (cfDNA/ctDNA) 檢測 |
| --- | --- | --- |
| **標的來源** | 完整白血球細胞核 | 腫瘤/胎兒/正常細胞**凋亡**釋放至血液 |
| **最佳檢體** | **全血** (Whole blood) | **血漿** (Plasma) |
| **抗凝血劑** | **EDTA** 首選 (禁用 Heparin) | **EDTA** 或 **cfDNA 專用管** (禁用 Heparin) |
| **血清檢體適用性** | 差 (細胞已破壞，產量不穩) | **極差** (白血球破裂釋出 gDNA **嚴重干擾**) |
| **檢體前處理與保存** | 全血可直接冷凍保存 | **2-4小時內盡速離心**分離血漿，僅冷凍血漿 |
| **目標分子量** | **高分子量** (>10,000 bp) | **低分子量/短片段** (~166 bp 及其倍數) |
