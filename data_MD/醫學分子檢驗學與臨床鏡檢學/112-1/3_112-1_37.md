## 題目

37.血清白蛋白濃度 3.2 g/ dL；血清總蛋白濃度 8 g/dL；腹水白蛋白濃度 2.4 g/dL，則可推測此腹水為下列何者？

滲出液（exudate）
濾出液（transudate）
二者混合液
無法判斷

> [!success]- 正確解答
> **A**

---

## 筆記與詳解

### (1) 快速破題

看到題目給出「血清白蛋白」與「腹水白蛋白」濃度，並且要判斷腹水性質，就要立刻想到計算 **SAAG (血清腹水白蛋白梯度)**。

* **解題思路**：公式 `SAAG = 血清白蛋白濃度 - 腹水白蛋白濃度`。
* **判斷標準**：以 **1.1 g/dL** 為切點。SAAG < 1.1 為滲出液（非門脈高壓）；SAAG $\ge$ 1.1 為濾出液（門脈高壓）。
* **計算本題**：3.2 - 2.4 = **0.8 g/dL**。因為 0.8 < 1.1，故直接選 (A) 滲出液。題目給的「血清總蛋白濃度 8 g/dL」為誘導性干擾資訊。

### (2) 關鍵字與知識點拆解

* **SAAG (Serum-Ascites Albumin Gradient)** → 評估腹水形成機轉的最準確生化指標。反映血管內外滲透壓差，主要用於區分「是否為門脈高壓」引起的腹水。
* **滲出液 (Exudate)** → 微血管通透性異常增加所致。液體外觀常為混濁、化膿或血性 [🖼️](https://www.google.com/search?q=exudative+fluid+gross+appearance&tbm=isch&udm=2)。
* **濾出液 (Transudate)** → 靜脈水壓升高或血漿膠體滲透壓降低所致，微血管通透性正常。液體外觀通常為清澈、淡黃色 [🖼️](https://www.google.com/search?q=transudative+fluid+gross+appearance&tbm=isch&udm=2)。
* **血清總蛋白** → 本題陷阱。在腹水鑑別中，總蛋白的參考價值極低，臨床主流已全面採用 SAAG。

### (3) 詳解與選項辨析

* **正確 (A) 滲出液 (exudate)**：本題 SAAG 計算結果為 0.8 g/dL。當 SAAG < 1.1 g/dL 時，代表腹腔內的白蛋白濃度相對較高，與血液中的濃度差異縮小。這意味著微血管通透性增加，導致大量大分子蛋白質（如白蛋白）「滲漏」至腹腔中，為典型的滲出液特徵。
* **錯誤 (B) 濾出液 (transudate)**：若為濾出液，其 SAAG 必須 $\ge$ 1.1 g/dL。濾出液的成因通常是靜水壓將水分「擠」出，或是低蛋白血症讓水分「漏」出，大分子蛋白質無法通過微血管壁，因此腹水中的白蛋白極少，導致血清與腹水的白蛋白濃度梯度差拉大（大於 1.1）。
* **錯誤 (C) 二者混合液**：在生化檢驗分類標準上，SAAG 切點 (1.1 g/dL) 為嚴格的二分法，不存在「二者混合液」的模糊診斷選項。
* **錯誤 (D) 無法判斷**：題目提供的血清與腹水白蛋白數值已完全滿足 SAAG 的計算需求，可做出絕對判斷。

### (4) 核心軸線與邏輯因果

腹膜發炎/惡性腫瘤/感染 → 局部微血管通透性 $\uparrow$ → 血管內大分子（白蛋白）漏出至腹腔 → 腹水白蛋白濃度 $\uparrow$ → (血清白蛋白 - 腹水白蛋白) 差異縮小 → **SAAG < 1.1 g/dL** → 診斷為：**滲出液 (Exudate)**。

### (5) 統整比較表

| 比較維度 | 滲出液 (Exudate) | 濾出液 (Transudate) |
| --- | --- | --- |
| **致病機轉** | **微血管通透性 $\uparrow$** (發炎/腫瘤) | **靜水壓 $\uparrow$** 或 **膠體滲透壓 $\downarrow$** |
| **肉眼外觀** | 混濁、化膿、血性 [🖼️](https://www.google.com/search?q=exudate+fluid+gross+appearance&tbm=isch&udm=2) | 清澈、淡黃色 [🖼️](https://www.google.com/search?q=transudate+fluid+gross+appearance&tbm=isch&udm=2) |
| **SAAG (腹水專用)** | **< 1.1 g/dL** (非門脈高壓) | **$\ge$ 1.1 g/dL** (門脈高壓) |
| **Light's Criteria (肋膜專用)** | 符合任一項：<br>1. 肋膜/血清總蛋白比 **> 0.5**<br>2. 肋膜/血清 LDH 比 **> 0.6** | 上述條件皆**不符合** |
| **比重 (Specific Gravity)** | **> 1.015** | **< 1.015** |
| **蛋白質總量** | **> 3 g/dL** | **< 3 g/dL** |
| **常見疾病** | 腹膜炎、惡性腫瘤、胰臟炎、結核病 | 肝硬化、心臟衰竭、腎病症候群 |
