## 題目

39.尿液化學檢查，下列何者不受維生素 C 的干擾？

血液
葡萄糖
酮體
亞硝酸鹽

> [!success]- 正確解答
> **C**

---

## 筆記與詳解

### (1) 快速破題

看到題目說「尿液化學檢查」且問「不受維生素 C 干擾」，必須立刻想到「維生素 C 是一種強還原劑」**。
在尿液試紙檢測中，只要原理依賴**「氧化還原反應」**或**「重氮反應 (Diazo)」**的項目，都會因為維生素 C 競爭性消耗試劑而產生**偽陰性 (False Negative)。選項中，血液與葡萄糖依賴氧化還原，亞硝酸鹽依賴重氮反應，皆會受嚴重干擾。唯獨「酮體」是使用硝普鈉反應，與還原力無關，因此完全不受影響，答案即為 C。

### (2) 關鍵字與知識點

* **維生素 C (Ascorbic acid)：** 知識點 → **強還原劑**。干擾過氧化物酶催化之氧化反應及重氮鹽偶合反應，造成化學呈色遭阻斷（結果呈現**偽陰性 ↓**）。
* **血液 (Blood)：** 知識點 → 利用血紅素的**過氧化物酶活性 (Peroxidase-like activity)**。
* **葡萄糖 (Glucose)：** 知識點 → 利用**葡萄糖氧化酶 (Glucose oxidase)** 產生 H₂O₂ 進行氧化呈色。
* **亞硝酸鹽 (Nitrite)：** 知識點 → 利用**格里斯反應 (Greiss reaction / 重氮反應)** 檢測細菌還原硝酸鹽之能力。
* **酮體 (Ketones)：** 知識點 → 利用**亞硝基鐵氰化鈉 (Sodium nitroprusside)** 在鹼性下檢測乙醯乙酸 (Acetoacetic acid)。

### (3) 詳解

* **(A) 血液：錯誤（會受干擾）。**
試紙條原理為紅血球中的血紅素具備過氧化物酶 (Peroxidase) 活性，可將試劑中的過氧化氫還原，進而使呈色劑氧化變色。當尿液中存在高濃度維生素 C（強還原劑）時，會搶先還原呈色劑或過氧化氫，造成**偽陰性**。若臨床上出現肉眼可見的血尿表徵 [🖼️](https://www.google.com/search?q=%22Gross+hematuria%22+clinical+sign&tbm=isch&udm=2)，或尿沉渣鏡檢發現大量紅血球 [🖼️](https://www.google.com/search?q=%22RBC%22+urine+sediment+microscopy&tbm=isch&udm=2)，但試紙卻呈陰性，極可能是維生素 C 造成。
* **(B) 葡萄糖：錯誤（會受干擾）。**
試紙採用「葡萄糖氧化酶/過氧化物酶」雙酵素法。同理，第二階段的氧化呈色步驟會被維生素 C 的強還原力截斷，導致糖尿檢測出現**偽陰性**。
* **(D) 亞硝酸鹽：錯誤（會受干擾）。**
試紙採用格里斯反應，本質上是一種重氮化偶合反應 (Diazo coupling)。維生素 C 會與試劑中的重氮鹽發生反應，阻止其與呈色劑結合，導致泌尿道感染 (UTI) 的檢測出現**偽陰性**（即便尿沉渣塗片已可見明顯細菌群 [🖼️](https://www.google.com/search?q=%22Bacteriuria%22+urine+sediment+Gram+stain&tbm=isch&udm=2)）。
* **(C) 酮體：正確（不受干擾）。**
酮體測試 (Legal's test) 的原理是利用亞硝基鐵氰化鈉與尿中的乙醯乙酸在鹼性環境下結合，生成紫色複合物。此反應機制**非**氧化還原反應，亦**非**重氮反應，故完全**不受**維生素 C 的還原性干擾。

### (4) 核心軸線與表格統整

**邏輯因果（縱向聯想）：**
病患攝取高劑量 Vit C → 尿液排泄高濃度還原劑 (Ascorbic acid) → 競爭性消耗尿液試紙上的**氧化劑 (如 H₂O₂)** 或**重氮鹽** → 阻斷「氧化呈色」與「重氮偶合呈色」→ 導致依賴該原理的項目全數呈現**偽陰性** → 臨床出現「鏡檢有異常，試紙卻全陰性」的醫檢矛盾。

**記憶口訣：**
維生素 C 造成的干擾項目：「**白血病、糖尿、亞膽**」（白血球、血液、葡萄糖、亞硝酸鹽、膽紅素），全部都會被干擾成**偽陰性**。
