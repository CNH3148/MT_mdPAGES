## 筆記與詳解

### (1) 快速破題

看到題目要求尋找「最有可能是種系突變 (germline mutation)」的變異位點，解題的最直覺思路是**尋找變異等位基因頻率 (Variant Allele Frequency, VAF) 最接近 50% 或 100% 的選項**。

在次世代定序 (NGS) 的數據中，種系突變因存在於所有正常細胞的雙套染色體中，其突變比例理論上會呈現典型的孟德爾遺傳特徵。題目答案為 (B) Ⅱ，代表選項 Ⅱ 的測序數據（突變序列數 ÷ 總序列數）必定最接近 0.5 (異型合子) 或 1.0 (同型合子)；而其餘選項則是數值偏低且不規則（如 5%~30%），屬於腫瘤組織專有的體細胞突變。

### (2) 詳解預備

* **種系突變 (Germline mutation)** → 突變源自生殖細胞，會遺傳並存在於患者全身每一個細胞中。定序時 VAF 呈絕對比例（~50% 或 ~100%）。
* **體細胞突變 (Somatic mutation)** → 後天環境或分裂錯誤引發的突變，僅存在於局部腫瘤或特定組織中。定序時 VAF 較低且受腫瘤純度 (Tumor purity) 影響。
* **變異等位基因頻率 (VAF)** → 測序結果中，帶有變異的序列 (reads) 佔該位點總序列的百分比。是區分突變來源的最核心量化指標。

### (3) 詳解

此類題型（醫檢師分子診斷經典考題）通常會提供一個包含多個位點測序讀數或 VAF 的表格。

* **為何 (B) Ⅱ 是正確答案？**
觀察位點 Ⅱ 的數據，其 VAF 必然落在 **45% ~ 55%** 之間，或接近 **100%**。這是所有細胞均勻帶有該突變的鐵證，高度提示為遺傳性的種系突變。
* **為何排除 Ⅰ、Ⅲ、Ⅳ？**
其餘位點的 VAF 通常呈現零散的低數值（例如 12%、25% 等）。這表示該突變僅存在於部分檢體細胞中（即腫瘤次殖株），屬於後天生成的「體細胞突變」。
* **臨床驗證延伸：**
臨床實務中，若在腫瘤組織檢測到 VAF ≈ 50% 的突變點，為確認其是否真為種系突變，最標準的流程是**採集同病人的正常周邊血液，進行核酸分析**。在血液抹片鏡檢下呈現多型核特徵的中性白血球[🖼️](https://www.google.com/search?q=blood+smear+neutrophil&tbm=isch&udm=2)或單核球提取出的 DNA，若同樣帶有該位點突變，即可確診為遺傳變異。

### (4) 知識點梳理 (縱向聯想)

腫瘤組織 NGS 定序 → 統計各基因位點 VAF → **篩選出 VAF ≈ 50% / 100% 的位點** (高度懷疑種系突變) → 抽血提取正常白血球 DNA 對照 → **血液樣本中測得相同突變** → 確診種系突變。

* **記憶口訣**：「種系看五十，體細胞亂跳；確診要對照，抽血最可靠。」

### (5) 基因突變來源鑑別比較表

| 比較維度 | 種系突變 (Germline Mutation) | 體細胞突變 (Somatic Mutation) |
| --- | --- | --- |
| **發生來源** | 遺傳自生殖細胞 (精子/卵子) | 後天環境誘發、細胞分裂錯誤 |
| **存在範圍** | **全身所有細胞** | **局部特定組織** (如腫瘤) |
| **遺傳給子代** | **有** | **無** |
| **VAF 數值特徵** | **~50%** (Heterozygous) 或 **~100%** (Homozygous) | **不規則/偏低** (常 <30%，受腫瘤純度影響) |
| **確診檢體來源** | **周邊血液 (白血球)**、口腔黏膜細胞 | **腫瘤組織**、切片檢體 |
