## 題目

53.承上題，臨床檢驗實務上，如果要更進一步確認該變異位點確實為種系突變，下列何種驗證最有效？

重複進行次世代定序檢驗，確認再現性
將樣本進行巨觀刮取（macro-dissection），提高腫瘤含量至90%以上，再次檢驗
使用第三平台檢驗，例如即時定量聚合酶連鎖反應進行變異位點確認
採取同病人血球 DNA，進行該位點的變異分析

> [!success]- 正確解答
> **D**

---

## 筆記與詳解

### (1) 快速破題

看到題目要求確認是否為「**種系突變 (Germline mutation)**」→ 核心思路應立刻連結到「種系突變存在於**全身所有正常細胞**，而不僅限於腫瘤細胞」。因此，解題關鍵就是「尋找**非腫瘤的正常組織**來作為比對」。選項中只有 (D) 採取同病人的血球 (代表正常細胞) 符合此生物學邏輯。

### (2) 詳解預備

* **種系突變 (Germline mutation) → 遺傳/先天發生**：突變源自生殖細胞，因此人體全身每一個細胞 (包含腫瘤與正常細胞) 皆帶有此突變。
* **體細胞突變 (Somatic mutation) → 後天發生**：突變僅發生在特定組織或器官的局部細胞中 (如腫瘤細胞)，正常組織不會有。
* **巨觀刮取 (Macro-dissection) → 提高腫瘤純度**：在病理玻片上觀察實體腫瘤組織的巨觀形態 [🖼️](https://www.google.com/search?q=tumor+macrodissection+FFPE&tbm=isch&udm=2)，並手動將非腫瘤的正常組織刮除。這會**富集腫瘤細胞**，通常用於尋找微量的體細胞突變。
* **再現性 / 第三平台檢驗 → 分析確效 (Analytical validation)**：用以確認「檢驗技術本身沒有出錯」(排除偽陽性)，證明檢體中確實有該突變，但無法告訴你這個突變是天生還是後天長出來的。

### (3) 詳解

* **(A) 錯誤**：重複進行 NGS 只能驗證「分析準確度 (再現性)」，證明檢體內確實有該變異，並非機器判讀錯誤或隨機雜訊。但若原檢體是腫瘤切片，重複定序依然無法確認該突變是否也存在於正常細胞中。
* **(B) 錯誤**：巨觀刮取的目的是盡可能排除正常細胞，**提高腫瘤含量**。這對於尋找低頻率的「體細胞突變」很有幫助。但若要證明是「種系突變」，我們反而極度需要正常細胞來證明「正常細胞裡也有突變」，因此刮除正常組織完全是背道而馳的做法。
* **(C) 錯誤**：使用 qPCR 或 Sanger 定序等第三平台稱為「正交驗證 (Orthogonal testing)」。與 (A) 選項相同，其目的是排除單一檢驗平台 (如 NGS) 的系統性誤差，證明突變真實存在，但無法鑑別該突變是 Somatic 還是 Germline。
* **(D) 正確**：這是臨床上鑑別的黃金標準。直接抽取病人的周邊血，利用血液抹片中呈現正常形態的白血球 [🖼️](https://www.google.com/search?q=peripheral+blood+smear+normal+leukocytes&tbm=isch&udm=2) 所萃取出的 DNA 進行檢驗。因為白血球代表「非腫瘤的正常組織」(假設病人非血癌)，若在此能測到同一突變，即可鐵證其為先天的「種系突變」。

### (4) 核心軸線與橫向比較

**核心邏輯軸線**：腫瘤組織檢出突變 → 懷疑為遺傳性癌症基因 (如 BRCA1/2) → 抽取周邊血 (正常組織) 進行定序比對 → 若血液檢出相同突變 → 確診**種系突變** → 啟動家族遺傳諮詢與標靶藥物 (如 PARP inhibitor) 評估。

**【記憶口訣】**：「種系全身有，抽血就能求；體細胞長瘤，切片找源頭。」

| 比較維度 | 種系突變 (Germline Mutation) | 體細胞突變 (Somatic Mutation) |
| --- | --- | --- |
| **突變來源** | **遺傳** (源於精卵或受精卵早期) | **後天獲得** (環境、老化、複製錯誤) |
| **存在細胞** | **全身所有細胞** (含腫瘤與正常細胞) | **僅限腫瘤/病變細胞** |
| **確診最佳檢體** | **周邊血白血球**、口腔黏膜細胞、唾液 | **腫瘤組織切片**、細胞游離DNA (ctDNA) |
| **變異對偶基因頻率 (VAF)** | 接近 **50%** (異型合子) 或 **100%** (同型合子) | 變異大 (通常 **<50%**，取決於腫瘤純度與異質性) |
| **檢體處理重點** | 無需富集腫瘤，只需一般 DNA 萃取 | 需透過 **巨觀/微觀刮取 (Microdissection)** 提高腫瘤純度 |
| **臨床意義** | 具**家族遺傳性** (需遺傳諮詢)、標靶用藥 | 無遺傳性、多用於評估標靶用藥或預後 |
