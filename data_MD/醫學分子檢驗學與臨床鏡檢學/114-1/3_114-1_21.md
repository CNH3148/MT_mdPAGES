## 題目

21.下列那一項疾病與滲出液（exudates）的形成較無關？

淋巴瘤
結核病
腎病症候群
病毒感染

> [!success]- 正確解答
> **C**

---

## 筆記與詳解

### (1) 解題思路

看到題目要求找出與「滲出液（exudates）」形成較**無關**的疾病，就必須立刻在腦中切換體液積水的兩大分類：「滲出液（Exudate）」與「漏出液（Transudate）」。
只要是**發炎、感染、腫瘤**，都會導致局部血管通透性增加，產生「滲出液」；而**系統性壓力失衡**（如心臟沒力導致靜水壓上升，或肝臟/腎臟病導致血清蛋白流失使膠體滲透壓下降），則會單純把水分擠出血管，產生「漏出液」。選項中只有 (C) 腎病症候群 是蛋白質流失造成的壓力失衡，屬於漏出液，因此為答案。

### (2) 關鍵字與知識點拆解

* **Exudates (滲出液)** → 局部微血管通透性增加（發炎、腫瘤、感染引起），富含蛋白質與細胞，外觀常呈現混濁或膿狀 [🖼️](https://www.google.com/search?q="purulent+exudate"+body+fluid&tbm=isch&udm=2)。
* **Transudates (漏出液)** → 系統性流體動力學改變（靜水壓 ↑ 或 膠體滲透壓 ↓），不涉及發炎，外觀通常清澈透明 [🖼️](https://www.google.com/search?q="clear+transudate"+body+fluid&tbm=isch&udm=2)。
* **腎病症候群 (Nephrotic syndrome)** → 腎絲球過濾膜受損 → 大量白蛋白自尿液流失 → 血清白蛋白下降（低蛋白血症）→ 血管內膠體滲透壓 ↓ → 組織嚴重水腫（如：凹陷性水腫 [🖼️](https://www.google.com/search?q="pitting+edema"+clinical+sign&tbm=isch&udm=2)）與漏出液積水。
* **淋巴瘤 (Lymphoma)** → 惡性腫瘤侵犯胸膜/腹膜或阻塞淋巴回流，引發局部發炎與血管通透性改變。
* **結核病 / 病毒感染** → 病原體引發免疫細胞聚集與釋放細胞激素（如組織胺），直接導致血管通透性大幅上升。

### (3) 詳解與選項辨析

* **(A) 淋巴瘤 (錯誤，有關)：** 惡性腫瘤細胞會分泌血管內皮生長因子 (VEGF) 等發炎介質，或實質破壞組織結構，導致微血管通透性增加，形成蛋白質與細胞含量極高的滲出液（有時會因淋巴阻塞而形成乳糜液）。
* **(B) 結核病 (錯誤，有關)：** 結核分枝桿菌會引發強烈的遲發型過敏反應（肉芽腫性發炎），釋放大量發炎物質，使血管通透性上升，形成富含淋巴球的滲出液。
* **(C) 腎病症候群 (正確，較無關)：** 此疾病的本質是「流失蛋白質」，導致血液無法「抓住」水分（膠體滲透壓下降）。過程中沒有局部的發炎反應，也沒有微血管通透性（孔隙）本身的擴張，水分是單純漏出血管外，形成**漏出液**，故與滲出液形成無關。
* **(D) 病毒感染 (錯誤，有關)：** 微生物感染必定引起局部急性或慢性免疫反應，發炎介質導致血管擴張與通透性增加，使血漿蛋白與白血球大量滲出，形成典型的滲出液。

### (4) 核心軸線與邏輯因果

* **滲出液成因軸線：** 局部發炎/感染/腫瘤 → 血管內皮細胞收縮或受損 → 通透性 **↑** → 大分子蛋白與白血球滲漏 → 體液指標：**Total Protein ↑, LDH ↑** → 臨床判定：**滲出液 (Exudate)**
* **漏出液成因軸線：** 心肝腎系統性疾病 → 靜水壓 **↑** 或 膠體滲透壓 **↓** → 血管孔隙正常，但水分因物理壓力差流失 → 體液指標：**Total Protein ↓, LDH ↓** → 臨床判定：**漏出液 (Transudate)**

> 💡 **記憶口訣：**
> * **「滲」**出液：**「深」**入發炎/腫瘤（有實質病灶、外觀深/混濁、蛋白質高）。
> * **「漏」**出液：單純**「漏」**水（無局部發炎，因心肝腎壓力失衡，清澈如水）。
>

### (5) 體液積水鑑別重點比較表 (Light's Criteria)

| 比較維度 | 漏出液 (Transudate) | 滲出液 (Exudate) |
| --- | --- | --- |
| **致病機轉** | 靜水壓 **↑** 或 膠體滲透壓 **↓** (系統性) | 微血管通透性 **↑** 或 淋巴阻塞 (局部性) |
| **常見疾病** | 鬱血性心衰竭、肝硬化、**腎病症候群** | **感染** (結核/病毒/細菌)、**腫瘤**、自體免疫 |
| **外觀特徵** | 清澈透亮、淡黃色 [🖼️](https://www.google.com/search?q="clear+transudate"+body+fluid&tbm=isch&udm=2) | **混濁**、黃綠/血色/乳糜狀 [🖼️](https://www.google.com/search?q="cloudy+exudate"+body+fluid&tbm=isch&udm=2) |
| **自發性凝固** | **否** (缺乏纖維蛋白原) | **是** (富含纖維蛋白原) |
| **比重 (SG)** | **< 1.015** | **> 1.015** |
| **白血球數 (WBC)** | **< 1000** /µL (主要為單核球) | **> 1000** /µL (常為**嗜中性球**或淋巴球) |
| **總蛋白質 (TP)** | **< 3.0** g/dL | **> 3.0** g/dL |
| **體液 / 血清 TP 比值** | **< 0.5** | **> 0.5** |
| **體液 / 血清 LDH 比值** | **< 0.6** | **> 0.6** |

*(註：表格中最後兩項 TP 與 LDH 比值即為臨床上極具鑑別度的 **Light's criteria**，只要符合任一項「>」之標準，即可判定為滲出液。)*
