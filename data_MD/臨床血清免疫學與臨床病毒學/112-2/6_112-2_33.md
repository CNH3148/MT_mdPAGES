## 題目

人類的主要組織相容性複合體I(MHC class I)在染色體上的排列順序為何?

A.A、B、C
B.B、C、A
C.C、A、B
D.C、B、A

> [!success]- 正確解答
> **B**

---

## 筆記與詳解

### (1) 快速破題

看到題目問 MHC class I 的染色體基因排列順序，最快的方式是記住**染色體由著絲點向端粒延伸**的大方向：

* 大區塊順序：**Class II** $\rightarrow$ **Class III** $\rightarrow$ **Class I**。
* 在最靠近外側（端粒端）的 Class I 區間中，其古典三個基因的線性物理順序是 **B、C、A**（一般人常直覺背成 A、B、C，此處為經典的「反直覺」考點）。
* 故本題答案直接選 **(B) B、C、A**。

---

### (2) 關鍵字與知識點

* **MHC (主要組織相容性複合體) / HLA (人類白血球抗原)**：人類的 MHC 基因群位於**第 6 號染色體短臂 (6p21.3)** 上 [🖼️](https://www.google.com/search?q="HLA+locus+map"+chromosome+6&tbm=isch&udm=2)。
* **MHC Class I 古典基因**：包括 **HLA-A、HLA-B、HLA-C**，負責編碼與第 15 號染色體產物 $\beta_2\text{-microglobulin}$ ($\beta_2\text{m}$) 結合的重鏈 [🖼️](https://www.google.com/search?q="MHC+class+I+structure"+ribbon+diagram&tbm=isch&udm=2)。
* **染色體物理排列向性**：沿著染色體由內側「著絲點（Centromere）」朝向外側「端粒（Telomere）」延伸。

---

### (3) 選項詳解與辨析

* **(B) B、C、A 為正確選項**：
* 沿著第 6 號染色體短臂，由著絲點往端粒（近 $\rightarrow$ 遠）的方向，MHC 基因區段依序為 Class II $\rightarrow$ Class III $\rightarrow$ Class I。
* 在最外側的 Class I 區域中，基因從最內側（靠近 Class III）到最外側（靠近端粒端）的實際物理線性排列順序為：**HLA-B $\rightarrow$ HLA-C $\rightarrow$ HLA-A**。這意味著 HLA-B 離 Class III 最近，而 HLA-A 最靠近端粒端。


* **(A) A、B、C / (C) C、A、B / (D) C、B、A 均錯誤**：
* 這些選項皆非實際物理空間上的排列順序，僅為干擾考生直覺背誦習慣（如：口語習慣講 A、B、C）的錯誤排列。



---

### (4) 邏輯與橫向比較

**【縱向邏輯因果】**
第 6 號染色體短臂（6p21.3）定位 $\rightarrow$ 由內向外排列（Class II $\rightarrow$ III $\rightarrow$ I） $\rightarrow$ Class I 內排列（**B $\rightarrow$ C $\rightarrow$ A**） $\rightarrow$ 表現對應 HLA 重鏈 $\rightarrow$ 呈遞內源性抗原給 **CD8+ T 細胞** [🖼️](https://www.google.com/search?q="MHC+class+I"+TCR+interaction&tbm=isch&udm=2)。

**【橫向對比表格】**

| 比較維度 | **MHC Class I (古典)** | **MHC Class II** | **MHC Class III (非抗原呈遞)** |
| --- | --- | --- | --- |
| **主要基因座** | **HLA-A、HLA-B、HLA-C** | **HLA-DP、HLA-DQ、HLA-DR** | **C2、C4、Bf (補體)、TNF (腫瘤壞死因子)** |
| **染色體物理順序**<br>(由著絲點 $\rightarrow$ 端粒) | **最靠近端粒端 (B $\rightarrow$ C $\rightarrow$ A)** | **最靠近著絲點端 (DP $\rightarrow$ DQ $\rightarrow$ DR)** | **夾在 Class I 與 Class II 之間** |
| **細胞表現分佈** | **所有具核細胞** (紅血球、滋養層除外) | **專職抗原呈現細胞 (APC)** (B細胞、巨噬細胞、樹突細胞) | 分泌型蛋白、無細胞膜表面限制 |
| **分子結構** | **重鏈 $\alpha$ 鏈** + **$\beta_2\text{-microglobulin}$** (15號染色體編碼) | **$\alpha$ 鏈** + **$\beta$ 鏈** (皆由 6 號染色體編碼) | 多樣化可溶性蛋白 (補體、細胞激素、HSP) |
| **主要呈遞對象/功能** | **CD8+ T 細胞** / 呈遞內源性抗原 | **CD4+ T 細胞** / 呈遞外源性抗原 | **補體系統活化**、發炎反應、免疫調節 |

**【記憶口訣】**

* **MHC Class I 順序**：**「B、C、A」** $\rightarrow$ 想像買車由入門等級 **B** 級、**C** 級，一路升級到頂規 **A** 級（由內到外）。
* **MHC Class II 順序**：**「DP、DQ、DR」** $\rightarrow$ 直接依英文字母 P $\rightarrow$ Q $\rightarrow$ R 的順序物理排列（由內到外）。
