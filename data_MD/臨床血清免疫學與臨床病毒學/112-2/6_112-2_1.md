## 題目

為逃避CD8 T細胞的毒殺,巨細胞病毒(cytomegalovirus)的蛋白會結合下列何種分子以阻礙 MHC class I的組成?

A.al domain
B.a2 domain
C.a3 domain
D.β2-microglobulin

> [!success]- 正確解答
> **D**

---

## 筆記與詳解

### (1) 快速破題

看到題目問「巨細胞病毒 (CMV) 阻礙 MHC class I 組成」→ 就要想到 MHC class I 的結構包含一條重鏈（具有 $\alpha_1$, $\alpha_2$, $\alpha_3$ 三個結構域）與一條輕鏈（$\beta_2$-microglobulin, $\beta_2\text{m}$）。若要全面且高效地阻礙整個 MHC class I 分子的正確折疊與組裝，病毒蛋白最常鎖定的目標就是非共價結合的 **$\beta_2$-microglobulin**（或者是與其結合的胜肽結合槽），導致重鏈無法穩定並被降解，進而逃避 CD8+ T 細胞的毒殺。

---

### (2) 專有名詞與關鍵字解析

* **巨細胞病毒 (Cytomegalovirus, CMV)** → 具備強大的免疫逃脫機制（Immune evasion），能製造多種蛋白質（如 US2, US3, US11 等）干擾宿主的 MHC class I 表現。
* **MHC class I** → 表現於所有具核細胞表面，負責向 CD8+ 胞毒性 T 細胞（CTL）呈現內源性抗原。
* **$\beta_2$-microglobulin ($\beta_2\text{m}$)** → MHC class I 的非多型性輕鏈，為 MHC class I 分子能正確折疊並轉運至細胞表面所必需。

---

### (3) 選項辨析與詳解

* **A, B, C. $\alpha_1, \alpha_2, \alpha_3$ domain**：
* $\alpha_1$ 與 $\alpha_2$ 共同組成 MHC class I 的抗原結合槽（Peptide-binding cleft）。
* $\alpha_3$ 則是與 CD8+ T 細胞的 CD8 共受體結合的區域。
* 雖然 CMV 的某些蛋白（如 US2 和 US11）會結合新合成的 MHC class I 重鏈（胞質側或管腔側），促使其易位回細胞質並透過蛋白酶體降解，但在「阻礙整個分子基本組成/組裝」的機制中，結合或破壞 $\beta_2\text{m}$（如部分病毒蛋白會競爭結合位、或直接阻止 $\beta_2\text{m}$ 與重鏈結合）是導致 MHC class I 結構無法完整建構的核心關鍵。


* **D. $\beta_2$-microglobulin ($\beta_2\text{m}$)**【正確答案】：
* CMV 的特定免疫逃避蛋白（例如 **US3**）會在內質網（ER）中滯留 MHC class I 重鏈，而其他蛋白則會直接干擾重鏈與 **$\beta_2\text{m}$** 的組裝。沒有 $\beta_2\text{m}$ 的結合，$\alpha$ 重鏈就無法形成穩定的三度空間結構，進而無法運送到細胞表面。因此，阻礙 $\beta_2\text{m}$ 的參與是 CMV 瓦解 MHC class I 呈現途徑的經典手段。



---

### (4) 邏輯因果與橫向比較

#### 縱向聯想

CMV 侵入細胞 → 製造免疫逃逸蛋白（如 US 家族） → 結合並干擾 **$\beta_2\text{m}$** 與重鏈組裝 → MHC class I 無法正確折疊 → 細胞表面 MHC class I 表現量 **↓** → CD8+ T 細胞無法辨識辨認 → 病毒成功逃脫毒殺。

> 💡 **記憶口訣**：
> 「巨細胞（CMV）太狡猾，**阻礙 $\beta_2$（$\beta_2\text{m}$）** 讓 MHC I 散了架，T 細胞看它沒辦法。」
