## 題目

下列何者與類風濕性關節炎相關性最大?

A.HLA-B8
B.HLA-B27
C.HLA-DR3
D.HLA-DR4

> [!success]- 正確解答
> **D**

---

## 筆記與詳解

### (1) 快速破題

看到題目問「類風濕性關節炎 (Rheumatoid Arthritis, RA)」與哪種 HLA (人類白血球抗原) 基因型相關，就要直覺聯想「**HLA-DR4**」。在自體免疫疾病的遺傳考點中，最常考的經典配對就是「僵直性脊椎炎配 B27」、「類風濕性關節炎配 DR4」、「SLE 配 DR2/DR3」。既然題目問 RA，答案即為 D。

### (2) 關鍵字列表與知識點

* **類風濕性關節炎 (RA)** → 系統性自體免疫疾病，主要攻擊關節滑液膜，導致天鵝頸變形 (swan neck deformity)[🖼️](https://www.google.com/search?q=%22swan+neck+deformity%22+clinical+sign&tbm=isch&udm=2) 或出現類風濕結節 (rheumatoid nodule)[🖼️](https://www.google.com/search?q=%22rheumatoid+nodule%22+clinical+sign&tbm=isch&udm=2)；專一性診斷抗體為 Anti-CCP。
* **HLA-B8** → MHC Class I，主要與重症肌無力 (Myasthenia gravis)、葛瑞夫茲氏症 (Graves' disease) 相關。
* **HLA-B27** → MHC Class I，國考必考之「僵直性脊椎炎 (Ankylosing spondylitis)」標誌。
* **HLA-DR3** → MHC Class II，常與全身性紅斑狼瘡 (SLE)、乾燥症 (Sjögren's syndrome)、第1型糖尿病相關。
* **HLA-DR4** → MHC Class II，高度專一於類風濕性關節炎 (RA)、第1型糖尿病。

### (3) 詳解

* **(A) HLA-B8 錯誤**：主要與重症肌無力 (MG) 或乳糜瀉相關，與 RA 無關。
* **(B) HLA-B27 錯誤**：看到 B27 必須立刻聯想血清陰性脊椎關節病變 (Seronegative spondyloarthropathies)，最經典為**僵直性脊椎炎**。其影像學特徵為竹節狀脊椎 (bamboo spine)[🖼️](https://www.google.com/search?q=%22bamboo+spine%22+X-ray&tbm=isch&udm=2)。
* **(C) HLA-DR3 錯誤**：DR3 與全身性紅斑狼瘡 (SLE) 較為相關，SLE 的經典視覺表徵為臉部蝴蝶斑 (malar rash)[🖼️](https://www.google.com/search?q=%22malar+rash%22+clinical+sign&tbm=isch&udm=2)。
* **(D) HLA-DR4 正確**：RA 的發生與 HLA-DR4 (及 HLA-DR1) 基因型具強烈相關性，這是臨床上判斷該疾病遺傳易感性最重要的標記。

### (4) 核心軸線與邏輯因果

* **致病機轉軸線**：
遺傳易感性 (**HLA-DR4**) ＋ 環境因子 (如抽菸) → 免疫耐受性打破 (出現瓜胺酸化蛋白質) → 活化 CD4+ T 細胞與 B 細胞 → 產生自體抗體 (**RF, Anti-CCP**) → 免疫複合物沉積與發炎細胞浸潤關節滑液膜 (Synovium) → 形成**血管翳 (Pannus)** → 軟骨與骨骼破壞 → 關節不可逆畸形。
* **記憶口訣**：
* **DR4 治 RA** (Doctor for RA)：DR4 → RA。
* **27 僵直**：B27 → 僵直性脊椎炎。
* **3 狼 4 糖**：DR3 → 紅斑性狼瘡 (SLE)；DR3 & DR4 皆與第1型糖尿病 (T1DM) 相關。
* **B8 無力**：B8 → 重症肌無力。



### (5) 常見自體免疫疾病與 HLA 基因型比較矩陣

| HLA 基因型 | 所屬 MHC 類別 | 主要相關自體免疫疾病 | 關鍵診斷抗體 / 檢驗指標 | 典型臨床或影像表徵 |
| --- | --- | --- | --- | --- |
| **HLA-DR4** | Class II | **類風濕性關節炎 (RA)**、第1型糖尿病 (T1DM) | **Anti-CCP** (專一性最高)、RF | 天鵝頸變形、關節對稱性發炎、血管翳 (Pannus) |
| **HLA-DR3** | Class II | **全身性紅斑狼瘡 (SLE)**、乾燥症、T1DM | **Anti-dsDNA**、**Anti-Sm** | 蝴蝶斑 (malar rash)、圓盤狀紅斑 |
| **HLA-DR2** | Class II | **多發性硬化症 (MS)**、SLE、肺出血腎炎綜合症 (Goodpasture's) | CSF **Oligoclonal bands** (MS) | 神經髓鞘脫失、肺出血＋血尿 (Goodpasture's) |
| **HLA-B27** | Class I | **僵直性脊椎炎**、反應性關節炎 (Reiter's) | HLA-B27 基因檢測 (RF 為**陰性**) | 竹節狀脊椎 (bamboo spine)、葡萄膜炎 |
| **HLA-B8** | Class I | **重症肌無力 (MG)**、葛瑞夫茲氏症、乳糜瀉 | **Anti-AChR** (MG)、Anti-tTG (乳糜瀉) | 眼瞼下垂 (MG)、甲狀腺腫大伴眼球突出 |
