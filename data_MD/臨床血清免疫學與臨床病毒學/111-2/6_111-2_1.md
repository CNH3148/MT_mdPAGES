## 題目

抗原呈獻細胞的胞內體(endosome)中,下列何種分子會先占據MHC class II的胜肽結合位,之後HLA-DM再媒介細胞外抗原與此分子交換結合位?

A.TAP (transporter associated with antigen processing)
B.Calreticulin
C.CLIP (class II-associated invariant chain peptide)
D.HLA-DO

> [!success]- 正確解答
> **C**

---

## 筆記與詳解

### (1) 快速破題

看到題目問「**MHC class II**」、「**胞內體 (endosome)**」中「**先佔據胜肽結合位**」並在之後與細胞外抗原交換的分子，就要立刻反射聯想到 **CLIP (class II-associated invariant chain peptide)**。

* **解題直覺**：MHC class II 在內質網剛合成時，為了防止在裡面就誤綁到細胞本身的內源性胜肽，會被一個防護罩 **Invariant chain (Ii)** 擋住結合槽。當它被運送到胞內體（MHC class II 途徑 [🖼️](https://www.google.com/search?q=MHC+class+II+antigen+presentation+pathway&tbm=isch&udm=2)）準備與外源抗原結合時，Ii 會被酵素剪碎，最後只留下一小段還塞在結合槽裡，這段「殘留的防護罩碎片」就是 **CLIP**。隨後，**HLA-DM** 會像鑰匙一樣把 CLIP 踢掉，換上外來的抗原胜肽。因此本題答案直接選 **C**。

---

### (2) 專有名詞與關鍵字解析

* **MHC class II** → 專門呈現**外源性抗原**（如細菌、胞外寄生蟲）給 **CD4+ T 細胞**的膜蛋白。
* **Invariant chain (Ii / CD74)** → 伴護蛋白，在內質網中與 MHC class II 結合，負責**阻擋結合槽**並導引其走向胞內體。
* **CLIP** → Invariant chain 被降解後的**殘留片段**，直接塞在 MHC class II 的胜肽結合槽中。
* **HLA-DM** → 胞內體中的催化分子，負責**踢走 CLIP**，協助高親和力外源抗原裝載至 MHC class II。
* **HLA-DO** → HLA-DM 的**負調控因子**，會與 HLA-DM 結合並抑制其活性。

---

### (3) 詳解

* **A. TAP (transporter associated with antigen processing)**
* *錯誤*。TAP 屬於 **MHC class I 途徑**。它位於內質網膜上，負責將細胞質中被蛋白酶體降解的內源性胜肽「主動運輸」送進內質網腔，供 MHC class I 結合，與 MHC class II 胞內體無關。


* **B. Calreticulin (鈣網蛋白)**
* *錯誤*。Calreticulin 是 **MHC class I 途徑**在內質網中的伴護蛋白（chaperone），負責協助未成熟的 MHC class I 重鏈與 $\beta_2$-microglobulin 摺疊並維持穩定。


* **C. CLIP (class II-associated invariant chain peptide)**
* *正確*。當 Invariant chain (Ii) 在胞內體被 Cathepsin 等蛋白酶降解後，僅剩一小段胜肽（即 CLIP）依然結合在 MHC class II 的胜肽結合槽中，直到 HLA-DM 介入將其交換為外源抗原。


* **D. HLA-DO**
* *錯誤*。HLA-DO 雖然存在於胞內體，但它是用來結合並**抑制 HLA-DM** 活性的調節蛋白，並不直接佔據 MHC class II 的胜肽結合位。



---

### (4) 核心軸線與橫向比較

#### 縱向因果邏輯（MHC class II 裝載路徑）

> 內質網合成 MHC class II ＋ **Ii** 結合（防止誤用） $\rightarrow$ 輸送至胞內體（Endosome） $\rightarrow$ 酵素降解 Ii，僅剩 **CLIP** 擋在結合槽 $\rightarrow$ **HLA-DM** 催化將 CLIP 移除並裝載外源抗原 $\rightarrow$ 送至細胞表面呈遞給 CD4+ T 細胞。

#### 記憶口訣

> * **「一」路 TAP 運（MHC class I 仰賴 TAP 運輸內源抗原）**
> * **「二」姑 CLIP 擋（MHC class II 依靠 CLIP 阻擋結合位，再由 DM 踢走）**
> 
> 

#### 橫向比較：MHC 兩大抗原呈現途徑

| 比較維度 | MHC Class I 途徑 | MHC Class II 途徑 |
| --- | --- | --- |
| **抗原來源** | **內源性**抗原 (如病毒、腫瘤蛋白) | **外源性**抗原 (如吞噬之細菌) |
| **抗原降解部位** | 細胞質中的**蛋白酶體** (Proteasome) | 胞內體/溶酶體**酸性水解酶** |
| **關鍵轉運/伴護蛋白** | **TAP**、Calnexin、**Calreticulin**、ERp57 | Invariant chain (**Ii** / CD74) |
| **胜肽裝載位置** | **內質網腔** (ER lumen) | **胞內體/溶酶體** (MIIC / Endosome) |
| **槽口暫時阻擋物** | **無** (由伴護蛋白維持槽口敞開) | **CLIP** (由 Ii 降解殘留而來) |
| **胜肽交換/催化分子** | **Tapasin** (協助篩選高親和力胜肽) | **HLA-DM** (踢除 CLIP，裝載抗原) |
| **調節/抑制分子** | 無特定同源抑制分子 | **HLA-DO** (與 HLA-DM 結合並抑制之) |
| **呈遞與辨識對象** | **CD8+** T 細胞 (胞毒性 T 細胞) | **CD4+** T 細胞 (輔助性 T 細胞) |
