## 題目

下列有關過敏原皮膚試驗中,以過敏原刺激個體所誘發過敏反應之敘述,何者錯誤?

A.立即性反應是因為組織胺作用在血管或神經末梢的H2受體而導致風疹塊及紅斑反應(wheal-and-flare)
B.發生立即性反應的個體,大約50%也會出現後期反應,嚴重者可能會持續24小時或更久
C.除組織胺外,肥大細胞所釋放之前列腺素(prostaglandins)也參與立即性反應之發生
D.肥大細胞所釋放之CGRP(calcitonin gene-related peptide)及VEGF(vascular endothelial growth factor)與後期反應之發生有關

> [!success]- 正確解答
> **A**

## 筆記與詳解

### (1) 快速破題

看到「過敏原皮膚試驗」與「風疹塊及紅斑反應 (wheal-and-flare)」，必須直覺聯想 Type I 過敏反應。題目要求選出「錯誤」敘述，選項 A 出現了組織胺作用於「**H2 受體**」的描述，這是一個經典且極易辨識的藥理學與免疫學陷阱。引發過敏性鼻炎、蕁麻疹及血管擴張的主要路徑是 **H1 受體**；而 H2 受體主要負責分泌胃酸。因此直接鎖定答案為 A。

### (2) 關鍵字與知識點拆解

* **過敏原皮膚試驗 (Skin test)** → 測試第一型過敏反應 (Type I hypersensitivity) 的標準臨床方法，仰賴 IgE 交聯與肥大細胞 (Mast cell) 活化。
* **風疹塊及紅斑 (Wheal-and-flare)[🖼️**](https://www.google.com/search?q=%22wheal-and-flare%22+clinical+sign&tbm=isch&udm=2) → 立即性反應的典型視覺特徵。「Wheal (風疹塊)」為微血管通透性↑導致血漿滲出；「Flare (紅斑)」為周邊微血管擴張。
* **組織胺 (Histamine) 受體** → **H1**：介導過敏、血管擴張、平滑肌收縮；**H2**：刺激胃壁細胞分泌胃酸。
* **前列腺素 (Prostaglandins, 如 PGD2)** → 肥大細胞活化後膜磷脂質代謝「新合成 (Newly synthesized)」的脂質介質，與白血球三烯 (Leukotrienes) 共同放大**立即性反應**。
* **後期反應 (Late-phase reaction)** → 發生於接觸過敏原 2~24 小時後，由細胞激素、趨化因子吸引**嗜酸性白血球 (Eosinophils)** 等發炎細胞浸潤所致。
* **CGRP & VEGF** → 降鈣素基因相關胜肽與血管內皮生長因子。由肥大細胞分泌，參與血管新生、組織重塑與維持**後期反應**的神經性發炎。

### (3) 詳解

* **選項 A (錯誤)**：引發風疹塊及紅斑反應的主要機制是組織胺結合到血管內皮與神經末梢的 **H1 受體**，而非 H2。H1 受體活化導致微血管擴張、通透性增加 (引發 wheal) 及神經末梢刺激 (引發搔癢與 flare)。H2 受體主要分布於胃部，負責調節胃酸分泌。
* **選項 B (正確)**：Type I 過敏反應具有雙相性 (Biphasic)。經歷立即性反應的個體中，約有 50% 會在 2~24 小時後發展出後期反應 (Late-phase)，特徵為大量白血球浸潤，嚴重時組織發炎與腫脹可持續數天。
* **選項 C (正確)**：肥大細胞活化不僅釋放預先儲存的顆粒 (如組織胺)，也會迅速透過花生四烯酸代謝路徑新合成脂質介質 (Lipid mediators)，如前列腺素 (PGD2) 及白血球三烯 (LTC4, LTD4, LTE4)。這些物質作用迅速，同樣是促成血管擴張、支氣管收縮等「立即性反應」的關鍵樞紐。
* **選項 D (正確)**：在後期反應階段，肥大細胞會透過轉錄轉譯分泌多種細胞激素 (Cytokines) 與生長因子。其中 CGRP (神經胜肽) 與 VEGF 參與了局部的血管通透性調節、血管新生及長期的組織重塑 (Tissue remodeling)，是驅動後期發炎反應的重要介質。

### (4) 核心邏輯與延伸比較

**縱向核心生理機轉：**
過敏原交聯 IgE → 肥大細胞去顆粒 (Mast cell degranulation)[🖼️](https://www.google.com/search?q=%22mast+cell+degranulation%22+histology&tbm=isch&udm=2) → 釋放組織胺結合 **H1 受體** → 微血管擴張/通透性↑ → 臨床表徵：**Wheal-and-flare** (立即性) → 數小時後轉錄分泌 Cytokines/VEGF/CGRP → 嗜酸性白血球浸潤 → 組織重塑與持續發炎 (後期)。

**💡 記憶口訣：**
組織胺受體分布：「**1 鼻 2 胃**」
(H**1** 負責過敏流**鼻**水/打噴嚏/紅疹；H**2** 負責**胃**酸分泌)

#### Type I 過敏反應：階段與介質比較矩陣

| 比較維度 | 立即性反應 (Immediate Phase) | 後期反應 (Late Phase) |
| --- | --- | --- |
| **發生時間** | 數分鐘內 (~30分鐘達高峰) | 2 ~ 24 小時 |
| **主要效應細胞** | **肥大細胞 (Mast cells)** | **嗜酸性白血球 (Eosinophils)**、Th2、中性白血球 |
| **核心介質類型** | **預成型顆粒** (Preformed) + **新合成脂質** | **新合成蛋白質** (De novo cytokines/peptides) |
| **關鍵化學介質** | **Histamine**、**Prostaglandins (PGD2)**、Leukotrienes | **IL-4, IL-5**, TNF-α, **VEGF**, **CGRP** |
| **生理/組織變化** | 微血管擴張、通透性↑、平滑肌收縮 (支氣管痙攣) | 發炎細胞浸潤、上皮細胞損傷、**組織重塑** |
| **視覺/臨床表徵** | **風疹塊與紅斑 (Wheal-and-flare)**[🖼️](https://www.google.com/search?q=%22wheal-and-flare%22+clinical+sign&tbm=isch&udm=2)、急性氣喘 | 組織紅腫痛、慢性氣喘、氣道高反應性 |
| **治療標的** | **抗 H1 組織胺** (Anti-histamines)、Epinephrine | 皮質類固醇 (Corticosteroids)、Anti-IgE (Omalizumab) |

#### 組織胺受體 (Histamine Receptors) 橫向比較

| 比較維度 | H1 受體 (本題核心考點) | H2 受體 |
| --- | --- | --- |
| **主要分布** | 血管內皮、平滑肌 (呼吸道/腸胃)、感覺神經末梢 | 胃壁細胞 (Parietal cells)、心肌、血管平滑肌 |
| **G蛋白偶聯** | Gq (活化 PLC → IP3/DAG → **Ca2+↑**) | Gs (活化 Adenylate cyclase → **cAMP↑**) |
| **促發效應** | 微血管通透性↑、血管擴張、支氣管收縮、**搔癢/疼痛** | **胃酸分泌↑**、心跳速率稍↑ |
| **臨床對應藥物** | Cetirizine, Loratadine, Diphenhydramine | Cimetidine, Famotidine, Ranitidine |
| **治療適應症** | **過敏性鼻炎、蕁麻疹**、動暈症 | **消化性潰瘍**、胃食道逆流 (GERD) |
