## 題目

下列有關B細胞受體(BCR)複合體相關的訊息傳遞,何者錯誤?

A.CD21/19/81是BCR複合體之輔助受體
B.Syk激酶能與BCR複合體 Igβ已磷酸化的酪胺酸基結合
C.ZAP-70激酶能與BCR複合體 Iga已磷酸化的酪胺酸基結合
D.CD40是BCR複合體之輔助受體

> [!success]- 正確解答
> **C**

---

## 筆記與詳解

### (1) 快速破題

看到題目考「B細胞受體 (BCR)」的訊息傳遞，且選項中同時出現 `Syk` 和 `ZAP-70`，就要立刻想到免疫學最經典的「T細胞與B細胞訊息傳遞分子對接」考點。**Syk 是 B細胞專屬**，而 **ZAP-70 是 T細胞專屬**。因此，看到將 ZAP-70 搭配 BCR 的敘述，即可直接抓出錯誤，答案選 C。

### (2) 關鍵字解析

* **BCR複合體 (BCR complex)** → 由負責結合抗原的膜免疫球蛋白 (mIg) 與負責傳遞訊號的 **Igα/Igβ (CD79a/CD79b)** 雙體組成。
* **ITAM (免疫受體酪胺酸活化基序)** → 位於 Igα/Igβ 胞內段。當受體活化時，其上的酪胺酸 (Tyrosine) 會被磷酸化，成為下游激酶的結合位點。
* **Syk 激酶** → **B細胞**訊息傳遞的核心 Tyrosine kinase，具有雙 SH2 結構域，能結合已磷酸化的 ITAM。
* **ZAP-70 激酶** → **T細胞**專屬的 Tyrosine kinase，功能與 B 細胞的 Syk 完全對等，負責結合在 TCR 複合體 (CD3) 磷酸化的 ITAM 上。
* **CD21/19/81** → 經典的 **B細胞輔助受體 (Co-receptor)**。能將 B 細胞活化的抗原閾值大幅降低。

### (3) 詳解

* **A (正確)：** CD19、CD21 (又稱 CR2，可結合補體 C3d)、CD81 (TAPA-1) 共同組成 B 細胞的輔助受體複合體。當 BCR 結合抗原，且 CD21 結合抗原上的補體時，CD19 的長胞內尾段會傳遞強烈的協同活化訊號。
* **B (正確)：** 當 BCR 結合抗原後，細胞膜上的 Src 家族激酶 (如 Lyn, Fyn, Blk) 會將 Igα/Igβ 上的 ITAM 磷酸化。接著，B 細胞內專屬的 **Syk 激酶** 會透過其 SH2 domain 結合到 Igβ (及 Igα) 已磷酸化的 ITAM 上，啟動下游訊息級聯反應。
* **C (錯誤，本題答案)：** **ZAP-70** (Zeta-chain-associated protein kinase 70) 是 **T細胞** 專屬的激酶。在 T 細胞活化時，ZAP-70 會結合到 TCR 複合體中 CD3 (特別是 ζ chain) 已磷酸化的 ITAM 上。B 細胞對應的分子是 Syk，絕非 ZAP-70。
* **D (相對正確)：** 嚴格的免疫學定義中，CD19/21/81 才是「輔助受體 (Co-receptor)」，而 CD40 屬於「共刺激分子 (Costimulatory molecule)」，負責接收 T 細胞 CD40L 傳來的第二訊號以進行 isotype switching。但在考題語境中，CD40 常被廣義歸類為輔助 BCR 活化與存活的受體。相較之下，選項 C 將 T 細胞專屬分子套用於 B 細胞，是 100% 絕對錯誤的致命傷，故答案選 C。

### (4) 核心軸線與邏輯因果 (縱向聯想)

**B細胞活化機轉：** 抗原交聯 BCR → 膜上 Src 家族激酶 (Lyn/Fyn/Blk) 活化 → 磷酸化 Igα/Igβ 的 ITAM → 招募並活化 **Syk** → 活化 PLC-γ2 → 分解 PIP2 產生 DAG & IP3 → 活化 PKC & 釋放內質網鈣離子 → 轉錄因子活化 (NF-κB, NFAT) → B細胞增生與分化。

*補充臨床檢驗聯想*：T細胞與B細胞在常規血液抹片中皆呈現極為相似的小淋巴球型態 (高核質比、深染的圓形細胞核與極少的嗜鹼性細胞質)[🖼️](https://www.google.com/search?q=small+lymphocyte+blood+smear+Wright+stain&tbm=isch&udm=2)，無法由視覺區分，必須仰賴流式細胞儀偵測表面標記 (如 CD3 vs CD19) 與內部訊息分子 (如 ZAP-70 vs Syk) 來進行淋巴瘤分型。

### (5) T細胞與B細胞訊息傳遞分子對照表 (橫向比較)

| 比較維度 | T細胞 (T Cell) | B細胞 (B Cell) |
| --- | --- | --- |
| **抗原辨識受體** | TCR (α/β 或 γ/δ) | mIg (IgM, IgD) |
| **訊號傳導次單元** | **CD3** (γ, δ, ε) 與 **ζ chain** | **Igα / Igβ** (CD79a / CD79b) |
| **活化基序** | ITAM (多個) | ITAM (各1個) |
| **經典輔助受體 (Co-receptor)** | CD4 / CD8 | **CD19 / CD21 / CD81** |
| **起始磷酸化激酶 (Src family)** | Lck, Fyn | Lyn, Fyn, Blk |
| **結合 ITAM 之關鍵激酶** | **ZAP-70** | **Syk** |
| **共刺激受體 (Costimulatory)** | CD28 (配體: B7-1/B7-2) | CD40 (配體: CD40L) |
