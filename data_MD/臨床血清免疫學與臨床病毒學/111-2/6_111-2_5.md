## 題目

下列有關B細胞作為抗原呈獻細胞時的特性,何者錯誤?

A.B細胞表面穩定表現MHC class II
B.B細胞表面穩定表現高量的CD80和CD86
C.比樹突細胞不易活化naive T細胞
D.B細胞以表面免疫球蛋白結合抗原,進行胞吞

> [!success]- 正確解答
> **B**

---

## 筆記與詳解

### (1) 快速破題

看到題目探討「專業抗原呈獻細胞 (Professional APCs)」之間的差異，並要求選出「錯誤」敘述，解題核心在於區分 **樹突細胞 (DC)** 與 **B細胞** 的定位差異：

* **樹突細胞 (DC)** 是 APC 界的「啟動大師」，專門負責活化 **Naive T cell (未活化 T 細胞)**，因此其表面會表現**高量**的共刺激分子 (CD80/CD86)。
* **B細胞** 的主要目的是向「已活化的輔助 T 細胞 (Effector Th cell)」尋求幫助以產生抗體。為了避免誤觸發未成熟的免疫反應，**未活化的休息狀態 B 細胞，其表面的 CD80/CD86 表現量極低**，必須等到結合抗原被活化後才會大量表現。
* 因此，看到「B細胞表面『穩定表現高量』的 CD80/CD86」即可判定為錯誤描述。

### (2) 關鍵字與知識點拆解

* **Professional APCs (專業抗原呈獻細胞)** → 包含樹突細胞、巨噬細胞、B細胞。共同特徵是表面皆有 **MHC class II**。
* **MHC class II** → 負責展示外源性抗原給 **CD4+ T 細胞**（提供活化 Signal 1）。
* **CD80 / CD86 (又稱 B7-1 / B7-2)** → APC 表面的**共刺激分子 (Co-stimulatory molecules)**，與 T 細胞表面的 CD28 結合（提供活化 Signal 2）。活化 Naive T cell 絕對需要強烈的 Signal 2。
* **Naive T 細胞 (未活化)** → 尚未遇過抗原的 T 細胞，門檻極高，**只有樹突細胞 (DC)** 具備足夠的 CD80/86 能夠有效活化它。
* **表面免疫球蛋白 (sIg / BCR)** → B細胞專屬的「抗原捕捉器」，具有極高專一性，可藉由**受體媒介胞吞作用 (Receptor-mediated endocytosis)** 抓取極低濃度的特定抗原。

### (3) 詳解與選項辨析

* **(A) B細胞表面穩定表現MHC class II：正確。** B細胞需要隨時準備將抗原呈現給周邊的輔助型 T 細胞 (Th cell) 辨識，因此平時就會穩定表現 (Constitutive expression) MHC class II。
* **(B) B細胞表面穩定表現高量的CD80和CD86：錯誤，本題答案。** 未活化的 B 細胞 (Resting B cell) 表面 CD80/CD86 表現量**極低**。只有當 B 細胞的 BCR 結合了抗原，或受到其他信號（如 CD40-CD40L 交互作用、TLR 刺激）活化後，才會被誘發 (Induced) 大幅上調 CD80/CD86 的表現。
* **(C) 比樹突細胞不易活化 naive T細胞：正確。** 因為休息狀態的 B 細胞缺乏足夠的共刺激分子 (CD80/CD86)，無法提供足夠的 Signal 2，因此無法有效活化 Naive T cell。活化 Naive T cell 的主力是成熟的樹突細胞。
* **(D) B細胞以表面免疫球蛋白結合抗原,進行胞吞：正確。** 巨噬細胞與樹突細胞多利用吞噬 (Phagocytosis) 或胞飲 (Pinocytosis) 非專一性地吃入抗原；而 B 細胞則是利用其表面的抗體 (BCR/sIg) **專一性地**結合抗原並進行胞吞 (Endocytosis)，這種方式讓 B 細胞能濃縮極微量的特定抗原。

### (4) 核心軸線與橫向比較表格

**免疫活化邏輯軸線 (縱向聯想)：**
病原體入侵 → **樹突細胞 (DC)** 吞噬並成熟 (呈現星狀樹枝狀突起形態[🖼️](https://www.google.com/search?q=dendritic+cell+morphology+SEM&tbm=isch&udm=2)) → 上調 MHC-II 與 **高量 CD80/CD86** → 游走至淋巴結活化 **Naive T cell** → 分化為效應 T 細胞 (如 Tfh)。
同時 → **B細胞** 利用 **BCR** 專一性捕捉同源抗原 → 胞吞分解後藉由 **MHC-II** 呈獻 → 與剛才被 DC 活化的效應 T 細胞 (Tfh) 互動 → 獲得幫助信號 → B細胞完全活化、增殖並產生抗體。

**三大專業抗原呈獻細胞 (Professional APCs) 比較表：**

| 比較維度 | 樹突細胞 (Dendritic Cell, DC) | B細胞 (B Cell) | 巨噬細胞 (Macrophage) |
| --- | --- | --- | --- |
| **抗原攝取方式** | 吞噬、胞飲 (非專一性) | **受體媒介胞吞 (BCR, 極高專一性)** | 吞噬作用 (非專一性) |
| **MHC-II 表現狀態** | 常態表現 (成熟時 **↑↑**) | **常態表現 (穩定)** | 誘導性表現 (平時低，受干擾素 γ 刺激 ↑) |
| **CD80/CD86 表現** | 成熟時 **常態高表現** | **誘導性表現** (未活化時極低) | 誘導性表現 |
| **主要活化目標** | **Naive T cells (未活化 T 細胞)** | **Effector T cells (已活化，如 Tfh)** | **Effector T cells (已活化，如 Th1)** |
| **免疫核心任務** | **啟動**初次免疫反應 (最強 APC) | 尋求 Th 協助以**產生抗體** (體液免疫) | 尋求 Th1 協助以**增強殺菌力** (細胞免疫) |
| **記憶口訣** | **D**C = **D**iscover (啟動者) | **B** cell = **B**ind specific (專一結合者) | **M**acrophage = **M**urder (殺手/清道夫) |
