## 題目

下列有關一般的CD8T細胞之敘述,何者錯誤?

A.辨識由抗原呈獻細胞上MHC class II呈獻之抗原片段
B.可直接毒殺被病毒感染的細胞
C.IL-2可促其增生
D.第一型干擾素可促其增生

> [!success]- 正確解答
> **A**

---

## 筆記與詳解

### (1) 快速破題

看到題目問「CD8 T細胞」並尋找「錯誤」選項 $\rightarrow$ 優先檢查其 **MHC 辨識限制性 (MHC restriction)**。

* **免疫學經典口訣（乘積為 8 規則）**：
* **CD8** $\times$ **MHC class I** $= 8$
* **CD4** $\times$ **MHC class II** $= 8$



因此，選項 A 敘述「CD8 T細胞辨識 MHC class II 呈獻之抗原」違反此基本規則，可直接秒殺選出答案 **A**。

---

### (2) 專有名詞與關鍵字解析

* **CD8+ T細胞 (Cytotoxic T Lymphocyte, CTL)** $\rightarrow$ 胞毒性 T 細胞，主要負責殺傷胞內病原體（如病毒）感染的細胞及腫瘤細胞。
* **MHC restriction (MHC 限制性)** $\rightarrow$ T 細胞必須同時辨識自身 MHC 分子與其所呈現的抗原胜肽。
* **MHC class I**：表現於所有具核細胞，呈現內源性抗原（如病毒蛋白、腫瘤抗原）。
* **MHC class II**：僅表現於專業抗原呈獻細胞 (APC)，呈現外源性抗原（如細菌、細胞外蛋白質）。


* **IL-2 (Interleukin-2, 白介素-2)** $\rightarrow$ 關鍵的 T 細胞生長因子，促進活化後 T 細胞（含 CD4+ 與 CD8+）進行克隆增殖。
* **第一型干擾素 (Type I Interferon, IFN-$\alpha$/$\beta$)** $\rightarrow$ 抗病毒反應的核心細胞激素，可作為 CD8+ T 細胞活化的「第三訊號」，促進其存活與分化。

---

### (3) 選項詳解

* **A. 辨識由抗原呈獻細胞上MHC class II呈獻之抗原片段 (錯誤)**
* **解析**：CD8+ T 細胞僅能辨識由 **MHC class I** 呈現之抗原。辨識 MHC class II 呈現抗原的是 **CD4+ T 細胞**（輔助型 T 細胞，Th）。


* **B. 可直接毒殺被病毒感染的細胞 (正確)**
* **解析**：活化後的效應 CD8+ T 細胞（CTL）與被病毒感染的靶細胞形成免疫突觸 [🖼️](https://www.google.com/search?q=immunological+synapse+CTL&tbm=isch&udm=2)，釋放穿孔素 (Perforin) [🖼️](https://www.google.com/search?q=perforin+pore+electron+microscopy&tbm=isch&udm=2) 於靶細胞膜上打孔，並注入顆粒酶 (Granzyme) 引發靶細胞凋亡，或透過表面 FasL 與靶細胞 Fas 結合啟動凋亡路徑。


* **C. IL-2可促其增生 (正確)**
* **解析**：T 細胞活化後會自分泌或旁分泌 **IL-2**，並與表面的高親和力 IL-2 接受器（CD25）結合，驅動細胞進入細胞週期進行大規模有絲分裂（增生）。


* **D. 第一型干擾素可促其增生 (正確)**
* **解析**：在病毒感染早期，受感染細胞釋放的 **Type I IFN (IFN-$\alpha$/$\beta$)** 可直接作用於 CD8+ T 細胞，提供關鍵的存活訊號並協同促進其克隆擴張 (Clonal expansion)。



---

### (4) 核心邏輯軸線與記憶口訣

* **縱向邏輯因果軸線**：
細胞受病毒感染 $\rightarrow$ 胞內合成病毒蛋白（內源性抗原） $\rightarrow$ 經蛋白酶體降解 $\rightarrow$ 進入內質網裝載至 **MHC class I** $\rightarrow$ 呈遞至細胞表面 $\rightarrow$ Naive CD8+ T 細胞之 TCR 結合 MHC-I [🖼️](https://www.google.com/search?q=CD8+TCR+MHC+class+I+interaction&tbm=isch&udm=2)（訊號 1） $\rightarrow$ 共刺激分子結合（訊號 2） $\rightarrow$ **IL-2 / Type I IFN** 刺激（訊號 3） $\rightarrow$ **CD8+ T 細胞大規模增殖並分化為 CTL** $\rightarrow$ 靶細胞毒殺與凋亡。
* **記憶口訣**：
> **CD 八一，CD 四二，乘積皆為八。**
> （CD8 結合 MHC class I；CD4 結合 MHC class II）

### (5) 橫向對比表格（T 細胞核心特徵比較）

| 比較維度 | CD4+ T 細胞 (輔助型, Th) | CD8+ T 細胞 (胞毒型, CTL) |
| --- | --- | --- |
| **MHC 限制** | **MHC class II** | **MHC class I** |
| **主要功能** | 分泌細胞激素、輔助 B/M$\phi$ 細胞 | **直接毒殺**受感染細胞與腫瘤細胞 [🖼️](https://www.google.com/search?q=cell+mediated+cytotoxicity+assay&tbm=isch&udm=2) |
| **抗原來源** | **外源性抗原** (胞吞作用) | **內源性抗原** (胞質內合成) |
| **輔助受體結合位置** | MHC class II $\beta2$ 區域 | MHC class I $\alpha3$ 區域 |
| **核心增殖因子** | **IL-2**、IL-4 | **IL-2**、**Type I IFN (IFN-$\alpha/\beta$)** |
| **效應釋放物質** | Cytokines (IFN-$\gamma$, IL-4, IL-17) | **穿孔素 (Perforin)**、**顆粒酶 (Granzyme)**、**FasL** |
| **臨床相關疾病** | HIV 感染主要受體 (**CD4 數量$\downarrow$**) | 病毒慢性感染（CD8 耗竭）、移植排斥 |
