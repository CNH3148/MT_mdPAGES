## 題目

下列有關主要組織相容性複合體(major histocompatibility complex,MHC)的敘述,何者錯誤?

A.MHC class I由一條a chain和一條β2-microglobulin組合而成
B.MHC class II由一條a chain和一條β chain組合而成
C.MHC class I可結合8~10個胺基酸組成的胜肽抗原
D.MHC class II可結合2~5個胺基酸組成的胜肽抗原

> [!success]- 正確解答
> **D**

---

## 筆記與詳解

### (1) 快速破題

看到題目考「MHC class I 與 class II」的結構與功能比較，並且要找「錯誤」選項 → 就要直接聯想到這兩者最核心的差異：**結構鏈組合**以及**胜肽結合槽的物理限制（容納長度）**。

MHC class I 結合槽兩端封閉，只能容納較短的胜肽（8~10 個胺基酸）；MHC class II 結合槽兩端開放，可容納較長的胜肽（通常 13~24 個）。選項 D 敘述的「2~5 個胺基酸」實在太短，在免疫學上無法提供足夠的抗原專一性（容易發生隨機結合），違背生理常理，因此可以直指答案是 D。

---

### (2) 詳解預備：關鍵字與知識點拆解

* **MHC class I** → **內生性抗原呈現**。存在於幾乎所有「有核細胞」表面。結構為一條重鏈（α chain）搭配一條輕鏈（β2-microglobulin）。
* **MHC class II** → **外生性抗原呈現**。僅存在於專業抗原呈現細胞（APC，如巨噬細胞、樹突細胞、B 細胞）。結構為 α chain 與 β chain 兩條等長鏈。
* **胜肽結合槽 (Peptide binding groove)**[🖼️](https://www.google.com/search?q=MHC+peptide+binding+groove+structure&tbm=isch&udm=2) → MHC 分子頂端與抗原胜肽結合的凹槽。Class I 像「口袋」（兩端封閉），Class II 像「熱狗堡」（兩端開放）。

---

### (3) 詳解與選項辨析

* **A (正確)**：MHC class I 的分子結構是由一條具有多型性、穿越細胞膜的 **α chain**，與一條高度保守、不穿膜的 **β2-microglobulin** 以非共價鍵結合而成。
* **B (正確)**：MHC class II 的分子結構是由兩條皆穿越細胞膜的蛋白質鏈所組成，分別稱為 **α chain** 與 **β chain**。
* **C (正確)**：MHC class I 的胜肽結合槽由 α1 和 α2 結構域構成，由於其物理結構**兩端閉合**，存在空間限制，因此只能結合約 **8~10 個**胺基酸長度的短片段胜肽。
* **D (錯誤，為本題解答)**：MHC class II 的胜肽結合槽由 α1 和 β1 結構域構成，結構**兩端開放**，胜肽可以突出於凹槽兩端，因此能容納 **13~24 個**胺基酸（最常見約 15~18 個）的較長胜肽片段。2~5 個胺基酸過於短小，無法穩定結合且不具備足夠的抗原鑑別度。

---

### (4) 核心軸線與邏輯因果

**結構決定功能 (縱向聯想)：**
MHC 胜肽結合槽兩端封閉 (Class I) → 空間受限 → 只能夾住 **短胜肽 (8-10 AA)** → 適合呈現細胞內部經 Proteasome 降解的細碎蛋白質（如病毒蛋白）；
MHC 胜肽結合槽兩端開放 (Class II) → 空間彈性大 → 可夾住 **長胜肽 (13-24 AA)** → 適合呈現吞噬體由外吞入、經溶酶體初步分解的較大段蛋白質。

**記憶口訣：**

1. **長短對應**：「**一短二長**」→ Class I 裝短胜肽 (8~10)，Class II 裝長胜肽 (13~24)。
2. **神仙配對法則 (Rule of 8)**：MHC 級數與 T 細胞 CD 標記數字相乘必須等於 8。
* MHC Class **I** × CD**8** T cell = 8
* MHC Class **II** × CD**4** T cell = 8

### (5) MHC Class I 與 Class II 綜合橫向比較表

| 比較維度 | MHC Class I | MHC Class II |
| --- | --- | --- |
| **表現細胞** | 所有**有核細胞**、血小板 | **專業抗原呈現細胞 (APC)** (巨噬細胞、樹突細胞、B細胞) |
| **結構組成** | **α chain** + **β2-microglobulin** | **α chain** + **β chain** |
| **結合槽構成** | α1 與 α2 結構域 | α1 與 β1 結構域 |
| **結合槽物理特性** | **兩端封閉** (口袋狀) | **兩端開放** (熱狗堡狀) |
| **結合胜肽長度** | **8 ~ 10** 個胺基酸 | **13 ~ 24** 個胺基酸 |
| **抗原來源** | **內生性** (如：病毒、腫瘤蛋白) | **外生性** (如：吞噬之細菌、毒素) |
| **抗原降解途徑** | 細胞質蛋白酶體 (**Proteasome**) | 溶酶體 (**Lysosome**) |
| **抗原呈現對象** | **CD8+** 胞毒性 T 細胞 (Tc) | **CD4+** 輔助性 T 細胞 (Th) |
