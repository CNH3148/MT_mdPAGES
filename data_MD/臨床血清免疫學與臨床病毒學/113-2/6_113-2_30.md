## 題目

下列有關淋巴球微毒殺試驗(lymphocyte microcytotoxicity test)敘述,何者錯誤?

A.使用B細胞評估第二型組織抗原相容程度
B.使用T細胞評估第一型組織抗原相容程度
C.加入anti-human globulin (AHG)抗體,以增加檢測靈敏度
D.待測樣本為補體,檢測試劑為捐贈者的細胞

> [!success]- 正確解答
> **D**

---

## 筆記與詳解

### (1) 快速破題

看到「淋巴球微毒殺試驗 (CDC test)」，要直覺聯想這是器官移植前必做的「HLA 分型」或「交叉比對」。
解題核心：釐清實驗的「抗原、抗體、補體」三大要角是誰。題目問「錯誤」選項，直接檢視實驗設計的樣本來源。常規交叉比對的待測樣本是受贈者的「**血清**」(測抗體)，試劑是捐贈者的「**細胞**」(抗原)，補體只是額外加入的「**效應媒介**」(通常買現成的兔補體)，因此說「待測樣本為補體」絕對錯誤，直接選 D。

### (2) 詳解預備：關鍵字 → 知識點

* **淋巴球微毒殺試驗 (Lymphocyte microcytotoxicity test, CDC)** → 組織相容性試驗的經典方法，原理為：抗原 + 抗體結合 → 活化補體 → 導致目標細胞死亡破裂。
* **T 細胞** → 表面只表現 **HLA Class I** (無 Class II)，為第一型 HLA 分型/交叉比對的標準用細胞。
* **B 細胞** → 表面同時具備 HLA Class I 與 **Class II**，實驗中用來針對第二型 HLA 進行評估。
* **AHG (Anti-human globulin)** → 抗人類球蛋白，作為「橋樑」可拉近並聚集更多細胞表面的抗體，藉此強效活化補體，提升微弱抗體的偵測率。

### (3) 詳解

* **A 正確**：HLA Class II 主要表現在抗原呈現細胞 (APC) 上。實驗室會利用純化的 B 細胞作為標的，來評估受血/受贈者血清中是否有針對 HLA Class II 的特異性抗體。
* **B 正確**：所有有核細胞均表現 HLA Class I，但純化的 T 細胞表面只有 Class I 而無 Class II。使用 T 細胞測試可完全排除 Class II 抗體的干擾，專一性評估 第一型組織抗原 的相容程度。
* **C 正確**：部分抗體 (如低效價或不擅長結合補體的 IgG) 無法有效啟動補體級聯反應。加入 AHG 後，AHG 會結合在已辨識 HLA 的 IgG 上形成大型複合體，大幅增加補體 (C1q) 的結合位點，此改良法稱為 AHG-CDC，可有效提升**檢測靈敏度**。
* **D 錯誤**：在交叉比對試驗中：
* **待測樣本**：受贈者的 **血清** (檢查是否含 anti-HLA 抗體)。
* **檢測試劑**：捐贈者的 **淋巴球** (作為標的抗原來源)。
* 補體 (通常為兔補體) 只是標準化添加的**次要試劑**，用來引發毒殺反應，絕非「待測樣本」。



### (4) 核心軸線與邏輯因果

**【CDC 檢驗機轉與視覺結果】**
受贈者血清 (含特異性抗體) + 捐贈者淋巴球 (表面 HLA) → 形成抗原抗體複合體 → 加入兔補體 (Rabbit Complement) → 補體活化形成膜攻擊複合體 (MAC) 在細胞膜打孔 → 細胞死亡，喪失膜通透性選擇 → 加入染劑 (Trypan blue 或 Eosin Y) → 染劑進入細胞內，鏡下可見**死亡細胞染上藍色或紅色** [🖼️](https://www.google.com/search?q=Trypan+blue+Eosin+Y+stained+dead+lymphocyte+microscope+CDC+test&tbm=isch&udm=2) → 判讀為陽性反應 (不相容)。
*(記憶口訣：**T1B2** → **T**細胞測 Class **1**；**B**細胞測 Class **2**。)*

### (5) 表格統整：HLA 交叉比對技術比較

| 比較維度 | 標準 CDC (淋巴球微毒殺) | AHG-CDC (抗球蛋白增強型) | FCXM (流式細胞交叉比對) |
| --- | --- | --- | --- |
| **標的細胞** | **T** 細胞 (Class I) / **B** 細胞 (Class II) | 同左 | 同左 |
| **偵測目標** | 補體結合型 **IgG**, **IgM** | **低效價 IgG** | **所有特異性 IgG** (無需補體活化) |
| **靈敏度** | **低** | **中高** | **最高** |
| **實驗媒介/放大器** | 兔補體 (Rabbit Complement) | 兔補體 + **AHG** | 螢光標示二級抗體 (FITC/PE) |
| **判讀方式** | 染劑進入死細胞顯色比率 (顯微鏡) | 染劑進入死細胞顯色比率 (顯微鏡) | **螢光強度 (MFI)** 偏移量 |
| **偽陽性干擾** | 非 HLA 抗體 (如自體抗體)、**IgM** | 自體抗體 | 自體抗體、**非特異性 Fc 結合** |
