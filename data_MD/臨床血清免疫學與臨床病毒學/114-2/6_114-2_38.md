## 題目

下列何者不是liposome 使用於疫苗載體之特性?

A.可以包裹蛋白質或DNA
B.一般利用 MHC class II呈獻抗原
C.可活化毒殺型T細胞
D.抗原可經由TAP 運送到內質網

> [!success]- 正確解答
> **B**

---

## 筆記與詳解

**(1) 快速破題**
看到題目考「Liposome (脂質體) 作為疫苗載體」，核心解題思路在於「為何要特別用脂質體包裝？」**。
一般游離蛋白質抗原注射入體內，會被抗原呈獻細胞 (APC) 胞吞，走外源性途徑 (MHC-II → CD4+ T細胞)。而 Liposome 具備脂質雙層結構，能與細胞膜融合，將抗原**直接送入細胞質 (Cytosol)**。抗原一但進入細胞質，必定走向**內源性途徑 (Endogenous pathway) → 經 **TAP** 進入內質網 → 結合 **MHC-I** → 活化 **CD8+ 毒殺型 T 細胞**。
題目問「不是」其特性，選項 C、D 為內源性途徑的必然特徵，選項 B 描述外源性途徑 (MHC-II)，明顯違背使用 Liposome 載體的最大免疫學優勢，故選 B。

**(2) 詳解預備 (關鍵字拆解)**

* **Liposome (脂質體)**：人工合成的雙層脂質球體[🖼️](https://www.google.com/search?q="lipid+bilayer"+liposome+microscopy+TEM&tbm=isch&udm=2)，內部水相空間可包覆親水性物質 (DNA/蛋白質)，疏水層可包覆脂溶性物質。
* **MHC class I**：表現於所有有核細胞表面，負責呈獻**細胞質內 (內源性)** 抗原。
* **MHC class II**：僅表現於專業抗原呈獻細胞 (APC)，負責呈獻**胞吞入內體 (外源性)** 抗原。
* **TAP (Transporter associated with antigen processing)**：內質網 (ER) 膜上的運輸蛋白，負責將細胞質中被降解的胜肽送進 ER 內與 MHC-I 結合。
* **毒殺型T細胞 (CD8+ Tc cell)**：辨識 MHC-I 抗原複合體，負責毒殺受感染或異常的細胞。

**(3) 詳解**

* **A 正確**：Liposome 載體結構具備親水核心與疏水雙層膜，可有效包裹保護蛋白質、核酸 (DNA/mRNA) 或佐劑，避免抗原在體內過早被酵素分解。
* **C、D 正確**：兩者為連續的生理事件。Liposome 使抗原得以進入細胞質，細胞質內的抗原會被蛋白酶體 (Proteasome) 降解 → 經由 **TAP** 運送到內質網 (選項 D) → 結合 MHC-I 呈獻於細胞表面 → 被 CD8+ T 細胞辨識，活化**毒殺型 T 細胞** (選項 C)。
* **B 錯誤**：一般游離的純蛋白質疫苗（未經載體）通常由 APC 吞噬，走的是 MHC-II 途徑，主要引發體液免疫（抗體）。使用 Liposome 作為載體的**最大特色與目的**，是為了打破此限制，透過**交叉呈獻 (Cross-presentation)** 或直接細胞質內遞送，將抗原導向 **MHC class I** 途徑，以誘發強大的細胞免疫 (Cellular immunity)。因此「一般利用 MHC class II」並非其獨特或主要的載體特性。

**(4) 核心軸線與邏輯因果**

* **Liposome 疫苗機轉**：包裹抗原 → 與細胞膜融合/內體逃脫 → 抗原釋放至**細胞質** → 蛋白酶體降解 → **TAP** 傳入**內質網 (ER)** → 結合 **MHC-I** → 細胞表面呈獻 → 活化 **CD8+ 毒殺型 T 細胞**。
* **記憶口訣**：
* **一質毒**：MHC-**1** + 細胞**質** (內源) + **毒**殺 T 細胞 (CD8+)
* **二外輔**：MHC-**2** + 細胞**外** (外源) + **輔**助 T 細胞 (CD4+)



**(5) MHC 呈獻途徑對比**

| 比較維度 | 內源性途徑 (Endogenous Pathway) | 外源性途徑 (Exogenous Pathway) |
| --- | --- | --- |
| **抗原來源** | **細胞質內** (病毒、腫瘤、**Liposome載體遞送**) | **細胞外環境** (細菌、游離蛋白質疫苗) |
| **負責細胞** | **所有有核細胞** | **專業 APC** (巨噬、樹突、B 細胞) |
| **降解酵素** | 蛋白酶體 (**Proteasome**) | 溶酶體酵素 (**Lysosomal enzymes** / Cathepsins) |
| **關鍵運輸/結構** | **TAP** (送入內質網) | **Invariant chain (Ii)** (佔據 MHC-II 結合位) |
| **胜肽結合位置** | **內質網 (ER)** | 吞噬溶酶體 (**Phagolysosome** / MIIC) |
| **呈獻分子** | **MHC class I** | **MHC class II** |
| **活化細胞** | **CD8+** 毒殺型 T 細胞 (Tc) | **CD4+** 輔助型 T 細胞 (Th) |
| **主要免疫反應** | **細胞免疫** (直接毒殺異常細胞) | **體液免疫** (協助 B 細胞產生抗體) |
