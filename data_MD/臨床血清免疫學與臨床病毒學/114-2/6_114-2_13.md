## 題目

下列何種分子一般不會表現在B細胞上?

A.CD14
B.CD20
C.CD40
D.CD45

> [!success]- 正確解答
> **A**

---

## 筆記與詳解

(1) 看到題目問「B細胞表面分子」且要找「不會表現」的，就要直覺想到各類白血球專屬的 CD marker (分化抗原簇)。選項中的 CD20、CD40 皆為 B 細胞經典的發育或活化標記，CD45 為所有白血球共有；而 CD14 是單核球/巨噬細胞的特異性標記，與 B 細胞無關，因此秒選 A。

(2)

* **CD14** → 單核球/巨噬細胞標記，為 LPS (內毒素) 結合受體的一部分。
* **CD20** → B 細胞專一性標記，涵蓋多數發育階段，為標靶藥物 (如 Rituximab) 標的。
* **CD40** → APC (含 B 細胞) 的共刺激分子，負責接收 T 細胞訊號以進行同型轉換 (Isotype switching)。
* **CD45** → 白血球共同抗原 (LCA)，存在於所有有核造血細胞。

(3)

* **選項 A (CD14)：錯誤 (本題答案)。** CD14 主要表現於單核球 (Monocyte)[🖼️](https://www.google.com/search?q=Monocyte+blood+smear+morphology&tbm=isch&udm=2) 與巨噬細胞 (Macrophage)[🖼️](https://www.google.com/search?q=Macrophage+morphology+microscopy&tbm=isch&udm=2) 表面。其核心生理功能是作為 TLR4 的共同受體 (Co-receptor)，負責辨識革蘭氏陰性菌外膜的 LPS (脂多醣)，**一般不會表現在 B 細胞上**。
* **選項 B (CD20)：正確。** CD20 是一跨膜蛋白，幾乎存在於所有 B 細胞發育階段（從 pro-B 晚期到成熟 B 細胞），是 B 細胞最重要的特異性鑑別標記之一。須注意，當 B 細胞最終分化為漿細胞 (Plasma cell)[🖼️](https://www.google.com/search?q=Plasma+cell+blood+smear+morphology&tbm=isch&udm=2) 時，CD20 表現會消失。
* **選項 C (CD40)：正確。** CD40 屬於 TNF 受體家族，廣泛存在於抗原呈現細胞 (APC，包含 B 細胞、巨噬細胞、樹突細胞)。在 B 細胞上，CD40 會與活化輔助型 T 細胞 (Th cell) 表面的 CD40L (CD154) 結合，提供 B 細胞活化、增殖及**抗體同型轉換 (Isotype switching)** 所需的關鍵第二訊號。
* **選項 D (CD45)：正確。** CD45 即白血球共同抗原 (Leukocyte Common Antigen, LCA)，表現於所有有核的造血系細胞表面 (包含 B 細胞、T 細胞、單核球等，但不含紅血球與血小板)，主要參與受體訊息傳遞及調節激酶活性。

(4)

* **縱向邏輯軸線：** 造血幹細胞分化 → 表現泛白血球標記 (**CD45**) → 淋巴系分化為 B 細胞 → 表現 B 細胞專一性標記 (**CD19, CD20, CD21**) 與活化共刺激受體 (**CD40**) → 若為骨髓系分化為單核球 → 則表現吞噬與內毒素辨識標記 (**CD14**)。
* **記憶口訣：** * B細胞標記：「**B** 哥 (B cell) 愛喝**酒** (**19**)，**雙十** (**20**) 弱**冠** (**21**) 慶祝**四十** (**40**) 大壽」。
* 單核球標記：「**一世** (**14**) 孤**單** (單核球)」。



(5)

| 細胞類別 | 核心 CD Marker | 功能特性 / 臨床意義 |
| --- | --- | --- |
| **B 細胞** | **CD19, CD20, CD21** | CD21 為 **EBV** 及補體 **C3d** 受體 |
| **B 細胞 (APC)** | **CD40** | 結合 Th 細胞 **CD40L (CD154)** → **Isotype switching** |
| **T 細胞 (泛)** | **CD2, CD3** | CD2 為 **SRBC** 受體 (**E-rosette** 形成) |
| **Th / Tc 細胞** | **CD4** / **CD8** | 結合 **MHC II** / 結合 **MHC I** |
| **NK 細胞** | **CD16, CD56** | CD16 (FcγRIII) 參與 **ADCC** 作用 |
| **單核球 / 巨噬細胞** | **CD14** | 結合 **LPS** (與 TLR4 協同作用) |
| **造血幹細胞 (HSC)** | **CD34** | **幹細胞移植**與純化標記 |
| **所有有核白血球** | **CD45** (LCA) | **泛白血球**標記 (排除 RBC、血小板) |
