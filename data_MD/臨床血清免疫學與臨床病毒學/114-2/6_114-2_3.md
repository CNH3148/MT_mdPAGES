## 題目

下列何種抗原呈獻屬於 cross presentation?

A.樹突細胞上的MHC class I呈獻外源性分子活化CD8 T細胞
B.病毒感染時細胞上的MHC class II呈獻外源性分子活化CD4 T細胞
C.病毒感染時細胞上的MHC class I呈獻內源性分子活化CD8 T細胞
D.樹突細胞上的MHC class II呈獻外源性分子活化CD4 T細胞

> [!success]- 正確解答
> **A**

---

## 筆記與詳解

### (1) 快速破題思路

看到題目出現關鍵字「cross presentation (交叉呈獻)」，立刻想到免疫學中的「反常路線」。
在正常免疫常識中：

* 內源性抗原 (Endogenous) → 找 MHC class I → 活化 CD8 T 細胞。
* 外源性抗原 (Exogenous) → 找 MHC class II → 活化 CD4 T 細胞。

**Cross-presentation 的定義就是「打破常規」**：抗原呈現細胞（主要是樹突細胞）將吞噬來的「外源性」分子，放到了「MHC class I」**上，並交給**「CD8 T 細胞」。
因此，選項中只要尋找 **「外源性分子」+「MHC class I」+「CD8」** 的組合，即可秒殺本題，答案為 A。

---

### (2) 關鍵字與知識點拆解

* **Cross-presentation (交叉呈獻)** → **知識點**：少數專業抗原呈現細胞（Professional APC，如樹突細胞）的特異功能，能將吞噬的外源性抗原「漏出」至細胞質，進入 MHC class I 途徑，藉此活化 CD8+ 胞毒性 T 細胞 (CTL)。這對清除「不直接感染 APC」的病毒或腫瘤至關重要。
* **MHC class I (第一型主要組織相容性複合體)** → **知識點**：表現於所有有核細胞表面。負責呈獻細胞內部合成的蛋白質（如病毒蛋白、腫瘤突變蛋白）。遵守「相乘等於8」法則（I × 8 = 8）。
* **MHC class II (第二型主要組織相容性複合體)** → **知識點**：僅表現於 APC（樹突細胞、巨噬細胞、B 細胞）表面。負責呈獻從細胞外吞噬進來的蛋白質（如細菌、毒素）。遵守「相乘等於8」法則（II × 4 = 8）。
* **Dendritic cell (樹突細胞)** → **知識點**：最強大的專業 APC（細胞表面具有星狀或樹枝狀突起的膜延伸特徵 [🖼️](https://www.google.com/search?q=dendritic+cell+morphology+SEM&tbm=isch&udm=2)），是唯一能有效進行 Cross-presentation 並活化「初始 (naive)」 T 細胞的細胞。

---

### (3) 詳解與選項辨析

* **(A) 正確**：樹突細胞吞噬**外源性分子**後，不走傳統的 MHC II，反而將其呈獻在 **MHC class I** 上，藉此活化 **CD8 T 細胞**。這完全符合 Cross-presentation 的定義。
* **(B) 錯誤**：MHC class II 呈獻外源性分子給 CD4 T 細胞，這是「傳統外源性途徑 (Classical exogenous pathway)」，屬於一般 APC 的標準作業流程，並非交叉呈獻。
* **(C) 錯誤**：病毒感染細胞後，細胞利用 MHC class I 將病毒的內源性分子呈獻給 CD8 T 細胞。這是「傳統內源性途徑 (Classical endogenous pathway)」，是一般有核細胞被感染時呼叫 CD8 T 細胞來毒殺自己的正常機制。
* **(D) 錯誤**：與 B 選項意義相同，樹突細胞利用 MHC class II 呈獻外源性抗原給 CD4 T 細胞，為傳統的 helper T cell 活化途徑，不是交叉呈獻。

---

### (4) 核心軸線與邏輯因果

**生理機轉與邏輯 (為何需要 Cross-presentation？)**
周邊組織細胞感染病毒 (或突變成腫瘤) → 該細胞不具備游走至淋巴結的能力，且 MHC I 只能吸引已活化的 effector CD8，無法活化 Naive CD8 → 樹突細胞到達現場 → 吞噬死掉的感染細胞碎片 (**外源性抗原**) → 若只走傳統路線，只能活化 CD4 (無法產生殺手細胞) → 因此樹突細胞啟動 **Cross-presentation** → 將外源抗原送上 **MHC I** → 成功活化 **Naive CD8 T 細胞** (此過程稱為 Cross-priming) → 產生大量具毒殺能力的 CD8 T 細胞去清除病灶。

> **💡 記憶口訣：雙八定律與「外一八」**
> * **正常雙八**：MHC **I** × CD**8** = 8；MHC **II** × CD**4** = 8。
> * **交叉呈獻 (Cross)**：**「外一八」** → **外**源性抗原 + MHC **I** + CD**8**。
