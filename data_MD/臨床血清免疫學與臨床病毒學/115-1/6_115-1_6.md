## 題目

有關第二型 MHC 分子(MHC class II)的抗原呈獻,下列敘述何者錯誤?

A.被內吞的抗原通常被表面MHC class II呈獻
B.MHC class II組裝需要ERp57、tapasin、calreticulin 等分子協助
C.MHC class II從內質網到late endosome 運送過程需要 invariant chain 協助
D.HLA-DM可以幫助短胜肽被呈獻出來

> [!success]- 正確解答
> **B**

---

## 筆記與詳解

### (1) 快速破題：解題思路

這是一道經典的免疫學國考陷阱題。看到題目問「**MHC class II**」，且要求選出「**錯誤**」的敘述，直接掃視選項中的專有名詞。
只要看到 **ERp57、tapasin、calreticulin** 這組蛋白，就要立刻聯想到它們是 **MHC class I** 內源性抗原呈現途徑的「胜肽裝載複合體 (Peptide-loading complex)」。題目把它們張冠李戴到 MHC class II 身上，因此秒選 (B)。

---

### (2) 詳解預備：關鍵字拆解與知識點

* **MHC class II**：負責呈現「外源性抗原」給 CD4+ Helper T cell。主要表現於專業抗原呈獻細胞 (APC)，如巨噬細胞、B 細胞，以及具有典型星狀偽足突起形態的樹突細胞 (Dendritic cell)[🖼️](https://www.google.com/search?q=Dendritic+cell+morphology+SEM&tbm=isch&udm=2)。
* **Endocytosed antigen (內吞抗原)**：細胞外病原體被吞噬後，在內體/溶體 (Endosome/Lysosome) 的酸性環境中降解而成的胜肽。
* **ERp57, tapasin, calreticulin**：**MHC class I 的專屬伴護蛋白**。負責在內質網 (ER) 穩定 MHC-I 結構，並協助接收由 TAP 運送進來的內源性胜肽。
* **Invariant chain (恆定鏈, Ii / CD74)**：**MHC class II 的專屬保鑣兼導航**。在 ER 中會先佔據 MHC-II 的抗原結合槽 (防止誤載內源性胜肽)，並導引 MHC-II 走向晚期內體 (Late endosome)。
* **HLA-DM**：**MHC class II 的催化劑**。當 Invariant chain 在內體被水解剩下一小段 CLIP 時，HLA-DM 負責將 CLIP 踢掉，讓真正的外源抗原胜肽裝載上去。

---

### (3) 詳解選項分析

* **(A) 敘述正確**：外源性抗原經由內吞作用 (Endocytosis) 進入細胞降解後，就是走 MHC class II 途徑，最終呈獻於細胞表面。
* **(B) 敘述錯誤，本題答案**：MHC class II 的組裝主要依賴 calnexin 及 **Invariant chain**。選項中的 `ERp57`、`tapasin`、`calreticulin` 加上 `TAP`，是構成 **MHC class I** 胜肽裝載複合體的關鍵分子。
* **(C) 敘述正確**：MHC class II 剛在內質網 (ER) 合成時，必須與 Invariant chain 結合。Invariant chain 具有特殊排序訊號，能引導 MHC class II 經高基氏體正確運送至晚期內體 (MIIC 區塊)。
* **(D) 敘述正確**：在內體中，Invariant chain 會被降解剩下 CLIP 佔據結合槽。此時必須依靠 HLA-DM 的協助，移除 CLIP，外來的短胜肽才能成功結合到 MHC class II 上。

---

### (4) 核心軸線與記憶口訣

**MHC-II 縱向生理機轉軸線：**
細胞外病原體侵入 → APC 吞噬 (Endocytosis) → 形成晚期內體/溶體 (降解抗原)
*(同時)* ER 合成 MHC-II + Invariant chain (防護結合槽) → 運送至晚期內體 → Invariant chain 降解剩 CLIP → **HLA-DM 移除 CLIP**並裝載外源胜肽 → 運送至細胞膜 → 活化 CD4+ T cell。

> 💡 **必背口訣：「1 內 8 TAP，2 外 4 恆 DM」**
> * **1 內 8 TAP**：MHC-**I** / **內**源抗原 / 活化 CD**8** / 關鍵蛋白：**TAP** (含 tapasin, calreticulin 等)
> * **2 外 4 恆 DM**：MHC-**II** / **外**源抗原 / 活化 CD**4** / 關鍵蛋白：**恆**定鏈 (Invariant chain)、HLA-**DM**

### (5) 橫向比較表格：MHC Class I vs. Class II 呈獻途徑

| 比較維度 | MHC Class I (第一型) | MHC Class II (第二型) |
| --- | --- | --- |
| **表現細胞** | **所有有核細胞** | **專業 APC** (樹突細胞、巨噬、B 細胞) |
| **抗原來源** | **內源性** (如：病毒蛋白、腫瘤突變蛋白) | **外源性** (如：胞外細菌、毒素) |
| **活化對象** | **CD8+** Cytotoxic T cell (Tc) | **CD4+** Helper T cell (Th) |
| **胜肽降解處** | cytosol 的 **Proteasome (蛋白酶體)** | **Endolysosome (內體/溶體)** |
| **ER 內輔助/伴護蛋白** | **Calreticulin, ERp57, Tapasin** | Calnexin, **Invariant chain (Ii)** |
| **抗原運送至 ER 載體** | **TAP** (Transporter associated with antigen processing) | (無，抗原不在 ER 裝載) |
| **裝載胜肽位置** | **內質網 (ER)** | **晚期內體 (Late endosome / MIIC)** |
| **結合槽保護/催化機制** | (無特定護衛蛋白) | **Invariant chain (保鑣) → CLIP**<br>**HLA-DM (催化置換 CLIP)** |
