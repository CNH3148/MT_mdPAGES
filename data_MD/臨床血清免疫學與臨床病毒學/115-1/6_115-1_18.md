## 題目

主要組織相容性複合體(MHC)與KIRs(killer cell immunoglobulin-like receptor)結合後,下列何者不會抑制 NK細胞的毒殺作用?

A.HLA-A
B.HLA-B
C.HLA-C
D.HLA-DR

> [!success]- 正確解答
> **D**

---

## 筆記與詳解

## (1) 快速破題

看到題目問「**MHC 與 KIR 結合**」且能「**抑制 NK 細胞毒殺**」 → 立刻聯想到 **Missing-self 假說**：NK 細胞表面的抑制型 KIR 專門辨識 **MHC Class I**，只要抓到足量 MHC Class I 就代表是健康自體細胞，進而按兵不動。

* **掃視選項：** A、B、C 全都是古典 **MHC Class I**（HLA-A, -B, -C）；只有 D 是 **MHC Class II**（HLA-DR）。
* **直接鎖定：** HLA-DR 身為 MHC Class II，根本不是 KIR 的配體，自然「**不會**」參與抑制 NK 細胞的機制，秒選 **D**。

---

## (2) 詳解預備：關鍵字與知識點

* **NK 細胞 (Natural Killer cell)：** 自然殺手細胞。屬於先天免疫的核心猛將，不需抗原預先致敏，專職清除病毒感染細胞與腫瘤細胞。
* **KIR (Killer cell Immunoglobulin-like Receptor)：** 殺手細胞免疫球蛋白樣受體。表現於 NK 細胞表面，分為活化型與抑制型。本題聚焦於帶有胞內 ITIM 結構的**抑制型 KIR**。
* **MHC Class I (人類稱 HLA-A, -B, -C)：** 表現於「所有具核細胞」表面。除了向 CD8+ T 細胞呈現內源性抗原外，也是給 NK 細胞檢查的「免死金牌」。
* **MHC Class II (人類稱 HLA-DR, -DP, -DQ)：** 僅表現於「專業抗原呈現細胞 (APC)」。主要負責向 CD4+ T 細胞呈現外源性抗原。

---

## (3) 詳解

NK 細胞的毒殺活性取決於其表面「活化受體」與「抑制受體」所接收訊號的拉鋸戰（平衡理論）。

* **正解機制：** 正常健康的具核細胞會源源不絕地在表面表達 **MHC Class I (HLA-A, -B, -C)**。當這些 MHC Class I 與 NK 細胞上的**抑制型 KIR** 結合時，會啟動抑制性下游訊號，**阻斷** NK 細胞釋放穿孔素 (Perforin) 與顆粒酶 (Granzyme)，細胞因而存活。
* **選項分析：**
* **A. HLA-A / B. HLA-B / C. HLA-C：** 皆為 MHC Class I 分子，能與抑制型 KIR 結合，**會**抑制 NK 毒殺。
* **D. HLA-DR：** 屬於 **MHC Class II** 分子，其正常生理功能是與 CD4+ T 細胞的 TCR 結合。NK 細胞表面的 KIR 並不與其發生交互作用，因此 HLA-DR **不會**對 NK 細胞產生抑制訊號。



---

## (4) 縱向聯想與橫向比較

### 縱向邏輯軸線

細胞健康穩定 → 表達 **MHC Class I** → 結合 NK 表面**抑制型 KIR** → 啟動胞內 **ITIM 途徑** → **抑制**毒殺作用（細胞安全無事）。
細胞癌變/病毒感染 → **MHC Class I 表達下降** (Missing-self) → **KIR 失去訊號** → 活化受體佔上風 → **NK 細胞活化** → 釋放穿孔素/顆粒酶 → **目標細胞溶解凋亡**。

> 💡 **秒殺記憶口訣：** > 「**一**隻殺手（NK / CD8+ T）愛吃 **ABC**；**二**個骨幹（CD4+ T）專挑 **DP/DQ/DR**」
> （MHC I 對應 HLA-A,B,C 及 NK/CD8；MHC II 對應 HLA-DP,DQ,DR 及 CD4）
