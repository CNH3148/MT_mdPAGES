## 題目

有關干擾素(interferons)之敘述,下列何者錯誤?

A.現今已經發現 type I、type II及type III等三類干擾素
B.type I干擾素能夠透過 RIG-I 及MDA-5並與下游 MAVS 轉接蛋白之作用而產生
C.type II干擾素能夠透過 cGAS 辨識病毒核酸成分並活化下游 STING 而產生
D.type I干擾素能夠誘導 PKR kinase 之產生導致eIF2α磷酸化,達到抑制病毒複製之功效

> [!success]- 正確解答
> **C**

---

## 筆記與詳解

### (1) 快速破題：解題思路

看到題目考「干擾素 (Interferons) 的分類與產生機制」，且要求尋找「**錯誤**」敘述：

* **直覺聯想**：只要看到細胞內感測病毒核酸（如 RIG-I 感測 RNA、cGAS 感測 DNA），其下游誘發的絕對是負責「**抗病毒**」的 **Type I** 干擾素（IFN-α/β）。
* **破綻鎖定**：選項 C 提及 `cGAS` 與 `STING`，這正是細胞感測游離 DNA 並誘發干擾素的經典途徑，但它產生的應該是 **Type I**，而非 Type II。Type II (IFN-γ) 是由免疫細胞 (NK/Th1) 分泌用來活化巨噬細胞的。因此，(C) 敘述張冠李戴，為錯誤選項。

---

### (2) 詳解預備：關鍵字與知識點拆解

* **Type I vs. Type II Interferons**：
* **Type I (IFN-α/β)**：由受感染細胞產生，核心功能為「抗病毒狀態 (Antiviral state)」。
* **Type II (IFN-γ)**：由 T 細胞或 NK 細胞產生，核心功能為「免疫調節與巨噬細胞活化」。


* **RIG-I / MDA-5 → MAVS**：偵測細胞質內「病毒 RNA」的感測器 (Sensors) 與粒線體上的轉接蛋白 (Adapter)，負責啟動 Type I 干擾素。
* **cGAS → STING**：偵測細胞質內「異常 DNA」(如 DNA 病毒感染) 的酵素與內質網轉接蛋白，同樣負責啟動 Type I 干擾素。
* **PKR kinase / eIF2α**：Type I 干擾素作用後誘發的抗病毒蛋白。PKR 會磷酸化 eIF2α，直接「關閉細胞的蛋白質轉譯」，阻止病毒合成零件。

---

### (3) 詳解選項分析

* **(A) 現今已經發現 type I、type II及type III等三類干擾素 → 正確。**
目前學界將干擾素分為三大類：Type I (α, β, ω 等)、Type II (γ) 以及近年發現專司黏膜防禦的 Type III (λ)。
* **(B) type I干擾素能夠透過 RIG-I 及MDA-5並與下游 MAVS 轉接蛋白之作用而產生 → 正確。**
這是 RNA 病毒感染時最經典的免疫反應路徑。RIG-I 辨識帶有 5'-triphosphate 的短雙股 RNA；MDA-5 辨識長雙股 RNA，兩者皆會將訊號傳遞給 MAVS，最終活化轉錄因子 IRF3/7，產生 Type I IFN。
* **(C) type II干擾素能夠透過 cGAS 辨識病毒核酸成分並活化下游 STING 而產生 → 錯誤 (本題答案)。**
cGAS-STING 途徑辨識細胞質內的 DNA 後，活化的是 IRF3 與 NF-κB，最終轉錄出的是 **Type I** 干擾素（而非 Type II）。Type II (IFN-γ) 主要是在免疫反應中，由受到 IL-12 刺激的 NK 細胞與 Th1 細胞所分泌。
* **(D) type I干擾素能夠誘導 PKR kinase 之產生導致 eIF2α 磷酸化，達到抑制病毒複製之功效 → 正確。**
Type I IFN 釋放後會結合周邊細胞表面的受體，促使細胞表現數百種「干擾素刺激基因 (ISGs)」。其中 PKR 活化後會導致 eIF2α 磷酸化，使核糖體停工，終止病毒蛋白質轉譯。當病毒感染嚴重時，細胞亦可能走向凋亡，此過程常可在實驗室培養中觀察到明顯的細胞病變效應 (CPE)[🖼️](https://www.google.com/search?q="Cytopathic+effect"+viral+culture&tbm=isch&udm=2)。

---

### (4) 核心軸線與邏輯因果

* **縱向聯想 (細胞抗病毒防線)**：
病毒入侵釋出核酸 → 細胞質感測器抓取 (**RNA找RIG-I** / **DNA找cGAS**) → 活化轉接蛋白 (**MAVS** / **STING**) → 轉錄因子 (IRF3) 入核 → 大量分泌 **Type I IFN** → 結合相鄰細胞受體 → 啟動 ISGs (如 **PKR**) → 抑制轉譯 / 阻斷病毒複製。
*(補充：若病毒成功逃避干擾素機制並大量複製，常會在細胞內留下特徵性痕跡，例如：巨細胞病毒 (CMV) 感染形成的「貓頭鷹眼」核內包涵體[🖼️](https://www.google.com/search?q="Owl+eye+inclusion+body"+CMV+histology&tbm=isch&udm=2) ，或狂犬病毒在神經細胞形成的內基氏體[🖼️](https://www.google.com/search?q="Negri+bodies"+Rabies+histology&tbm=isch&udm=2) )*
* **記憶口訣**：
**「一生抗毒 (Type I)，RNA找MAVS，DNA找STING；二哥殺菌 (Type II)，NK與T來相挺！」**

### (5) 橫向比較：三大類干擾素 (Interferons) 總整理

| 比較維度 | Type I IFN | Type II IFN | Type III IFN |
| --- | --- | --- | --- |
| **主要成員** | **IFN-α**, **IFN-β** | **IFN-γ** | IFN-λ (1~4) |
| **主要分泌細胞** | **所有有核細胞** (特別是 pDC) | **NK cells**, **Th1 cells**, CD8+ T cells | 上皮細胞、樹突細胞 |
| **上游誘發訊號** | 病毒核酸 (RNA/DNA)<br>TLR3/7/8/9 活化 | **IL-12**, **IL-18** 刺激<br>抗原辨識 (TCR) | 病毒核酸 (類似 Type I) |
| **細胞內傳遞路徑** | RNA: **RIG-I/MDA-5** → **MAVS**<br>DNA: **cGAS** → **STING** | 非直接由細胞內核酸感測器誘發 | TLRs, RLRs 路徑 |
| **核心功能** | 強效**抗病毒** (干擾轉譯)<br>↑ MHC class I | 強效**免疫調節**<br>活化**巨噬細胞**殺菌<br>↑ MHC class I & **II** | **黏膜**局部抗病毒 (呼吸道/腸道/肝臟) |
| **下游效應分子** | ↑ **PKR** (停滯轉譯)<br>↑ OAS/RNase L (降解 RNA) | ↑ iNOS, ROS (增強毒殺)<br>促進 Th1 分化 | 類似 Type I ISGs (但受體分布侷限) |
