B.②③⑤

> [!success]- 正確解答
> **B**

---

## 筆記與詳解

### (1) 快速破題

看到題目問「表現 MHC class II 的最主要細胞」，應立即直覺聯想到**專業抗原呈現細胞 (Professional Antigen-Presenting Cells, pAPCs)**。在免疫學常識中，pAPCs 的經典三大成員即為**樹突細胞 (DC)**、**巨噬細胞**與 **B 細胞**。因此，直接選出包含 ②③⑤ 的選項，答案即為 **B**。

---

### (2) 關鍵字與知識點對照

* **MHC class II** → 主要表達於專業抗原呈現細胞 (pAPCs)，負責呈現外源性抗原給 CD4+ T 細胞。
* **MHC class I** → 表達於所有**具核細胞**，負責呈現內源性抗原給 CD8+ T 細胞。
* **Naïve T 細胞 / NK 細胞** → 屬於抗原辨識或效應細胞，主要表達 MHC class I，非 MHC class II 的主要表達者。

---

### (3) 選項徹底詳解

* **正確項分析 (②、③、⑤)**
* **② B細胞**：屬於 pAPC。利用表面受體 (BCR) 結合特異性抗原，內化處理後透過 MHC class II 呈現給 Helper T 細胞 (Th)，以活化自身並進行抗體基因轉變。
* **③ 巨噬細胞 (macrophages)**：屬於 pAPC。具強大吞噬能力，外源性病原體經吞噬後，抗原胜肽與 MHC class II 結合呈遞。鏡檢特徵為富含空泡與吞噬顆粒的胞質 [🖼️](https://www.google.com/search?q=macrophage+microscopy&tbm=isch&udm=2)。
* **⑤ 樹突細胞 (dendritic cells)**：**功能最強的 pAPC**。主要功能是活化初級 (Naïve) T 細胞。其視覺特徵為擁有眾多放射狀的樹突狀突起 [🖼️](https://www.google.com/search?q="dendritic+cell"+morphology&tbm=isch&udm=2)，能大幅增加與 T 細胞接觸的表面積。


* **錯誤項分析 (①、④、⑥)**
* **① naïve CD4 T細胞 & ④ naïve CD8 T細胞**：屬於淋巴效應細胞。其表面的 TCR 負責「接受」由 MHC 呈現的抗原，本身並非主要抗原呈現者（僅具核表達 MHC class I）。
* **⑥ 自然殺手細胞 (NK cells)**：屬於先天免疫的毒殺細胞，主要負責殺傷缺乏 MHC class I 的變異細胞或病毒感染細胞，自身不屬於 pAPC，不表達 MHC class II。



---

### (4) 縱橫邏輯梳理與橫向對比

* **縱向生理機轉軸線**：
外源性病原體入侵 → pAPC 吞噬內化 → 溶酶體水解成胜肽 → 結合 **MHC class II** → 呈遞至胞膜 → 活化 **CD4+ T 細胞** → 引發後續免疫應答。
* **記憶口訣**：
* **「2 × 4 = 8；1 × 8 = 8」**：MHC class **II** 結合 CD**4** T 細胞；MHC class **I** 結合 CD**8** T 細胞。
* **「pAPC 三劍客：B 居 (巨) 樹」**：**B**細胞、**巨**噬細胞、**樹**突細胞。



| 比較維度 | MHC class I | MHC class II |
| --- | --- | --- |
| **主要表達細胞** | **所有具核細胞** (內含 T、B、NK、上皮細胞等) | **專業抗原呈現細胞 (pAPCs)** |
| **經典細胞代表** | 紅血球以外的體細胞、淋巴細胞 | **樹突細胞** [🖼️](https://www.google.com/search?q="dendritic+cell"+morphology&tbm=isch&udm=2)、**巨噬細胞**、**B細胞** |
| **抗原辨識受體** | **CD8**+ T 細胞 (毒殺性 T 細胞, CTL) | **CD4**+ T 細胞 (輔助性 T 細胞, Th) |
| **抗原來源機轉** | **內源性抗原** (病毒複製蛋白、腫瘤抗原) | **外源性抗原** (內吞/吞噬之細菌、外來蛋白) |
| **主要免疫功能** | 標記異常細胞以利**毒殺清除** | 啟動**免疫活化**與抗體產生訊息 |
