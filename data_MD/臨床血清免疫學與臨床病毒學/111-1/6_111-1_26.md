## 題目

下列何項試驗是經皮下注射小量的結核菌素以評估是否曾感染結核分枝桿菌?

A.Mantoux test
B.Skin prick test
C.Patch test
D.Skin puncture test

> [!success]- 正確解答
> **A**

---

## 筆記與詳解

### (1) 快速破題

看到題目關鍵字「結核菌素 (Tuberculin)」與「注射」，應直覺聯想結核病篩檢的 **Mantoux test (曼托試驗)**。選項中的 Prick test 與 Patch test 皆為常見的過敏原測試，與結核桿菌感染無關。

⚠️ **錯誤糾正：題目敘述有誤！**
題目敘述為「經**皮下**注射 (Subcutaneous)」，這在醫學與護理操作上是**絕對錯誤**的觀念。Mantoux test 必須是**皮內注射 (Intradermal injection, ID)**，若打入皮下組織會導致偽陰性且無法準確判讀。

### (2) 關鍵字與知識點拆解

* **結核菌素 (Tuberculin) / PPD (Purified Protein Derivative)** → 知識點：自結核分枝桿菌提煉的蛋白質衍生物。注射後若曾受感染，會引發遲發型超敏反應，呈現局部突起的硬結 (Induration)[🖼️](https://www.google.com/search?q=Mantoux+test+induration+clinical+sign&tbm=isch&udm=2)。
* **Mantoux test (曼托試驗)** → 知識點：標準結核菌素測試法，採用**皮內注射**，於 48-72 小時後量測「硬結」直徑以判定陽性與否。
* **Skin prick test / Skin puncture test (皮膚針刺試驗)** → 知識點：用於測試 Type I 超敏反應（如花粉、塵蟎等 IgE 介導過敏）。表皮微刺後迅速產生膨疹與紅暈 (Wheal and Flare)[🖼️](https://www.google.com/search?q=skin+prick+test+wheal+flare+clinical+sign&tbm=isch&udm=2)。
* **Patch test (貼布試驗)** → 知識點：用於測試 Type IV 超敏反應中的「接觸性皮膚炎」（如金屬過敏）。敷貼過敏原 48 小時後觀察是否有局部紅斑或水泡 (Erythema/Vesicles)[🖼️](https://www.google.com/search?q=patch+test+erythema+vesicles+clinical+sign&tbm=isch&udm=2)。

### (3) 詳解與選項辨析

* **正確選項 A (Mantoux test)**：
利用純化蛋白衍生物 (PPD) 進行**皮內注射 (ID)**。測試原理為第四型超敏反應（遲發型、細胞介導免疫）。若受試者曾感染結核分枝桿菌（或曾接種 BCG 疫苗），體內記憶 T 細胞會聚集至注射處釋放細胞激素，引發巨噬細胞浸潤，形成可觸摸的硬結。
* **錯誤選項 B (Skin prick test) 與 D (Skin puncture test)**：
兩者為同義詞。將過敏原滴於前臂或背部，用針尖輕挑表皮但不流血。評估的是 Type I 過敏反應，反應極快，約 15-20 分鐘即可判讀，與結核病無關。
* **錯誤選項 C (Patch test)**：
雖然同樣屬於 Type IV 遲發型反應，但 Patch test 是將過敏原「貼」在皮膚表面，完全沒有「注射」行為，臨床專用於診斷接觸性皮膚炎，而非結核病。

### (4) 核心軸線與橫向比較

**縱向生理機轉軸線：**
結核菌暴露史 (自然感染/BCG疫苗) → 體內產生特異性記憶 T 細胞 → 實施 Mantoux test (皮內注射 PPD) → 抗原呈現細胞啟動記憶 T 細胞 → 釋放 IFN-γ 等細胞激素 → 48~72小時引發局部發炎與細胞浸潤 → 形成可觸測的**硬結 (Induration)**[🖼️](https://www.google.com/search?q=Mantoux+test+induration+clinical+sign&tbm=isch&udm=2) → 依直徑大小 (如 >5, >10, >15 mm) 判讀風險等級。

💡 **記憶口訣：**

* **Mantoux (曼托) 慢吞吞**：遲發型 (Type IV) 需等 48 小時，**皮內**打針防結核。
* **Prick (刺) 快腫**：微刺破皮，15 分鐘速發過敏 (Type I)。
* **Patch (貼) 皮膚**：貼布抓接觸性皮膚炎。
