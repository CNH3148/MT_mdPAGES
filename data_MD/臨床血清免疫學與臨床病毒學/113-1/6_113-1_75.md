## 筆記與詳解

### (1) 快速破題

看到題目考「病毒培養與檢測」並要找『錯誤』的 → 就要聯想到腸病毒屬（尤其是小兒麻痺病毒）是**歷史上最早被成功大規模細胞培養的病毒之一**（當年 Salk 研發疫苗就是依靠大量猴腎細胞培養）。它們在細胞培養中極易生長，且會產生典型、易於觀察的細胞病變 (CPE)[🖼️](https://www.google.com/search?q="Poliovirus"+"CPE"+cell+culture&tbm=isch&udm=2)。因此看到「無法以細胞株培養」這類違背病毒學歷史常識的敘述，即可秒殺選 A。

### (2) 關鍵字與知識點拆解

* **小兒麻痺病毒 (Poliovirus)** → 屬於微小核糖核酸病毒科 (Picornaviridae) 腸病毒屬 (Enterovirus)。為無外套膜的 (+)ssRNA 病毒，主要引起急性無力肢體麻痺 (AFP)[🖼️](https://www.google.com/search?q="acute+flaccid+paralysis"+clinical+sign&tbm=isch&udm=2)。
* **細胞株培養 (Cell culture) 與 CPE** → 此病毒極易在人類或靈長類細胞株（如 HeLa, Vero, PMK）生長，造成細胞變圓、折光性增加、最終皺縮剝落的典型細胞病變 (CPE)[🖼️](https://www.google.com/search?q="Enterovirus"+"CPE"+microscopy&tbm=isch&udm=2)。
* **VDPV (Vaccine-derived poliovirus)** → 疫苗衍生小兒麻痺病毒。口服沙賓疫苗 (OPV) 因屬「活體減毒」，病毒在腸道複製過程中有極低機率發生基因突變而恢復神經毒性。
* **Type 1、2、3** → 依據病毒衣殼蛋白中和抗原 (VP1) 差異，分為三種血清型。目前 Type 2 與 Type 3 野生株已宣布全球根除。
* **IRES (Internal ribosome entry site)** → 小兒麻痺病毒基因組的 5' 端非轉譯區 (UTR) 缺乏真核生物常見的 5' cap，而是利用 IRES 結構直接招募宿主核糖體進行轉譯。

### (3) 詳解

* **選項 (A) 錯誤 (本題答案)**：小兒麻痺病毒**非常容易**透過細胞株培養，且生長迅速，幾天內即可觀察到明顯的細胞病變 (CPE)[🖼️](https://www.google.com/search?q="Poliovirus"+"CPE"&tbm=isch&udm=2)。雖然現代醫學常使用 RT-PCR 進行快速分型，但「無法以細胞株培養」是完全錯誤的敘述。
* **選項 (B) 正確**：沙賓疫苗 (OPV) 是減毒活疫苗，能引發良好的腸道局部免疫，但病毒在人體腸道複製時，可能發生基因突變而恢復神經毒性，造成疫苗衍生小兒麻痺病毒 (VDPV) 感染。這也是目前許多國家（包含台灣）全面改用死毒疫苗 (IPV) 的主因。
* **選項 (C) 正確**：小兒麻痺病毒具有 3 種血清型（Type 1、2、3），疫苗必須涵蓋這些血清型才能提供全面保護。
* **選項 (D) 正確**：身為 Picornavirus，其 (+)ssRNA 缺乏 5'-cap 結構，但 5' UTR 具有特殊的二級結構 **IRES**。宿主核糖體可直接辨識並結合 IRES，不須依賴 cap-dependent 的轉譯機制即可合成病毒蛋白。

### (4) 核心軸線與橫向比較

**縱向病理邏輯：**
無外套膜 (+)ssRNA → **耐酸/耐膽鹽** → 糞口傳染 → 腸道淋巴組織 (Peyer's patches) 複製 → 釋放至血液 (微小病毒血症) → 突破血腦屏障/順向神經傳導侵犯**中樞神經前角運動神經元** → 下運動神經元受損 → **弛緩性麻痺 (Flaccid paralysis)**[🖼️](https://www.google.com/search?q="acute+flaccid+paralysis"+polio&tbm=isch&udm=2)。

**橫向比較：小兒麻痺疫苗 (Salk vs. Sabin) 國考必考重點**

| 比較維度 | 沙克疫苗 (Salk / IPV) | 沙賓疫苗 (Sabin / OPV) |
| --- | --- | --- |
| **疫苗類型** | **不活化 (Killed/Inactivated)** | **減毒活毒 (Live-attenuated)** |
| **給藥途徑** | **肌肉注射** (IM) | **口服** (Oral) |
| **誘導抗體** | 血液 **IgG** 為主 | 血液 IgG + **腸道局部 IgA** |
| **阻斷腸道傳染** | **差** (病毒仍能在腸道繁殖並隨糞便排出) | **極佳** (產生強大腸道免疫) |
| **VDPV 風險** | **無** (死毒無突變風險) | **有** (極低機率突變恢復毒力) |
| **免疫缺陷者適用** | **可** (安全) | **絕對禁忌** (恐引發重症) |

> 💡 **記憶口訣：**
> * **Salk** (沙克) = **S**afe (安全，死毒無VDPV)、**S**hot (注射)。
> * **Sabin** (沙賓) = **In** (吃進去/口服)、**In**testine (腸道局部 **IgA** 免疫極強)。
