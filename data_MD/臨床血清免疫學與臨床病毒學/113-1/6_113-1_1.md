## 題目

下列何種主要組織相容性複合體(MHC)表現在胎兒衍生的胎盤細胞(fetus-derived placental cells)，可以抑制NK細胞對胎盤細胞的毒殺?

HLA-F
HLA-G

> [!success]- 正確解答
> **D**

## 筆記與詳解

### (1) 快速破題

看到題目關鍵字「胎兒衍生的胎盤細胞 (fetus-derived placental cells)」與「抑制 NK 細胞毒殺」，就要直接聯想到 **「母胎免疫耐受性 (Maternal-fetal immune tolerance)」**。
在人類白血球抗原 (HLA) 系統中，傳統的 HLA-A、B 容易引發排斥；而胎盤為了避免被母體免疫系統攻擊，特化出表現「非傳統型 MHC class I」的機制。其中，最具代表性、專門負責抑制母體 NK 細胞的就是 **HLA-G**。

* **記憶口訣**：HLA-**G** 代表 **G**estation (懷孕) 或 **G**uard (保護胎兒)。

### (2) 專有名詞與關鍵字拆解

* **MHC (主要組織相容性複合體) / HLA (人類白血球抗原)** → 辨識「自我」與「非我」的抗原呈現分子。人類的 MHC 稱為 HLA。
* **胎盤細胞 (Placental cells)** → 主要是指滋養層細胞 (Trophoblasts)，在組織切片下可觀察到多核的合體滋養層 (syncytiotrophoblast)[🖼️](https://www.google.com/search?q="syncytiotrophoblast"+histology&tbm=isch&udm=2) 與單核的細胞滋養層 (cytotrophoblast)[🖼️](https://www.google.com/search?q="cytotrophoblast"+histology&tbm=isch&udm=2) 交錯排列。它們是胎兒與母體血液接觸的最前線。
* **NK 細胞 (自然殺手細胞)** → 具有「Missing-self」辨識機制：若細胞表面缺乏 MHC class I，NK 細胞就會將其視為異常並直接毒殺。
* **HLA-G** → 屬於非傳統型 MHC class I (MHC class Ib)，多型性極低，專門結合並活化 NK 細胞上的抑制性受體 (如 ILT2, KIR2DL4)。

### (3) 詳解與選項辨析

胎兒擁有來自父系的基因，對母體而言屬於「半同種異體移植物 (semi-allograft)」。為了不被母體免疫系統排斥，胎盤細胞的免疫策略如下：

1. **關閉傳統 MHC-I (避免 T 細胞攻擊)**：胎盤細胞表面**不表現**高度多型性的 HLA-A 與 HLA-B。這能避免母體的 CD8+ 胞毒性 T 細胞將胎盤細胞認作外來物。
2. **開啟非傳統 MHC-I (避免 NK 細胞攻擊)**：由於缺乏 HLA-A/B，胎盤細胞會觸發 NK 細胞的「Missing-self」攻擊機制。為了解決這個問題，胎盤細胞大量表現 **HLA-G** (以及部分 HLA-E、HLA-C)。HLA-G 能對母體的 NK 細胞發出強烈的抑制訊號，阻止毒殺反應。

* **(A) HLA-A (錯誤)**：屬於傳統型 MHC class Ia，具有極高的多型性。若胎盤表現 HLA-A，會被母體 CD8+ T 細胞辨識為外來抗原並引發強烈排斥。
* **(B) HLA-D (錯誤)**：實際上是指 MHC class II (包含 HLA-DP, DQ, DR)。主要表現在抗原呈現細胞 (APC) 上，活化 CD4+ 輔助型 T 細胞，與胎盤抑制 NK 細胞無直接關聯。
* **(C) HLA-F (錯誤)**：同屬非傳統型 MHC class Ib，雖然在懷孕期間也有表現，但其主要功能偏向於發炎反應調節與免疫受體的結合，**HLA-G** 才是公認負責抑制 NK 細胞毒殺、建立母胎耐受性的絕對核心分子。
* **(D) HLA-G (正確)**：胎盤滋養層細胞的標誌性抗原，專職結合 NK 細胞抑制性受體，維持懷孕穩定。

### (4) 邏輯因果與重點統整

* **母胎免疫耐受性機轉核心軸線：**
胎兒帶父系基因 (半同種異體) → 胎盤滋養層↓ HLA-A/B**表現 → 逃避母體 **CD8+ T** 細胞毒殺 → 觸發母體 **NK** 細胞「Missing-self」機制 (企圖攻擊) → 胎盤滋養層代償性**↑ HLA-G表現 → 結合 NK 細胞抑制性受體 → 抑制毒殺作用 → **胎兒成功存活**。
