## 題目

下列關於淋巴球微毒殺試驗(lymphocyte microcytotoxicity test)的特點,何者錯誤?

A.待測樣本為捐贈者的細胞
B.使用T細胞為待測樣本時,可評估第二型組織抗原相容
C.能被應用於檢測排斥反應
D.欲增加檢測靈敏度,可加做anti-human globulin (AHG)抗體步驟

> [!success]- 正確解答
> **B**

---

## 筆記與詳解

### (1) 快速破題

看到「淋巴球微毒殺試驗 (LCT)」與「T細胞/第二型組織抗原」，直覺必須聯想到 HLA (人類白血球抗原) 分布的經典考點：**HLA Class I 表現於所有有核細胞 (包含 T 細胞、B 細胞)**；而 **HLA Class II 僅表現於抗原呈現細胞 (APC，如 B 細胞、巨噬細胞)**。因此，若要評估「第二型組織抗原 (Class II)」，必須分離出 **B 細胞** 來作為標的細胞。題目敘述用 T 細胞評估第二型，明顯張冠李戴，故選 B。

### (2) 關鍵字 → 知識點

* **淋巴球微毒殺試驗 (Lymphocyte microcytotoxicity test, LCT)**：又稱補體依賴性毒殺試驗 (CDC)。利用特異性抗體結合細胞表面 HLA 抗原，加入補體後導致細胞膜破裂，死細胞失去膜完整性而染上染劑（如 Eosin Y 或 Trypan blue，鏡下呈現深色、脹破的細胞型態 [🖼️](https://www.google.com/search?q=trypan+blue+dead+lymphocyte+microscopy&tbm=isch&udm=2)），藉此判定抗原抗體反應。
* **T 細胞 vs B 細胞**：T 細胞僅具 HLA Class I；B 細胞同時具備 HLA Class I 與 Class II。分離特定淋巴球可分別檢測不同型別的 HLA。
* **排斥反應 (Rejection)**：移植前利用此試驗進行「交叉配合試驗 (Crossmatch)」，檢測受贈者血清中是否含有對抗捐贈者的**預造抗體 (DSA)**，以預防最致命的**超急性排斥 (Hyperacute rejection)**。
* **AHG (Anti-human globulin)**：針對效價較低或不易活化補體的抗體（如部分 IgG），加入 AHG 促進抗體交聯，大幅提升補體活化效率與檢驗靈敏度。

### (3) 詳解

* **(A) 待測樣本為捐贈者的細胞：正確。**
在器官移植前的交叉配合試驗 (Crossmatch) 中，標準作法是抽取「**捐贈者**」的淋巴球作為標的細胞，並與「**受贈者**」的血清（評估是否有潛在抗體）混合。
* **(B) 使用T細胞為待測樣本時,可評估第二型組織抗原相容：錯誤，本題解答。**
T 細胞表面**只有 HLA Class I** (HLA-A, B, C)。若要評估 HLA Class II (HLA-DR, DQ, DP)，必須使用表現 Class II 的抗原呈現細胞，也就是要使用 **B 細胞** 作為待測樣本。
* **(C) 能被應用於檢測排斥反應：正確。**
這項試驗是實體器官移植前的黃金標準之一。若測試結果為陽性（細胞大量死亡），代表受贈者體內已有對抗捐贈者 HLA 的抗體，這會導致移植後立刻發生**超急性排斥反應**，臨床上視為移植絕對禁忌症。
* **(D) 欲增加檢測靈敏度,可加做anti-human globulin (AHG)抗體步驟：正確。**
此技術進階版稱為 AHG-CDC。受贈者血清中的抗體若為 IgG，其活化補體的能力往往弱於 IgM。加入 AHG 後，AHG 會結合在 IgG 尾端形成橋接，擴大空間立體結構，強力啟動古典補體路徑，使死細胞特徵（細胞染劑攝入與膨脹 [🖼️](https://www.google.com/search?q=%22complement-dependent+cytotoxicity%22+eosin+Y+staining&tbm=isch&udm=2)）更為明顯，有效降低偽陰性。

### (4) 邏輯因果與橫向比較

**【縱向生理機轉與檢驗邏輯】**
受贈者曾因輸血/懷孕/移植產生 HLA 抗體 → 抽取**捐贈者**淋巴球 + **受贈者**血清混合 → 抗原與抗體結合 → 加入兔補體 (Rabbit complement) → 補體活化形成膜攻擊複合體 (MAC) → 細胞膜穿孔 → 染劑進入死細胞內 → 顯微鏡下觀察**染色細胞比例** → 評估**超急性排斥**風險。

**【記憶口訣】**

* **1T 2B**：Class **1** 用 **T** cell 測 (其實 T、B 都有，但分離 T 比較純粹)；Class **2** 只能用 **B** cell 測。
* **CDCA**：**CDC** 加 **A**HG = 靈敏度 **↑**。

**【橫向比較表格：HLA 交叉配合試驗方法比較】**

| 比較維度 | 標準 CDC (LCT) | AHG-CDC | 流量細胞儀交叉配合 (FCXM) |
| --- | --- | --- | --- |
| **檢驗原理** | 抗原抗體反應 + **補體活化** | 加入 AHG **增強補體活化** | 螢光抗體標記 + **雷射偵測** |
| **靈敏度** | **最低** | 中等 | **最高** (約為 CDC 的 10-100倍) |
| **細胞存活率要求** | **高** (需活細胞才能看出染劑通透性差異) | **高** | 低 (可排除死細胞干擾) |
| **偵測抗體類型** | 僅限**能活化補體**的抗體 (主要是 IgM, 部分 IgG) | **能活化補體**的抗體 + 弱效 **IgG** | **所有 IgG** (無論是否活化補體) |
| **假陽性風險** | 易受自體抗體或非 HLA 抗體干擾 | 較高 | 低 |
| **臨床意義** | 陽性絕對**不可移植** (超急性排斥) | 抓出低效價 IgG | 極高靈敏度，有助於高風險病患評估 |
