## 題目

下列何種細胞不會表現主要組織相容性複合體(MHC) class I?

A.Red blood cells
B.Macrophages
C.Neutrophils
D.Kidney cells

> [!success]- 正確解答
> **A**

---

## 筆記與詳解

### (1) 快速破題

看到題目問「不會表現 MHC class I」的細胞，最直接的解題思路就是聯想「MHC class I 分布於幾乎所有『有核細胞』」**。因此，只要在選項中尋找**「無核細胞」即可。人體中最典型的無核細胞就是成熟的紅血球，故答案秒選 A。

### (2) 關鍵字與知識點拆解

* **MHC class I (主要組織相容性複合體第一型)** → 分布於全身所有**有核細胞**及**血小板**表面；功能為向 CD8+ 毒殺型 T 細胞呈現**內生性抗原**（如病毒蛋白、腫瘤突變蛋白）。
* **Red blood cells (成熟紅血球)** → 在骨髓成熟過程中會將細胞核排出，成為無核的雙凹圓盤狀細胞[🖼️](https://www.google.com/search?q=mature+red+blood+cell+blood+smear&tbm=isch&udm=2)；因**無細胞核**，故不表現 MHC I。
* **Macrophages (巨噬細胞) / Neutrophils (嗜中性白血球)** → 皆為具備明顯細胞核型態的白血球（如多葉核[🖼️](https://www.google.com/search?q=neutrophil+blood+smear+morphology&tbm=isch&udm=2)），屬有核細胞，必然表現 MHC I。
* **Kidney cells (腎臟組織細胞)** → 一般體細胞，具細胞核，表現 MHC I 以便受到感染時能呼叫免疫系統清除自己。

### (3) 詳解與選項辨析

* **A. Red blood cells (正確)**：成熟的紅血球為了極大化攜帶血紅素與氧氣的空間，捨棄了細胞核與大部分胞器。因為缺乏細胞核的轉錄與表現機制，紅血球表面不帶有 MHC class I 分子。這也是為何某些感染紅血球的病原（如瘧原蟲[🖼️](https://www.google.com/search?q=Plasmodium+falciparum+ring+form+blood+smear&tbm=isch&udm=2)）能藉此躲避毒殺型 T 細胞 (CD8+ T cell) 的偵測與毒殺。
* **B. Macrophages (錯誤)**：巨噬細胞不僅是有核細胞（必然有 MHC I），它更是**專業抗原呈現細胞 (APC)**。這意味著它同時具備 MHC class I 與 MHC class II，能全方位地將抗原呈現給免疫系統。
* **C. Neutrophils (錯誤)**：屬於有核的白血球，具備 MHC class I。
* **D. Kidney cells (錯誤)**：腎臟細胞是普通的體細胞（有核），理所當然會表現 MHC class I。若腎臟細胞被病毒感染或發生癌變，會透過 MHC I 將異常胜肽展現於細胞膜表面，引導 CD8+ T 細胞來啟動細胞凋亡機制。

### (4) 核心軸線與橫向比較表

**核心邏輯因果：** 細胞具備細胞核 → 轉錄轉譯 MHC I 蛋白 → 表現於細胞膜表面 → 結合胞內突變或病毒抗原 → 活化 CD8+ T 細胞進行胞毒作用。相反地，無核細胞 (如 RBC) 缺乏此機制，因此仰賴自然殺手細胞 (NK cell) 來巡邏「缺乏 MHC I」的異常狀況（但成熟 RBC 被 NK cell 免疫耐受，不會被攻擊）。

**💡 記憶口訣：** **「1×8 = 2×4 = 8」** (MHC **I** 搭配 CD**8**；MHC **II** 搭配 CD**4**)

| 比較維度 | MHC class I | MHC class II |
| --- | --- | --- |
| **表現細胞** | 所有**有核細胞**、血小板 | 僅限**專業 APC** (樹突細胞、巨噬細胞、B 細胞) |
| **抗原來源** | **內生性** (Endogenous；如病毒、腫瘤) | **外生性** (Exogenous；如吞噬的細菌) |
| **抗原裂解路徑** | **Proteasome** (蛋白酶體) | **Lysosome** (溶體) |
| **對接 T 細胞** | **CD8+** Cytotoxic T cells (Tc) | **CD4+** Helper T cells (Th) |
| **分子結構** | **α 鏈** (3個 domains) + **β2-microglobulin** | **α 鏈** (2個 domains) + **β 鏈** (2個 domains) |
| **結合胜肽長度** | 較短 (**8~10** 個胺基酸) | 較長 (**13~18** 個胺基酸) |
