## 題目

使用peptide-MHC class I tetramer的技術,可以定量下列何種專一性免疫細胞的數量?

A.CD4 T細胞
B.B細胞
C.NK細胞
D.CD8 T細胞

> [!success]- 正確解答
> **D**

---

## 筆記與詳解

### (1) 快速破題

看到題目出現「MHC class I」，請直覺反射免疫學核心法則「Rule of 8」：**MHC-I 配對 CD8** (1 x 8 = 8)，**MHC-II 配對 CD4** (2 x 4 = 8)。四聚體 (Tetramer) 技術是利用人工合成的螢光標記 MHC-peptide 複合體來捕捉具專一性 TCR 的 T 細胞，因題目明示「MHC class I」，故唯一能與之結合並被定量的就是 CD8 T 細胞。

### (2) 關鍵字與知識點拆解

* **Peptide-MHC tetramer (四聚體技術) → 專一性 T 細胞定量**：單一 MHC 與 TCR 結合力 (affinity) 極弱，易脫落；將四個 MHC 分子結合在同一個骨架 (通常為 Streptavidin) 上，可大幅提升結合性 (avidity)，使螢光標記的複合體能穩定結合在特定 T 細胞上，再透過流式細胞儀螢光散布圖[🖼️](https://www.google.com/search?q=%22flow+cytometry%22+tetramer+plot&tbm=isch&udm=2) 進行精準計數。
* **MHC class I (第一型主要組織相容性複合體) → 內生性抗原**：表現於所有有核細胞表面，負責將細胞內處理過的外來胜肽（如病毒、腫瘤抗原）呈現給毒殺性 T 細胞。
* **CD8 T 細胞 (Tc) → 毒殺作用**：利用 TCR 辨識抗原胜肽，並利用 CD8 共受體結合 MHC class I 分子。
* **CD4 T 細胞 (Th) → 輔助作用**：利用 TCR 辨識抗原胜肽，並利用 CD4 共受體結合 MHC class II 分子。

### (3) 詳解

* **(D) CD8 T細胞：正確。** CD8 T 細胞表面的 TCR 專門辨識 MHC class I 呈現的抗原胜肽，加入螢光標記的 Peptide-MHC class I tetramer 後，可專一性結合並被定量。
* **(A) CD4 T細胞：錯誤。** 受限於 MHC 限制性 (MHC restriction)，CD4 T 細胞只認得 MHC class II。若要定量特定 CD4 T 細胞，必須使用 Peptide-MHC **class II** tetramer。
* **(B) B細胞：錯誤。** B 細胞受體 (BCR) 直接辨識**原生構形 (native conformation)** 的游離抗原或病原體表面抗原，不需要經過抗原呈現細胞 (APC) 處理，也不受 MHC 分子限制。
* **(C) NK細胞：錯誤。** 自然殺手細胞屬於先天免疫，表面**沒有** TCR，不具抗原專一性。其辨識機制為「Missing self 假說」——負責毒殺**缺乏 MHC class I** 表現的異常細胞，因此無法用 MHC tetramer 技術定量。

### (4) 核心邏輯與橫向比較

**縱向聯想：**
細胞內感染/突變 → 蛋白質經 Proteasome 降解 → 裝載於 ER 內的 **MHC class I** → 移至細胞膜表面呈現 → 加入螢光 Peptide-MHC-I tetramer → 穩定結合 **CD8+ T 細胞**的 TCR → 流式細胞儀雷射激發螢光 → 專一性細胞計數。

**記憶口訣：**

* **Rule of 8**：1(MHC-I) × 8(CD8) = 8；2(MHC-II) × 4(CD4) = 8。

**免疫細胞受體與抗原辨識比較表**

| 比較維度 | CD8+ T 細胞 (Tc) | CD4+ T 細胞 (Th) | B 細胞 | NK 細胞 |
| --- | --- | --- | --- | --- |
| **辨識受體** | TCR + CD8 | TCR + CD4 | BCR (膜表 IgG/IgM) | KIR, KAR (**無** TCR) |
| **MHC 限制性** | **MHC class I** | **MHC class II** | **無限制** | 辨識 **MHC-I 缺失** |
| **抗原來源** | **內生性** (病毒、腫瘤) | **外源性** (細菌、寄生蟲) | 原生游離抗原 | 無專一抗原 |
| **定量技術** | Peptide-**MHC-I** tetramer | Peptide-**MHC-II** tetramer | 螢光標記原生抗原 | 特異性表面抗原 (CD56+, CD3-) |
| **主要功能** | 分泌穿孔素/顆粒酶毒殺 | 分泌細胞激素調控免疫 | 分化漿細胞產抗體 | 先天免疫直接毒殺 |
