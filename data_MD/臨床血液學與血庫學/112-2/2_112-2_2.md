## 題目

有關惡性貧血(pernicious anemia)的敘述,下列何者錯誤?

A.病人的transcobalamin II (TC-II)減少
B.是一種慢性疾病
C.Vitamin B12缺乏
D.病人的IF(intrinsic factor)減少

> [!success]- 正確解答
> **A**

---

## 筆記與詳解

### (1) 快速破題

看到題目「惡性貧血 (Pernicious anemia)」，必須立刻條件反射想到核心病理機轉：**「自體免疫破壞胃壁細胞 → 內在因子 (IF) 缺乏 → 維生素 B12 無法吸收」**。
選項中 B、C、D 皆完美符合上述典型特徵；而 A 提到的 Transcobalamin II (TC-II) 是血液中負責「運送」B12 的蛋白質，其先天性缺乏屬於另一種罕見遺傳疾病，與惡性貧血的「吸收障礙」無關，故錯誤選項為 A。

### (2) 關鍵字與知識點拆解

* **惡性貧血 (Pernicious anemia)** → **自體免疫慢性病**：產生抗胃壁細胞抗體 (Anti-parietal cell Ab) 或抗內在因子抗體 (Anti-IF Ab)。
* **IF (Intrinsic factor, 內在因子)** → **B12 吸收的鑰匙**：由胃壁細胞分泌，與 B12 結合後，才能在「迴腸末端」被吸收。惡性貧血患者的 IF 會顯著減少。
* **Vitamin B12 (Cobalamin)** → **DNA 合成與神經髓鞘形成關鍵**：缺乏時導致巨母紅血球性貧血 (Megaloblastic anemia) 與不可逆的神經病變。
* **Transcobalamin II (TC-II)** → **B12 血液運輸蛋白**：負責將 B12 運送至骨髓等組織。其減少/缺乏屬於遺傳性疾病 (TC-II deficiency)，並非惡性貧血的特徵。

### (3) 詳解與選項辨析

* **(A) 病人的 transcobalamin II (TC-II) 減少 (錯誤，本題答案)**：惡性貧血的根本問題在於「腸道無法吸收 B12 (因缺乏 IF)」，而不是「血液運送蛋白 (TC-II) 減少」。TC-II 缺乏症是另一種體染色體隱性遺傳疾病，患者體內 B12 總量可能正常，但無法運送至細胞利用。
* **(B) 是一種慢性疾病 (正確)**：由自體免疫引起，屬於慢性且漸進性發展的疾病，若不補充 B12 (通常需藉由肌肉注射，避開腸道吸收障礙) 將會持續惡化。
* **(C) Vitamin B12 缺乏 (正確)**：這是惡性貧血最直接的結果。因為沒有 IF 的保護與引導，飲食中的 B12 無法在腸道被吸收。
* **(D) 病人的 IF (intrinsic factor) 減少 (正確)**：自體免疫抗體破壞胃壁細胞，導致 IF 分泌量大幅下降。

### (4) 核心軸線與邏輯因果

* **縱向病理邏輯**：
自體免疫反應 (抗體攻擊胃壁細胞) → **↓ 內在因子 (IF)** ＆ 胃酸缺乏 → 迴腸無法吸收 B12 → **↓ 體內 Vit B12** → DNA 合成受阻 ＆ 髓鞘受損 → **巨母紅血球性貧血** ＆ **神經學症狀**。
* **記憶口訣**：**「惡胃內缺B12，神經大腸有問題」**
(**惡**性貧血 → **胃**壁細胞受損 → **內**在因子缺乏 → **B12** 吸收不良 → 併發**神經**症狀 + 血液抹片見**大**紅血球與多葉狀核嗜中性白血球 + 吸收位點在迴**腸**)。
