## 題目

11.下列何種方法不適用於檢測 paroxysmal nocturnal hemoglobinuria（PNH）？

Osmotic fragility test
Sucrose hemolysis test
Acid Ham's test
Flow cytometry immunofluorescent staining

> [!success]- 正確解答
> **A**

---

## 筆記與詳解

### (1) 快速破題

看到題目測驗「PNH (陣發性夜間血紅素尿症)」的檢驗方法並要求選出「**錯誤 / 不適用**」者，必須立刻想到：PNH 的致病核心是**紅血球對「補體」異常敏感**，因此傳統檢驗皆與活化補體（如酸性環境、低離子強度）有關，現代則依賴流式細胞儀。而選項中的 **Osmotic fragility test (滲透壓脆弱性試驗)** 測量的是紅血球對「低張溶液」的抵抗力，這是**遺傳性球形紅血球增多症 (Hereditary Spherocytosis, HS)** 的專屬確診試驗，與補體機制完全無關，故秒選 (A)。

### (2) 關鍵字與知識點拆解

* **Paroxysmal nocturnal hemoglobinuria (PNH):** 後天性 *PIGA* 基因突變，導致造血幹細胞缺乏 GPI 錨定蛋白（**CD55**, **CD59**），使紅血球易受補體攻擊而破裂。臨床典型表徵為早晨第一泡尿呈深褐色/可樂色尿液 [🖼️](https://www.google.com/search?q="hemoglobinuria"+urine+clinical+sign&tbm=isch&udm=2)。
* **Osmotic fragility test (OFT):** 測試紅血球膜在低張食鹽水中的耐受度。若提早破裂（耐受度↓），代表紅血球表面積/體積比下降，專用於診斷 HS。
* **Sucrose hemolysis test (蔗糖水試驗):** 利用低離子強度的蔗糖溶液促進補體與紅血球結合。為 PNH 的**篩檢試驗 (Screening test)**。
* **Acid Ham's test (酸性血清溶血試驗):** 利用弱酸性 (pH 6.2) 環境活化補體替代途徑。為 PNH 的**傳統確認試驗 (Confirmatory test)**。
* **Flow cytometry (流式細胞術):** 利用螢光抗體標定，直接量測細胞表面是否缺乏 CD55、CD59。為目前診斷 PNH 的**黃金標準 (Gold standard)**。

### (3) 詳解

* **(A) Osmotic fragility test (正確答案，因為不適用)：** 此試驗與補體活化無關，而是檢測紅血球膜的物理結構穩定性。HS 患者因紅血球骨架蛋白異常，形成球形紅血球 [🖼️](https://www.google.com/search?q="spherocytes"+blood+smear+wright+stain&tbm=isch&udm=2)，對低張溶液極度敏感，OFT 會呈現**陽性（脆性增加）**。PNH 患者的紅血球形態通常正常，OFT 無異常。
* **(B) Sucrose hemolysis test：** 適用。低離子強度的蔗糖溶液會增強補體對 PNH 紅血球的破壞，產生溶血。敏感度高但特異性較低，因此作為第一線篩檢。
* **(C) Acid Ham's test：** 適用。模擬人體夜間睡眠時二氧化碳蓄積所造成的微酸性環境，此環境會活化補體。PNH 紅血球因缺乏 CD59 無法抑制 MAC (膜攻擊複合體) 形成，導致大量溶血。特異性高，為傳統確診工具。
* **(D) Flow cytometry immunofluorescent staining：** 適用。直接且精準定量缺乏 GPI-anchored proteins (如 CD55、CD59、FLAER) 的白血球或紅血球比例，不僅能確診，還能評估克隆大小與疾病嚴重度，已完全取代 Ham's test 成為現代臨床標準。

### (4) 核心機轉軸線與記憶口訣

* **縱向病理軸線 (PNH)：**
*PIGA* 基因突變 → GPI anchor 合成障礙 → 紅血球缺乏補體調節蛋白 (**CD55 / CD59**) → 睡眠時呼吸變慢、$CO_2$ 蓄積 → 血液 pH 值微幅下降 (微酸) → 補體替代途徑活化 → 膜攻擊複合體 (MAC) 形成並穿刺紅血球 → **血管內溶血 (Intravascular hemolysis)** → 釋放游離血紅素 → 腎臟過濾排出產生**血紅素尿 (Hemoglobinuria)**。
* **記憶口訣：**
* **PNH 檢驗「流水酸」**（**流**式細胞儀、蔗糖**水**、**酸**性 Ham's test）。
* **球形紅血球**才怕**滲透**（Osmotic fragility test 專屬 HS）。



### (5) PNH 與 HS 核心鑑別比較表

| 比較維度 | Paroxysmal Nocturnal Hemoglobinuria (PNH) | Hereditary Spherocytosis (HS) |
| --- | --- | --- |
| **疾病性質** | **後天**獲得性基因突變 (*PIGA*) | **先天**遺傳性基因突變 (體染色體顯性) |
| **缺陷機轉** | 缺乏 **GPI anchor** → 無法固定補體調節蛋白 | 紅血球**骨架蛋白** (如 Spectrin, Ankyrin) 缺陷 |
| **關鍵缺失蛋白** | **CD55** (DAF), **CD59** (MIRL) | Ankyrin, Band 3, Spectrin |
| **溶血機制** | 對**補體**異常敏感 (MAC 攻擊破壞) | 對**滲透壓**耐受下降 (脾臟巨噬細胞吞噬) |
| **溶血場所** | **血管內**溶血 (Intravascular) | **血管外**溶血 (Extravascular - 脾臟) |
| **視覺特徵 / 抹片** | 早晨**可樂色血紅素尿** [🖼️](https://www.google.com/search?q="hemoglobinuria"+urine+clinical+sign&tbm=isch&udm=2) / 尿液含鐵血黃素顆粒 | **球形紅血球** (Spherocytes) [🖼️](https://www.google.com/search?q="spherocytes"+blood+smear+wright+stain&tbm=isch&udm=2) 無中心淡染區 |
| **篩檢試驗** | **Sucrose** hemolysis test | MCHC **↑** (細胞脫水濃縮) |
| **確診試驗** | **Flow cytometry** (現代) / **Acid Ham's** (傳統) | **Osmotic fragility test** (OFT) **陽性** (脆性↑) |
