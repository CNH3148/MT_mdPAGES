## 核心趨勢與高頻考點
本單元的核心命題邏輯始終圍繞著「**形態-機轉-疾病**」三者間的強連結。考生必須能從血液抹片的影像或文字描述中，精準辨識出特定的紅血球形態或包涵體，並迅速連結其背後的形成機制（如：膜缺陷、骨髓浸潤、脾功能、氧化壓力）與對應的臨床疾病。

**高頻考點分析與預測：**

1.  **影像判讀是王道**：歷屆考題反覆出現血液抹片圖，直接指向特定細胞或包涵體要求辨識。未來趨勢將更強調「鑑別診斷」，例如在同一視野下呈現多種異常形態（如海洋性貧血的靶狀血球與嗜鹼性點彩），測驗考生綜合判斷能力。
2.  **脾臟功能是隱藏主角**：脾臟的 Pitting (去核/修飾) 與 Culling (清除) 功能是理解包涵體（特別是 Howell-Jolly bodies）為何出現的關鍵。考題常以「脾切除後」的情境出現，預測未來可能結合自體免疫疾病造成的「功能性無脾」概念出題。
3.  **染色法的對應關係**：這是絕對的考點。考生必須牢記「**哪種異常、用哪種染色法看**」。最經典的對比是：
    *   **一般染色 (Wright/Liu's)**：可見 Howell-Jolly、Basophilic stippling、Pappenheimer bodies。
    *   **超活體染色 (Supravital stain)**：專門用來看 **Heinz bodies** 和 **Reticulocyte**。
    *   **特殊染色 (Prussian blue)**：確認 **Pappenheimer bodies** 的鐵質成分。
    這個邏輯鏈 ( **異常** → **成分** → **對應染色法** ) 會反覆以不同形式出現。
4.  **因果鏈的推理**：考題不再是單點記憶，而是測驗推理能力。例如：
    *   **生理機轉 → 形態變化 → 疾病**
    *   鉛中毒 → 抑制RNA降解酶 → RNA聚集 → **嗜鹼性點彩 (Basophilic stippling)**
    *   G6PD缺乏 → 氧化壓力 → 血紅素變性 → **Heinz bodies** → 被脾臟巨噬細胞咬掉 → **Bite cell**
