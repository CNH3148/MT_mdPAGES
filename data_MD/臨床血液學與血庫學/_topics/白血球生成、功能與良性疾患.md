## 類群定義

> 涵蓋各類白血球（顆粒球、單核球、淋巴球）的正常發育、成熟過程與生理功能（如吞噬、趨化、抗原呈現）。包含非腫瘤性的白血球數量異常（如嗜中性球增多／減少、淋巴球增多）與功能/形態異常的遺傳性疾病（如Pelger-Huët anomaly, Chédiak-Higashi syndrome, chronic granulomatous disease, LAD-I）。亦跨學科整合基礎造血生理、凝血常規數值與部分血庫抗原學。

## 類群說明

身為資深醫事檢驗師，我為您分析本章節的歷屆與最新考題。本類群除了經典的良性白血球疾患外，近年更大量融合了「表面抗原標記 (CD markers) 的精準判讀」以及「跨章節血液學基本功」，考生必須具備高度的系統性整合能力。

## 核心趨勢與高頻考點
綜合分析歷屆考題，本單元命題趨勢高度集中在「**鑑別診斷**」與「**功能、形態與表面標記的連結**」。出題者期望考生能從一張血液抹片、一個檢測數據或一段臨床描述，反推出背後的細胞生理或病理機制。

1.  **反應性 vs. 腫瘤性：** 「**#類白血病反應 (Leukemoid Reaction)**」是絕對的命題核心。考題圍繞其與慢性骨髓性白血病 (CML) 的鑑別，形成一個黃金考點組合：
    *   **生理機轉 (嚴重感染/發炎) → 血液學變化 (白血球↑、核左移、毒性變化) → 關鍵檢測 (LAP score ↑↑) → 鑑別診斷 (排除 CML)**。

2.  **遺傳性功能/形態異常：** 重點在於「**基因缺陷 → 細胞異常 → 臨床表現/診斷測試**」的因果鏈。
    *   **CGD (慢性肉芽腫病)**：**NADPH oxidase** 基因缺陷 → 呼吸爆發失敗 (殺菌力↓) → **NBT 試驗陰性**。
    *   **Chédiak-Higashi 症候群**：**LYST** 基因缺陷 → 溶酶體運輸融合障礙 → 細胞內**巨大顆粒** [🖼️](https://www.google.com/search?q=Chediak-Higashi+syndrome+giant+granules+blood+smear&tbm=isch&udm=2)、部分白化症。
    *   **Pelger-Huët 異常**：Lamin B receptor 基因變異 → 核分葉不全 → **雙葉核 (夾鼻眼鏡)** [🖼️](https://www.google.com/search?q=Pelger-Huet+anomaly+pince-nez+neutrophil&tbm=isch&udm=2)，但**功能正常** (異型合子多見此表現)。
    *   **LAD-I (白血球黏著缺乏症第一型)**：**β2-integrin (CD18)** 基因缺陷 → 嗜中性球無法穿透血管壁 → 臍帶延遲脫落，但其**吞噬與殺菌功能依然正常**。

3.  **傳染性單核球增多症 (Infectious Mononucleosis)**：此為獨立的明星考點，常見的陷阱題設計如下：
    *   **病原體 (EBV) → 感染標的 (B 細胞) → 免疫反應 (CD8+ T 細胞活化增生) → 抹片所見 (反應性/非典型淋巴球)** [🖼️](https://www.google.com/search?q=reactive+lymphocyte+Downey+cell+blood+smear&tbm=isch&udm=2)。必須清楚抹片上看到的並非被感染的 B 細胞，而是反應性的 T 細胞。

4.  **細胞發育與造血生理：** 
    *   **成熟特徵分水嶺**：顆粒球發育中，**#骨髓球 (Myelocyte)** 是「有絲分裂池 (mitotic pool)」的最後一個階段，過了此期便失去增生能力；同時它也是「**次級顆粒 (specific granules)**」最早開始出現的階段。
    *   **生長因子對應**：**Stem cell factor (SCF)** 最直接作用於最源頭的造血幹細胞 (HSCs)；**EPO** 主要作用於 CFU-E → Pronormoblast 階段；**IL-5** 則專責誘導嗜酸性球 (Eosinophil)。

5.  **表面標記 (CD Markers) 與精準分型：** 近年考題在此越鑽越深。
    *   **正常比例**：周邊血中 T 細胞約佔 **80%**，B 細胞約佔 **20%**。
    *   **特化 T 細胞**：Regulatory T cell (Treg) 的標記為 **CD4+ / CD25+ / FOXP3+ / TCRαβ**。
    *   **白血病精準分型**：Early T-cell precursor ALL (ETP-ALL) 的標記為 **CD7+ / CD1a- / CD8-**。
    *   **特異抗原定位**：顆粒白血球抗原 HNA-1 是位於 **FcRIIIb (CD16b)** 分子上。
    *   **單核球**：主要標記為 **CD14**，抹片呈現灰藍色胞質與腎形核 [🖼️](https://www.google.com/search?q=Monocyte+blood+smear&tbm=isch&udm=2)。

## 未來考點預測與易混淆陷阱
*   **脾臟功能的陷阱**：考題常故意混淆脾臟狀態與血球數量的關係。**脾功能亢進 (Hypersplenism)** 會因過度吞噬造成破壞增加，引發 #Neutropenia；而切除脾臟或 **脾功能低下 (Hyposplenism)** 則會導致血球無法被正常清除，數量反而上升，**絕對不會**造成低下。
*   **血清學測試的陷阱**：診斷傳染性單核球增多症的 Monospot test 偵測的是 **異嗜性抗體 (Heterophile antibody)**，並非 EBV 的特異性抗體。
*   **細胞棲地的陷阱**：雖然嗜鹼性球 (Basophil) 與肥大細胞 (Mast cell) 都有過敏相關的顆粒，但 Basophil 釋放至周邊血時**已是成熟型**；Mast cell 則**正常情況下絕對不會出現在血液循環中**，只存在於組織。
*   **跨學科鑑別診斷**：要區分遺傳性球性紅血球症 (HS) 與自體免疫溶血 (AIHA)，最重要的武器是 **直接抗球蛋白試驗 (DAT)**。若題幹涉及正常凝血數值，務必記住 Fibrinogen (纖維蛋白原) 正常值為 **1.5～4.0 g/L**。
