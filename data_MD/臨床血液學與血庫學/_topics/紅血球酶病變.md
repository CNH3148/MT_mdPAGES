## 類群定義

> 涵蓋因紅血球代謝路徑中特定酵素的遺傳性缺乏所引起的溶血性貧血。核心知識點為G6PD缺乏症和丙酮酸激酶（PK）缺乏症，測驗其致病機轉、遺傳方式、臨床表現（特別是氧化壓力引發的溶血）及實驗室診斷特徵（如Heinz bodies）。

## 類群說明

好的，身為資深醫檢師，我將為您分析「紅血球酶病變」此一國考知識點。歷屆考題高度集中在 G6PD 缺乏症與丙酮酸激酶 (PK) 缺乏症的比較，特別是其根本生化機轉如何導致不同的臨床與檢驗表現。

---

## 核心趨勢與高頻考點
本單元的出題邏輯極為清晰，幾乎完全圍繞著兩大代謝路徑的「因果關係」進行命題。考生必須建立以下的思維鏈：

1.  **G6PD 缺乏症的因果鏈**：
    *   **生理機轉**：G6PD 是 **己糖單磷酸分流 (HMP shunt)** 的關鍵酵素 → 缺乏 G6PD → **NADPH** 生成不足 → 無法還原氧化態的穀胱甘肽 (GSSG) 成還原型穀胱甘肽 (GSH) → 紅血球對**氧化壓力**的防禦能力下降。
    *   **臨床觸發**：接觸氧化性藥物、感染或食用蠶豆 (Fava beans) → 氧化壓力驟增。
    *   **細胞病變**：血紅素 (Hemoglobin) 被氧化變性 → 聚集成 **Heinz bodies (海因茲氏體)**。
    *   **檢驗與後果**：
        *   Heinz bodies 需用**超活體染色 (Supravital stain)** 才能觀察到，Wright 氏染色不可見。
        *   帶有 Heinz bodies 的紅血球通過**脾臟**時，會被巨噬細胞咬掉一部分，形成 **Bite cells** (咬痕細胞)。
        *   溶血主要為**急性、血管內溶血** (伴隨部分血管外溶血)，導致血紅素尿、血漿中 haptoglobin 下降。

2.  **PK 缺乏症的因果鏈**：
    *   **生理機轉**：PK 是**糖解路徑 (EMP pathway)** 的末端酵素 → 缺乏 PK → **ATP** 生成不足。
    *   **細胞病變**：ATP 不足 → 細胞膜上的陽離子幫浦 (Na+/K+ pump) 失能 → 細胞內鉀離子流失、水分喪失 → 紅血球脫水、僵硬，形成 **Echinocytes (圓棘球)**。
    *   **檢驗與後果**：
        *   僵硬的紅血球提早被**脾臟**的網狀內皮系統清除，屬於**慢性、血管外溶血**。
        *   臨床上不會有特定誘發因子，長期溶血導致脾臟腫大、黃疸，且網狀紅血球 (Reticulocyte) 會代償性**顯著升高**。

**預測考法趨勢**：
*   **情境題**：給定一個「接觸蠶豆後尿液變深」的案例，要求判斷最可能的酵素缺乏、實驗室數據變化 (如 Haptoglobin ↓, Indirect bilirubin ↑) 或血液抹片所見 (Bite cells)。
*   **圖像題**：直接給予 Wright 氏染色 (可見 Bite cells) 與超活體染色 (可見 Heinz bodies) 的對比圖，要求判斷疾病或解釋染色原理。
*   **機制比較題**：直接比較 G6PD 與 PK 缺乏症在「影響的代謝路徑」、「缺乏的關鍵產物 (NADPH vs. ATP)」、「遺傳方式」上的差異。

## 關鍵字反射表

| 比較維度 | G6PD 缺乏症 (蠶豆症) | 丙酮酸激酶 (PK) 缺乏症 |
| :--- | :--- | :--- |
| **影響路徑** | **己糖單磷酸分流 (HMP shunt)** | **糖解路徑 (Embden-Meyerhof)** |
| **缺乏產物** | ↓ **NADPH** | ↓ **ATP** |
| **致病機轉** | 無法抵抗**氧化壓力** | 陽離子幫浦失效、細胞膜不穩 |
| **遺傳方式** | **性聯遺傳 (X-linked)** | **體染色體隱性 (AR)** |
| **溶血模式** | **急性、陣發性** | **慢性、持續性** |
| **主要溶血** | 血管內 & 血管外 | **血管外** |
| **誘發因子** | **有** (氧化性藥物、蠶豆、感染) | 無特定因子 |
| **RBC內含物** | **Heinz bodies** (海因茲氏體) | 無特定內含物 |
| **觀察方式** | **超活體染色 (Supravital stain)** | 一般染色 |
| **特殊 RBC 形態** | **Bite cells** (咬痕細胞), Eccentrocytes | Echinocytes (圓棘球), Polychromasia↑↑ |
| **臨床表現** | 急性黃疸、血紅素尿、背痛 | 慢性貧血、黃疸、**脾腫大** |
| **清除器官** | **脾臟** | **脾臟** |
