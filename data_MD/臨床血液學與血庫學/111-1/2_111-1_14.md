## 題目

有關紅血球之生理代謝,下列敘述何者最適當?

A.經由粒線體進行糖解作用產生能量
B.經G6PD(glucose-6-phosphate dehydrogenase)之作用產生NADP以穩定血色素
C.2,3-DPG (2,3-diphosphoglycerate)濃度上升時可提升血色素與氧之結合力
D.Pyruvate kinase缺乏會造成慢性溶血

> [!success]- 正確解答
> **D**

---

## 筆記與詳解

### (1) 快速破題

看到「成熟紅血球代謝」，第一直覺必須是它們「沒有細胞核與粒線體」，所有能量代謝皆在細胞質進行（無氧糖解）。接著，紅血球的三大代謝旁路對應三個唯一目的：

1. **EMP 路徑 (缺 PK)** → 產 **ATP** 失敗 → 幫浦停擺、細胞破裂 → **溶血**。
2. **HMP 路徑 (缺 G6PD)** → 產 **NADPH** 失敗 → 抗氧化失敗 → 血紅素變性沉澱 → **溶血**。
3. **Rapoport-Luebering 路徑 (2,3-DPG)** → 逼迫血色素交出氧氣 → **親和力下降**。
掌握這三個對應關係，即可判定選項 D 正確，其餘皆犯了基礎機制的錯誤。

### (2) 關鍵字與知識點

* **成熟紅血球 (Mature RBC)** → 內部無粒線體，無法進行 TCA cycle 或電子傳遞鏈，100% 依賴無氧糖解作用產能。
* **粒線體 (Mitochondria)** → 有氧呼吸的胞器。紅血球沒有它，所以糖解作用只能在「細胞質」發生。
* **G6PD (Glucose-6-phosphate dehydrogenase)** → HMP 旁路的關鍵酵素，負責還原產生 **NADPH**，提供抗氧化保護。
* **2,3-DPG (2,3-diphosphoglycerate)** → 結合於血色素 β 鏈中心，會**降低** Hb 對氧的親和力，促使組織釋放氧氣。
* **Pyruvate kinase (PK)** → EMP 糖解路徑最後一步產能酵素，負責生成 **ATP**。

### (3) 詳解

* **A 錯誤：** 成熟紅血球**沒有粒線體**。紅血球的糖解作用（Embden-Meyerhof pathway, EMP）完全在「細胞質」內進行，以無氧的方式分解葡萄糖產生能量 (ATP)。
* **B 錯誤：** G6PD 的作用是還原產生 **NADPH**，而不是 NADP。NADPH (帶有 H，具還原力) 能維持細胞內穀胱甘肽 (GSH) 的還原態，藉此對抗氧化壓力，防止血色素氧化變性並保護細胞膜完整。
* **C 錯誤：** 2,3-DPG 濃度上升時，會穩定血色素的「去氧態 (T state)」，這會「降低」 (而非提升) 血色素與氧氣的結合力（即氧氣解離曲線右移），目的是讓紅血球更容易在微血管將氧氣釋放給缺氧的組織。
* **D 正確：** Pyruvate kinase (PK) 缺乏會阻斷 EMP 糖解作用，導致 **ATP 生成大幅減少**。缺乏 ATP 會讓紅血球膜上的 Na⁺/K⁺ 幫浦失去動力，造成細胞內鉀離子流失、鈣離子堆積、水分流失而脫水變形，最終提早被脾臟巨噬細胞清除，引發**慢性非球形紅血球溶血性貧血**。

### (4) 核心軸線與邏輯因果 (縱向聯想)

* **能量代謝軸（PK 缺乏）：** 糖解異常 (PK↓) → **ATP↓** → Na⁺/K⁺ 幫浦停擺 → 細胞脫水/膜僵硬形成棘紅血球 Echinocyte [🖼️](https://www.google.com/search?q=echinocyte+blood+smear&tbm=isch&udm=2) → 遭脾臟巨噬細胞吞噬 → **慢性溶血 (血管外)**。
* **抗氧化代謝軸（G6PD 缺乏）：** HMP 旁路異常 (G6PD↓) → **NADPH↓** → 穀胱甘肽 (GSH) 無法再生 → 遭遇氧化壓力 → Hb 沉澱形成 Heinz body [🖼️](https://www.google.com/search?q=%22Heinz+body%22+supravital+stain&tbm=isch&udm=2) → 巨噬細胞咬除沉澱物形成 Bite cell [🖼️](https://www.google.com/search?q=%22Bite+cell%22+blood+smear&tbm=isch&udm=2) → **急性溶血 (血管內+外)**。
* **氧氣調節軸（2,3-DPG）：** 組織缺氧 → R-L 旁路活化 → **2,3-DPG↑** → 結合去氧 Hb → **親氧力↓** (曲線右移) → 釋放 O₂。
