## 題目

有關血小板抗原的敘述,下列何者正確?

A.血小板存在大量Lewis抗原
B.ABO不符合之血小板輸血後,血小板之上升量可能會稍差
C.血小板特異性抗原大多存在於細胞表面的脂蛋白
D.血小板上不存在HLA-A抗原

> [!success]- 正確解答
> **B**

---

## 筆記與詳解

### (1) 快速破題

看到題目詢問「血小板抗原」，直覺應聯想血小板表面的三大重要抗原系統：**ABO、HLA Class I (A, B)、HPA (血小板特異性抗原)**。
解題思路：

* 題目若提「脂蛋白」必為陷阱，HPA 本質為「**醣蛋白 (Glycoprotein)**」。
* 題目若說「無 HLA」必錯，血小板富含 **HLA Class I (A, B)**，這是造成輸血無效的主因。
* 血小板上有 ABO 抗原，若給予 ABO 不合的血小板，抗原抗體反應會加速血小板清除，導致上升量 (CCI) 較差。因此一眼即可判定答案為 B。

### (2) 關鍵字與知識點拆解

* **ABO 抗原** → 血小板本身會表現並從血漿中吸附 ABO 抗原。ABO 不合時輸注，存活期變短。
* **HLA-A / HLA-B 抗原 (HLA Class I)** → 大量存在於血小板表面。是引起「**血小板輸血無效 (Platelet Refractoriness)**」的最主要免疫原因。*(註：血小板無 HLA Class II)*。
* **血小板特異性抗原 (HPA)** → 存在於血小板膜上的**醣蛋白 (Glycoprotein, GP)**（如 GPIIb/IIIa, GPIb/IX）。與新生兒同種免疫血小板低下症 (NAIT) 及輸血後紫斑症 (PTP) 密切相關。
* **Lewis 抗原** → 血小板會從血漿中吸附微量抗原，但量極少，臨床意義低，不能稱作「大量存在」。

### (3) 詳解與辨析

* **(A) 錯誤**：血小板表面的主要紅血球相關抗原為 ABO。雖然會微量吸附血漿中的可溶性物質（如 Lewis 或 Chido/Rodgers），但量非常少，絕非「大量存在」。
* **(B) 正確**：血小板膜上具有 ABO 抗原。當進行 ABO 不相合 (ABO-incompatible) 的輸血時，受血者的同種血球凝集素（如 anti-A, anti-B）會結合血小板上的 ABO 抗原並加速其清除，導致輸血後的血小板增加量 (Platelet increment) 及存活時間稍差。
* **(C) 錯誤**：血小板特異性抗原 (HPA) 大多位於細胞表面的「**醣蛋白 (Glycoproteins)**」，而不是脂蛋白。例如最常引起免疫反應的 HPA-1a 位於 GPIIb/IIIa 上。
* **(D) 錯誤**：血小板表面帶有大量豐富的 **HLA Class I 抗原**（包含 HLA-A 與 HLA-B）。這正是為何多次接受血小板輸血的病人容易產生同種抗體 (alloantibodies) 而導致後續輸血無效的原因。

### (4) 核心邏輯與生理機轉

**縱向聯想（免疫性血小板破壞機轉）：**
懷孕或多次輸血暴露 → 免疫系統辨識外來 **HLA Class I / HPA** → 產生同種抗體 (Alloantibody) → 再次輸注或胎兒抗原進入母體 → IgG 抗體結合血小板抗原 → 巨噬細胞吞噬清除 → **PLT 計數急遽↓** → 臨床出現皮膚紫斑/瘀點[🖼️](https://www.google.com/search?q=purpura+petechiae+clinical+sign&tbm=isch&udm=2)或嚴重出血 → 評估指標：**CCI (校正血小板增加數) < 5000** 判定為輸血無效。

**記憶口訣：**
血小板抗原三大巨頭：**「A、H、P」**

* **A**BO (會吸附，不合會稍降)
* **H**LA-A/B (Class **I**，輸血無效**No.1**殺手)
* **P**rotein (**HPA**是**醣蛋白**Glycoprotein，專惹紫斑NAIT/PTP)
