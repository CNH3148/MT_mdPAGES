## 題目

45.下列有關品管之敘述，何者錯誤？

外部品管可以偵測系統偏差（bias）
內部品管發生異常主要為隨機誤差（random error）
內部品管主要是偵測準確度
外部品管主要是與真值做比較

> [!success]- 正確解答
> **C**

---

## 筆記與詳解

### (1) 快速破題

看到「內部品管 (IQC)」與「外部品管 (EQA)」的比較，必須直覺條件反射其核心對應關係：**內部品管 = 監控精密度 (Precision) / 隨機誤差**；**外部品管 = 監控準確度 (Accuracy) / 系統誤差**。題目要求選出「錯誤」的敘述，選項 C 將內部品管與準確度連上關係，明顯違背核心定義，故直接選 C。

### (2) 關鍵字與知識點拆解

* **內部品管 (IQC)** → 知識點：評估**精密度 (Precision)** 與再現性。透過每日檢測確保儀器狀態穩定，主要使用 Levey-Jennings chart 監控。
* **外部品管 (EQA/PT)** → 知識點：評估**準確度 (Accuracy)**。與其他實驗室 (同儕均值) 或參考實驗室比對，找尋與「真值」的偏差 (Bias)。
* **隨機誤差 (Random Error)** → 知識點：影響**精密度**。數據呈現無方向性的離散，常因人為操作不穩定、氣泡、電壓波動引起。
* **系統誤差 (Systematic Error)** → 知識點：影響**準確度**。數據呈現單一方向的偏移 (Shift 或 Trend [🖼️](https://www.google.com/search?q=Levey-Jennings+chart+trend+shift&tbm=isch&udm=2))，常因試劑變質、校正不良或光源老化引起。

### (3) 詳解辨析

* **(A) 外部品管可以偵測系統偏差 (bias) - 正確：** 外部品管 (如 CAP 能力試驗) 是將未知檢體發放給各實驗室，比對後計算出的同儕均值即視為「真值」。若自身實驗室結果長期偏離此均值，即可偵測出實驗室內部未察覺的**系統偏差**。
* **(B) 內部品管發生異常主要為隨機誤差 - 正確：** 雖然內部品管也能透過 Westgard rules 抓出系統誤差 (如 2-2s, 10x)，但在日常單次異常的發生上 (如最常見的 1-2s 警告或 1-3s 違規)，主要來源多為單次操作手法變異、微量檢體吸取不均等**隨機誤差**。相較於 (C)，此敘述在實務統計上屬合理描述。
* **(C) 內部品管主要是偵測準確度 - 錯誤 (本題答案)：** 內部品管使用的是「已知濃度範圍」的品管液，其首要目的是確認儀器在每一天、每一批次的測量結果是否具備高度的**精密度 (再現性)**。要確認「準確度」(即結果對不對)，必須依賴校正 (Calibration) 或外部品管與真值比對。
* **(D) 外部品管主要是與真值做比較 - 正確：** 外部品管的核心機制就是利用同儕均值 (Peer group mean) 或參考實驗室測量值作為統計學上的**真值替代品**，藉此評估各實驗室的檢驗準確度。

### (4) 核心軸線與橫向比較

* **縱向邏輯軸線：**
* 實驗室環境/試劑變質 → 發生**系統誤差 (Bias)** → 測值整體偏移 → **準確度 (Accuracy) ↓** → **外部品管 (EQA)** 表現不佳 / IQC 圖表呈現 Shift/Trend [🖼️](https://www.google.com/search?q=Levey-Jennings+chart+trend+shift&tbm=isch&udm=2)。
* 人為操作手法不一/電壓波動 → 發生**隨機誤差 (Random Error)** → 測值離散度大 → **精密度 (Precision) ↓** → **內部品管 (IQC)** 呈現 SD 變大 / 觸發 1-3s, R-4s 違規。


* **記憶口訣：** **「內精隨、外準系」** (內部品管-精密度-隨機誤差；外部品管-準確度-系統誤差)。

### (5) 內部與外部品管核心比較表

| 比較維度 | 內部品管 (Internal QC) | 外部品管 (External QC / PT) |
| --- | --- | --- |
| **主要評估目標** | **精密度 (Precision)**、再現性 | **準確度 (Accuracy)** |
| **主要監控誤差** | **隨機誤差 (Random Error)** (輔助看系統誤差) | **系統誤差 (Systematic Error / Bias)** |
| **對照基準** | 實驗室自訂的 **Mean ± SD** | 同儕實驗室均值 (**Peer group mean**) / 真值 |
| **執行頻率** | **高** (每日 / 每批次) | **低** (每月 / 每季 / 每年) |
| **圖表/統計工具** | **Levey-Jennings chart**、**Westgard rules** | **SDI** (Standard Deviation Index)、**Z-score** |
| **常見異常成因** | 氣泡、電壓不穩、人為微量操作差異 | **試劑變質**、**校正液 (Calibrator) 偏差**、濾光片老化 |
