## 筆記與詳解

### (1) 快速破題

看到題目給定「VWF multimers 電泳圖」[🖼️](https://www.google.com/search?q="VWF+multimer"+electrophoresis&tbm=isch&udm=2)，且圖 B (病人) 明顯**缺失了上方與中間的條帶**（代表缺乏高分子量 HMW 與中分子量 IMW），只要看到「大、中分子量皆消失」，直覺就要想到 **Type 2A VWD**。若只消失最高分子量，則偏向 Type 2B；若所有條帶都在但顏色變淡為 Type 1；若完全無條帶則是 Type 3。

### (2) 關鍵字解析

* **VWF multimers (Von Willebrand Factor multimers)** → 類血友病因子多聚體。VWF 在血漿中以多種大小的多聚體存在，分子量越「大」，其促凝血（結合血小板）的活性越強。
* **Type 1 VWD** → 部分「量」的缺乏（最常見）。
* **Type 2 VWD** → 「質」的異常（包含亞型 2A, 2B, 2M, 2N）。
* **Type 3 VWD** → 嚴重「量」的缺乏（幾乎測不到）。

### (3) 詳解

* **圖譜判讀：** 圖 A (正常人) 呈現完整的階梯狀電泳條帶，包含底部（低分子量 LMW）、中部（中分子量 IMW）與頂部（高分子量 HMW）。圖 B (病人) 僅剩底部的 LMW 條帶，**HMW 與 IMW 均消失**。
* **(A) Type 2A VWD (正確)：** 特徵為 VWF 易被 ADAMTS13 酵素過度分解，或組裝異常，導致在電泳圖上**同時缺乏 HMW 與 IMW 多聚體**。此為本題圖 B 最典型的表現。
* **(B) Type 1 VWD (錯誤)：** 電泳圖譜的分布應與正常人相同（HMW, IMW, LMW 皆存在），但整體條帶顏色會較淡（因濃度均勻下降）。
* **(C) Type 2B VWD (錯誤)：** 特徵為 VWF 突變導致對血小板 GPIb 親和力異常升高，HMW 多聚體會在血流中自發性與血小板結合而被消耗。電泳圖譜通常**僅缺乏 HMW 多聚體**（IMW 通常保留）。
* **(D) Type 3 VWD (錯誤)：** VWF 幾乎完全缺乏，電泳圖上會呈現**完全空白（無條帶）**。

### (4) 核心軸線與邏輯因果

**縱向因果：** 基因突變 (VWF 結構異常) → 易被 ADAMTS13 酵素過度切割或組裝不良 → 血液中缺乏大、中型 VWF multimers 電泳條帶 [🖼️](https://www.google.com/search?q="VWF+multimer"+electrophoresis&tbm=isch&udm=2) → 血小板黏附功能 (VWF:RCo) 大幅下降 → 臨床表徵：黏膜出血、皮下出血斑點 (Petechiae) [🖼️](https://www.google.com/search?q=petechiae+purpura+clinical+sign&tbm=isch&udm=2)。

**Type 2 亞型記憶口訣：**

* 2**A**：**A**bsent (HMW & IMW 雙雙消失)。
* 2**B**：**B**inding 增強 (與血小板親和力↑，消耗掉最大的 HMW)。
* 2**M**：**M**ultimer 正常 (結構/電泳正常，但功能極差)。
* 2**N**：**N**ormal multimer (結合 Factor VIII 異常，臨床表現類似血友病 Hemophilia)。
