## 筆記與詳解

### (1) 快速破題

看到題目提及「先天性血小板疾病」且檢驗工具為「流式細胞儀（Flow cytometry）」，第一時間就要想到這是用來偵測**血小板表面醣蛋白（Glycoprotein, GP）缺陷**的黃金標準。
解題核心：如果直方圖顯示缺乏 **CD41 (GPIIb)** 或 **CD61 (GPIIIa)** 的表現，就是 Glanzmann thrombasthenia (GT)；若顯示缺乏 **CD42b (GPIb)** 則是 Bernard-Soulier syndrome (BSS)。既然答案為 A，圖中必然顯示 CD41 或 CD61 的螢光訊號大幅偏低（患者峰值左移至陰性區）。

### (2) 關鍵字與知識點拆解

* **流式細胞儀 (Flow cytometry)** → 利用螢光單株抗體標記特定 CD markers，直接定量血小板表面的膜醣蛋白表現量。
* **Glanzmann thrombasthenia (GT)** → 體染色體隱性遺傳。缺乏 **GPIIb/IIIa (CD41/CD61)**。血小板無法與 Fibrinogen 結合，導致「**血小板凝集 (Aggregation)**」功能障礙。
* **Bernard-Soulier syndrome (BSS)** → 體染色體隱性遺傳。缺乏 **GPIb-IX-V (CD42b/CD42a)**。血小板無法與 vWF 結合，導致「**血小板黏附 (Adhesion)**」功能障礙。周邊血液抹片常見巨大血小板 [🖼️](https://www.google.com/search?q="Giant+platelet"+blood+smear&tbm=isch&udm=2)。
* **May-Hegglin anomaly (MHA)** → **MYH9** 基因突變。周邊血液特徵為巨大血小板 [🖼️](https://www.google.com/search?q="Giant+platelet"+blood+smear&tbm=isch&udm=2) 以及嗜中性白血球內出現 Dohle-like bodies [🖼️](https://www.google.com/search?q="Dohle-like+bodies"+blood+smear&tbm=isch&udm=2)。
* **Platelet-type von Willebrand disease (PT-VWD)** → GPIb 基因突變，使其對 vWF 具有異常的**高親和力**（Gain-of-function），臨床表現極似 Type 2B VWD。

### (3) 詳解與選項辨析

* **(A) 正確**：GT 肇因於 GPIIb/IIIa 缺陷。流式細胞儀檢查會呈現 **CD41 / CD61 缺乏或低下**。在血小板凝集試驗中，除了 Ristocetin 反應正常外，加入 ADP、Collagen、Epinephrine 皆**無法**引發血小板凝集。本題依據流式細胞儀的缺陷圖譜（CD41/61 陰性）即可確診。
* **(B) 錯誤**：BSS 肇因於 GPIb/IX/V 缺陷，流式細胞儀應顯示 **CD42 (CD42b, CD42a 等)** 缺乏。凝集試驗中，加入 Ristocetin 無反應（且加入正常血漿無法矯正），但加入 ADP、Collagen 等致效劑則凝集正常。
* **(C) 錯誤**：May-Hegglin  anomaly 的血小板膜表面醣蛋白表現通常正常，流式細胞儀不會出現特定的 CD marker 缺失。其診斷仰賴抹片鏡檢觀察特異性的白血球包涵體。
* **(D) 錯誤**：PT-VWD 的 GPIb 數量通常正常或僅輕微下降，無法單憑流式細胞儀的受體缺失圖譜確診，必須仰賴低濃度 Ristocetin (Low-dose RIPA) 誘發異常凝集來證實其對 vWF 的高親和力。

### (4) 邏輯軸線與記憶口訣

* **邏輯軸線**：基因突變 → 血小板表面受體 (Glycoprotein) 缺失 → 流式細胞儀對應的 CD marker 螢光強度下降 → 搭配凝集試驗反應 → 確診特定先天性血小板疾病。
* **記憶口訣**：
* **GT 23, 4161**：GT 缺乏 GP **II**b/**III**a，對應流式細胞儀 CD**41**/CD**61**。
* **BSS 19, 42**：BSS 缺乏 GP **I**b-**IX**-V，對應流式細胞儀 CD**42**。（且 **B**SS 有 **B**ig platelets）。
