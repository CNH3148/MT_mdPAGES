## 題目

下列何者最可能是Type 3 VWD的檢驗結果?

A.血漿中缺乏高分子量VWF,但低分子量VWF正常
B.血漿中缺乏低分子量VWF,但高分子量VWF正常
C.VWF:RCo與VWF:Ag檢驗結果異常
D.Factor VIII活性正常

> [!success]- 正確解答
> **C**

---

## 筆記與詳解

### (1) 快速破題

看到題目出現「Type 3 VWD」，要立刻聯想到「最嚴重、最徹底的缺乏」。Type 3 代表 VWF 蛋白**幾乎完全消失**。既然甚麼都沒有，那跟 VWF 有關的抗原定量 (VWF:Ag) 和功能測試 (VWF:RCo) 必定都是**異常（極低或測不到）**。同時，因為 VWF 是 Factor VIII (FVIII) 的保護載體，VWF 不見了，FVIII 也會被快速代謝掉而低下。選項中只有 C 符合此全面低下的病理特徵。

### (2) 關鍵字與知識點拆解

* **Type 3 VWD**：第三型溫韋伯氏疾病，屬於「嚴重定量缺陷」，患者體內幾乎無 VWF。
* **高/低分子量 VWF (HMW/LMW)**：VWF 藉由雙硫鍵形成多聚體，HMW (高分子量) 具有最強的凝血功能。VWF 多聚體分析電泳圖[🖼️](https://www.google.com/search?q="VWF+multimer"+gel+electrophoresis&tbm=isch&udm=2)的異常分布主要用來判定 **Type 2** 的質變亞型。
* **VWF:Ag (Antigen)**：評估 VWF 的「絕對數量」。
* **VWF:RCo (Ristocetin Cofactor)**：評估 VWF 與血小板 GPIb 結合的「功能活性」。
* **Factor VIII (FVIII)**：凝血因子 VIII。在血漿中必須與 VWF 結合才能穩定存在（免於被蛋白酶切割），否則半衰期極短。

### (3) 詳解與辨析

* **A 與 B 錯誤**：選項敘述的是 VWF 多聚體分布異常。當血漿中「缺乏高分子量 (HMW) VWF」時，屬於 **Type 2A 或 Type 2B VWD** 的典型特徵，而非 Type 3。Type 3 是「全部 (無論高低分子量)」都極度缺乏。
* **C 正確**：Type 3 為最嚴重的 VWF 數量缺乏，因此其抗原量 (VWF:Ag) 與功能活性 (VWF:RCo) 兩者皆會呈現顯著異常，數值通常低於實驗室偵測極限。
* **D 錯誤**：因為血液中幾乎沒有 VWF 可以保護 FVIII，導致 FVIII 快速被降解。因此 Type 3 VWD 患者的 FVIII 活性會**顯著下降** (通常 <10%)，甚至可能出現類似 A 型血友病的深部關節積血[🖼️](https://www.google.com/search?q=hemarthrosis+clinical+sign&tbm=isch&udm=2)症狀。

### (4) 核心軸線與邏輯因果

基因突變 (常染色體隱性) → VWF 蛋白合成失敗 (數量趨近 0) → 缺乏保護載體 → FVIII 快速降解 (半衰期大幅縮短) → 導致「初級止血 (血小板無法黏附)」與「次級止血 (凝血瀑布異常)」雙重障礙 → 實驗室數據全面低下 (VWF:Ag ↓↓, VWF:RCo ↓↓, FVIII ↓↓) → 臨床表現出嚴重的皮膚黏膜出血[🖼️](https://www.google.com/search?q=mucocutaneous+bleeding+clinical+sign&tbm=isch&udm=2)與深層肌肉關節出血。

*記憶口訣*：**Type 3 = 3無** (無Ag、無RCo、無FVIII保護)。
