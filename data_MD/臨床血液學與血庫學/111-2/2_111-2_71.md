## 題目

有關血小板輸注之血品選擇,下列敘述何者最適當?

A.需以相同ABO血型血小板輸注最主要是因血小板的ABO抗原表現很強
B.分離術(apheresis)血小板輸注時,無需考慮ABO血型,因為暴露之ABO抗體量少
C.發生血小板輸注無效(refractoriness)後,再輸注時需優先考慮HLA class II型相合的血小板
D.早產兒輸注減除白血球血小板血品時,血品應先執行輻射照射後才能使用

> [!success]- 正確解答
> **D**

---

## 筆記與詳解

### (1) 快速破題

看到題目出現「早產兒」、「免疫不全」或「近親輸血」+ 輸血不良反應預防 → 就要立刻想到預防致死率極高的 **TA-GVHD (輸血相關移植物抗宿主疾病)**。預防 TA-GVHD 的唯一有效手段是「**輻射照射 (Irradiation)**」。
題目選項 D 故意將「減除白血球 (Leukoreduction)」與「輻射照射」並列，考驗是否能釐清兩者目的：「減白」只能防發燒 (FNHTR) 與 CMV 感染，無法完全消滅 T 細胞；只有「輻射照射」能破壞 T 細胞 DNA 預防 TA-GVHD。此外，看到「血小板輸注無效」就要想到與 **HLA Class I** (而非 Class II) 相關。由此可快速選出正確答案 D。

### (2) 關鍵字與知識點拆解

* **血小板 ABO 抗原** → 表現**弱** (主要自血漿吸附)。
* **分離術血小板 (Apheresis platelets)** → 單袋即含大量血漿 (~200-300 mL)，內含大量 ABO 同型凝集素 (Anti-A/Anti-B)，不合型易致受血者紅血球溶血。
* **血小板輸注無效 (Refractoriness)** → 免疫性主因為受血者產生抗 **HLA Class I** 或抗 **HPA** (人類血小板抗原) 抗體。
* **HLA Class II** → 血小板**不表現** HLA Class II，僅表現 HLA Class I。
* **減除白血球 (Leukoreduction)** → 濾除 99.9% 白血球，預防 **FNHTR (發熱性非溶血性輸血反應)**、預防 **CMV** 傳染、減少 HLA 異體免疫。
* **輻射照射 (Irradiation)** → 破壞 **T 淋巴球** DNA，使其無法增殖，專門用於預防 **TA-GVHD** (臨床表徵包含斑丘疹[🖼️](https://www.google.com/search?q=TA-GVHD+maculopapular+rash+clinical+sign&tbm=isch&udm=2)、發燒、肝功能異常)。

### (3) 詳解

* **(A) 錯誤**：血小板的 ABO 抗原表現其實**很弱**。盡量使用 ABO 同型輸注，主要是為了避免血品中夾帶的異型血漿 (含 Anti-A 或 Anti-B) 攻擊病人的紅血球造成溶血，且同型血小板的存活期與輸注效益略佳。
* **(B) 錯誤**：分離術血小板來自單一捐血者，含有較多量的高濃度血漿 (常達 200 mL 以上)。若輸給非同型受血者，暴露的 ABO 抗體量**很大**，極易引發溶血反應，因此**非常需要**考慮 ABO 血型相容性 (特別是血漿與受血者紅血球的相容性)。
* **(C) 錯誤**：血小板表面**只有表現 HLA Class I** (HLA-A, B, C) 以及 HPA，**沒有表現 HLA Class II** (HLA-DR, DP, DQ)。當發生免疫性血小板輸注無效時，應優先尋找 **HLA Class I** 相合或交叉試驗陰性的血小板。
* **(D) 正確**：早產兒的免疫系統尚未發育完全，無法清除輸入體內的異體 T 淋巴球，是發生 TA-GVHD 的極高風險群。「減除白血球」血品仍殘留微量 T 細胞 (足以引發疾病)，因此必須先執行**輻射照射 (25 Gy)** 去活化 T 細胞後才能安全使用。

### (4) 核心軸線與邏輯因果

**縱向聯想：** 特殊體質/狀態 (早產兒 / 免疫低下 / 近親捐血) → 無法清除外來 T 細胞 → 捐血者 T 細胞反客為主攻擊宿主組織 → 爆發 TA-GVHD (高致死率) → 唯一解方：事前以 **Gamma 射線照射**血品 → 破壞 T 細胞 DNA 使其失去增殖能力。

**橫向比較：** 血品特殊處理方法與血小板輸注無效成因是國考常客，需嚴格區分「減白」、「輻射」與「洗滌」的目的，以及「免疫性」與「非免疫性」輸注無效的鑑別。

### (5) 表格統整

**表一、血液成分特殊處理比較 (防護對象與機制)**

| 處理方法 | 去除/破壞目標 | 預防之不良反應 | 核心適應症 |
| --- | --- | --- | --- |
| **減除白血球 (Leukoreduction)** | 濾除 >99.9% 白血球 (殘留 $<5 \times 10^6$) | **FNHTR**、**CMV** 傳染、HLA 異體免疫 | 曾有 FNHTR 病史、需長期輸血者、器官移植候選人 |
| **輻射照射 (Irradiation)** | 破壞 **T 淋巴球** DNA (使其無法增殖) | **TA-GVHD** (輸血相關移植物抗宿主疾病) | **早產兒**、免疫不全 (化療/骨髓移植)、**一等親/近親**血品、HLA 配對血品 |
| **洗滌 (Washing)** | 去除 **血漿蛋白** (如 IgA)、過多鉀離子 | **過敏反應 (Allergic reaction)**、高血鉀 | **IgA 缺乏症**且具抗 IgA 抗體者、曾發生嚴重過敏性休克 (Anaphylaxis) 者 |
