## 題目

有關血庫抗體篩檢之敘述，下列何者正確？

A.冷型自體抗體多屬於 IgG
B.會以病患血清和試劑紅血球進行抗體篩檢
C.流程中包括 20℃試驗階段
D.不可使用新鮮凝固的血清來進行測試

> [!success]- 正確解答
> **B**

---

## 筆記與詳解

### (1) 快速破題

看到「血庫抗體篩檢 (Antibody screening)」，核心解題思路就是「找病患體內的意外抗體」。抗體存在於血清或血漿中，所以測試物必定為「病患血清/血漿」，而偵測工具必為帶有已知抗原的「試劑紅血球」(通常為 O 型血)。

* 看到選項 A 考抗體特性，記住口訣「**冷M溫G** (冷麥溫居)」即可知道冷型是 IgM。
* 看到選項 C 考反應溫度，臨床有意義的抗體都在人體體溫 (**37°C**) 破壞紅血球，因此篩檢重點是 37°C 而非 20°C。
* 看到選項 D 檢體要求，血庫測試中新鮮血清與血漿皆可使用。
因此，選項 B 直觀且完全符合原理。

### (2) 關鍵字與知識點拆解

* **血庫抗體篩檢 (Antibody screening)** → 尋找「非 ABO 血型系統」的意外抗體 (unexpected antibodies)，目的為預防溶血性輸血反應。
* **冷型/溫型自體抗體** → 依最佳反應溫度區分。冷型對應 IgM (大分子、自然發生)；溫型對應 IgG (小分子、免疫刺激產生)。
* **試劑紅血球 (Screening cells)** → 必須是 **O 型**，為的是避免與病患血清中的 anti-A 或 anti-B 發生干擾，且表面須帶有已知且完整的各類血型抗原 (如 Rh, Kell, Duffy, Kidd 等)。
* **新鮮凝固血清 / EDTA 血漿** → 兩者皆可用作檢體。新鮮血清保留了補體活性；EDTA 血漿則可螯合鈣離子，避免檢體在體外因冷型抗體活化補體而造成偽陽性干擾。

### (3) 選項詳解與辨析

* **(A) 錯誤**：冷型自體抗體 (Cold autoantibodies) 主要為 **IgM**；溫型自體抗體 (Warm autoantibodies) 才是 **IgG**。
* **(B) 正確**：抗體篩檢的基本原理 (間接抗球蛋白試驗 IAT) 即是將「**病患血清/血漿** (含未知抗體)」與「**試劑紅血球** (具已知抗原)」混合，觀察是否出現紅血球凝集 (Hemagglutination) [🖼️](https://www.google.com/search?q=hemagglutination+tube+test&tbm=isch&udm=2) 或溶血反應。
* **(C) 錯誤**：常規抗體篩檢是為了找出「具臨床意義 (會引發體內溶血)」的抗體，這些抗體大多在體溫下最活躍。因此常規流程主要包含 **37°C 溫育階段** 以及 **抗人類球蛋白 (AHG) 階段**，並不包含 20°C 試驗 (20°C 通常用於鑑定無重大臨床意義的冷型抗體)。
* **(D) 錯誤**：進行測試 **可以使用** 新鮮凝固的血清 (Fresh clotted serum)。新鮮血清含有活性補體，有助於偵測高度依賴補體活化的抗體 (如 Kidd 系統抗體)；不過現代血庫也常規使用 EDTA 血漿以減少冷凝集素的體外活化干擾。

### (4) 核心邏輯與橫向比較

**核心邏輯軸線 (抗體篩檢機轉)：**
輸血/懷孕等免疫刺激 → 產生具臨床意義的 **IgG 溫型意外抗體** → 抗體游離於病患血清/血漿中 → 抽血加入 **O型試劑紅血球** + **37°C** 溫育 + 加入 **AHG** 試劑 → 促使 IgG 橋接紅血球產生巨觀凝集 → 判定抗體篩檢陽性。

**【血庫常見抗體特性比較表】**

| 比較維度 | 溫型抗體 (Warm Antibodies) | 冷型抗體 (Cold Antibodies) |
| --- | --- | --- |
| **主要免疫球蛋白** | **IgG** (單體，小分子) | **IgM** (五聚體，大分子) |
| **最佳反應溫度** | **37°C** (體溫) | **< 25°C** (室溫或 4°C) |
| **臨床意義** | **高** (引發 HTR, HDN) | 低 (多為無害的自然發生抗體) |
| **通過胎盤能力** | **是** (會引起新生兒溶血症) | 否 |
| **IAT 檢驗陽性階段** | **AHG 階段** | 室溫立即離心 (IS) 階段 |
| **血型系統代表** | **Rh, Kell, Duffy, Kidd** | **Lewis, M, N, P1, I** |
| **體內致病表徵** | 血管外溶血 (脾臟吞噬破壞) | 血管內溶血 / 肢端發紺 (Acrocyanosis) [🖼️](https://www.google.com/search?q=acrocyanosis+clinical+sign&tbm=isch&udm=2) |

*(註：HTR = 溶血性輸血反應, HDN = 新生兒溶血症)*
