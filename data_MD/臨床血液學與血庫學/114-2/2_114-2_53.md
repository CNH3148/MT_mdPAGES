## 題目

有關血小板的dense tubular system,下列敘述何者最不適當?

為殘留的內質網(residual endoplasmic reticulum)結構
可儲存血小板活化時所須的鈣離子
對血小板活化時的 prostaglandin (eicosanoid) signaling pathway 相當重要
是一種與血小板膜表面相連而延伸進入到血小板內的 membrane system

> [!success]- 正確解答
> **D**

---

## 筆記與詳解

(1) 快速破題
看到考血小板內部結構，並且要找「錯誤」，解題思路就是立刻將「兩大膜系統」對立起來：**DTS (緻密管系統)** 與 **SCCS/OCS (表面連接小管系統 / 開放性小管系統)**。
題目問「DTS」，選項 D 卻描述「與表面相連 (connected to surface)」，這顯然是「表面連接小管系統 (SCCS)」的專屬特徵。只要把握「DTS = 內部封閉 / SCCS = 外部相通」這個二分法，就能直覺選出 D 是錯誤描述。

(2) 詳解預備 (關鍵字 → 知識點)

* **Dense tubular system (DTS)** → 知識點：巨核細胞的平滑內質網遺跡 (封閉系統)、血小板的 $Ca^{2+}$ 儲存庫、前列腺素 (Prostaglandin) 及 TXA2 合成酶 (如 COX) 的所在位置。
* **Surface-connected canalicular system (SCCS) / Open canalicular system (OCS)** → 知識點：血小板細胞膜向內凹陷形成 (開放系統)；作為血小板活化後，顆粒物質「釋放/分泌」的對外通道。
* **Prostaglandin signaling pathway** → 知識點：血小板膜磷脂質 $\rightarrow$ Arachidonic acid $\rightarrow$ PGG2/PGH2 (在 DTS 經 COX 催化) $\rightarrow$ TXA2 (強效促凝集物質)。

(3) 詳解

* **選項 A (正確)**：DTS 確實是源自於巨核細胞 (Megakaryocyte) 的平滑內質網 (SER) 遺跡，屬於封閉的細胞內膜系統。
* **選項 B (正確)**：與肌肉細胞的肌漿網類似，DTS 是血小板內最重要的 $Ca^{2+}$ 儲存庫。血小板活化時會釋放 $Ca^{2+}$ 以誘發細胞骨架收縮與胞內顆粒移動。
* **選項 C (正確)**：DTS 膜上富含 Cyclooxygenase (COX) 與 Thromboxane synthase，是前列腺素代謝路徑與合成 Thromboxane A2 (TXA2) 的核心樞紐。
* **選項 D (錯誤，本題答案)**：與「血小板膜表面相連」並延伸至內部的膜系統稱為 **SCCS / OCS (表面連接小管系統 / 開放小管系統)**。SCCS 在電子顯微鏡下呈現向外開口的管狀網絡[🖼️](https://www.google.com/search?q=platelet+"open+canalicular+system"+electron+microscopy&tbm=isch&udm=2)，負責將胞內釋放的顆粒內容物排至胞外；而 DTS 則是完全封閉、不與細胞膜相連的。

(4) 邏輯因果 (縱向聯想)
**血小板活化與膜系統協同機轉：**
血管內皮受損 $\rightarrow$ 致效劑結合膜受體 $\rightarrow$ **DTS** 釋放 $Ca^{2+}$ 並合成 TXA2 $\rightarrow$ 胞內高 $Ca^{2+}$ 激發微管/微絲收縮 (形狀改變) $\rightarrow$ Alpha/Dense 顆粒向中央集中 $\rightarrow$ 顆粒膜與 **SCCS** 融合 $\rightarrow$ 內容物經 **SCCS** 通道大量排出至血液 $\rightarrow$ 招募更多血小板形成血栓 (Primary hemostasis)。
在常規的 Wright stain 血液抹片中，未活化的血小板呈現帶有紫紅色中心顆粒的淡藍色碎片[🖼️](https://www.google.com/search?q=platelets+Wright+stain+blood+smear&tbm=isch&udm=2)，活化後則會伸出偽足。

(5) 血小板超微結構統整表

| 結構分區 (Zone) | 包含構造 | 核心機轉 / 特徵 | 關鍵字 / 標記物質 |
| --- | --- | --- | --- |
| **周邊區 (Peripheral)** | 醣萼 (Glycocalyx)、細胞膜 | 黏附 (Adhesion)、聚集 (Aggregation) | **GPIb/IX/V** (結合vWF)<br>**GPIIb/IIIa** (結合Fibrinogen) |
| **溶膠凝膠區 (Sol-gel)** | 微管 (Microtubules)、微絲 | 維持盤狀外觀、活化時**骨架收縮變形** | **Tubulin**、**Actin**、Myosin |
| **胞器區 (Organelle)** | **$\alpha$ 顆粒 ($\alpha$ granule)** | 數量**最多**；釋放凝血及血管修復因子 | **vWF**、**Fibrinogen**、PF4、PDGF |
|  | **緻密體 (Dense body)** | 活化/招募周邊血小板 (提供能量與傳導) | **ADP**、**ATP**、**$Ca^{2+}$**、**Serotonin** |
|  | 溶體 (Lysosomes) | 胞內消化作用 | 酸性水解酶 |
| **膜系統 (Membrane)** | **緻密管系統 (DTS)** | **封閉**系統；**巨核細胞平滑內質網**遺跡 | **儲存 $Ca^{2+}$**、**合成 TXA2** (COX所在) |
|  | **開放小管系統 (OCS/SCCS)** | **開放**系統；**細胞膜內凹**延伸形成 | **釋放通道** (與外界相通) |

**【記憶口訣】**

* **DTS (Dense) = Deep & $Ca^{2+}$** (藏在深處不與外界相通，負責存鈣與合成 TXA2)
* **OCS / SCCS (Open/Surface) = Out** (對外開放，是顆粒物質 **O**ut 釋放的出口)
* **緻密體 (Dense body) 裝什麼？** 口訣：「**C**hinese **A**rt **S**ociety」 $\rightarrow$ **C**alcium, **A**DP/ATP, **S**erotonin。
