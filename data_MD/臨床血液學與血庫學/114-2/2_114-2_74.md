## 題目

有關血小板之抗原,下列敘述何者最不適當?

具有ABO抗原
具有血小板上表現第I型人類白血球抗原(human leukocyte antigen),不具有第II型抗原
人類血小板抗原(human platelet antigen,HPA)之抗體可以造成新生兒血小板減少症
AstraZeneca COVID-19 疫苗注射會產生對抗血小板PF3 之抗體

> [!success]- 正確解答
> **D**

---

## 筆記與詳解

### (1) 快速破題

看到題目出現「AstraZeneca COVID-19 疫苗」與「對抗...之抗體」，必須立刻聯想到 **VITT (疫苗引起的免疫血栓性血小板低下症)**。此疾病的致病機轉與臨床上常見的 **HIT (肝素引起之血小板低下症)** 高度相似，其標靶抗原都是 **PF4 (Platelet Factor 4)**，絕對不是參與凝血反應的磷脂質表面 PF3。因此，看到 PF3 即可直接選出錯誤選項。

### (2) 關鍵字與知識點拆解

* **血小板 ABO 抗原** → **吸附性抗原**：非血小板自身製造，是由血漿中吸附而來，輸血時仍建議盡量同型。
* **HLA (人類白血球抗原)** → **Class I 專屬**：有核細胞與血小板皆具備 Class I (A, B, C)；Class II (DP, DQ, DR) 僅存在於抗原呈現細胞 (APC)。
* **HPA (人類血小板抗原)** → **血小板特異性抗原**：同種異體抗體反應的標的，常引發新生兒血小板低下 (NAIT) 及輸血後紫斑症 (PTP)。
* **PF3 (Platelet Factor 3)** → **磷脂質表面**：提供凝血因子 (如 Factor X、Prothrombin) 活化的平台。
* **PF4 (Platelet Factor 4)** → **肝素/疫苗誘發血栓標靶**：儲存於血小板 α-顆粒，釋放後若與 Heparin 或疫苗載體結合，易引發自體免疫抗體攻擊。

### (3) 詳解辨析

* **A 正確**：血小板表面確實具有 ABO 抗原。這些抗原主要從血漿中吸附而來，在臨床進行血小板輸注時，若能 ABO 同型輸注，存活期與效果最佳。
* **B 正確**：血小板會表現 **HLA Class I**。由於血小板缺乏細胞核且非 APC (抗原呈現細胞)，故**不具備 HLA Class II**。多次輸血的病人容易產生 anti-HLA Class I 抗體，這是造成**血小板輸血無效 (Platelet Refractoriness)** 最常見的原因。
* **C 正確**：當母體缺乏某種 HPA (最常見為 HPA-1a 陰性)，懷孕時若胎兒帶有該抗原 (HPA-1a 陽性)，母體會產生 IgG 抗體。此抗體穿過胎盤破壞胎兒血小板，導致**新生兒同種異體免疫血小板減少症 (NAIT)**，新生兒出生後會出現全身性出血斑點或紫斑 [🖼️](https://www.google.com/search?q=neonatal+petechiae+purpura+clinical+sign&tbm=isch&udm=2)。
* **D 錯誤，本題答案**：AstraZeneca COVID-19 疫苗 (腺病毒載體) 注射後引發的 VITT，其產生的自體抗體是針對 **PF4** 而非 PF3。此類 IgG 抗體與 PF4 結合後，會透過血小板表面的 FcγRIIa 受體強烈活化血小板，導致血栓形成並大量消耗血小板。

### (4) 核心軸線與邏輯因果

* **VITT/HIT 免疫血栓機轉** (縱向聯想)：
誘發物進入體內 (Heparin / 腺病毒載體疫苗) → 與血小板釋放的 **PF4** 形成免疫複合物 → 刺激 B 細胞產生 **Anti-PF4 IgG** → 該抗體結合血小板表面 **FcγRIIa 接受器** → 血小板過度活化與凝集 (微血管血栓形成) → 大量消耗血小板 (實驗室數值：**PLT ↓**) → 併發出血與血栓雙重危機。
* **💡 記憶口訣**：
**AZ 疫苗 / 肝素** 打完出事，氣到**死 (4)** → 標靶是 **PF4**。

### (5) 血小板相關抗原與臨床意義比較表

| 抗原/因子名稱 | 存在位置 / 特性 | 臨床相關疾病/事件 | 核心致病機轉或影響 |
| --- | --- | --- | --- |
| **ABO 抗原** | 表現於紅血球；**吸附**於血小板表面 | 血小板輸血效能 | 血小板**存活期縮短** (若 ABO 不相容) |
| **HLA Class I** | 所有有核細胞、血小板 | **血小板輸血無效 (Refractoriness)** | 多次輸血產生 **Anti-HLA 異體抗體** → 破壞輸入的血小板 |
| **HLA Class II** | 僅存於 **APC** (巨噬、樹突、B細胞) | (血小板無此抗原) | N/A |
| **HPA** | 血小板**特異性**抗原 (如 HPA-1a) | **NAIT** (新生兒)、**PTP** (輸血後紫斑) | 母親/受血者產生 **Anti-HPA IgG** → 攻擊胎兒/自體血小板 |
| **PF3** | 活化血小板膜上的**磷脂質表面** | 凝血機制異常 | 無法提供 **Tenase / Prothrombinase complex** 結合平台 |
| **PF4** | 血小板 **α-顆粒 (α-granules)** 釋放 | **HIT** (肝素誘發)、**VITT** (疫苗誘發) | 形成 **PF4-大分子複合物** → 誘發 **Anti-PF4 IgG** → 血栓併低下 |
