## 題目

出現下列何種變異之慢性淋巴性白血病(chronic lymphocytic leukemia)病人,預後最好?

hypermutated IGVH(immunoglobulin heavy chain variable region) gene
Binet stage C
deletion 17p
high ZAP expression

> [!success]- 正確解答
> **A**

---

## 筆記與詳解

**(1) 快速破題**
看到題目提到「CLL (慢性淋巴性白血病)」並且要找「預後『最好』」，第一直覺必須聯想到「**基因突變 (mutated)**」或是染色體「**del(13q)**」。在 CLL 的知識點中，帶有超突變 (`hypermutated IGVH`) 代表 B 細胞成熟度高，預後最好；反之，缺乏突變 (`unmutated`)、帶有高危險染色體缺失 (`del(17p)`, `del(11q)`) 或表現特定抗原 (`high ZAP-70`, `CD38`) 者，細胞具高增殖性與抗藥性，預後極差。

**(2) 詳解預備 (關鍵字與知識點)**

* **CLL (Chronic Lymphocytic Leukemia)**：成熟 B 細胞惡性增生疾病，周邊血液抹片最典型的視覺特徵為易破裂的抹滅細胞 (smudge cells)[🖼️](https://www.google.com/search?q="smudge+cells"+blood+smear&tbm=isch&udm=2)。
* **IGVH 基因**：免疫球蛋白重鏈可變區 (Immunoglobulin heavy chain variable region)。若發生體細胞超突變 (somatic hypermutation)，代表該 B 細胞已經過生發中心 (germinal center) 洗禮，屬較成熟穩定的記憶型 B 細胞，疾病進程緩慢。
* **ZAP-70 & CD38**：流式細胞儀常測的蛋白質標記，高表現量與「未突變 (unmutated)」IGVH 高度相關，屬於預後不良的替代指標。
* **del(17p)**：17 號染色體短臂缺失，直接導致重要的抑癌基因 **TP53** 喪失功能，是 CLL 中最致命的基因變異。
* **Binet 分期系統**：CLL 的歐洲系統臨床分期，依淋巴結腫大區域及是否有貧血/血小板低下分為 A、B、C 期。C 期為末期。

**(3) 詳解**

* **(A) hypermutated IGVH (正確)**：CLL 細胞若帶有 >2% 的 IGVH 基因突變，表示其起源於成熟的記憶型 B 細胞。此類病人病程和緩 (indolent)，對治療反應佳、存活期極長，為**最佳預後指標**。
* **(B) Binet stage C (錯誤)**：定義為血紅素 (Hb) < 10 g/dL 或血小板 (PLT) < 100,000/μL，代表骨髓已被大量白血病細胞佔據，屬於嚴重晚期，存活期短，**預後極差**。
* **(C) deletion 17p (錯誤)**：帶有此突變會造成 p53 抑癌蛋白缺乏，使得腫瘤細胞對傳統化學治療 (如 Fludarabine) 產生強烈抗藥性，是所有細胞遺傳學變異中**預後最差**的。
* **(D) high ZAP-70 expression (錯誤)**：ZAP-70 是一種激酶，在 CLL 中若高量表現 (通常定義為 >20%)，強烈暗示其 IGVH 為**未突變 (unmutated)** 狀態，代表細胞較原始、疾病具高侵襲性，**預後差**。

**(4) 核心軸線與邏輯因果**

* **生理機轉與預後連動**：B 細胞進入生發中心 → 產生 IGVH 基因超突變 (Somatic hypermutation) → 形成高成熟度記憶 B 細胞 → 惡性轉化為 CLL → 細胞增殖慢、凋亡路徑保留 → **臨床預後佳**。
* **檢驗標記連動**：缺乏 IGVH 突變 (Unmutated) ⇆ 伴隨高表現 ZAP-70 / CD38 ⇆ 預後差。
* **記憶口訣**：CLL 預後評估：「**好**的**突**然**13**歲 (Mutated, del(13q) 預後好)，**壞**的**原**始**11、17**歲找麻煩 (Unmutated, del(11q), del(17p) 預後極差)，外加 **ZAP / CD38** 兩個壞朋友」。

**(5) CLL 預後指標橫向比較表**

| 比較維度 | 🟢 預後良好 (Good Prognosis) | 🔴 預後不良 (Poor Prognosis) |
| --- | --- | --- |
| **IGVH 基因狀態** | **Hypermutated** (>2% 突變) | **Unmutated** (<2% 突變) |
| **細胞遺傳學 (FISH)** | **del(13q)** (單獨出現最佳)、正常核型 | **del(17p)** (最差，**TP53** 喪失)、**del(11q)** |
| **流式細胞儀指標** | **↓** ZAP-70 (<20%)<br>**↓** CD38 (<30%) | **↑** ZAP-70 (>20%)<br>**↑** CD38 (>30%) |
| **細胞起源特徵** | 成熟記憶型 B 細胞 (Post-germinal center) | 原始/未成熟 B 細胞 (Pre-germinal center) |
| **臨床分期** | Binet Stage **A** / Rai Stage **0, I** | Binet Stage **C** / Rai Stage **III, IV** |
| **生化/血液數據** | 正常血球數值、淋巴球倍增時間 **>12個月** | 嚴重貧血、**PLT↓**、淋巴球倍增時間 **<6個月** |
