## 題目

有關B與T淋巴球的敘述,下列何者錯誤?

A.正常情況下,血液中的T淋巴球數量大於B淋巴球
B.成熟的T淋巴球表達 surface CD3,而成熟的B淋巴球表達CD10
C.B淋巴球與活化的T淋巴球均可以表達surface HLA class I 與class II 抗原
D.B淋巴球可以分化成plasma cells,與antibody之製造有關

> [!success]- 正確解答
> **B**

---

## 筆記與詳解

### (1) 快速破題

看到題目討論「淋巴球分類」與「表面標記 (Surface Markers)」的配對，解題思路應直指 **CD markers 的特異性與成熟度差異**。在免疫學常識中，T 細胞的泛標記是 CD3，而成熟 B 細胞的標記是 CD19、CD20。看到選項出現 **CD10**，就必須警覺：CD10 (CALLA) 是「未成熟」或生發中心 B 細胞的標記，不該出現在「成熟」的周邊血 B 細胞描述中，由此可直接抓出錯誤選項。

### (2) 詳解預備：關鍵字與知識點拆解

* **T淋巴球數量大於B淋巴球** → **知識點**：周邊血液淋巴球比例。正常生理下 T 細胞佔絕對多數（約 70-80%），B 細胞佔少數（約 10-20%）。
* **CD3** → **知識點**：Pan-T cell marker（泛 T 細胞表面抗原），所有成熟 T 細胞皆會表現。
* **CD10 (CALLA)** → **知識點**：Common Acute Lymphoblastic Leukemia Antigen，主要表現在**未成熟 B 細胞 (Pre-B)** 或淋巴結中的生發中心 (Germinal center) B 細胞。
* **HLA class I 與 class II** → **知識點**：
* Class I 存在於「所有有核細胞」。
* Class II 存在於「抗原呈現細胞 (APC)」（如 B 細胞、巨噬細胞）。
* **陷阱考點**：靜止的 T 細胞無 Class II，但**活化的 T 細胞會表現 HLA-DR (Class II)**。


* **Plasma cells (漿細胞)** → **知識點**：B 細胞受刺激後的最終分化型態，細胞質豐富且核偏一側 [🖼️](https://www.google.com/search?q="plasma+cell"+morphology+smear&tbm=isch&udm=2)，功能為專職分泌抗體。

### (3) 詳解

* **(A) 正確**：正常成人的周邊血液中，淋巴球的分佈以 T 細胞為主（約 70-80%），其次為 B 細胞（約 10-20%），最少的是 NK 細胞（約 5-10%）。
* **(B) 錯誤 (本題答案)**：前半句正確，成熟的 T 細胞會表達 CD3；但後半句錯誤，**成熟的 B 細胞主要表達 CD19、CD20、CD21**。CD10 屬於**早期未成熟** B 細胞或生發中心 B 細胞的特徵，隨著 B 細胞成熟並釋放至周邊血，CD10 的表現會消失。
* **(C) 正確**：這是一個經典國考陷阱。HLA class I 存在於所有有核細胞，故 B、T 細胞皆有。HLA class II 雖然是 B 細胞 (APC) 的標記，但 T 細胞在**被活化後 (Activated T cells)** 表面也會誘導出 HLA class II (主要為 HLA-DR) 的表現。
* **(D) 正確**：B 淋巴球負責體液免疫，辨識抗原後會進行株落擴增，並最終分化為漿細胞 (Plasma cells)，開始大量合成並分泌特異性抗體 (Antibodies)。

### (4) 核心邏輯與記憶口訣

* **縱向邏輯軸線**：造血幹細胞分化 → 進入不同器官成熟（胸腺 T / 骨髓 B） → 隨著成熟度改變表面 CD 標記 → 進入周邊血執行任務 → 遇抗原活化（T 表現 HLA-DR / B 分化為漿細胞） → 免疫防禦。
* **國考記憶口訣（CD標記）**：
* **T 細胞**：「**3** 歲的 **T**oy」 (CD**3** 是所有 T 細胞的基底標記)。
* **B 細胞**：「**19**、**20** 歲的 **B**oy 才算成熟」 (成熟 B 細胞標記為 CD**19**、CD**20**)。
* **CD10**：「還沒 **10** 歲，不成熟」 (CD**10** 為未成熟/早期 B 細胞指標)。



### (5) 淋巴球特性橫向比較表

| 比較維度 | T 淋巴球 (T cells) | B 淋巴球 (B cells) | 自然殺手細胞 (NK cells) |
| --- | --- | --- | --- |
| **成熟器官** | 胸腺 (Thymus) | 骨髓 (Bone Marrow) | 骨髓 (Bone Marrow) |
| **泛細胞標記 (Pan-marker)** | **CD3** | **CD19, CD20** | CD56, CD16 |
| **其他特殊標記** | CD4 (Th), CD8 (Tc) | **CD10 (未成熟)**, CD21 (EBV受體) | 缺乏 CD3, CD19 |
| **周邊血佔比** | **70-80%** (絕對多數) | 10-20% | 5-10% |
| **HLA Class II 表現** | 靜止期：無 / **活化期：有** | **天生具備** (屬 APC) | 無 |
| **核心功能與機轉** | 細胞介導免疫 (Cell-mediated) | 體液免疫 (轉化漿細胞產抗體) | 先天免疫 (抗病毒/腫瘤)、**ADCC作用** |
