## 題目

發生下列何種情況會造成血清的陰離子間隙(anion gap)增加?

A.低白蛋白血症(hypoalbuminemia)
B.高血鈣(hypercalcemia)
C.高血鉀(hyperkalemia)
D.乳酸中毒(lactic acidosis)

> [!success]- 正確解答
> **D**

---

## 筆記與詳解

### (1) 快速破題

看到題目詢問「陰離子間隙 (Anion Gap, AG) 增加」，應立即聯想到**高 AG 代謝性酸中毒**。

* 解題思路：在醫學與護理常識中，高 AG 最經典的記憶口訣是 **MUDPILES**，其中的 **L** 代表 **L**actic acidosis（乳酸中毒）。因此看到「乳酸中毒」即可秒選 D。
* 反向排除：AG 反映的是血液中「未測定陰離子 (UA)」與「未測定陽離子 (UC)」的差值。A 是陰離子減少，B、C 是陽離子增加，這三者在數學邏輯上都會導致 AG **下降**。

### (2) 關鍵字與知識點拆解

* **陰離子間隙 (AG)**：常規計算公式為 $AG = [Na^+] - ([Cl^-] + [HCO_3^-])$。正常值約為 8-12 mEq/L。其本質為未測定陰離子扣除未測定陽離子 ($AG = UA - UC$)。
* **未測定陰離子 (UA)**：血漿中最大的 UA 是**白蛋白 (Albumin)**。異常時則為乳酸、酮體、毒物代謝產物。
* **未測定陽離子 (UC)**：主要為**鈣 ($Ca^{2+}$)**、**鎂 ($Mg^{2+}$)**、**鉀 ($K^+$)** 及免疫球蛋白 (如 IgG)。
* **乳酸中毒 (Lactic acidosis)**：無氧代謝或組織灌流不足導致乳酸堆積，解離出氫離子 ($H^+$) 與乳酸根。

### (3) 深入詳解

陰離子間隙的變化嚴格取決於 $AG = UA - UC$ 的平衡：

* **A. 低白蛋白血症 (hypoalbuminemia)**：**錯誤**。白蛋白帶負電，是血液中最主要的 UA。當白蛋白降低時，UA 下降，會導致 **AG 下降**。（臨床上低白蛋白極易掩蓋真實的高 AG 狀態，因此重症單位常需使用白蛋白校正公式：$校正後 AG = 測量 AG + 2.5 \times (4.5 - 血清白蛋白濃度)$）。
* **B. 高血鈣 (hypercalcemia)**：**錯誤**。鈣離子帶正電，屬於 UC。當血中未測定陽離子增加時，會抵銷原有的陰離子缺口，導致 **AG 下降**。
* **C. 高血鉀 (hyperkalemia)**：**錯誤**。鉀離子帶正電，同樣屬於 UC（常規 AG 計算不包含鉀）。鉀離子濃度過高會使 **AG 下降**。
* **D. 乳酸中毒 (lactic acidosis)**：**正確**。乳酸解離產生大量 $H^+$，消耗血中的 $HCO_3^-$ 導致代償性下降；同時，解離出的「乳酸根 (Lactate)」作為額外的未測定陰離子 (UA) 留在血液中。UA 大量增加，導致 **AG 上升**。

### (4) 核心邏輯與延伸統整

**縱向生理機轉聯想**：
組織缺氧或休克 → 進行無氧代謝產生異常酸 (乳酸) → 釋放 $H^+$ 消耗 $HCO_3^-$ (數值↓) → 血漿中留下大量乳酸根 (UA ↑) → $AG ↑$ → 臨床診斷為**高 AG 代謝性酸中毒**。嚴重酸中毒時，患者常伴隨深而快的庫斯莫耳氏呼吸 (Kussmaul breathing)[🖼️](https://www.google.com/search?q="Kussmaul+breathing"+clinical+sign&tbm=isch&udm=2) 以進行呼吸代償排出 $CO_2$。

**橫向比較：陰離子間隙 (AG) 異常狀態統整**

| AG 狀態 | 核心機轉 ($AG = UA - UC$) | 代表病因 / 臨床狀態 | 關鍵生化 / 鏡檢特徵 |
| --- | --- | --- | --- |
| **AG 增加** | **UA ↑** (異常酸增加) | 乳酸中毒、DKA、尿毒症 | $HCO_3^-$ **↓**、$pH$ **↓** |
|  |  | 乙二醇 (Ethylene glycol) 中毒 | 尿沉渣見**草酸鈣結晶** (Calcium oxalate crystals)[🖼️](https://www.google.com/search?q="Calcium+oxalate+crystals"+urine+sediment&tbm=isch&udm=2) |
|  | **UC ↓** (陽離子流失) | 低血鈣、低血鎂 | $Ca^{2+}$ **↓**、$Mg^{2+}$ **↓** |
| **AG 下降** | **UA ↓** (陰離子流失) | **低白蛋白血症** (最常見病因) | Albumin **↓** (引發**偽陰性**高 AG) |
|  | **UC ↑** (異常陽離子增加) | 高血鈣、高血鎂、高血鉀 | $Ca^{2+}$ **↑**、$K^+$ **↑** |
|  |  | 多發性骨髓瘤 (Multiple Myeloma) | 血液抹片見**錢幣狀排列** (Rouleaux formation)[🖼️](https://www.google.com/search?q="Rouleaux+formation"+blood+smear&tbm=isch&udm=2) |
| **AG 正常** | 腸道/腎臟直接流失 $HCO_3^-$ | 嚴重腹瀉、腎小管酸中毒 (RTA) | $HCO_3^-$ **↓**、代償性 $Cl^-$ **↑** (高氯性酸中毒) |

**💡 高 AG 代謝性酸中毒記憶口訣 (MUDPILES)：**

* **M**ethanol (甲醇)
* **U**remia (尿毒症)
* **D**KA (糖尿病酮酸中毒)
* **P**ropylene glycol (丙二醇)
* **I**soniazid / **I**nfection (異煙肼/感染)
* **L**actic acidosis (乳酸中毒)
* **E**thylene glycol (乙二醇)
* **S**alicylates (水楊酸/阿斯匹靈中毒)
