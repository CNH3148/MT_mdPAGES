## 題目

有關測量腎絲球過濾率(GFR)的指標cystatin C之敘述,下列何者最不適當?

A.為內生性的指標
B.為silver standard
C.為 cysteine protease inhibitor
D.甲狀腺機能亢進時體內產量增加

> [!success]- 正確解答
> **B**

---

## 筆記與詳解

## (1) 快速破題

看到題目問「Cystatin C 的敘述何者最不適當」，應立刻啟動以下核心反射：

* **關鍵字直接對決**：Cystatin C 是臨床評估 GFR 的重要**內生性指標**（由全核細胞恆定分泌），其生化本質為**半胱氨酸蛋白酶抑制劑 (Cysteine protease inhibitor)**。
* **干擾項識別**：醫學界針對 GFR 的量測，公認的「黃金標準 (Gold standard)」為**菊糖清除率 (Inulin clearance)**，生化檢驗領域中不存在「銀標準 (Silver standard)」之法定術語，此選項為刻意製造的文字陷阱，故直接選 **B**。
* **高階考點應變**：必須謹記 Cystatin C 的非腎性干擾因素——**甲狀腺機能亢進時體內產量會增加**（屬於國考極常出現的鑑別度考點）。

---

## (2) 詳解預備

* **Cystatin C (半胱氨酸蛋白酶抑制劑 C)** → 分子量小 (13 kDa) 的鹼性蛋白質，體內所有**具細胞核的細胞**皆能以恆定速率產生。
* **GFR (腎絲球過濾率)** → 評估腎臟排泄功能最具代表性的指標。
* **Gold Standard (黃金標準)** → 臨床定義上清除率等於 GFR 的指標，法定唯一為外生性的**菊糖 (Inulin)**。
* **Cysteine Protease Inhibitor** → Cystatin C 的生化功能分類，負責抑制細胞內外的半胱氨酸蛋白酶，保護組織免受降解。
* **Hyperthyroidism (甲狀腺機能亢進)** → 全身細胞代謝率異常亢進，導致內源性蛋白質（含 Cystatin C）合成與釋放量顯著**上升**。

---

## (3) 詳解

* **A. 為內生性的指標**：**正確**。Cystatin C 由體內有核細胞自行合成並釋放至血液中，不需像菊糖 (Inulin) 清除率試驗一樣進行外源性靜脈輸注。
* **B. 為silver standard**：**錯誤（最不適當，故為本題答案）**。醫學文獻與教科書中，僅有外生性的菊糖 (Inulin) 被尊稱為「黃金標準 (Gold Standard)」。Cystatin C 與肌酸酐 (Creatinine) 皆屬於常見的「內生性替代指標（Alternative marker）」，並無「銀標準」這種正式的分級稱謂。
* **C. 為 cysteine protease inhibitor**：**正確**。此為其基本的生物化學結構與功能命名（Cystatin 家族成員）。
* **D. 甲狀腺機能亢進時體內產量增加**：**正確**。雖然 Cystatin C 的產量不受肌肉量、年齡、性別影響，但會受到甲狀腺功能的干擾。**甲狀腺機能亢進**時，細胞轉錄與代謝加快，Cystatin C 產量**增加**（導致血清濃度上升，造成 eGFR 偽性降低）；反之，甲狀腺機能低下時，其產量則會**減少**。

---

## (4) 縱向聯想與橫向比較

### 縱向邏輯因果聯想

甲狀腺機能亢進 [🖼️](https://www.google.com/search?q=%22hyperthyroidism%22+clinical+sign&tbm=isch&udm=2) 臨床表徵 → 全身有核細胞轉錄與代謝率 **↑** → Cystatin C 合成與釋放總量 **↑** → 血清 Cystatin C 偵測濃度 **↑** → 臨床公式估算之腎絲球過濾率 (eGFR) **偽性 ↓** → 誤判為腎功能惡化。

### 記憶口訣

> **「菊糖居金位，肌酸看肌肉，希斯核甲催。」**
> （菊糖是黃金標準；肌酸酐受肌肉量影響；Cystatin C 來自有核細胞、受甲狀腺調節、本質是抑制催化酶）
