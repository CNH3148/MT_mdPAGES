## 題目

有關動脈血氣體分析的檢體處理敘述,下列何者最不適當?

A.通常使用含有 heparin 抗凝劑的採血管
B.若沒有妥善密封,氧分壓將會上升
C.運送過程中若使用過低的溫度保存,將導致氧分壓下降
D.檢體放置過久,二氧化碳分壓將會上升

> [!success]- 正確解答
> **C**

---

## 筆記與詳解

**(1) 快速破題**
看到題目關於「動脈血氣體分析 (ABG) 檢體處理」，並且要找「最不適當」的描述 → 就要立刻想到影響檢體品質的三大元兇：「**細胞代謝**」、「**氣泡/接觸空氣**」與「**抗凝劑比例**」。題目中提到「溫度」，直覺反應應是「低溫保存在冰水中是為了**抑制細胞代謝**」。血球細胞代謝會消耗 $O_2$、產生 $CO_2$；若保存溫度夠低，能「阻止」氧分壓下降。因此選項 C 稱過低溫度會「導致」氧分壓下降是因果完全相反的敘述，直接秒殺選 C。

**(2) 詳解預備**

* **動脈血氣體分析 (ABG)** → 評估病患酸鹼平衡 ($pH$)、通氣狀況 ($PCO_2$) 與氧合狀態 ($PO_2$) 的關鍵檢驗。
* **Heparin (肝素)** → ABG 唯一建議使用的抗凝劑，通常預先塗佈於塑膠採血針內壁，不會改變血液的酸鹼值 (相較於 EDTA 或 Citrate 等液體抗凝劑)。
* **妥善密封 (排空氣泡)** → 檢體若接觸室內空氣 (室溫空氣的 $PO_2$ 約 150 mmHg，$PCO_2$ 近乎 0 mmHg)，會造成氣體發生濃度梯度擴散。
* **低溫保存 (冰浴)** → 傳統上用來減緩有核細胞 (如白血球) 的有氧代謝。

**(3) 詳解**

* **A 正確**：ABG 採血標準配備是使用含有 lyophilized (凍乾) 或鋰鹽 Heparin 抗凝劑的專用採血針[🖼️](https://www.google.com/search?q=heparinized+blood+gas+syringe&tbm=isch&udm=2)。Heparin 不會影響血液的 $pH$ 值，是氣體分析的最佳選擇。
* **B 正確**：若未妥善密封或檢體內混有氣泡[🖼️](https://www.google.com/search?q=air+bubbles+in+blood+gas+syringe&tbm=isch&udm=2)未排出，由於大氣中的氧分壓 ($PO_2$) 通常高於靜脈血或部分動脈血，且二氧化碳分壓 ($PCO_2$) 遠低於血液，氣體將遵循濃度梯度交換。這會導致檢體的 $PO_2$ **上升**、$PCO_2$ **下降**。
* **C 錯誤 (本題答案)**：低溫保存的初衷是為了「抑制」血球細胞代謝，從而「**防止**」 $PO_2$ 下降與 $PCO_2$ 上升。此外，若使用現代高透氣性的塑膠採血針並置於冰水中，因低溫下氧氣溶解度增加，反而可能促使外界氧氣穿透管壁進入血液，導致 $PO_2$ **假性上升**。因此，過低溫度保存絕對不會「導致氧分壓下降」。
* **D 正確**：檢體放置過久，血液中的有核細胞會持續進行有氧代謝，消耗氧氣並產生二氧化碳與乳酸，導致 $PO_2$ **下降**、$PCO_2$ **上升**、$pH$ 值**下降**。

**(4) 邏輯梳理與橫向比較**

* **核心軸線**：檢體暴露環境條件 → 氣體擴散方向 或 細胞代謝速率改變 → $pH$、$PCO_2$、$PO_2$ 數值產生偽性偏移。
* **記憶口訣**：
* **接觸空氣 (Air)** = 趨向大氣特性 (氧多、二氧化碳極少) → **$PO_2$ ↑、$PCO_2$ ↓** ($pH$ ↑)
* **細胞代謝 (Metabolism)** = 模擬人體呼吸 (吸氧、吐二氧化碳、生乳酸) → **$PO_2$ ↓、$PCO_2$ ↑** ($pH$ ↓)



**(5) 檢體處理不當之干擾總表**

| 處理不當情況 | 發生機轉 | $pH$ | $PCO_2$ | $PO_2$ |
| --- | --- | --- | --- | --- |
| **放置過久 (室溫)** | 白血球持續有氧代謝 (消耗 $O_2$ / 產生 $CO_2$ 與乳酸) | **↓** | **↑** | **↓** |
| **未密封 / 混入氣泡** | 與大氣發生氣體交換 ($O_2$ 進入 / $CO_2$ 逸散) | **↑** | **↓** | **↑** (若原 $PO_2$ < 150) |
| **Heparin 比例過高** | 液態抗凝劑稀釋效應 (若非使用凍乾肝素) | **↓** | **↓** | **↑** (趨向室內空氣) |
| **冰水浴 + 塑膠針筒** | 低溫增氧氣溶解度 + 塑膠透氣性 ($O_2$ 滲入) | - | - | **↑** |
