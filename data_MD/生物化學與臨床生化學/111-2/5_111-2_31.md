## 題目

31.有關糖尿病之敘述，下列何者錯誤？

第一型糖尿病多與人類白血球抗原（human leukocyte antigens, HLAs）有關
第一型糖尿病多因蘭氏小島的β細胞被破壞所引起
第二型病人多於青少年發病
第二型糖尿病主要是因胰島素阻抗（insulin resistance）所致

> [!success]- 正確解答
> **C**

---

## 筆記與詳解

### (1) 快速破題

看到題目比較「第一型」與「第二型」糖尿病，且要找「錯誤」的敘述 → 應立即啟動「發病年齡」、「致病機轉」與「胰島素狀態」的經典對比。傳統醫學與國考常識中，最核心的區別是「第一型＝青少年發病（幼年型）、第二型＝中老年發病（成人型）」。題目選項 C 敘述「第二型病人多於青少年發病」完全顛倒了兩者的典型好發年齡，因此直接鎖定為錯誤答案。

### (2) 關鍵字與知識點

* **第一型糖尿病 (Type 1 DM) →** 知識點：主要為自體免疫疾病，特徵為胰島素「絕對缺乏」，極易引發糖尿病酮酸中毒 (DKA)，患者尿液試紙常呈酮體陽性[🖼️](https://www.google.com/search?q=urine+dipstick+ketone+positive&tbm=isch&udm=2)。
* **人類白血球抗原 (HLAs) →** 知識點：與免疫系統辨識有關的基因群。Type 1 DM 的遺傳易感性與第 6 對染色體上的 HLA 基因（尤其是 HLA-DR3 與 HLA-DR4）高度相關。
* **蘭氏小島 β 細胞 (Islets of Langerhans β cells) →** 知識點：胰臟中負責分泌胰島素的內分泌細胞。在 Type 1 DM 中，這些細胞被自體抗體（如 ICA、GADA）錯誤攻擊並摧毀。
* **第二型糖尿病 (Type 2 DM) →** 知識點：因不良生活型態、肥胖與遺傳綜合導致，特徵為胰島素「相對缺乏」，常見臨床表徵為因嚴重胰島素阻抗引起的黑色棘皮症 (Acanthosis nigricans)[🖼️](https://www.google.com/search?q="Acanthosis+nigricans"+clinical+sign&tbm=isch&udm=2)。
* **胰島素阻抗 (Insulin resistance) →** 知識點：周邊組織（如骨骼肌、脂肪細胞、肝臟）對胰島素訊號的敏感度下降，是 Type 2 DM 最核心的初期病理機轉。

### (3) 詳解

* **(A) 第一型糖尿病多與人類白血球抗原（HLAs）有關：敘述正確。** 第一型糖尿病具有基因易感性，研究證實其發病與特定 HLA 基因型（如 HLA-DR3, HLA-DR4, HLA-DQ8）有強烈關聯，這使得患者的免疫系統容易對自身的胰島細胞產生異常反應。
* **(B) 第一型糖尿病多因蘭氏小島的β細胞被破壞所引起：敘述正確。** 此為第一型糖尿病的根本致病機轉。患者體內會產生自體抗體（如抗胰島細胞抗體 ICA、抗麩胺酸脫羧酶抗體 GADA）攻擊並摧毀負責分泌胰島素的 β 細胞，最終導致胰島素**絕對缺乏**。
* **(C) 第二型病人多於青少年發病：敘述錯誤，為本題正確解答。** 傳統上，第二型糖尿病被稱為「成人型糖尿病」，**好發於中老年人**（通常 > 40 歲）。雖然近年來因兒童肥胖率上升，導致青少年罹患第二型的比例增加，但在醫學分類與考題的「典型特徵」設定上，**第一型才是主要於兒童或青少年期發病**的類型。
* **(D) 第二型糖尿病主要是因胰島素阻抗（insulin resistance）所致：敘述正確。** 第二型糖尿病患者初期胰臟仍能分泌胰島素，但身體細胞對胰島素的反應遲鈍（胰島素阻抗），葡萄糖無法順利進入細胞。為克服此阻抗，胰臟初期會代償性地分泌更多胰島素（高胰島素血症），長期下來導致 β 細胞過勞、功能衰竭，最終演變為胰島素相對不足。

### (4) 核心軸線與邏輯因果

**第一型糖尿病 (T1DM) 發展軸線：**
HLA 基因易感性 + 環境誘發因素 → 自體免疫系統錯誤激活 → 產生自體抗體 (ICA, GADA) → **直接破壞胰臟 β 細胞** → 胰島素**絕對缺乏** → 血糖飆升 + 脂肪大量分解代謝 → 產生酮體 → 引發**糖尿病酮酸中毒 (DKA)**

**第二型糖尿病 (T2DM) 發展軸線：**
家族遺傳 + 肥胖/不良飲食型態 → 周邊組織產生**胰島素阻抗** → 胰臟代償性過度分泌胰島素 (高胰島素血症) → 長期代償導致 **β 細胞過勞衰竭** → 胰島素**相對缺乏** → 血糖持續升高 → 易引發**高滲透壓高血糖狀態 (HHS)**

### (5) T1DM 與 T2DM 核心重點比較

| 比較維度 | 第一型糖尿病 (T1DM) | 第二型糖尿病 (T2DM) |
| --- | --- | --- |
| **典型發病年齡** | **< 30歲** (兒童/青少年為主) | **> 40歲** (中老年為主) |
| **致病機轉** | **自體免疫** (破壞β細胞) | **胰島素阻抗** (合併β細胞衰竭) |
| **典型體型** | **消瘦** | **肥胖** (> 80%) |
| **胰島素狀態** | **絕對缺乏** | 初期偏高/正常，後期**相對缺乏** |
| **C-peptide** | **顯著下降** 或 測不到 | 正常 或 **升高** |
| **遺傳因子** | **HLA-DR3 / DR4** 相關 | 多基因遺傳 (家族史關聯性**極強**) |
| **自體抗體 (ICA/GADA)** | **陽性 (+)** | **陰性 (-)** |
| **急性併發症** | **糖尿病酮酸中毒 (DKA)** | **高滲透壓高血糖狀態 (HHS)** |
| **典型特徵/表徵** | 三多一少、尿液酮體陽性[🖼️](https://www.google.com/search?q=urine+dipstick+ketone+positive&tbm=isch&udm=2) | 黑色棘皮症[🖼️](https://www.google.com/search?q="Acanthosis+nigricans"+clinical+sign&tbm=isch&udm=2)、多無早期症狀 |
| **核心治療原則** | **必須依賴外源胰島素注射** | 飲食/運動控制、**口服降糖藥**、後期胰島素 |
