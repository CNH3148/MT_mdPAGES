## 題目

下列何者不是造成 type B lactic acidosis 的病理因素?

A.組織缺氧
B.氰化物中毒
C.糖尿病酸中毒
D.肝衰竭

> [!success]- 正確解答
> **A**

---

## 筆記與詳解

### (1) 快速破題

看到題目區分乳酸性酸中毒（Lactic acidosis）的「Type A」與「Type B」，核心解題思路在於「是否有巨觀的組織缺氧 (Tissue hypoxia) 與低血流灌流 (Hypoperfusion)」。

* **Type A** = **A**bnormal oxygenation（缺氧引起）。
* **Type B** = **B**yproducts/Biochemical（非缺氧引起，多為代謝異常、毒物、器官衰竭）。

題目問「何者**不是** Type B」，即反向尋找「何者是 Type A」。選項 A「組織缺氧」正是 Type A 的絕對定義條件，因此直接選 A。

---

### (2) 關鍵字 → 知識點

* **Type A 乳酸性酸中毒** → 知識點：**組織缺氧 (Tissue hypoxia)** 或低灌流驅動的無氧糖解。常見於休克、心衰竭、嚴重貧血。
* **Type B 乳酸性酸中毒** → 知識點：**非缺氧性代謝異常**。細分為 B1 (全身系統性疾病，如肝腎衰竭、糖尿病)、B2 (藥物或毒物干擾，如 Metformin、氰化物)、B3 (先天性代謝缺陷)。
* **氰化物 (Cyanide) 中毒** → 知識點：毒物干擾粒線體電子傳遞鏈 (抑制 Cytochrome c oxidase)，導致細胞無法利用氧氣（組織氧分壓正常但細胞內窒息），歸類為 **Type B2**。
* **肝衰竭 (Liver failure)** → 知識點：肝臟負責透過柯氏循環 (Cori cycle) 將乳酸轉化為葡萄糖。肝衰竭導致「乳酸清除率下降」而堆積，歸類為 **Type B1**。

---

### (3) 詳解辨析

乳酸 (Lactate) 是細胞進行無氧糖解 (Anaerobic glycolysis) 的終產物。臨床上根據其蓄積的病理機轉分為兩型：

* **(A) 組織缺氧 (Tissue hypoxia)：為本題正確答案。**
組織缺氧是 **Type A** 乳酸性酸中毒的根本原因。當心輸出量不足、嚴重貧血或呼吸衰竭導致氧氣無法送達周邊組織時，細胞被迫轉向無氧代謝，大量產生乳酸。嚴重缺氧患者臨床上常表現發紺 (Cyanosis)[🖼️](https://www.google.com/search?q=Cyanosis+clinical+sign&tbm=isch&udm=2)。因其非屬 Type B，故選此項。
* **(B) 氰化物中毒：錯誤（屬 Type B）。**
氰化物屬於毒物引起的 **Type B2** 酸中毒。雖然它會造成「細胞內窒息 (Histotoxic hypoxia)」，但患者巨觀的血液氧氣運輸與組織灌流是**正常**的。氰化物阻斷粒線體利用氧氣的能力，導致靜脈血含氧量異常偏高，呈現鮮紅色 (Cherry-red venous blood)[🖼️](https://www.google.com/search?q=Cyanide+poisoning+cherry+red+blood&tbm=isch&udm=2)，而非真正的組織缺血。
* **(C) 糖尿病酸中毒 (Diabetic ketoacidosis, DKA)：錯誤（屬 Type B）。**
糖尿病屬於基礎代謝疾病引起的 **Type B1** 酸中毒。在嚴重代謝紊亂時，除了酮體產生，也會伴隨糖解作用失衡與組織代謝異常，引發乳酸堆積。
* **(D) 肝衰竭：錯誤（屬 Type B）。**
肝臟是人體代謝乳酸的主要器官。肝臟嚴重受損（如肝硬化、猛爆性肝炎併發黃疸 (Jaundice)[🖼️](https://www.google.com/search?q=Jaundice+clinical+sign&tbm=isch&udm=2) 時），清除血中乳酸的能力劇降，導致乳酸滯留，歸類於 **Type B1** 酸中毒。

---

### (4) 核心軸線與橫向比較

**生理機轉軸線：**

* **產量過多 (Overproduction)：** 組織灌流不足 (Type A) 或 毒物/代謝異常強制無氧糖解 (Type B2/B3)。
* **清除下降 (Decreased clearance)：** 肝衰竭 (Type B1) 導致柯氏循環停滯。

**記憶口訣：**

* **Type A** = **A**sphyxia / **A**nemia / **A**rrhythmia (休克/缺氧相關)
* **Type B** = **B**ad liver/kidney (器官衰竭) / **B**ad drugs (毒藥物) / **B**orn errors (先天異常)

| 分型 / 生理狀態 | 致病機轉 | 血氧與組織灌流狀態 | 常見病因 (舉例) | 實驗室/臨床特徵 |
