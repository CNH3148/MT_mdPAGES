## 題目

有關尿酸的檢測方法,下列何者正確?

A.可與 picrate 反應,在510 nm 測吸光值
B.可與 phosphotungstate反應,vitamin C 會造成偽陽性
C.可用 uricase-UV法,測吸光值的增加
D.可用 uricase-peroxidase法,vitamin C 會造成偽陽性

> [!success]- 正確解答
> **B**

---

## 筆記與詳解

### (1) 快速破題

看到「尿酸 (Uric acid) 檢測」與「Vitamin C 干擾」，必須立刻聯想到 Vitamin C 是**強還原劑**。在檢驗醫學中，Vitamin C 的干擾方向取決於反應原理：

* 遇到**化學還原法** (如 Phosphotungstate)：Vitamin C 會「代打」加入還原反應，產生更多顏色 → **偽陽性**。
* 遇到**酵素呈色法** (如 Uricase-peroxidase)：Vitamin C 會「搶奪」反應中間產物 $H_2O_2$ (使其無法順利氧化呈色劑) → **偽陰性**。
此外，看到 Picrate 就要反射想到是測 Creatinine (肌酸酐)；看到 Uricase-UV 法就要知道是測尿酸吸收峰的「消失」(吸光值下降)。綜合以上，直接選 (B)。

---

### (2) 詳解預備

* **Picrate (苦味酸) 反應** → 知識點：Jaffe reaction，此為 Creatinine (肌酸酐) 的專屬經典化學檢測法，非測量尿酸。
* **Phosphotungstate (磷鎢酸) 反應** → 知識點：尿酸傳統化學檢測法。利用尿酸的還原力將磷鎢酸還原成鎢藍 (Tungsten blue)[🖼️](https://www.google.com/search?q=tungsten+blue+solution&tbm=isch&udm=2)。
* **Vitamin C (Ascorbic acid)** → 知識點：常見的內源性強還原劑，是臨床化學檢驗中最經典的干擾物質。
* **Uricase-UV 法** → 知識點：利用尿酸本身在 293 nm 有強烈紫外光吸收，經 Uricase (尿酸酶) 作用後轉為無吸收峰的尿囊素 (Allantoin)。
* **Uricase-peroxidase 法** → 知識點：Trinder reaction (耦合酵素呈色法)，利用 Uricase 產生的 $H_2O_2$，再由 Peroxidase (過氧化物酶) 催化氧化呈色劑發色。

---

### (3) 詳解

* **(A) 錯誤**：Picrate 反應 (Jaffe reaction) 是用來檢測 **Creatinine (肌酸酐)**，反應後會生成橘紅色複合物[🖼️](https://www.google.com/search?q="Jaffe+reaction"+orange+red&tbm=isch&udm=2)，測量波長約在 510~520 nm。
* **(B) 正確**：Phosphotungstate 法為化學還原法。尿酸作為還原劑使試劑呈色。若檢體中含有高濃度的 Vitamin C (強還原劑)，會一併還原磷鎢酸，產生額外的吸光值，導致**偽陽性**。
* **(C) 錯誤**：Uricase-UV 法是監測尿酸在 293 nm 的吸收峰。隨著尿酸被 Uricase 分解成尿囊素，293 nm 的吸光值會**下降**，因此是測量吸光值的「減少」而非增加。
* **(D) 錯誤**：Uricase-peroxidase 法中，呈色關鍵是反應生成的 $H_2O_2$。Vitamin C 會消耗反應中的 $H_2O_2$，使後續呈色劑無法被氧化發色，導致吸光值偏低，造成**偽陰性**。

---

### (4) 邏輯因果與表格統整

**邏輯因果軸線：**
維生素C的化學本質 (強還原力) → 干預檢測反應途徑 → 導致不同檢測法的誤差方向

* **化學法 (Phosphotungstate)**：試劑需要被還原才能發色 → Vit C 主動還原試劑 → 顏色異常加深 → **偽陽性**。
* **酵素呈色法 (Uricase-POD)**：試劑需要 $H_2O_2$ 來氧化發色 → Vit C 搶走/還原 $H_2O_2$ → 顏色無法生成 → **偽陰性**。

> **💡 記憶口訣**：「化學加碼(偽陽)、酵素扯後腿(偽陰)」
> (Vitamin C 在化學還原法中幫忙加碼產色，在酵素氧化法中扯後腿搶走反應物。)

**常見 NPN (非蛋白氮) 檢測法與 Vitamin C 干擾橫向對比：**

| 檢驗項目 (方法) | 核心原理 | 測量波長 / 訊號變化 | Vitamin C 干擾方向 |
| --- | --- | --- | --- |
| **Uric acid** (Phosphotungstate 化學法) | 尿酸**還原**磷鎢酸呈色 | 700 nm / 吸光值↑ | **偽陽性** (強還原力干擾) |
| **Uric acid** (Uricase-UV 直接酵素法) | 尿酸被分解為尿囊素 (無吸收) | 293 nm / 吸光值**↓** | 無顯著干擾 (不依賴氧化還原呈色) |
| **Uric acid** (Uricase-Peroxidase 耦合酵素法) | 產物 $H_2O_2$ **氧化**呈色劑發色 | 500 nm / 吸光值↑ | **偽陰性** (消耗 $H_2O_2$) |
| **Creatinine** (Jaffe reaction 苦味酸法) | Creatinine 與 Picrate 形成橘紅色複合物 | 510-520 nm / 吸光值↑ | **偽陽性** (Vit C/Glucose/酮體皆干擾) |
| **Glucose** (GOD-POD 氧化酶法) | (同 Uricase-POD 反應架構) | 500 nm / 吸光值↑ | **偽陰性** (消耗 $H_2O_2$) |
