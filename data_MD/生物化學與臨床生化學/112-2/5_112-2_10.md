## 題目

有關 cystatin C 的敘述,下列何者錯誤?

A.屬於GFR 內源性標誌的bronze standard
B.分子量比白蛋白小
C.在腎小管不會被分泌
D.理論上 cystatin C 測出eGFR 比肌酸酐廓清率(creatinine clearance rate)高

> [!success]- 正確解答
> **D**

---

## 筆記與詳解

### (1) 快速破題

看到題目在比較「Cystatin C」與「肌酸酐 (Creatinine)」，並且要找「錯誤」的敘述 → 就要直覺想到這兩種標誌物在「腎小管」的代謝差異（是否被分泌）。
因為肌酸酐除了被腎絲球過濾，還會被腎小管**分泌 (secretion)** 進入尿液，導致公式算出來的廓清率會「高估」真實的腎絲球濾過率 (GFR)。而 Cystatin C **不會**被腎小管分泌，因此其測出的 eGFR 較貼近真實 GFR。既然肌酸酐會高估，那 Cystatin C 算出的數值理應比肌酸酐廓清率「低」。因此立刻判斷 D 為錯誤敘述。

### (2) 詳解預備

* **Cystatin C (半胱氨酸蛋白酶抑制劑 C) →** 存在於所有有核細胞的低分子量蛋白質，以恆定速率產生。因分子量小能自由通過腎絲球，是優秀的 GFR 內源性指標，不受肌肉量或飲食影響。
* **Creatinine clearance rate (CCr, 肌酸酐廓清率) →** 臨床最基礎的 eGFR 評估標準。但因肌酸酐有 10~20% 來自腎小管分泌，會導致結果高估 GFR。
* **Bronze standard (銅牌標準) →** GFR 標誌物的分級俗稱。金牌 (Gold) 是菊糖 (Inulin)，銀牌 (Silver) 是外源性放射或造影劑 (如 Iohexol)，銅牌 (Bronze) 則為內源性指標 (如 Creatinine、Cystatin C)。
* **分泌 (Secretion) vs. 再吸收 (Reabsorption) →** 分泌是物質從血液進入尿液管腔；再吸收是從尿液管腔回到血液或管壁細胞內。

### (3) 詳解

* **A. 屬於 GFR 內源性標誌的 bronze standard：正確。** 評估 GFR 的「Gold standard (黃金標準)」是完全不被分泌與再吸收的外源性菊糖 (Inulin)；而人體自行產生的內源性指標 (Cystatin C、Creatinine) 由於存在少許誤差或受生理條件限制，在評估準確度層級上被歸類為「Bronze standard (銅牌標準)」。
* **B. 分子量比白蛋白小：正確。** Cystatin C 分子量極小（約 13.3 kDa），而白蛋白 (Albumin) 分子量約 66~69 kDa。因為分子量極小且帶正電，Cystatin C 得以自由穿透腎絲球濾過膜。
* **C. 在腎小管不會被分泌：正確。** Cystatin C 在腎絲球自由濾過後，會在近端腎小管被細胞「再吸收」並直接在細胞內「分解 (catabolized)」，因此它**不會**被腎小管分泌到尿液中，也不會以原形回到血液中。
* **D. 理論上 cystatin C 測出 eGFR 比肌酸酐廓清率 (creatinine clearance rate) 高：錯誤 (本題答案)。** 由於肌酸酐 (Creatinine) 會被近端腎小管**分泌**（使尿中排泄總量增加約 10~20%），導致算出的肌酸酐廓清率 (CCr) 會**高估**真實的 GFR。Cystatin C 由於不被分泌，其 eGFR 數值較貼近真實 GFR。因此，理論上 Cystatin C 測出的 eGFR 會比被高估的肌酸酐廓清率還要「低」。

### (4) 核心軸線與邏輯因果

**【縱向聯想：GFR 標誌物的假性誤差機轉】**

* **Creatinine 路徑：** 肌肉代謝產生 → 腎絲球自由濾過 + **腎小管額外分泌 (10-20%)** → 尿中總量虛高 → 代入廓清率公式 → **高估**真實 GFR。
* **Cystatin C 路徑：** 有核細胞恆定產生 → 腎絲球自由濾過 + 腎小管再吸收並直接分解 (**無分泌**) → 血液濃度純粹反映濾過率 → 算出的 eGFR **貼近**真實 GFR。
* **臨床演進：** 當各項 GFR 標誌物顯示腎功能嚴重衰退進入慢性腎臟病晚期 (ESRD)，尿流速極度變慢，此時尿液沉渣鏡檢常可見反映腎小管擴張變性的寬圓柱體 (broad casts) [🖼️](https://www.google.com/search?q=%22broad+casts%22+urine+sediment&tbm=isch&udm=2) 及代表圓柱體長期滯留退化的蠟狀圓柱體 (waxy casts) [🖼️](https://www.google.com/search?q=%22waxy+casts%22+urine+sediment&tbm=isch&udm=2)。

**【記憶口訣】**
**「肌凸 (Cr) 估高，C 妹 (CysC) 貼近」**

* **肌**酸酐因被分泌而**凸**出（多排出了），導致算出來的 GFR 被**估高**。
* **C**ystatin C 不被分泌，數值**貼近**真實 GFR（數值相對較低）。
