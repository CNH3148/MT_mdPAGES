## 題目

含有 valinomycin 之離子交換膜(ion exchange membrane)是建構下列何種離子選擇性電極的成分?

A.鈉
B.鉀
C.氯
D.重碳酸根

> [!success]- 正確解答
> **B**

---

## 筆記與詳解

### (1) 快速破題

看到題目出現「**valinomycin**」，直覺反射秒選「**鉀離子 (K⁺)**」。在生化自動分析儀的離子選擇性電極（ISE）中，各個離子都有其專屬的「離子載體（Ionophore）」。Valinomycin 是一種環狀胜肽抗生素，其立體的孔徑與電荷分布與鉀離子完美契合，因此被專門用來製造測量鉀離子的液態薄膜電極。

### (2) 關鍵字與知識點拆解

* **Ion-selective electrode (ISE) 離子選擇性電極** → 利用特定離子載體或薄膜材質，對待測離子產生具專一性的電位變化，並遵循 Nernst equation 計算濃度。
* **Ion exchange membrane 液態聚合物膜** → 將具有辨識能力的離子載體溶於非水溶性溶劑，並混入 PVC 薄膜中，作為電極感測面。
* **Valinomycin 纈胺黴素** → 一種天然的環狀胜肽抗生素（ionophore），對鉀的選擇性是鈉的 10,000 倍，專用於 K⁺ ISE。

### (3) 詳解

* **(A) 鈉 (Na)**：錯誤。鈉離子 ISE 主要是使用**特殊配方的玻璃薄膜（Glass membrane，含鋁矽酸鹽）**，若使用液態薄膜則其離子載體為 **Monensin** 或 ETH 227。
* **(B) 鉀 (K)**：正確。液態薄膜中加入 **Valinomycin**，對鉀離子有極高的特異性親和力，為國考常考之絕對考點。
* **(C) 氯 (Cl)**：錯誤。氯離子 ISE 多使用**固態薄膜（Solid-state membrane）**，主要成分為 **AgCl/Ag₂S** 晶體混合物；或是使用含季銨鹽類（Quaternary ammonium salts）的液態膜。
* **(D) 重碳酸根 (HCO₃⁻)**：錯誤。臨床常用來推算體內重碳酸根的檢驗為 pCO₂，使用的是 **Severinghaus 電極**（氣體感測電極，利用 Teflon 膜讓 CO₂ 通過並改變內部 pH 值）。

### (4) 核心邏輯與聯想

* **縱向原理軸線**：檢體接觸 ISE 薄膜 → 特定離子載體（如 Valinomycin）選擇性捕捉目標離子（K⁺） → 膜兩側產生**電位差（Potentiometry）** → 依據 **Nernst 方程式**計算 → 得知檢體中離子活性/濃度。
* **記憶口訣 (國考必背 ISE 載體對應)**：
* **V**alinomycin → **鉀 (K)** (聯想：**V**ery **K**ing / 偉大的國王)
* **M**onensin → **鈉 (Na)** (聯想：**M**o**N**ic**a** / 莫妮卡)
* **D**i-(n-octylphenyl) phosphate → **鈣 (Ca)** (聯想：**D**octor **Ca**re / 醫生照護)



### (5) 離子選擇與氣體感測電極 (ISE & Gas Electrodes) 總整理

| 待測物 | 電極類型 | **核心材質 / 離子載體 (Ionophore)** |
| --- | --- | --- |
| **鉀 (K⁺)** | 液態薄膜 (Liquid) | **Valinomycin** |
| **鈉 (Na⁺)** | 玻璃薄膜 (Glass) / 液態薄膜 | **鋁矽酸鹽 (Aluminosilicate)** / **Monensin** |
| **氯 (Cl⁻)** | 固態薄膜 (Solid) / 液態薄膜 | **AgCl/Ag₂S 晶粒** / **季銨鹽 (Quaternary ammonium)** |
| **鈣 (Ca²⁺)** | 液態薄膜 (Liquid) | **Di-(n-octylphenyl) phosphate** 或 ETH 1001 |
| **鋰 (Li⁺)** | 液態薄膜 (Liquid) | **ETH 149** (Crown ether 衍生物) |
| **pH (H⁺)** | 玻璃薄膜 (Glass) | **特殊矽酸鹽玻璃 (Silicate glass)** |
| **pCO₂** | 氣體感測 (Gas-sensing) | **Severinghaus 電極** (Teflon膜包覆 pH電極) |
| **pO₂** | 電流式 (Amperometric) | **Clark 電極** (白金陰極 + Ag/AgCl陽極) |
