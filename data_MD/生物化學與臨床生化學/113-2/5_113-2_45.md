## 題目

45.下列何種類型的腎小管酸中毒（renal tubular acidosis, RTA）是起因於近曲小管功能不良，無法回收重碳酸鹽，造成代謝性酸中毒？

RTA type I
RTA type II
RTA type III
RTA type IV

> [!success]- 正確解答
> **B**

---

## 筆記與詳解

### (1) 快速破題

看到題目中「**近曲小管 (proximal tubule)**」與「**重碳酸鹽 (bicarbonate) 回收不良**」，請直接對應 **Type II**。

* **直覺聯想**：英文字母「**B**i」carbonate 代表數字「**2 (II)**」。
* **反向排除**：若題目提到「遠曲小管 (Distal)」與「無法排酸 ($H^+$)」，則為 Type I；若提到「高血鉀 (Hyperkalemia)」，必為 Type IV。

### (2) 關鍵字解析

* **腎小管酸中毒 (RTA)**：腎臟功能異常導致無法有效酸化尿液，引發**正常陰離子間隙 (Normal Anion Gap)** 的高氯性代謝性酸中毒。
* **近曲小管 (Proximal tubule)**：正常生理下負責回收超過 85% 濾過的重碳酸鹽 (HCO₃⁻)。
* **遠曲小管 (Distal tubule) / 集尿管**：負責分泌氫離子 (H⁺) 以排出酸性物質。

### (3) 詳解

* **(A) 錯誤 - RTA type I (遠端型)**：病灶位於遠曲小管 (Distal tubule) 及集尿管的 $\alpha$-間質細胞，缺陷為**無法排泌氫離子 (H⁺)**。導致尿液無法酸化，尿液 pH 值恆定 > 5.5。
* **(B) 正確 - RTA type II (近端型)**：病灶位於近曲小管 (Proximal tubule)，缺陷為**無法回收重碳酸鹽 (HCO₃⁻)**。大量 HCO₃⁻ 從尿液流失導致酸中毒。
* **(C) 錯誤 - RTA type III (混合型)**：極罕見，是 Type I 與 Type II 的混合表現。通常因碳酸酐酶 II (Carbonic anhydrase II) 基因突變引起，臨床上常伴隨大理石骨病 (Osteopetrosis) 的緻密骨骼 X 光表徵[🖼️](https://www.google.com/search?q=osteopetrosis+radiography&tbm=isch&udm=2)。
* **(D) 錯誤 - RTA type IV (高血鉀型)**：病灶在遠端集尿管，肇因於**醛固酮 (Aldosterone) 缺乏或阻抗**。醛固酮作用低下導致無法回收鈉 (Na⁺)，同時無法排除鉀 (K⁺) 與氫 (H⁺)，是**唯一會出現高血鉀**的 RTA 分型。

### (4) 邏輯軸線與口訣

* **縱向生理邏輯**：病灶位置 $\rightarrow$ 離子失衡機制 $\rightarrow$ 血液鉀離子變化 $\rightarrow$ 尿液酸化能力 (pH 值) $\rightarrow$ 併發症。
* **記憶口訣**：
* **Type 1** = **1** 根管子末端塞住 (**D**istal) $\rightarrow$ 酸排不出去 $\rightarrow$ 尿液偏鹼易生結石 (Nephrocalcinosis)[🖼️](https://www.google.com/search?q=nephrocalcinosis+ultrasound&tbm=isch&udm=2)。
* **Type 2** = **2** 個 **B** (**B**icarbonate 漏在 **P**roximal) $\rightarrow$ 常合併 Fanconi 症候群。
* **Type 4** = **4** 肢無力 $\rightarrow$ 因為**高血鉀** (Aldosterone 罷工)。



### (5) 腎小管酸中毒 (RTA) 核心鑑別表格

| 比較維度 | RTA Type I (遠端型) | RTA Type II (近端型) | RTA Type IV (高血鉀型) |
| --- | --- | --- | --- |
| **病灶位置** | **遠曲小管** (Distal) | **近曲小管** (Proximal) | 皮質集尿管 |
| **致病機轉** | **↓ H⁺ 分泌** | **↓ HCO₃⁻ 回收** | **↓ 醛固酮 (Aldosterone)** 作用 |
| **血鉀 (K⁺) 數值** | ↓ (低血鉀) | ↓ (低血鉀) | **↑ (高血鉀)** |
| **尿液 pH 值** | **> 5.5** (絕對無法低於 5.5) | < 5.5 (達穩定態後可酸化) | < 5.5 |
| **腎結石 / 鈣化風險** | **極高** | 無 | 無 |
| **臨床特色 / 併發症** | 腎鈣化 (Nephrocalcinosis)[🖼️](https://www.google.com/search?q=nephrocalcinosis+ultrasound&tbm=isch&udm=2)、骨軟化 | **Fanconi syndrome** (尿液流失葡萄糖/胺基酸)、多發性骨髓瘤 | **糖尿病腎病變**、慢性腎衰竭、NSAIDs 使用 |
