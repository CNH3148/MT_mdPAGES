## 題目

72.銅的缺乏會造成病人黑色素降低，是因為缺少銅參與下列何種酵素活性？

藍胞漿素（ceruloplasmin）
細胞色素 C 氧化酶（cytochrome C oxidase）
超氧化物歧化酶（superoxide dismutase）
酪胺酸酶（tyrosinase）

> [!success]- 正確解答
> **D**

---

## 筆記與詳解

### (1) 快速破題

看到題目關鍵字「**銅 (Copper)**」加上「**黑色素 (Melanin) 降低**」，解題思路應直接聯結到**黑色素的合成途徑**。
黑色素的生成原料為酪胺酸 (Tyrosine)，催化此反應的關鍵限速酵素為「**酪胺酸酶 (Tyrosinase)**」，而該酵素必須依賴「**銅離子**」作為輔因子才能發揮作用。因此，缺銅會直接導致 Tyrosinase 活性下降，使黑色素減少。

### (2) 關鍵字與知識點拆解

* **銅 (Copper, Cu)：** 人體必需微量元素，為多種重要金屬蛋白酶 (Metalloenzymes) 的核心輔因子。
* **黑色素 (Melanin)：** 由黑色素細胞合成，為決定皮膚與毛髮顏色的主要色素 [🖼️](https://www.google.com/search?q=melanin+pigmentation+clinical+sign&tbm=isch&udm=2)，合成受阻會導致白化症 (Albinism)。
* **藍胞漿素 (Ceruloplasmin)：** 血液中主要的「銅載體蛋白」，同時具備亞鐵氧化酶 (Ferroxidase) 功能，將 Fe²⁺ 氧化為 Fe³⁺ 以利鐵質運送。
* **細胞色素 C 氧化酶 (Cytochrome C oxidase)：** 粒線體電子傳遞鏈 (Complex IV) 的末端酵素，負責結合氧氣產生水與 ATP。
* **超氧化物歧化酶 (Superoxide dismutase, SOD)：** 細胞內的抗氧化防線，負責將超氧化物自由基轉換為過氧化氫 (H₂O₂)。
* **酪胺酸酶 (Tyrosinase)：** 黑色素合成途徑的第一步與限速酵素 (Tyrosine → DOPA → Melanin)。

### (3) 選項詳解

* **(A) 藍胞漿素（ceruloplasmin）：錯誤。** 雖然它是含銅蛋白，但其主要生理作用是「運送銅」及「參與鐵代謝」。缺銅會導致其總量下降，但它不參與黑色素的合成。
* **(B) 細胞色素 C 氧化酶（cytochrome C oxidase）：錯誤。** 這是粒線體中參與呼吸作用的含銅酵素。缺銅會造成此酵素活性下降，導致細胞能量 (ATP) 生成障礙、肌肉無力，與黑色素降低無因果關係。
* **(C) 超氧化物歧化酶（superoxide dismutase）：錯誤。** 人體細胞質內存在的是 Cu/Zn SOD，缺銅會使其清除自由基的抗氧化能力下降，引發氧化壓力增加，但與黑色素代謝無關。
* **(D) 酪胺酸酶（tyrosinase）：正確。** 這是生成黑色素的絕對關鍵酵素，其活性中心需要結合銅離子。當體內缺銅時，酪胺酸酶無法催化 Tyrosine 轉變為 DOPA，阻斷了黑色素的合成路徑，進而引發臨床上的色素脫失或白化特徵 [🖼️](https://www.google.com/search?q=hypopigmentation+clinical+sign&tbm=isch&udm=2)。

### (4) 核心軸線與橫向比較

**核心邏輯因果：**
銅攝取不足 / 基因缺陷導致吸收障礙 → 體內含銅酵素活性↓ → **Tyrosinase 活性↓** → Tyrosine 無法轉換為 DOPA → 黑色素合成中斷 → 皮膚、毛髮黑色素缺乏。

💡 **記憶口訣 (含銅重要酵素)：**
「**CT** **S**can **C**opper」(電腦斷層掃描銅)：

* **C**eruloplasmin (藍胞漿素)
* **T**yrosinase (酪胺酸酶)
* **S**OD (超氧化物歧化酶)
* **C**ytochrome C oxidase (細胞色素 C 氧化酶)
