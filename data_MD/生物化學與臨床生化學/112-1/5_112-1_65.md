## 題目

下列何者為人體含量最多的去碘酶(deiodinase),是正常血液循環T₃的主要來源?

type 1 iodothyronine 5'-deiodinase
type 2 iodothyronine 5'-deiodinase
type 3 iodothyronine 5'-deiodinase
type 4 iodothyronine 5'-deiodinase

> [!success]- 正確解答
> **A**

---

## 筆記與詳解

### (1) 快速破題

看到題目關鍵字「**人體含量最多**」且為「**血液循環 T₃ 的主要來源**」，就要立刻聯想到 **Type 1去碘酶 (D1)**。
解題直覺：

* 問「周邊血液循環 T₃ 主要來源 / 肝腎為主」→ 必選 **Type 1**。
* 問「局部組織胞內 T₃ / 腦下垂體 TSH 負回饋」→ 必選 **Type 2**。
* 問「去活化 / 產生無活性 rT₃ (reverse T3)」→ 必選 **Type 3**。
* 不存在 Type 4，可直接刪除。

### (2) 關鍵字與知識點拆解

* **Deiodinase (去碘酶)**：含硒酵素 (Selenoprotein)，負責移除甲狀腺素分子上的碘原子，是決定甲狀腺素是否具備生物活性 (T₃) 或去活化 (rT₃) 的關鍵開關。
* **Type 1 iodothyronine 5'-deiodinase (D1)**：進行外環去碘 (Outer ring deiodination)。將 T₄ 轉換為活性 T₃，大量表現於肝、腎，產物**直接釋放入血**。
* **Type 2 iodothyronine 5'-deiodinase (D2)**：同樣進行外環去碘。將 T₄ 轉為 T₃，但產物主要**留在細胞內**供局部使用（如中樞神經、棕色脂肪）。
* **Type 3 iodothyronine 5'-deiodinase (D3)**：進行內環去碘 (Inner ring deiodination)。將 T₄ 轉換為**無活性**的 rT₃，避免組織暴露於過多活性甲狀腺素中。

### (3) 詳解

甲狀腺腺體分泌的賀爾蒙以 T₄ (Thyroxine) 佔絕大多數 (>80%)，但真正與受體結合、發揮強大生理活性的主要是 T₃ (Triiodothyronine)。因此，周邊組織的「去碘酶」扮演著活化甲狀腺素的關鍵角色。

* **(A) type 1 iodothyronine 5'-deiodinase：正確。** D1 廣泛且大量分布於肝臟、腎臟與甲狀腺。其催化效率高，負責將血液中循環的 T₄ 轉化為 T₃，並將這些 T₃ 釋放回血液中，供應全身周邊組織使用，是循環中 T₃ 的最主要來源。
* **(B) type 2 iodothyronine 5'-deiodinase：錯誤。** D2 主要表現於大腦、腦下垂體前葉、棕色脂肪與骨骼肌。它的功能是維持「細胞內」局部的 T₃ 濃度。例如在腦下垂體，D2 產生的 T₃ 負責執行對 TSH 的負回饋抑制。其產物極少進入全身血液循環。
* **(C) type 3 iodothyronine 5'-deiodinase：錯誤。** D3 是「去活化酵素」，表現於胎盤、大腦及皮膚。它拔除的是內環的碘，將 T₄ 變成無活性的反三碘甲狀腺素 (rT₃)，或將 T₃ 降解為 T₂。主要作用為保護胎兒或特定組織免受過量甲狀腺素刺激。
* **(D) type 4 iodothyronine 5'-deiodinase：錯誤。** 生理學上人體去碘酶僅分為 Type 1、2、3，不存在 Type 4。

### (4) 核心軸線與橫向比較

**縱向生理邏輯：**
甲狀腺大量分泌前驅物 T₄ → 血液流經肝/腎等周邊組織 → **D1 酵素**執行外環去碘 (5'-去碘) → 產生大量活性 T₃ 釋放入血循環 → 結合細胞核受體，全身基礎代謝率 **↑** → 若路徑過度活化 (如甲亢)，可能出現典型的甲狀腺腫 Goiter[🖼️](https://www.google.com/search?q=thyroid+goiter+clinical+sign&tbm=isch&udm=2) 或 Graves' disease 造成的浸潤性凸眼 Exophthalmos[🖼️](https://www.google.com/search?q=Graves+disease+exophthalmos+clinical+sign&tbm=isch&udm=2) 等臨床表徵。

**橫向比較：去碘酶 (Deiodinases) 系統總整理**

| 比較維度 | Type 1 (D1) | Type 2 (D2) | Type 3 (D3) |
| --- | --- | --- | --- |
| **主要分布器官** | **肝、腎**、甲狀腺 | **大腦、腦下垂體**、棕色脂肪 | **胎盤**、大腦神經膠細胞 |
| **催化位置** | 外環 (5'-Outer) | 外環 (5'-Outer) | **內環** (5-Inner) |
| **主要產物** | T₄ → **T₃** | T₄ → **T₃** | T₄ → **rT₃** (無活性) |
| **生理功能定位** | 維持**全身血液循環** T₃ 濃度 | 維持**局部胞內** T₃ 需求、TSH **負回饋調控** | **去活化**、預防組織過度刺激 (如胎兒保護) |
| **對 PTU (抗甲狀腺藥) 反應** | **極敏感** (易被抑制) | **具抗性** (不受影響) | 不受影響 |
| **空腹/飢餓時變化** | 活性 **↓** (減少代謝消耗) | 活性 **↑** (維持腦部機能) | 活性變化不大 |

> **記憶口訣：**
> * **1 肝腎，補全身** (D1在肝腎，給全身用)
> * **2 腦袋，管回饋** (D2在腦和垂體，負責局部與負回饋)
> * **3 胎盤，踩煞車** (D3變出 rT3，去活化當煞車)
