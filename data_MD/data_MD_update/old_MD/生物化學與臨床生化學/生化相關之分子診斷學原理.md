## 核心趨勢與高頻考點
本單元的出題趨勢已從傳統的PCR、南方墨點法等基礎技術，全面轉向現代精準醫療的核心技術。考生必須理解，這些技術並非獨立存在，而是為了解決臨床問題而環環相扣。

其核心邏輯鏈如下：
**精準醫療需求 (如：癌症標靶治療)** → **需要更個人化、更即時的檢測工具** → **因應臨床需求，但缺乏已查驗登記的IVD產品時** → **實驗室開發檢測 (LDTs) 應運而生** → **LDTs 的實現依賴三大支柱：**
1.  **高靈敏度定量 (Quantification):** 從 Real-time PCR 的相對定量演進到 Digital PCR 的**絕對定量**，專攻微量檢體（如 ctDNA）的偵測。
2.  **高通量分析 (High-Throughput):** 次世代定序 (NGS) 取代傳統 Sanger 定序，能一次分析數十到數百個基因，常用於癌症基因套組檢測。
3.  **非侵入性採檢 (Non-Invasive):** 液態活檢 (Liquid Biopsy) 概念興起，以血液中的 **ctDNA** 或 **CTC** 取代傳統組織切片，用於監測治療反應與抗藥性。

**高頻考點預測：**
1.  **技術比較：** **Digital PCR vs. Real-time PCR** 的定量原理差異，以及 **CTC vs. ctDNA** 的生物特性與臨床應用比較，將會是重複出現的考題。
2.  **法規與品管：** LDTs 的法規（主管機關為**衛福部食藥署 TFDA**）、與 IVD 的差異、確效流程是必考重點，其規範主因是**檢測結果直接影響病人診斷與治療**。未來可能結合品管概念，例如 NGS 的 **Phred Quality Score (Q score)** 計算與意義。
3.  **臨床應用整合：** 題目會將技術與特定癌症連結，例如非小細胞肺癌 (NSCLC) 的 **EGFR** 突變檢測，就可能同時考到 LDTs、NGS 或液態活檢的應用。
4.  **新興技術原理：** 對於 aCGH（看拷貝數變異）、Microarray（看基因表現量）、Bisulfite sequencing（看**甲基化**）等技術，考生需掌握其「核心原理」與「判讀方式」。
5.  **基礎觀念深化：** 除了新技術，如 **Tm 值**的定義（50%雙股解鏈溫度）與**黏合溫度 (Annealing Temp.)** 對產物專一性與產量的影響，以及**抑癌基因**與**原癌基因**的基本定義，這類基礎觀念仍是出題重點。
6.  **技術的臨床選擇與限制：** 為何臨床上多用「癌症基因套組 (Cancer Panel)」而非「全基因體定序 (WGS)」？原因包含成本、數據分析複雜度、以及大量「臨床意義不明變異 (VUS)」的干擾。考生需理解不同技術的適用情境。

## 關鍵字反射表

| 技術/概念 | 原理/核心概念 | 定量方式 | 主要臨床應用 | 關鍵指標/結果判讀 | 鑑別重點/限制 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **Real-time PCR** | 螢光探針/染料，同步放大與偵測 | **相對定量** | 病毒量偵測、基因表現量 | **Ct 值** (Cycle threshold)，與初始模板量**成反比** | 需標準曲線才能絕對定量 |
| **Digital PCR (dPCR)** | **微滴/孔盤分割 (Partitioning)**，終點偵測 | **絕對定量** (基於**二進制定量**分析) | 稀有突變偵測 (ctDNA)、拷貝數變異 (CNV)、NGS Library 定量 | **陽性微滴數** + **Poisson 分佈**校正 | **靈敏度極高**，但通量較低、動態範圍窄 |
| **次世代定序 (NGS)** | **大量平行定序 (Massively Parallel)**，邊合成邊定序 | 定序計數 (Read counts) | 癌症基因套組、全外顯子/基因體定序 | **Phred Score (Q30=錯誤率1/1000)**、覆蓋深度 (Depth) | 數據分析複雜。**WGS** 成本高、數據解讀困難、易產生VUS，故臨床多用目標性更強的 **Cancer Panel**。 |
| **液態活檢 (Liquid Biopsy)** | 分析血液等體液中的腫瘤來源分子 (ctDNA, CTC, Exosome) | - | 癌症早期篩檢、治療監控、抗藥性偵測 | **非侵入性**為最大優點，**無法**直接分析**腫瘤組織**。CTC 與 ctDNA 皆為其分析標的。 |
| ↳ **ctDNA** | 腫瘤凋亡/壞死釋放的**片段化 DNA** | dPCR / NGS | **監控治療反應**、偵測微小殘存疾病 (MRD) | 半衰期**極短 (1-2hr)**，能即時反映腫瘤變化 | **無法**進行蛋白質層級或藥物敏感性測試 |
| ↳ **CTC** | 進入血液循環的**完整腫瘤細胞** | 細胞計數 / 分子分析 | 預後評估、轉移風險、建立細胞株做藥敏測試 | **極度稀少** (1顆 / 10⁹ 血球)，捕捉困難 | 可培養，但捕捉技術是瓶頸 |
| **LDTs** | **實驗室自行開發**、驗證、使用的檢測 | - | **精準醫療**、罕見疾病、新興傳染病 (當缺乏合適IVD時) | - | **不需**美國FDA或台灣TFDA**上市前審查 (PMA/510k)**，但需符合衛福部**食藥署(TFDA)**之**特管法**規範 |
| **IVD** | 體外診斷醫療器材，**商品化套組** | - | 常規檢驗項目 | - | 需主管機關**上市前審查許可**，品質穩定但開發週期長 |
| **Bisulfite Treatment** | 亞硫酸氫鹽將**未甲基化 C → U**，5-mC 不變 | - | DNA **甲基化** (Epigenetics) 分析 | C/T 比例 | DNA 修飾分析的**金標準** |
| **aCGH** | 檢體與對照組 DNA 競爭性結合至晶片探針 | - | 染色體**拷貝數變異 (CNV)** | Log2 ratio > 0 (**增益**), < 0 (**缺失**) | 解析度高於傳統染色體分析，**無法**偵測點突變 |
| **Microarray** | 檢體與對照組 cDNA/cRNA 競爭性結合 | - | **基因表現量**分析 | 紅色 (實驗組表現↑)、綠色 (對照組表現↑)、**黃色 (兩者表現量相近)** | 已逐漸被 RNA-seq 取代 |
| **熔解溫度 (Tm)** | DNA雙股螺旋有 **50%** 解離成單股時的溫度 | - | PCR/qPCR 反應條件設計 | 受 **GC含量**、DNA長度、鹽離子濃度影響。GC含量越高，Tm越高 | **黏合溫度 (Annealing)** 通常設定比引子Tm低約 5°C。溫度太低→非專一性產物；太高→產量下降 |
| **Nonsense Variant** | 點突變造成密碼子變為**終止密碼子** | - | 遺傳疾病、癌症 | 產生**截斷的蛋白質**，功能喪失 | UAA, UAG, UGA |
| **BCR-ABL1 [t(9;22)]** | 染色體**易位 (Translocation)** | - | 慢性骨髓性白血病 (CML) | 形成**融合基因** (Fusion gene) | 為CML的**經典標靶治療**目標 (I​​matinib) |
| **抑癌基因 (Tumor Suppressor Gene)** | 功能如同「煞車」，抑制細胞異常增生。需**兩個等位基因皆突變**才會失去功能 (Knudson's two-hit hypothesis)。 | - | 遺傳性癌症風險評估、癌症機制研究。 | 檢測有無功能喪失性突變 (Loss-of-function)。 | 與「油門」的原癌基因 (Proto-oncogene) 相對。常見例子：**TP53, RB1, BRCA1/2**。 |
| **原癌基因 (Proto-oncogene)** | 正常調控細胞生長與分裂的基因，如同「油門」。**單一等位基因**的功能增益性突變 (Gain-of-function) 即可致癌。 | - | 標靶治療的目標。 | 檢測有無活化性突變 (Activating mutation)。 | 突變後成為致癌基因 (Oncogene)。常見例子：**EGFR, HER2, RAS, MYC**。 |
