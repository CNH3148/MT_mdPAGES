## 類群定義

> 涵蓋基因體功能調控的進階主題。表觀遺傳學主要測驗DNA甲基化（Methylation）的分析方法（如重亞硫酸鹽處理）。蛋白質體學則測驗用於蛋白質分離與鑑定的技術（如二維電泳、質譜法）。隨著題庫擴充，本區塊亦納入了高度相關的「次世代定序(NGS)原理及生資分析」、「腫瘤標靶與突變分析(如TMB, LOH)」，以及少部分進階鏡檢特徵（如結核性腦脊髓液、微絲蟲）。

## 核心趨勢與高頻考點
本單元的命題趨勢除了原有的「表觀遺傳學」與「蛋白質體學」兩大經典主軸外，近幾年大量融入了「次世代定序與腫瘤分子診斷」以及「特殊鏡檢特徵」的跨領域考題。考生必須建立以下系統性邏輯：

1.  **表觀遺傳學：聚焦 DNA 甲基化分析與基因調控**
    *   **核心邏輯**：檢測 DNA 甲基化的關鍵在於「**創造可被偵測的差異**」。其邏輯鏈為：`原始 DNA (甲基化 C vs. 未甲基化 C) → 化學處理 (重亞硫酸鹽) → 未甲基化 C 轉變為 U，甲基化 C 不變 → 序列產生差異 → 利用 PCR 或定序技術偵測此差異`
    *   **高頻考點**：
        *   **重亞硫酸鹽 (Bisulfite)** 作用機制（未甲基化 C → U）為絕對必考。
        *   **啟動子 (Promoter) 甲基化意義**：高度甲基化通常伴隨基因「**表現受抑制 (Silencing)**」。
        *   **分析方法比較**：MethyLight (定量 TaqMan PCR)、MSP (定性 PCR)、甲基化晶片 (可同時高通量定量大量 CpG 位點)。

2.  **蛋白質體學：分離與鑑定黃金組合**
    *   **核心邏輯**：分析複雜蛋白質混合物的標準流程為「**分離 (2D-PAGE) → 酵素水解 → 鑑定 (質譜法 MS)**」。
    *   **高頻考點**：
        *   **二維電泳 (2D-PAGE)**：第一維**等電點聚焦 (IEF, 依 pI)**；第二維 **SDS-PAGE (依 MW)**。
        *   **質譜法 (MS)**：**MALDI-TOF** 的核心分離依據為「**質荷比 (m/z)**」，在臨床上極常應用於**細菌鑑定**。
        *   實驗操作順序：檢體製備 → 1D IEF → 2D SDS-PAGE → 膠體染色 → 酵素水解 (Trypsin) → 質譜分析。

3.  **次世代定序 (NGS) 與突變分析 (新興熱門)**
    *   **平台比較**：
        *   **Illumina**：橋式 PCR (Bridge PCR) + 可逆性中斷子定序。
        *   **Ion Torrent**：乳化 PCR (Emulsion PCR) + 偵測釋放的「**氫離子 (H⁺) / pH 值變化**」。
    *   **突變掃描策略**：要找出「**未知**」的點突變或短插刪，必須使用掃描型技術如 **CCM (化學切割)**、**SSCP (單股結構多型性)**、**DGGE (變性梯度膠體)**、**HRM (高解析度熔點分析)**。已知突變才用 ARMS、PCR-RFLP。
    *   **生物資訊流程**：序列質量裁剪 → 序列比對 (Alignment) → 基因變異辨認 (Variant calling) → 基因變異註釋 (Annotation)。

4.  **腫瘤分子診斷與應用 (新興熱門)**
    *   **異型合子喪失 (LOH)**：微衛星 (Microsatellite) 分析中，腫瘤組織比對正常組織，少了一個對偶基因波峰。
    *   **腫瘤突變負荷 (TMB)**：高 TMB 產生較多新抗原，預測對「**免疫治療**」反應較佳。
    *   **臨床靈敏度比較**：檢測微量殘存疾病 (MRD) 時，**FISH 靈敏度最低 (約 1-5%)**，流式細胞儀、qPCR、NGS 靈敏度較高。
    *   **用藥相關**：ERCC1 (DNA 修復) 表現高會對鉑類化療抗藥；同義突變 (Synonymous mutation, 如 p.Q787Q) 不改變胺基酸，通常與藥效無關。

5.  **跨領域綜合考點：特殊鏡檢與核酸擴增**
    *   結核性腦膜炎 (TB Meningitis)：CSF 淋巴/單核球為主、葡萄糖下降，冰存後出現**網狀薄膜 (Pellicle)**。
    *   寄生蟲特性：**羅阿絲蟲 (Loa loa)** 具有**日間週期性**。
    *   等溫擴增：**TMA** 放大 RNA，搭配 **HPA (雜交保護分析)** 偵測，不使用 TaqMan probe。

## 未來考點預測與易混淆陷阱
*   **陷阱 1 (甲基化方向)：** Bisulfite 處理是將「**未甲基化**」的 C 轉變成 U。已經甲基化的 C 是「**不反應 (不變)**」的。
*   **陷阱 2 (NGS 偵測訊號)：** Ion Torrent 是測 **H⁺ (氫離子/pH)**，**不是**測焦磷酸 (Pyrophosphate，那是 Pyrosequencing 焦磷酸定序的原理)。
*   **陷阱 3 (突變檢測策略選擇)：** 題目問「未知突變」時，絕對不能選 ARMS、ASO 等針對已知位點設計引子/探針的技術；必須選 HRM、SSCP、DGGE、CCM 等依靠物理/化學特性差異的掃描法。
*   **陷阱 4 (核酸染劑 vs. 細菌染色)：** Acridine orange 是一種螢光核酸染劑，不是專門用來鑑定隱球菌的 (隱球菌要找 India ink 的莢膜)。
*   **陷阱 5 (MRD 的盲區)：** FISH 因為是看細胞層級，靈敏度遠遠不及 PCR 或 NGS，無法作為極微量殘留疾病 (MRD) 的首選高靈敏方法。
